## 给数据化阶段的注记

1. 本章无 castInteraction、无 scoredChoice——节点类型只用 narration/dialogue/choice/sceneHeader/chapterEnd/**dressingUpdate（动态）**。
2. **动态本命卦盘（本章唯一引擎增量）**：幕四演卦的三个 dressingUpdate 节点不带静态 board，标 `dynamicNatalBoard: 'branches' | 'marks' | 'full'`——Player 遇到时调新 helper `natalDressingBoard(natal, stage)`（state.js，从 `paipan(natal.throws, 固定 dateParts)` 生成 board：branches=六爻支+五行 / marks=+世应 / full=+六亲）。排盘事实 date-independent（natalPalaceText 同款口径，六神/空亡不入 board 不受日期影响）；natal 缺失时 helper `console.warn` + 返回 null（board:null=清盘，fail loudly 不静默造假盘）。
3. **动态文本一处**：幕四卦名总述复用 `natalPalaceText(natal)` 输出，节点走 `dynamicNatal` 机制（**ch2 6.4 先例**：文本内〔…〕全角占位符经 Player 正则替换，fallback 为描述性中文——勿用花括号 token，不匹配会裸漏）。
4. **随机卦文本红线（全章骨）**：玩家本命卦是真随机——所有台词/旁白**不得指涉任何具体卦名、爻支、六亲名分**（动态节点除外）；她排的每一爻在盘上可见，嘴上不点破（「哪一爻是什么，你自己看」）。与 ch3 6.4「文本不指涉具体亲属名分」同源红线，本章升为结构约束。
5. **不解红线延续**：全章零断语零吉凶——「排给你看，是让你看清你手里有什么；断，还早」是本章的界碑句（策划案 § 5 原句）。旺衰四手势在幕二考较中以**乾为天**复习（教学卦，非本命卦）——本命卦上一个旺衰字都不出。
6. 好感账目（qiannang 同构）：幕二考较应答 +1、幕四观排风味 +1、幕五核心事件 **+5**（「她愿意把你的卦当回事」）＝好感满 +7；章通关灵力 +5（bonus 规格，无教学块无 CP）。
7. 背景/BGM 复用：修书房后院（幕一，bg-xiushufang）/ 明蓍堂（幕二至五）；BGM bgm-main → bgm-dawn（幕四起，钱囊夜谈同款温柔线）。
8. 称谓口径：进入本章的玩家好感 ≥35（信任档中段）——她**直呼 {{player}} 名**（ch4 幕六 pass 线已开）；无 favorBranch（单线，门槛已保证档位）。
9. 时间锚：守窗轮值期间的一个午后（ch4 章末「明日巳时，你先值」的延续窗口，ch4-ch5 之间浮动安全）。
10. **幕五开局清盘**：幕五第一个节点为 `dressingUpdate { board: null }`（她把新纸拿回＝收盘）——ch2/ch3 dressing 序列收尾先例，防 full 盘残留到 +5 情感终场。
11. **幕二考较三选项均为 terminal**：各自挂 `effects: { favor: 1 }`，错选的「重答」在响应文本内叙述收束（**无 loop-back 节点**）——三选项等值 +1、零双计（与 ch2/ch3 引导应答的 wrong 零分模式刻意不同：本章 bonus「不惩罚」规格）。
12. 未教不考自查：本章零新 KP；复习考较只考已 accepted 的 KP-LY-004/006/008/010（宫/世应/装亲/月建旺衰四问，全部 ch1-ch4 已教）；禁词表与 ch4 lint 同集（旬空/伏神/元神/忌神/六合等全禁，无预告位）。

> 场景：修书房后院，柴棚（bg-xiushufang 复用）。午后。
> 本幕功能：私教由头（守窗轮值中她来换值，撞见玩家自练）；触发情境自然生成。

*守窗的第三日。*

*柴棚里能看见那扇窗，窗看不见你——{{ta}}挑的位置，果然刁钻。巳时到未时，后院只有晒架上的浆布动，窗台上的灰纹丝没动。*

*你把卦纸在膝上铺开——不是案卦，是你自己那张。守窗的时辰长，你数它的爻玩：这一爻生那一爻，那一爻克这一爻——四章学下来的东西，在这张纸上转圈。*

**沈疏桐**：「背地里用功，被我撞见几回了？」

*你一惊。{{ta}}不知什么时候立在柴棚口，手里提着换值的灯笼——今日{{ta}}来得早。*

**{{player}}**：「……第三回。」

**沈疏桐**：「第四回。前日换值，你就蹲在这儿描八宫图——我没拆穿你。」

*{{ta}}走进棚里，目光落在你膝上的卦纸上——你下意识把纸往回收了半寸。*

*{{ta}}看见了这半寸。*

**沈疏桐**：「怕什么。规矩是我立的——不许解。我几时说过，不许**看**？」

*{{ta}}把灯笼搁下，在柴堆另一头坐了，隔着一棚的斜光看你。*

**沈疏桐**：「{{player}}。今日窗归我，你歇。——歇之前，回明蓍堂一趟。」

**{{player}}**：「上课？」

**沈疏桐**：「不上课。」*{{ta}}起身掸了掸衣角，*「四章的东西，你肚子里存了多少，我心里没数。今日盘一盘账——**盘完，给你看样东西。**」

## 第三幕 · 你的卦

> 场景：明蓍堂。傍晚。
> 本幕功能：本命卦上桌；「不解」界碑重立；玩家的犹豫与交付（信任的双向仪式）。
> ⚠️ 红线：全幕不指涉卦名/爻支/名分（随机卦约束，§ 注记 4）。

*你的手按在怀口上，没动。*

*那张纸在你怀里贴了四章。三枚旧钱摇出来的第一卦；你自己装的支，自己查的籍贯，自己排的名分。一层一层，像给一封信写完了姓名住址——始终没拆开。*

**{{player}}**：「{{senior}}说过——不许解。」

**沈疏桐**：「今日也不解。」

*{{ta}}答得极稳。*

**沈疏桐**：「解是解，排是排。你那张卦，支是你刚学装卦那阵装的，亲是你才安上没几日的——你的功夫一直在长，**那张纸没跟着长**。」

*{{ta}}把案面清出一块空来，动作不紧不慢。*

**沈疏桐**：「今日我替你，把它从头到尾**重排一遍**。用我的手，用你如今学过的所有规矩——支、宫、世应、名分，一样不落。」

**{{player}}**：「……为什么？」

*{{ta}}顿了一下。灯还没点，暮色从窗纸上渗进来，{{ta}}的脸在半明半暗里。*

**沈疏桐**：「**排给你看，是让你看清你手里有什么。断，还早。**」

*这句话{{ta}}说得很慢，像把一块界石放进土里。*

*你把手伸进怀里。纸的边角已经被体温焐软了。你把它取出来，展开，抚平——推到{{ta}}面前。*

*{{ta}}看了你一眼。这一眼没有话，可你读得懂：{{ta}}知道这张纸对你是什么，{{ta}}也知道你此刻把它交出来是什么。*

**沈疏桐**：「好。」

*第一步，装支。*

*{{ta}}执笔，从初爻起，一枚一枚往上装。内卦三支，外卦三支——{{ta}}落笔比你快得多，可每一笔都放得很正，像把六枚钉钉进它们各自等了很久的位置。*

【dressingUpdate · dynamicNatalBoard: 'branches'】
