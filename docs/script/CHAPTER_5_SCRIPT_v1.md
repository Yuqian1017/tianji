## 第二幕 · 粮道

> 场景：明蓍堂。次日上午。
> 本幕功能：**KP-LY-013 示范＋引导**（元神/忌神定义——粮道比喻）；仇神半句立名（§ 0 第 7 条白名单单点）。

*明蓍堂的案上，那张乾为天又摊开了。你都替它累。*

**沈疏桐**：「别嫌它老。」*{{ta}}看出了你的神色，*「兵书翻烂一册，胜过新书堆一架。今日的课，还在它身上。」

*{{ta}}的指尖落在世爻戌土上——四章下来，这枚字你闭眼都摸得到位置。*

**沈疏桐**：「旺衰课上，你学会了称它——月建一票，日辰一票。可你称的，始终是它**一个**。」

**沈疏桐**：「今日换个问法：这张卦上六个爻——**有没有谁，在帮它？有没有谁，在伤它？**」

*你俯身看卦。戌，土。生土的是火——你的目光顺着爻位爬：四爻，官鬼午火。*

**{{player}}**：「四爻午火——火生土。它在帮。」

**沈疏桐**：「再找。伤它的。」

**{{player}}**：「克土的是木……二爻，妻财寅木。它在伤。」

**沈疏桐**：「都找对了。现在给他们上名分。」

*{{ta}}提笔，在卦纸空白处写下两个词：*

***元神。忌神。***

**沈疏桐**：「**生用神的爻，叫元神**——原书的话：『生用神之神卽為元神』。**克用神的爻，叫忌神**——『忌神克用神之爻也』。」

**沈疏桐**：「打个比方。用神是阵前的**前锋**——月建日辰是天时，管它的令。可前锋不是一个人在打仗：身后有**粮道**，对面有**敌兵**。」

*{{ta}}的笔尖点在午火上：*

**沈疏桐**：「元神，就是粮道。粮道通，前锋打得动。」

*笔尖移到寅木：*

**沈疏桐**：「忌神，就是敌兵。敌兵至，前锋要挨刀。」

**{{player}}**：「所以看一个爻，先看天时，再看——粮道和敌兵？」

**沈疏桐**：「对。第四课教你**看天**，这一课教你**看阵**。」

【引导应答 · KP-LY-013 · 好感位】

**沈疏桐**：「趁热。若问的事用神在**初爻子水**——子孙子水。谁是它的元神，谁是它的忌神？」

- **选「生水者金——五爻申金元神；克水者土——世爻戌土、三爻辰土忌神」** → **沈疏桐**：「一口气找全了，粮道敌兵各归各位。……连忌神两现都没漏。看阵的眼，你有了。」（好感 +1）
- **选「生水者金——申金元神；克水者……火？」** → **沈疏桐**：「火克的是金。克**水**的是土——土克水，堤拦水。这卦里的土可不少：戌土、辰土，两路敌兵。重找一遍。」（引导重答后继续）
- **选「亥子相邻，亥水帮子水？」** → **沈疏桐**：「这卦里没有亥。别背卦，**看**卦——眼前六个爻，谁生水？申金。谁克水？辰戌两土。」（引导重答后继续）
