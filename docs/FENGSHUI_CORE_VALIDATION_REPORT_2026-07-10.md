## 1. 结论

现有风水工具壳层可以复用，但旧排盘核心不能直接保留。修复前，9 运 x 24 山的 216 张盘与独立实现对照，5,832 个共享字段中有 1,648 个不一致；错误集中在山盘和向盘顺逆。

修复后，项目内 216 张盘共 6,469 项检查 0 fail；`@soul-atelier/xuankong@0.2.1` 对照四盘与结构标签 6,048 字段 0 mismatch。`@soul-atelier/calendar@0.3.0` 的 tyme4ts 后端对 2020-2026 共 10,228 个时刻进行独立历法复核，三元九运太阳年、流年中宫和流月中宫共 30,684 字段 0 mismatch。

## 2. 修复的问题

### F-FS-001 二十四山基础表错误

旧表把每卦第一个山标成天元龙、第二个标成地元龙；正确顺序应为地元、天元、人元。子、辰、巽、巳、坤、戌、亥的玄空阴阳也标错。

### F-FS-002 山盘向盘顺逆错误

旧引擎直接看实际坐山或向首自身的阴阳。下卦法应先取运盘坐/向宫数字入中，再到该数字的洛书原宫中寻找同元龙山，以其阴阳定顺逆；只有五黄无原卦时才借实际山阴阳。

### F-FS-003 月飞星边界错误

旧函数把公历月份直接当流月。现按立春、惊蛰、清明等十二个“节”切换寅至丑月，并使用精确交节时刻。

### F-FS-004 年界与八宅命卦公式错误

三元九运和流年星新增真实立春边界辅助函数；年份式 UI 明确表示“立春后太阳年”。2000 年后的男女命卦常数各错一位，已按年份后两位数公式修复，并枚举 1900-2099 两性 400 个值。

### F-FS-005 下卦范围越界

旧工具把每山全部 15 度都套用下卦。现仅接受山中线左右各 4.5 度；兼向/替卦和出卦未实现时明确拒绝。

### F-FS-006 解释层越界

旧 UI 和 Prompt 把九星、组合、疾病财务断语及铜器钱币等化解建议当作结果。现运行时只显示四盘数字、三元龙和结构标签；解释层标为 `not_validated`。
