## 总说

肝主疏泄藏血,体阴用阳,喜条达恶抑郁。肝气、肝火、肝阳、肝风四者同源异流,常兼夹转化。肝郁络滞则胁痛;湿壅肝胆胆汁泛溢则黄疸;气血壅结则积聚;气血水互结则鼓胀;风阳上扰或阴血不承则头痛眩晕;风阳暴升气血逆乱上冲于脑则中风;肝郁痰瘀结于颈前则瘿病。

## 胁痛

**定义**:一侧或两侧胁肋部疼痛。基本病机肝络失和——不通则痛(气滞/血瘀/湿热,实多)与不荣则痛(阴血亏虚)。对应肝炎、胆囊炎、胆石症、肋间神经痛等。**右胁痛为主者多与肝胆疾患相关,应查肝功能、B超。**

**辨证**:胀痛游走、随情绪增减——在气;刺痛固定、拒按夜甚——在血;灼痛口苦、伴身热目黄——湿热;隐痛绵绵、遇劳加重——阴虚。

**分证论治**:肝郁气滞——柴胡疏肝散;肝胆湿热——龙胆泻肝汤(结石绞痛连肩背用大柴胡汤加金钱草海金沙郁金鸡内金等利胆排石;**胆绞痛/胆囊炎急性发作先就医**);瘀血阻络——血府逐瘀汤或复元活血汤(跌打伤);肝络失养——一贯煎。

**要诀**:疏肝柔肝并举——疏肝理气药多辛温香燥,久用耗肝阴,宜选轻灵之品(香附、苏梗、佛手、绿萼梅)并配白芍类柔肝。胁痛剧+呕吐蛔虫者先乌梅丸安蛔。

## 黄疸

**定义**:目黄、身黄、小便黄,**目睛黄染为要征**(与萎黄之别)。病机关键是湿——"黄家所得,从湿得之"。对应肝炎、肝硬化、胆石、溶血等各类黄疸,**新发黄疸一律就医查因**(胆红素分型、肝功、超声)。

**阳黄/阴黄纲领**:阳黄——黄色鲜明如橘,起病急病程短,伴身热口苦、苔黄腻脉弦数(热证);阴黄——黄色晦暗如烟熏,病程长势缓,伴纳少乏力、舌淡脉沉迟(虚寒证)。**急黄**为阳黄重症——疸色如金、高热烦渴、神昏出血,红线。

**治法大纲**:化湿邪、利小便("诸病黄家,但利其小便")。**大黄为退黄专功**,茵陈配大黄退黄更佳(教材临证要诀)。

**分证论治**:

| 证 | 治法 | 代表方 |
|---|---|---|
| 阳黄·热重于湿 | 清热通腑利湿退黄 | 茵陈蒿汤 |
| 阳黄·湿重于热 | 利湿化浊运脾佐清热 | 茵陈五苓散合甘露消毒丹;兼表麻黄连翘赤小豆汤 |
| 阳黄·胆腑郁热 | 疏肝泄热利胆退黄 | 大柴胡汤(砂石加金钱草海金沙) |
| **急黄·疫毒炽盛** | 清热解毒凉血开窍 | 《千金》犀角散(水牛角代)+安宫牛黄丸——**住院急救范畴** |
| 阴黄·寒湿阻遏 | 温中化湿健脾和胃 | 茵陈术附汤 |
| 阴黄·脾虚湿滞 | 健脾养血利湿退黄 | 黄芪建中汤加茵陈茯苓 |

**消退后调治**(黄退不等于病愈):湿热留恋——茵陈四苓散;肝脾不调——柴胡疏肝散或归芍六君子汤;气滞血瘀——逍遥散合鳖甲煎丸。失调可迁延成癥积、鼓胀。

**调护**:传染性肝炎自发病日隔离30~45天、餐具消毒;戒酒、忌辛热甘肥;急黄绝对卧床。**恶化征象:黄疸加深、斑疹吐衄、神昏痉厥——立即就医。**

**附·萎黄**:肌肤淡黄干萎无光而**目睛与小便不黄**,属气血不足(虫积食滞或失血病后),黄芪建中汤/人参养营汤调理——勿误作黄疸。

## 积聚

**定义**:腹内结块。**积**——结块固定不移、痛有定处,病在血分属脏病,渐成较重;**聚**——包块聚散无常、痛无定处,病在气分属腑病,较轻。病机:气滞血瘀。

**警示**:积块日趋肿大、坚硬不平——**必须影像+病理排除恶性**。与痞满鉴别:痞满无块可扪。

**分证论治**:聚证——肝气郁结(逍遥散/木香顺气散)、食滞痰阻(六磨汤);积证按初中末三期——气滞血阻(柴胡疏肝散合失笑散)、瘀血内结(膈下逐瘀汤配六君子汤,可服鳖甲煎丸)、正虚瘀结(八珍汤合化积丸)。

**治则要诀**:"初者任受攻,中者且攻且补,末者任受补";**"大积大聚,衰其大半而止"、"屡攻屡补,以平为期"**——攻伐、破血、香燥之品均不可过用久用。

## 鼓胀

> **定位:相当于肝硬化腹水等,中医四大难症(风痨臌膈)之一,须专科管理;变证为红线。**

**定义**:腹大如鼓、皮色苍黄、脉络显露(腹壁青筋)、脐孔突起。病机:肝脾肾受损,**气、血、水**互结("胀病亦不外水裹、气结、血瘀")。

**鉴别**:与水肿——水肿从眼睑或下肢起、遍及全身,鼓胀以腹为主四肢不肿(晚期方肿)且有肝病体征(面颈血痣赤缕、胁下癥积)。气臌(叩之如鼓)/水臌(按之如囊裹水)/血臌(青筋显露、刺痛、赤缕)。

**分证论治**:气滞湿阻——柴胡疏肝散合胃苓汤;水湿困脾——实脾饮;水热蕴结——中满分消丸合茵陈蒿汤;瘀结水留——调营饮;阳虚水盛——附子理苓汤或济生肾气丸;阴虚水停——六味地黄丸合一贯煎("阳虚易治,阴虚难调"——滋阴用甘寒淡渗,少佐桂附通阳)。

**攻逐水药(舟车丸/控涎丹/十枣汤/甘遂大戟芫花)**:教材明载三条铁律——①中病即止,衰其大半而止;②严密观察,呕吐腹痛腹泻即停;③**正虚体弱、发热、黄疸加深、有消化道溃疡或出血史者禁用**。此类药一律医师主导,**绝不自行使用**。

**变证(红线,立即急救)**:①大出血——骤然大量呕血、便血(食管静脉曲张破裂);②神昏(肝性脑病)——由嗜睡烦躁渐入昏迷;③由闭转脱。**鼓胀患者呕血黑便、性格行为改变、嗜睡,一律急诊。**

**调护**:忌盐酱过咸(古训"更断盐酱"——低盐),戒酒,忌粗硬食物(防曲张静脉破裂),情志舒畅;腹水消退后仍须调治防复发。

## 头痛

**定义**:以头痛为主症的病证。外感(风为主,夹寒/热/湿)与内伤(肝、脾、肾)两纲。

**红线——真头痛**:突发剧烈头痛持续不解、"旦发夕死"、伴喷射呕吐/项强/神昏——蛛网膜下腔出血、高血压危象等颅内急症,**立即急救**。新发头痛应测血压。

**经络定位与引经药(教材要诀)**:太阳头痛(后头连项)——羌活、蔓荆子、川芎;阳明头痛(前额眉棱骨)——葛根、白芷、知母;少阳头痛(两侧连耳)——柴胡、黄芩、川芎;厥阴头痛(巅顶或连目系)——吴茱萸、藁本。

**分证论治**:

| 证 | 特征 | 代表方 |
|---|---|---|
| 风寒头痛 | 痛连项背、拘紧感、遇风剧 | 川芎茶调散;巅顶痛干呕吐涎沫吴茱萸汤加减 |
| 风热头痛 | 胀痛如裂、面红目赤、口渴 | 芎芷石膏汤 |
| 风湿头痛 | 头痛如裹、肢体困重、苔白腻 | 羌活胜湿汤 |
| 肝阳头痛 | 昏胀痛两侧为重、烦怒、口苦 | 天麻钩藤饮 |
| 血虚头痛 | 隐痛昏晕、面色少华、遇劳加重 | 加味四物汤 |
| 痰浊头痛 | 昏蒙、胸脘满闷、呕恶苔腻 | 半夏白术天麻汤 |
| 肾虚头痛 | 空痛、眩晕耳鸣、腰膝酸软 | 大补元煎;阳虚右归丸 |
| 瘀血头痛 | 经久不愈、刺痛固定、或有外伤史 | 通窍活血汤 |

川芎为治头痛要药。顽固头痛可配虫类药(全蝎、蜈蚣、僵蚕、地龙——**有小毒,汤剂全蝎3~5g,研末1~2g,不可过用**)。偏头痛(偏头风)与肝阳肝火关系最密,平肝清热熄风通络。

**调护(A级)**:肝阳者忌肥甘辛辣,肝火头痛可冷毛巾敷头;痰浊者饮食清淡;禁烟酒;头部保健按摩可疏经活血。

## 眩晕

**定义**:眼花(眩)+头晕甚则旋转如坐车船(晕)。病位清窍,关乎肝脾肾;"诸风掉眩皆属于肝"、"无痰不作眩"(丹溪)、"无虚不能作眩"(景岳)。虚者居多。

**红线警示**:**"眩晕乃中风之渐"**——中年以上肝阳上亢者若眩晕头胀、面赤、肢麻震颤,警惕中风,监测血压;伴口齿不清、肢体无力立即急诊。眩晕原因多(高血压、低血压、贫血、耳源性、椎-基底动脉供血不足),就医查因(测血压是第一步,可用家用血压计)。

**鉴别**:中风(有昏仆偏瘫)、厥证(昏不知人)、痫证(昏仆抽搐吐沫)。

**分证论治**:肝阳上亢——天麻钩藤饮;气血亏虚——归脾汤;肾精不足——左归丸(阳虚右归丸);痰湿中阻——半夏白术天麻汤(化火黄连温胆汤);瘀血阻窍——通窍活血汤。

**调护(A级)**:发作时卧床、避免突然剧烈的体位改变与头颈转动;避免高空作业与驾驶;饮食忌过咸(伤肾/血压)、肥甘醇酒;颈椎病相关者锻炼颈肩肌肉。

## 中风

> **红线之首:突然口眼歪斜、半身不遂、言语不利、昏仆——无论轻重,立即呼叫急救(缺血性中风溶栓有严格时间窗)。中医所有内容均为住院救治之辅助与恢复期康复。**

**定义**:卒然昏仆、不省人事、半身不遂、口眼歪斜、语言不利;轻者无昏仆仅见歪斜不遂。病位在脑(心脑),病理基础肝肾阴虚,病理因素风、火、痰、瘀,病机总属**阴阳失调,气血逆乱,上冲于脑**。

**先兆(预防重点,教材古训)**:"眩晕者,中风之渐也";"凡大指、次指麻木或不用者,三年内有中风之患"——**头晕、一侧肢体麻木、一过性言语不利均为先兆,应就医控制血压、调摄起居饮食情志**。

**辨证框架**:
- **中经络**(神志清):风痰入络——真方白丸子;风阳上扰——天麻钩藤饮;阴虚风动——镇肝熄风汤。
- **中脏腑·闭证**(昏迷、牙关紧闭、两手握固):痰热腑实——桃仁承气汤(通腑泄热为要);**阳闭**(痰火瘀闭:面赤气粗、苔黄腻脉弦滑数)——羚角钩藤汤+安宫牛黄丸/至宝丹;**阴闭**(痰浊瘀闭:面白唇暗、静卧不烦、四肢不温、苔白腻)——涤痰汤+苏合香丸。
- **中脏腑·脱证**(目合口张、手撒肢冷、汗多、二便自遗、鼻息低微——阴竭阳亡)——参附汤合生脉散急救。闭证可恶变转脱,内闭外脱者开窍与固脱兼用。
- **恢复期/后遗症期**(发病2周~半年/半年以上):风痰瘀阻——解语丹;**气虚络瘀——补阳还五汤**(半身不遂经典方,无风阳痰热者用);肝肾亏虚——左归丸合地黄饮子。配合针灸与功能锻炼。

**鉴别·口僻(面瘫)**:仅口眼歪斜+耳后疼痛流涎,**无半身不遂与神志障碍**,风邪入络所致,任何年龄可患——牵正散(散剂效佳)。但**初发时无法自行区分面瘫与中风者,一律先按中风急诊**。

**调护**:昏迷期防闭脱转化、口腔护理、少量频服;恢复期尽早被动活动与语言训练、防患肢受压变形、防褥疮;饮食清淡忌肥甘辛辣、禁烟酒;控制血压、防复中。

## 瘿病

**定义**:颈前喉结两旁结块肿大(可随吞咽上下移动)。对应甲状腺肿、甲亢、甲状腺炎、腺瘤、癌等。病机:气滞、痰凝、血瘀壅结颈前;与情志、水土(碘)、体质(女性多发)相关。

**鉴别**:瘰疬——肿块在颈侧或颌下、累累如串珠而小;消渴——多饮多食多尿而颈无肿块(瘿病阴虚火旺型有多食易饥但伴瘿肿、眼突、烦热心悸)。**肿块短期迅速增大、坚硬不平、不可移——防恶变,立即专科检查**(超声+甲功)。

**分证论治**:气郁痰阻——四海舒郁丸;痰结血瘀——海藻玉壶汤(**本方海藻与甘草同用属十八反反药同用之特例,医师方剂范畴,见安全表**);肝火旺盛(烦热多汗、急躁、眼突、手颤)——栀子清肝汤合消瘰丸;心肝阴虚——天王补心丹或一贯煎。

**用药安全(教材明载,重要)**:①海带、海藻、海螵蛸、海蛤壳等**含碘量高**——缺碘性单纯甲状腺肿可用,**甲亢者慎用**;②**黄药子有小毒,长期服用损肝**,用量一般不超过10g,必须慎用。

**调护(A级)**:缺碘地区食用碘盐、适量海带;保持情志舒畅;观察瘿肿形态变化,经治不消、增大变硬即就医。

## 疟疾

> **定位:疟原虫感染性疾病,确诊靠血涂片,治疗以现代抗疟药为主(青蒿素为中医药重大成果);疑似疟疾(疫区旅行史+周期性寒热)立即就医。瘴疟(凶险型疟疾,神昏痉厥)为红线急症。**

**定义**:感受疟邪,寒战、壮热、头痛、汗出、**休作有时**(每日/间日/三日一发)。"疟不离少阳"。

**分证**:正疟(典型发作)——柴胡截疟饮/截疟七宝饮;温疟(热多寒少)——白虎加桂枝汤/白虎加人参汤;寒疟(寒多热少)——柴胡桂枝干姜汤合截疟七宝饮;**瘴疟**——热瘴(壮热神昏谵语)清瘴汤+紫雪丹,冷瘴(寒甚昏蒙)加味不换金正气散+苏合香丸——**均属急救住院范畴**;劳疟(久疟遇劳即发)——何人饮;**疟母**(久疟胁下痞块=脾肿大)——鳖甲煎丸,兼虚配八珍汤。

**要诀**:截疟药常山、青蒿、槟榔、马鞭草等;**服药宜在发作前2小时**,发作时服药易吐。青蒿治疟自《肘后方》始。

**调护**:防蚊灭蚊为本;寒战时保暖热饮,高热可冷敷;汗后温水擦身换衣避风;久疟饮食调补。
