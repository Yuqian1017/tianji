# 四诊(二):问诊、切诊

> 提炼自《中医诊断学》第二章第三、四节。

## 一、问诊

原则:确定**主诉**(主要矛盾)围绕询问、**问辨结合**(边问边辨,缺什么证据问什么)。内容:一般项目、主诉与病史(现病史/既往史/生活史/家族史)、现在症状(依**十问歌**:"一问寒热二问汗,三问头身四问便,五问饮食六问胸,七聋八渴俱当辨,九问旧病十问因,再兼服药参机变;妇女尤必问经期")。

### 问寒热(四型)

1. **但寒不热**:恶风(遇风怕冷避之可缓)=外感风邪/卫气虚;**恶寒**(加衣近火不减)=外感初起,实;寒战=恶寒之甚;**畏寒**(得温可缓)=里寒证(阳虚)。⇒ 鉴别眼目:加衣被能否缓解。
2. **但热不寒**:**壮热**(高热不退)=里实热;**潮热**三型——**阳明潮热**(日晡热甚,腹满便秘)=胃肠燥屎内结、**湿温潮热**(身热不扬:初扪不热久扪灼手,午后甚)=湿热、**阴虚潮热**(午后夜间低热、五心烦热、骨蒸)=阴虚;**微热**(长期低热)=温病余邪/气虚发热(劳累加重)/小儿夏季热。
3. **恶寒发热并见**=表证:寒重热轻=表寒;热重寒轻=表热;兼恶风自汗脉浮缓=表虚;无汗头身痛脉浮紧=表实。寒热俱轻=邪轻正盛,俱重=邪盛正实,寒重热轻亦可为邪盛正虚。
4. **寒热往来**(界线分明交替)=半表半里(少阳)、疟疾(定时发作)。

### 问汗

- 无汗:表证无汗=邪郁肌表(表实);里证无汗=津血亏。
- 有汗:表证有汗=中风表虚;大汗蒸蒸+口渴饮冷=里实热;**冷汗淋漓/汗出如油+肢厥脉微=脱汗(绝汗),亡阳危候**;**自汗**(醒时汗出动则甚)=气虚阳虚;**盗汗**(睡时汗出醒则止)=阴虚;**战汗**(先寒战后汗出)=邪正剧争之转折——汗后脉静身凉为佳,汗后烦躁脉疾为危。
- 局部:头汗=上焦邪热/中焦湿热/虚阳浮越;半身汗=中风先兆、痿证截瘫(患侧经络闭阻);手足心汗=郁热/阴虚阳亢。

### 问疼痛

**性质→病机**:胀痛=气滞;**刺痛(固定不移)=瘀血**;绞痛=实邪闭阻/寒凝;串痛(游走)=风邪/气滞;掣痛=筋脉失养/经脉阻滞;灼痛喜冷=火热/阴虚;冷痛喜温=寒凝/阳虚;重痛=湿困;空痛喜按=精血不足;隐痛绵绵=气血不足虚证。总纲:**不通则痛为实,不荣则痛为虚**。

**部位→归经归脏**:
- 头痛分经:后头连项=太阳,前额=阳明,两侧=少阳,巅顶=厥阴,痛连齿=少阴。外感痛剧无休止;内伤痛缓时止(气虚劳甚、血虚隐隐眩晕面白、肾虚空痛腰酸、血瘀刺痛、痰浊如裹泛呕、肝火胀痛口苦)。
- 胸痛:胸痹(憋闷引肩臂)=心脉不畅(阳虚痰浊气虚血瘀);**真心痛**(彻背剧痛、面青灰、手足青至节);壮热喘促鼻煽=热邪壅肺;潮热盗汗咳血=肺阴虚;胀痛走窜太息=肝郁;刺痛固定=血瘀。
- 胁痛=肝胆病(气郁胀痛、火郁灼痛、湿热身黄、瘀血刺痛、悬饮咳唾引痛)。
- 胃脘痛:冷痛得热减=寒邪犯胃;灼痛消谷口臭便秘=胃火;胀痛嗳气=肝气犯胃;刺痛固定=瘀血;胀痛嗳腐吞酸=食滞;隐痛呕清水=胃阳虚;灼痛嘈杂饥不欲食=胃阴虚。
- 腹痛:大腹属脾胃,脐腹属脾与小肠,小腹属膀胱胞宫大小肠,少腹属肝经。隐痛便溏喜温按=脾胃虚寒;小腹胀痛尿不利=癃闭(膀胱);小腹刺痛尿不利=蓄血;少腹冷痛引阴部=寒凝肝脉;绕脐痛起包块可移=虫积。暴急胀痛拒按得食甚=实;徐缓隐痛喜按得食减=虚;得热减=寒,喜冷=热。
- 腰痛:冷痛脊痛活动受限=寒湿痹;冷痛尿清长=肾虚;刺痛固定=闪挫瘀血;腰痛连腹绕如带状=带脉病。
- 四肢关节痛:串痛=风痹(行痹);困重=湿痹(着痹);剧痛得热减=寒痹(痛痹);灼痛红肿喜冷=热痹;足跟胫膝隐痛=肾虚。
- 周身痛:新病+表证=外邪束表;久病卧床=气血亏虚。

### 问其他不适

头晕(风火上扰/阴虚阳亢/血虚/中气不足/肾精不足/痰浊中阻);目眩=肝肾阴虚阳亢或气血不足;雀目(夜盲)=肝血肾阴亏;**耳鸣**:暴鸣声大按之不减=实(肝胆火),渐鸣声细按之减轻=虚(肾精亏);耳聋:新聋多实、渐聋多虚;胸闷=气机不畅;心悸怔忡(心阳虚/气血不足/阴虚火旺/水饮凌心/痰阻/瘀血);腹胀:上腹=脾胃,小腹=膀胱,胁下=肝胆,虚则气不运、实则气郁滞;麻木=气血不足或风痰湿瘀阻络。

### 问饮食口味

- 口渴:不渴=寒证/津未伤;**渴多饮**=津伤(实热、消渴、汗吐下后);**渴不多饮**=阴虚、湿热、痰饮、瘀血(津液输布障碍)。
- 食欲:纳呆=脾胃气虚/湿邪困脾;厌食=伤食(妇女停经厌食呕=妊娠恶阻);**饥不欲食**=胃阴不足;**消谷善饥**=胃火亢盛(兼消瘦见消渴);小儿异嗜泥土生米=虫积;喜热食=寒,喜冷食=热;食后稍安=虚,食后加重=实。病中食欲渐复=胃气渐复;**除中**(病重不能食忽暴食)=中气衰败回光返照。
- 口味:淡=脾胃气虚;甜=脾胃湿热;粘腻=湿困;酸=肝胆蕴热,酸腐=伤食;**苦=热证(肝胆郁热)**;咸=肾病寒证。

### 问二便

- 便秘=胃肠积热/气滞/气血津亏/阴寒凝结;溏泻=脾虚/肾阳虚(五更泄)/肝郁乘脾/伤食/湿热。
- 排便感:肛门灼热=大肠湿热;排便不爽=气滞(肝郁/伤食/湿热);**里急后重**=痢疾(湿热气滞);滑泻失禁=脾肾阳衰;肛门气坠=中气下陷。
- 尿:量多=虚寒/肾阳虚/消渴;量少=津亏/水肿癃闭;尿频=下焦湿热/下焦虚寒/肾气不固;**小便涩痛**=淋证(湿热);**癃闭**(点滴而出/不通)=实(湿热气结瘀石)或虚(肾阳衰);余沥不尽、失禁、遗尿=肾气不固/下元虚(神昏遗尿=危重)。

### 问睡眠、经带

- **失眠**(阳不入阴):虚——心脾两虚、心肾不交、阴虚阳亢;实——痰火扰心、食滞("胃不和则卧不安")。
- **嗜睡**:湿困清阳、脾气虚弱;**但欲寐**=心肾阳衰;昏睡不醒=热入营血邪陷心包/中风。
- 月经:先期=血热/气虚不摄;后期=血寒/血虚/血瘀;先后不定=肝郁/脾肾虚/瘀。量多=血热/气虚/瘀;量少=血虚/寒凝/血瘀。崩漏=血热、气虚(脾不统血)、瘀血。经闭须先排除妊娠等生理性。**痛经**:经前痛=实,经后痛=虚;按之痛甚=实,按之减=虚;得热减=寒;刺痛=瘀,胀甚于痛=气滞,痛甚于胀=血瘀,隐痛=血虚。
- 带下:白清稀无臭=虚寒(寒湿下注);黄稠臭秽=湿热下注;赤白带=肝经郁热。

## 二、脉诊

**部位**:独取寸口(腕后桡动脉)。高骨(桡骨茎突)内侧为**关**,腕端为**寸**,肘端为**尺**;浮中沉三候合为三部九候。分候脏腑:**左寸心、左关肝胆、左尺肾;右寸肺、右关脾胃、右尺肾(命门)**。
**方法**:平旦为佳(不拘泥);平臂仰掌;中指定关、三指弓形指腹按脉,总按与单按结合;**举(轻取)、按(重取)、寻(委曲求之)**;平息计至数;**五十动**(每侧不少于50跳,防漏促结代,约2-3分钟)。小儿一指定关法。

**平脉**:一息四至(闰以太息五至,72-80次/分),不浮不沉,从容和缓,尺脉沉取有力。三要素:**有胃**(从容和缓——脉无胃气为真脏脉,不治)、**有神**(柔和有力)、**有根**(尺脉沉取有力,肾气犹存)。生理变异:四时(春弦夏洪秋浮冬沉)、地域南软北实、女性濡弱略快(孕滑数冲和)、小儿快、瘦人浮胖人沉、六阴脉六阳脉、斜飞脉反关脉(桡动脉变异,非病)。

### 28 病脉分类(按位、数、形、势)

**浮脉类**(轻取即得):
| 脉 | 脉象 | 主病 |
|---|---|---|
| 浮 | 举之有余,如水漂木 | 表证(浮有力表实,浮无力虚阳外浮) |
| 洪 | 极大,来盛去衰,如波涛 | 里热炽盛(久病虚证见洪=正虚邪盛危候) |
| 濡 | 浮而细软,如帛在水 | 虚证、湿证 |
| 散 | 浮散无根,至数不齐,如杨花 | 元气离散,脏气将绝 |
| 芤 | 浮大中空,如按葱管 | 失血、伤阴 |
| 革 | 浮而搏指,中空外坚,如按鼓皮 | 亡血失精、半产漏下 |

**沉脉类**(重按乃得):沉(里证:沉有力里实、沉无力里虚)、伏(重按推筋着骨始得:邪闭、厥证、痛极)、弱(极软沉细:气血阴阳俱虚)、牢(沉按实大弦长坚牢:阴寒凝结、内实坚积)。

**迟脉类**(一息不足四至):迟(寒证:迟有力冷积、迟无力虚寒;邪热结聚亦可迟而有力,勿概认为寒)、缓(一息四至怠缓:湿证、脾胃虚;病中转缓和=正气恢复)、涩(往来艰涩如轻刀刮竹:精血亏[无力]、气滞血瘀痰食[有力])、**结**(缓而时一止、止无定数:阴盛气结、寒痰血瘀;虚证为脉气不继)。

**数脉类**(一息五至以上):数(热证:有力实热、无力虚热;浮数无根=虚阳外越)、疾(一息七八至:阳极阴竭、元阳将脱)、**促**(数而时一止、止无定数:阳热亢盛、气血痰食郁滞;促而无力=虚脱)、动(如豆厥厥动摇、滑数有力:痛证、惊证;孕妇可见)。

**虚脉类**(应指无力):虚(三部举按无力:虚证)、**细**(如线应指明显:气血两虚、诸虚劳损、湿)、微(极细软似有若无:阴阳气血诸虚、阳气衰微)、**代**(止有定数、良久方来:脏气衰微;风证痛证为实;妊娠初期可见)、短(首尾俱短:气病——有力气滞、无力气虚)。

**实脉类**(应指有力):实(三部举按均有力:实证)、**滑**(往来流利如珠走盘:痰饮、食积、实热;孕脉滑而和缓)、**弦**(端直以长如按琴弦:肝胆病、痰饮、痛证、疟疾;弦细劲如循刀刃=胃气全无难治)、**紧**(绷急如牵绳转索:寒证、痛证)、长(超过本位:阳有余;长而和缓为平)。

> 促、结、代鉴别:皆有歇止——促=数中一止无定数(热/郁),结=缓中一止无定数(阴盛寒瘀),代=止有定数良久方来(脏气衰微)。

**常见相兼脉主病**(相兼脉主病=各脉主病之和):浮紧=表寒/风痹;浮缓=表虚(太阳中风);浮数=表热;浮滑=风痰;沉迟=里寒;弦数=肝热肝火;滑数=痰热食积;洪数=气分热盛;沉弦=肝郁气滞/水饮;沉涩=血瘀;弦细=肝肾阴虚/肝郁脾虚;沉缓=脾虚水湿;沉细=阴虚血虚;弦滑数=肝火挟痰;沉细数=阴虚血虚有热;弦紧=寒痛/寒滞肝脉。

**脉症顺逆与从舍**:脉症相应为顺,相反为逆(实证见洪数滑实为顺,见细微弱为逆;暴病宜浮洪数实,久病宜沉微细弱)。脉症不应必有真假:**舍脉从症**(如腹满痛拒按便结苔黄燥而脉迟细——症真热、脉假)、**舍症从脉**(如热闭肢厥而脉滑数——脉真热、症假寒)。

## 三、按诊

- **按肌肤**:初按热甚久按转轻=表热,久按愈甚=里热;濡软喜按=虚,硬痛拒按=实;按之凹陷不即起=水肿,举手即起=气肿;疮疡:边硬顶软=已成脓,波动应手=有脓。
- **按手足**:手足背热=外感发热,**手足心热=内伤发热**;额热甚于手心=表热,手心甚于额=里热;阳衰证四肢犹温尚可治,厥冷则凶。
- **按虚里**(左乳下心尖搏动,候宗气):动而不紧缓而不急=常;微弱=宗气内虚;动而应衣=宗气外泄;按之弹手洪大而搏=危候。
- **按腹**:腹壁冷喜暖手按=虚寒,灼热喜冷=实热;痛喜按=虚,拒按=实;实满(按之充实叩之重浊)vs 气胀虚满(按之不实叩之空声);**水臌**(拍之有波动感、按有凹痕)vs**气臌**(叩之如鼓无波动);痞满按之柔软=虚、硬有抵抗压痛=实;**积**(痛有定处按之有形不移,病在血分)vs**聚**(痛无定处按之无形聚散不定,病在气分);左小腹按之累累硬块=宿粪,右小腹痛按有包块=肠痈;虫块如筋结、指下蠕动、聚散不定。
- **按腧穴**:脏腑病可在相应俞募穴出现结节压痛(肺俞中府、肝俞期门、胃俞足三里、阑尾穴等)。

## 经典补强(v2.0 经典五书反哺;详 [[43-经典-灵枢与难经要义]] 第七、十节)

- "**临病人问所便**"(灵枢·师传):问患者当下想要什么是问诊捷径——欲寒饮=中热,欲热饮=寒中(与八纲寒热问诊直接衔接);同篇医患沟通范式:"**告之以其败,语之以其善,导之以其所便,开之以其所苦**"(先讲利害、再给出路、顺其所安)。
- "**望而知之谓之神,闻而知之谓之圣,问而知之谓之工,切脉而知之谓之巧**"(难经六十一难)——四诊并重、切诊居末,不迷信摸脉的经典依据(本 Skill"脉只自测率与律"的立场与此同源)。
- **独取寸口**之理:"寸口者,脉之大会,手太阴之脉动也……五脏六腑之所终始"(难经一难);"五脏六腑之气味皆出于胃,变见于气口"(素问·五藏别论)。"人无胃气曰逆,逆者死"(平人气象)——脉贵有胃气,食欲存亡是家庭可观察的对应指标。
