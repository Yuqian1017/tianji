# 病因辨证与气血津液辨证

> 提炼自《中医诊断学》第四章第一、二节。病因辨证是外感病辨证的基础;气血津液辨证与脏腑辨证互参。

## 一、病因辨证

### 六淫疫疠证候

| 证 | 临床表现 | 辨证要点 |
|---|---|---|
| **风淫** | 发热恶风、汗出、咳嗽鼻塞流涕、苔薄白脉浮缓;或麻木、口眼歪斜、抽搐强直;或皮肤瘙痒 | 发病急、消退快、游走不定 |
| **寒淫** | 恶寒发热无汗、头身痛、喘咳、苔薄白脉浮紧(表);手足拘急、四肢厥冷脉微欲绝;或腹痛肠鸣泄泻呕吐(中里) | 寒象+凝滞收引之痛 |
| **暑淫** | **伤暑**:恶热汗出口渴疲乏尿黄、脉虚数;**中暑**:发热卒然昏倒、汗出不止、气急,甚昏迷惊厥、舌绛干燥脉濡数 | 夏令+热象+耗气伤津+多挟湿 |
| **湿淫** | **伤湿**(表湿):头胀痛、胸闷、口不渴、身重痛、发热体倦、苔白滑脉濡缓;**冒湿**:首如裹、遍体不舒;湿伤关节:酸痛重着屈伸不利(着痹) | 重着、粘滞、缠绵 |
| **燥淫** | **凉燥**:恶寒重发热轻、无汗头痛、咳嗽咽痒鼻塞、舌白干脉浮(似风寒+燥象);**温燥**:身热微恶风寒、少汗、口渴心烦、干咳痰少带血、舌干苔黄脉浮数(似风热+伤津) | 干燥伤津+咳 |
| **火淫** | 壮热口渴、面红目赤、汗出心烦,或谵妄、衄血吐血斑疹、躁扰发狂、痈脓,舌红绛脉洪数或细数 | 大热+伤津+动血生风+腐肉成痈 |
| **疫疠** | 病初恶寒发热俱重,继之壮热、面垢、烦躁神昏抽搐、舌红绛,苔黄厚燥或**白如积粉**、脉数有力 | 传染性强。大头瘟(头面颈红肿咽痛)、烂喉痧(咽烂+猩红皮疹)、疫喉/白喉(白膜犬吠咳)、疫咳/百日咳、疫毒痢(下痢赤白里急后重) |

### 七情证候(内伤杂病)

喜伤:精神恍惚、神志错乱哭笑无常、脉缓;怒伤:头晕胀痛面红目赤、胸闷善太息、胁胀窜痛、呃逆呕吐,甚呕血昏厥、脉弦;思伤:健忘心悸失眠多梦、食少腹胀便溏、消瘦、脉缓(伤脾+暗耗心血);忧伤:情志抑郁、神疲食少、脉涩;悲伤:面色惨淡、吁叹饮泣、精神萎靡、脉弱;恐伤:少腹胀满、遗精滑精、二便失禁;惊伤:情绪不安惶恐、心悸失眠、甚神乱。

### 饮食劳逸外伤证候

- 饮食所伤:伤胃(胃痛恶闻食臭、胸膈痞满、吞酸嗳腐、苔厚腻脉滑);伤肠(腹痛泄泻);误食毒物(吐泻交作、腹痛如绞甚痉挛昏迷)。
- 劳逸所伤:过劳(倦怠嗜卧懒言食少);过逸(体胖行动不便、动则喘喝心悸、肢软)。
- 房室所伤:头晕耳鸣、腰膝酸软消瘦;男遗精早泄阳痿,女梦交宫寒不孕、经少经闭、带下清稀。
- 外伤:金刃跌仆(青紫肿痛→伤筋折骨→内伤脏腑吐血下血→陷骨伤脑戴眼神昏);虫兽伤(见 05-病因)。

## 二、气血津液辨证

### 气病四证

| 证 | 辨证要点 | 表现 |
|---|---|---|
| **气虚** | 机能减退+**动则加剧** | 少气懒言、神疲乏力、头晕目眩、**自汗**、舌淡苔白、脉虚无力 |
| **气陷** | **内脏下垂** | 头晕目花、少气倦怠、久痢久泄、腹部坠胀、脱肛、子宫脱垂、脉弱 |
| **气滞** | **胀闷、疼痛、攻窜阵发** | 随部位:食积脘腹胀痛、肝郁胁肋窜痛;辨气滞须与辨因辨位结合 |
| **气逆** | 气机逆而向上 | 肺气逆=咳喘;胃气逆=呃逆嗳气恶心呕吐;肝气逆=头痛眩晕昏厥呕血 |

### 血病四证

| 证 | 辨证要点 | 表现 |
|---|---|---|
| **血虚** | 面唇爪甲舌**淡白失荣**+全身虚弱 | 面白无华或萎黄、头晕眼花、心悸失眠、手足发麻、经少色淡后期闭经、脉细无力 |
| **血瘀** | **刺痛定处拒按夜甚、肿块、紫暗、脉涩** | 症积、出血紫暗夹块、柏油便、面黧黑肌肤甲错、皮下紫斑丝状红缕腹部青筋、经闭、舌紫暗瘀斑 |
| **血热** | **出血+热象**(内伤杂病;外感血分证另见卫气营血) | 各种出血、月经先期量多、心烦口渴、舌红绛、脉滑数 |
| **血寒** | **手足少腹冷痛、肤色紫暗、得温减** | 喜暖恶寒、经期推迟、痛经、经色紫暗夹块、舌紫暗苔白、脉沉迟涩 |

### 气血同病五证

- **气滞血瘀**:胸胁胀满窜痛急躁+痞块刺痛拒按、经闭痛经、舌紫暗、脉弦涩(多关乎肝)。
- **气虚血瘀**:面淡白或晦滞、倦怠少气+刺痛不移拒按(多见心肝病位)、舌淡暗、脉沉涩。
- **气血两虚**:头晕目眩、少气懒言、乏力自汗+面淡白萎黄、心悸失眠、舌淡嫩、脉细弱。
- **气不摄血**:出血(吐血便血紫斑崩漏)+气短倦怠面白、舌淡脉细弱。
- **气随血脱**:大出血时突然面色苍白、四肢厥冷、大汗淋漓甚晕厥、脉微欲绝或浮大而散。

### 津液病辨证

- **津液不足**:口渴咽干、唇燥裂、皮肤干枯、尿少便结、舌红少津、脉细数。
- **水液停聚**:
  - **水肿·阳水**(急、实):眼睑头面先肿、上半身甚、来势迅速;风水偏寒(恶寒无汗苔薄白脉浮紧)/偏热(咽喉肿痛舌红脉浮数);水湿浸渍(全身肿按之没指、身重脘闷苔白腻脉沉)。
  - **水肿·阴水**(缓、虚):**足先肿、腰以下甚、按之凹陷不起**;脾虚(脘闷腹胀纳呆便溏)、肾阳虚(腰膝冷痛、四肢不温、面㿠白、舌淡胖苔白滑、脉沉迟无力)。
  - **痰证**:咳痰粘稠、胸脘满闷、纳呆呕恶、头晕目眩,或神昏癫狂喉中痰鸣、肢体麻木、瘰疬瘿瘤乳癖痰核,**苔白腻、脉滑**("诸般怪证皆属于痰")。
  - **饮证**:咳喘痰多而稀、胸闷心悸、倚息不得卧(饮停心肺);脘腹痞胀水声漉漉、泛吐清水(胃肠);肢体浮肿沉重、尿不利(四肢);**苔白滑、脉弦**。

## 经典补强(v2.0 经典五书反哺;详 [[45-经典-金匮要略杂病方证条文]] 第三节)

- **瘀血证候原文**(金匮·惊悸吐血下血胸满瘀血病):"病人胸满,**唇痿舌青**,口燥,**但欲嗽水,不欲咽**……腹不满,**其人言我满**,为有瘀血";"病者如热状,烦满,口干燥而渴,**其脉反无热**,此为阴伏,是瘀血也"——**漱水不欲咽、自觉胀满而按之不满、如热状而脉不热**三条,皆问诊即可捕捉的瘀血指征(与本文件血瘀证候表互参)。
- "**经为血,血不利则为水**"(金匮·水气病):妇人经水前断、后病水肿者名"血分"——血瘀致水肿的经典命题。
- 津液脱失分层(六脱)见 [[04-精气血津液]] 经典补强;"百病生于气"九气条(怒则气上/思则气结等)见 [[42-经典-素问要义]] 第五节举痛论条。
