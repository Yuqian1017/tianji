# 37 内科·肢体经络病证

## 痹证

**定义**:风寒湿热等邪闭阻经络,肢体筋骨关节肌肉疼痛、重着、麻木、屈伸不利,甚则肿大变形。对应风湿/类风湿关节炎、强直性脊柱炎、痛风等。"风寒湿三气杂至,合而为痹。"

**与痿证鉴别**:痹证**以痛为主**、因痛而活动受限;痿证**无痛**、肢体软弱无力。

**辨证**:行痹——痛游走不定(风胜);痛痹——痛剧有定处遇寒加重(寒胜);着痹——酸痛重着漫肿(湿胜);热痹——红肿灼热痛不可触(热胜);久病见皮下结节为痰、关节僵硬紫暗为瘀;日久肝肾亏虚。**关节红肿热痛伴高热者(热痹/化脓性/痛风急性发作)应就医查血。**

**分证论治**:

| 证 | 治法 | 代表方 |
|---|---|---|
| 行痹 | 祛风通络散寒除湿 | 防风汤 |
| 痛痹 | 散寒通络祛风除湿 | 乌头汤(**含川乌**) |
| 着痹 | 除湿通络祛风散寒 | 薏苡仁汤 |
| 久痹风寒湿不显 | 益气和营蠲痹 | 蠲痹汤 |
| 风湿热痹 | 清热通络祛风除湿 | 白虎加桂枝汤合宣痹汤;热毒炽盛五味消毒饮合犀黄丸 |
| 痰瘀痹阻 | 化痰行瘀蠲痹通络 | 双合汤 |
| 肝肾两虚 | 培补肝肾舒筋止痛 | 补血荣筋丸;正虚邪恋通用独活寄生汤 |

**引经用药(教材要诀)**:上肢——片姜黄、羌活、桂枝;下肢——独活、川牛膝、木瓜;颈项——葛根、伸筋草;腰——桑寄生、杜仲、淫羊藿;膝肿积液——土茯苓、车前子、薏苡仁;小关节肿灼——土贝母、蜂房、威灵仙。"治风先治血,血行风自灭。"

**有毒药物警示(教材原文,务必遵守)**:
- **附子、川乌、草乌**:生用毒大须炮制,内服常用5~12g,小量递增、不可久服;**文火久煎或与甘草同煎可缓毒**;中毒表现——唇舌发麻、头晕、心悸、恶心、脉迟,**即停服,绿豆甘草汤频饮,危重按中毒急救**。
- **雷公藤**:大毒;去皮根心、先煎1小时,常用10~25g;副作用——胃肠反应、**肝损害、白细胞血小板减少**、心悸心律紊乱、女子闭经,须监测。
- **马钱子**:大毒;炮制后入丸散0.2~0.6g;中毒——烦躁、颈项强硬、角弓反张,甚则死亡。
- 虫类搜风药(全蝎、蜈蚣等)辛温有毒,量不可大、不宜久服;焙干研末吞服可减量增效。

**痛风(教材专论)**:嘌呤代谢紊乱、尿酸沉积,下肢足趾关节红肿剧痛夜间发作——按痹证辨治,用凉血清热祛风除湿泄浊法;**调护关键(A级):控制高嘌呤食物(动物内脏、鱼虾海味),忌酒,避风寒过劳**。

**调护(A级)**:防风防寒防潮,汗出勿当风贪凉、勿乘热浴冷,汗湿内衣及时更换;适度锻炼;久病者保持关节功能位、防跌仆。

## 痉证

> **红线定位:项背强直、四肢抽搐、角弓反张=中枢神经系统急症的表现(脑膜炎、脑炎、高热惊厥、破伤风、肝昏迷、尿毒症等),一律立即急诊,病因治疗为根本;中医内容为配合抢救与辨证辅助。**

**定义**:项背强急、四肢抽搐,甚至口噤、角弓反张。刚痉(无汗)柔痉(有汗);病机——外感邪壅或热盛动风,内伤阴虚血少、筋脉失养("津亏血少,无以滋荣经脉则一")。

**鉴别**:痫证(发作片刻自止、反复发作史)、厥证(四肢逆冷无抽搐)、中风(偏瘫)、颤证(幅度小频率快之慢性颤动)、**破伤风**(有金疮破伤不洁史,先牙关紧闭苦笑面容后延及全身)。

**分证论治**:邪壅经络——羌活胜湿汤(寒甚葛根汤,风盛有汗栝蒌桂枝汤,湿热三仁汤加味);肝经热盛——羚角钩藤汤(神昏加安宫牛黄丸/至宝丹/紫雪丹);阳明热盛——白虎汤合增液承气汤;心营热盛——清营汤。内伤发痉以滋阴养血为大法。

**护理要点(现场,重要)**:发作时**保持呼吸道通畅、清除假牙异物防窒息**;**勿强行按压捆绑肢体**(防骨折);减少搬动与声光刺激;高热者降温。口张目瞪、戴眼反折、遗尿、汗出如油为预后不良征。

## 痿证

> **定位:肢体软弱无力、不能随意运动=迟缓性瘫痪,须神经科查因(重症肌无力、运动神经元病、多发性神经炎、脊髓病变等)。危重征:舌体瘫软、抬头无力、呼吸吞咽困难——立即急救。**

**定义**:肢体筋脉弛缓、软弱无力、不能随意运动,或伴肌肉萎缩,下肢多见。病机"肺热叶焦"、五脏虚损,以热证虚证为多。

**与痹证鉴别**:痿证不痛而无力;痹证因痛而活动受限。与偏枯(中风半身不遂伴语言謇涩口眼歪斜)鉴别。

**分证论治**:肺热津伤(热病后突发)——清燥救肺汤;湿热浸淫(困重痿软微肿)——加味二妙散;脾胃虚弱(渐起纳呆便溏)——参苓白术散合补中益气汤;肝肾亏损(腰膝酸软、腿胫大肉渐脱)——虎潜丸(阴阳两虚鹿角胶丸/加味四斤丸);脉络瘀阻——圣愈汤合补阳还五汤。

**治则要诀**:①**"治痿独取阳明"**——补益脾胃(养胃阴/健脾气)兼清胃火湿热;②**治痿慎用风药**("痿证断不可作风治而用风药"——发散伤阴血);③调畅气血,"气血流通即是补"。

**调护**:防褥疮、保持肢体功能位、防冻烫伤(知觉障碍者);适度锻炼拍打患肢促气血;清心寡欲、饮食清淡营养。

## 颤证

> **定位:对应帕金森病、特发性震颤、甲亢震颤、肝豆状核变性等——神经科/内分泌科确诊(必要时查铜代谢、甲功),中医长期调治可改善症状。**

**定义**:头部或肢体摇动颤抖、不能自制。多发中老年,隐袭起病渐进加重。基本病机:**肝风内动,筋脉失养**;本虚(阴津精血亏)标实(风火痰瘀)。

**与瘛疭(抽搐)鉴别**:颤证幅度小频率快呈慢性,无发热神昏;瘛疭抽搐持续、弛纵交替,多伴热病。

**分证论治**:风阳内动——天麻钩藤饮合镇肝熄风汤;痰热风动——导痰汤合羚角钩藤汤;气血亏虚——人参养荣汤;髓海不足——龟鹿二仙膏合大定风珠;阳气虚衰——温补肾阳方(附子、肉桂、巴戟天配熟地山萸肉、党参白术,白芍甘草缓急——**附子类B级**)。

**要诀**:各证均可配熄风止颤之品(钩藤、天麻、珍珠母、龙牡、全蝎、蜈蚣、僵蚕——虫类研末吞服为佳,入煎剂效逊);羚羊角可以山羊角代;**年高病久宜缓图,慎用攻伐**,能减轻症状、控制发展即应坚持。

**调护(A级)**:情绪稳定;太极拳、八段锦、内养功等锻炼;防跌倒;卧床者翻身按摩防褥疮;戒烟酒,忌肥甘。

## 腰痛

**定义**:腰脊或脊旁疼痛。病机:筋脉痹阻、腰府失养;"肾虚其本,风寒湿热痰瘀闪挫其标"。

**先排它病(教材明载)**:泌尿系感染结石(参淋证,伴尿频尿痛血尿)、肝胆疾病、妇科疾病、骨伤科疾病均可致腰痛——**"切忌腰痛治腰"**,伴发热、血尿、下肢放射痛麻木无力、夜间痛进行性加重者及时就医检查。

**辨证**:外感者起病急伴外感症状;内伤者隐袭酸痛缠绵(多肾虚);闪挫者起病急、痛处固定拒按、有外伤史。

**分证论治**:

| 证 | 特征 | 代表方 |
|---|---|---|
| 寒湿腰痛 | 冷痛重着、静卧不减、阴雨加重 | 甘姜苓术汤(肾着汤);久病独活寄生汤加附子 |
| 湿热腰痛 | 痛处伴热感、活动后或减、小便短赤 | 四妙丸 |
| 瘀血腰痛 | 痛如刺、固定拒按、日轻夜重 | 身痛逐瘀汤 |
| 肾阴虚 | 隐痛酸软、心烦咽干、手足心热 | 左归丸;相火旺知柏地黄丸 |
| 肾阳虚 | 隐痛局部发凉、肢冷畏寒、遇劳更甚 | 右归丸;无明显偏盛青娥丸 |

**调护(A级,实用)**:保持正确坐卧行姿势、勿强力负重、避免跌仆闪挫;避坐卧湿地;涉水冒雨汗出后及时换衣擦身,可服生姜红糖茶散寒;慢性腰痛注意保暖或用腰托;常做腰部自我按摩、太极拳;急性期及时治疗并休息。外治可配热熨(肉桂吴萸葱头花椒炒热布包熨痛处——A⁻须防烫伤)。
