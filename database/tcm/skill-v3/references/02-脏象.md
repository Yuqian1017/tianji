# 脏象(藏象)

> 提炼自《中医基础理论》第三章。脏象=内脏机能活动表现于外的征象,"据象定脏"。核心是**以五脏为中心的整体观**。

## 分类总纲

- **五脏**(心肝脾肺肾):藏精气而不泻,满而不能实。实体性器官。
- **六腑**(胆胃小肠大肠膀胱三焦):传化物而不藏,实而不能满,**以通为用、以降为顺**。
- **奇恒之府**(脑髓骨脉胆女子胞):形似腑(中空)、功似脏(藏精),藏而不泻。
- 七冲门:飞门(唇)、户门(齿)、吸门(会厌)、贲门(胃上口)、幽门(胃下口)、阑门(小肠大肠交)、魄门(肛门)。

## 五脏

### 心(五行火,阳中之阳,通夏)
- **主血脉**:推动血行,心气充沛+血液充盈+脉道通利为三前提。其华在面,充在血脉。
- **主神志(藏神)**:主思维意识("任物")、主宰生命活动(君主之官,五脏六腑之大主)。得神者昌失神者亡。
- 附**心包络**:代心受邪。热入心包=高热神昏谵语;痰浊蒙蔽心包=神昏意识障碍。
- 在志喜,在液汗,开窍舌,与小肠相表里。

### 肺(五行金,阳中之阴,通秋,华盖、娇脏)
- **主气司呼吸**:主呼吸之气+主一身之气(参与宗气生成、调节气机)。
- **主宣发肃降**:宣发(吸清呼浊、布散津液精微、宣发卫气调腠理汗孔);肃降(吸清、下布津液、通调水道、清肃洁净)。失宣→咳嗽鼻塞无汗;失降→喘促气逆咳痰。
- **通调水道**:肺为水之上源。**朝百脉、助心行血**。**主治节**(相傅之官)。
- 娇脏:不耐寒热燥湿,最畏火风,外邪多先犯肺。在志悲忧,在液涕,开窍鼻,外合皮毛,与大肠相表里。

### 脾(五行土,阴中之至阴,通长夏,后天之本)
- **主运化**:运化水谷(消化吸收转输精微,气血生化之源)+运化水湿(诸湿肿满皆属于脾;脾虚生湿、为生痰之源)。
- **主升清**:上输精微、维持内脏位置(脾气下陷/中气下陷→久泄脱肛内脏下垂)。
- **主生血统血**:生血(化源);统血(气摄血,脾不统血→便血尿血崩漏,尤下部出血)。
- 特性:**脾宜升则健、喜燥恶湿**(湿困脾土)。在志思,在液涎,开窍口其华在唇,主肌肉四肢,与胃相表里。

### 肝(五行木,阴中之阳,通春,刚脏、体阴用阳)
- **主疏泄**:①调畅气机 ②调节精神情志(因郁致病/因病致郁,七情之病必由肝起) ③促进消化(协调脾胃升降+分泌排泄胆汁,土得木而达) ④维持气血运行(气行则血行) ⑤调节水液(理气以行水) ⑥调节性与生殖(调理冲任——女子以肝为先天;调节精室开合)。
- **主藏血生血**:贮藏血液、调节血量(人动血运诸经,人静血归于肝,血海)、防止出血;肝血不足→目涩夜盲、筋脉拘急麻木、月经量少。
- 特性:喜条达恶抑郁、**为刚脏**(将军之官,气易亢逆,肝为五脏之贼)、**体阴用阳**(血为体气为用,治肝宜柔不宜刚、宜和不宜伐)。在志怒,在液泪,开窍目,主筋其华在爪,与胆相表里。

### 肾(五行水,阴中之阳,通冬,先天之本、封藏之本)
- **主藏精**:先天之精(生殖之精)+后天之精(五脏六腑之精)。功能:促生殖(天癸——男八女七节律)、促生长发育(生长壮老已)、参与生血(精生髓化血)、抵御外邪(藏精者春不病温)。
- **主水液**:靠肾阳气化,主持调节水液代谢(关门开阖:阖多开少→尿少水肿;开多阖少→尿多尿频)。
- **主纳气**:摄纳肺吸入之气。肾不纳气→呼多吸少、动则喘甚。咳喘"在肺为实,在肾为虚"。
- **主一身阴阳**:肾阴(元阴真水,滋养濡润)、肾阳(元阳真火,推动温煦)为脏腑阴阳之本。肾阴虚→五心烦热眩晕耳鸣腰膝酸软遗精;肾阳虚→神疲腰膝冷痛形寒肢冷阳痿宫寒水肿。阴阳互损→阴阳两虚。
- 附**命门**:诸说不一,主流认识命门之火=肾阳,命门之水=肾阴,强调肾中阴阳的重要性。
- 主骨生髓通脑,其华在发,开窍耳及二阴,在志恐,在液唾,与膀胱相表里。

## 奇恒之府(脑髓女子胞)

- **脑**:元神之府,髓海,主宰生命活动、主精神意识、主感觉运动。脏象学说将脑之功能统归于心而分属五脏,与肾关系最密切(肾精生髓充脑);脑病可从心/肺/脾/肝/肾论治。
- **髓**:脑髓脊髓骨髓,由肾精化生、后天精充养,功能:充养脑髓、滋养骨骼、化生血液(血虚可补肾填精)。
- **女子胞(子宫)**:主持月经、孕育胎儿。与肝(女子以肝为先天,藏血疏泄)、脾(化生固摄经血)、肾(天癸至竭)关系最密切;与冲(血海)、任(主胞胎)、督、带脉相关。冲为血海、任主胞胎,二者相资方能有子。男子对应**精室**(藏精生育,属肾主,与冲任相关)。

## 五体五官(五脏所主)

心主脉(华面)、肺主皮毛(华毛)、肝主筋(华爪)、脾主肉(华唇)、肾主骨(华发)。开窍:心舌、肺鼻、脾口、肝目、肾耳及二阴。
