# 证型 → 治法 → 选药 → 选方 总索引

> **跨书衔接层**:把《中医诊断学》证型(`11`~`14`)、《中药学》药群(`15`~`23`)、《方剂学》成方(`24`~`30`)串成一条可查询的链,并设病种(`31`~`37`)、针灸(`38`~`41`)、经典(`42`~`45`)三个直达索引。本表只做**路由**,不含细节——证型鉴别回 `11`~`14`,药物性能回 `16`~`22`,方剂条目(组成/方义/加减/禁忌)回 `25`~`30` 对应文件及章末速查表,**凡涉具体药物与成方必过 [[安全-配伍妊娠禁忌与毒性药]]**。
> 用法:辨证七步骤得出证名 → 查本表得治法与代表方 → 进对应文件核对辨证要点与类方鉴别 → 走涉药四关。带"※"者为教材仅提及方名的锚点(黄连阿胶汤、半夏秫米汤已补入 `28`;余待后续版本补入)。
