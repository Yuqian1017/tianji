# 脏腑之间的关系

> 提炼自《中医基础理论》第三章第五节。辨证中判断**两脏同病、病位传变**的核心依据。
> 总纲:五脏一体观。五脏生克制化"不是单向的链,也不是首尾相衔的环,而是球状的网";且**五脏互藏**——"凡五脏之气必互相灌溉,故各五脏之中,必各兼五气"(《景岳全书·真脏脉》)。教材强调:五脏关系早已超越五行生克范围,须从各脏生理功能阐释。

## 一、脏与脏(十对)

| 两脏 | 关系轴 | 生理机制 | 病理(常见证) |
|---|---|---|---|
| 心—肺 | **气与血** | 肺朝百脉助心行血;心主血以载气。**宗气**贯心脉而司呼吸,是联结心搏与呼吸的中心环节 | 肺失宣肃→血行失常;心血瘀阻→影响宣降。心肺气虚、气虚血瘀 |
| 心—脾 | **血之生成与运行** | 脾为生化之源以充心血,心血养脾且"心火生脾土";血行赖心气推动+脾气统摄("诸血皆运于脾");思发于脾而成于心 | 心脾两虚(心悸失眠多梦+食少倦怠、化源不足) |
| 心—肝 | **血与神志** | 肝藏血、心行之("肝藏血,心行之"——王冰);血为神志物质基础,心主神明与肝调畅情志相配合 | 心肝血虚、心肝火旺(均从阴血不足或火旺两端) |
| 心—肾 | **水火既济** | ①心火下降于肾使肾水不寒,肾水上济于心使心火不亢("心肾相交,全凭升降");②精血互生;③精神互用(积精全神、神全益精);④君相安位(君火以明,相火以位) | **心肾不交**(肾阴虚于下、心火亢于上:腰膝酸软遗精+心烦失眠);水气凌心;心肾阳虚 |
| 肺—脾 | **气与水** | 气之生成:"肺为主气之枢,脾为生气之源"(水谷精气+清气合为宗气);水液:脾之运湿赖肺宣降,肺之宣降赖脾转输 | **"脾为生痰之源,肺为贮痰之器"**——痰嗽病在肺而本在脾。脾肺两虚、痰湿阻肺 |
| 肺—肝 | **气机升降之枢转** | **肝从左升,肺从右降**("左右者,阴阳之道路也"),肝升才能肺降,肺降才能肝升 | 肝火犯肺(**木火刑金**:胁痛口苦烦躁+咳嗽甚则咯血) |
| 肺—肾 | **金水相生** | ①呼吸:"肺为气之主,肾为气之根",吸入之气经肺肃降下纳于肾;②水液:"其标在肺,其本在肾"(肺通调水道,肾主蒸腾气化);③阴液互生(肺阴输精于肾、肾阴上润于肺,"金水同源") | 肾不纳气(呼多吸少、动则喘甚——咳喘"初病治肺,久病治肾");肺肾阴虚、肺肾气虚 |
| 肝—脾 | **疏泄与运化、藏血与统血** | "**土得木而达**"(肝疏泄协调脾胃升降+分泌排泄胆汁助消化);"**木赖土以培之**"(脾化生精微滋养肝体);血:肝藏、脾生且统 | 木郁乘土→**肝脾不调**(胁胀+腹胀便溏)、**肝胃不和**(胁胀+嗳气呕恶);反之**土壅木郁** |
| 肝—肾 | **肝肾同源(乙癸同源)** | ①阴液互养:**水能涵木**,肾阴为主,肾阴足则肝阳不亢;②精血互生(精血同源于水谷精微);③同具相火(相火源于命门,寄于肝肾胆三焦);④**藏泄互用**:肝疏泄与肾闭藏相反相成,主司女子月经、男子排精 | 肝肾阴虚、肝阳上亢→滋水涵木;"东方之木,无虚不可补,补肾即所以补肝;北方之水,无实不可泻,泻肝即所以泻肾"(《医宗必读》) |
| 脾—肾 | **先后天互资** | "先天温养后天,后天补养先天":脾运化赖肾阳(命门火)温煦蒸化("釜爨方成"),肾精赖水谷精微充养;水液:肾司开阖、**脾土制水** | **脾肾阳虚**(消化失调+水液紊乱:五更泄、下利清谷、水肿)。补法之争:李东垣等主"补肾不若补脾",许叔微、严用和主"补脾不若补肾"——临床辨主次先后,或温肾兼补脾,或运脾兼温肾 |

## 二、脏与腑(表里配合)

配合依据有四:**经脉络属、结构相连、气化相通(脏行气于腑,腑输精于脏)、病理相关**。治法由此而生:脏病治腑、腑病治脏、脏腑同治。

| 表里 | 生理配合 | 病理互传(例) |
|---|---|---|
| 心—小肠 | 心火下移助小肠化物泌别;小肠化生之精微上奉心化赤为血 | **心火下移小肠**→小便赤涩尿血("心家有热,结于小肠,故小便血也");小肠实热上熏于心 |
| 肺—大肠 | 肺气肃降则大肠传导("理大便必须调肺气");大肠通降则肺气宣通;肺行水与大肠主津协调大便润燥 | 肺失肃降→便秘;大肠热结腑气不通→胸闷喘促 |
| 脾—胃 | **纳运相得**(胃纳脾运)、**升降相因**(脾升清胃降浊,居中为枢)、**燥湿相济**(脾喜燥恶湿、胃喜润恶燥,"太阴湿土得阳始运,阳明燥土得阴自安") | 纳运失调、升降反常(泄泻/呕逆)、燥湿不济 |
| 肝—胆 | 肝分泌胆汁、胆贮藏排泄;"肝气虽强,非胆不断,肝胆相济,勇敢乃成"(谋虑与决断) | 胆汁疏泄不利(黄疸口苦)、情志异常(胆怯易惊) |
| 肾—膀胱 | 膀胱贮尿排尿全赖肾之固摄气化("小便之利,膀胱主之,实肾气主之也") | 肾阳虚衰气化无权→小便不利、癃闭、尿频遗尿失禁——故小便异常常从肾治 |

## 三、腑与腑

六腑共成"化水谷、行津液"的传化链(受纳腐熟→泌别清浊→传导排泄;三焦总司气化),虚实更替,宜通不宜滞。

- **"六腑以通为用,腑病以通为补"**——此"补"指通泄使传化复常,仅适用于传化阻滞之**实证**;虚证(如胃阴不足)仍当甘寒滋养,不可通泄。
- 病理互传例:胃实热灼津→大肠燥结;大肠燥结→胃失和降(嗳气呕恶);胆火犯胃→呕吐苦水;脾胃湿热熏蒸肝胆→口苦黄疸。
