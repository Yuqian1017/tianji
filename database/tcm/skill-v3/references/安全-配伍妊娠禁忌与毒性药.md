## 一、配伍禁忌:十八反

**十八反歌**(张子和《儒门事亲》):"本草明言十八反,半蒌贝蔹芨攻乌,藻戟遂芫俱战草,诸参辛芍叛藜芦。"

| 主药 | 反(不可同用) |
|---|---|
| **乌头**(川乌、草乌、附子) | 贝母、瓜蒌、半夏、白及、白蔹 |
| **甘草** | 甘遂、大戟、海藻、芫花 |
| **藜芦** | 人参、丹参、玄参、沙参(苦参)、细辛、芍药 |

## 二、配伍禁忌:十九畏

**十九畏歌**(刘纯《医经小学》):"硫黄原是火中精,朴硝一见便相争;水银莫与砒霜见,狼毒最怕密陀僧;巴豆性烈最为上,偏与牵牛不顺情;丁香莫与郁金见,牙硝难合京三棱;川乌草乌不顺犀,人参最怕五灵脂;官桂善能调冷气,若逢石脂便相欺。"

| 药 | 畏 |
|---|---|
| 硫黄 | 朴硝(芒硝) |
| 水银 | 砒霜 |
| 狼毒 | 密陀僧 |
| 巴豆 | 牵牛 |
| 丁香 | 郁金 |
| 牙硝(芒硝) | 三棱 |
| 川乌、草乌 | 犀角 |
| 人参 | 五灵脂 |
| 官桂(肉桂) | 赤石脂 |

> **临床态度**:反药能否同用历代有争议(古方也有甘遂配甘草、乌头配半夏等反用取效者),但科学结论未定。**Skill 一律按禁忌处理**——凡问及十八反/十九畏组合,明确指出属配伍禁忌、不推荐同用,不做"可以反用"的建议。

## 三、妊娠用药禁忌

| 级别 | 药性特点 | 药物(总论所列,各论续补) | 处理 |
|---|---|---|---|
| **禁用** | 毒性强/药性猛烈 | 巴豆、牵牛、大戟、商陆、麝香、三棱、莪术、水蛭、斑蝥、雄黄、砒霜;各论续补:海马(善堕胎)、蟾酥、砒石、轻粉、木鳖子、樟脑、硫黄、常山、瓜蒂、罂粟壳(致新生儿窒息)、大蒜忌灌肠用 | 绝对不用 |
| **慎用**(收涩章续补) | 质重固涩 | 赤石脂、禹余粮 | 非必用则避免 |
| **慎用** | 通经祛瘀、行气破滞、辛热滑利 | 桃仁、红花、牛膝、大黄、枳实、附子、肉桂、干姜、木通、冬葵子、瞿麦 | 病情需要可斟酌("有故无殒"),非必用则避免 |

**《药典》毒性分级**:大毒、有毒、小毒。

## 六、服药饮食禁忌(忌口)

一般忌生冷、油腻、腥膻、辛辣刺激。分证:热病忌辛辣煎炸;寒病忌生冷清凉;胸痹忌肥肉动物内脏烟酒;肝阳上亢忌胡椒辣椒大蒜白酒;黄疸胁痛忌动物脂肪辛辣烟酒;脾胃虚弱忌油炸粘腻寒冷固硬;疮疡皮肤病忌鱼虾蟹发物及辛辣。

### 妊娠禁忌方剂(常见者)

- **禁用**:桃核承气汤、下瘀血汤、大黄䗪虫丸、抵当类、十枣汤、三承气(峻下)、失笑散、复元活血汤、血府逐瘀汤类(五逐瘀)、小金丹、紫金锭、滚痰丸、枳实导滞丸、木香槟榔丸。
- **慎用**:凉开三宝及苏合香丸(麝香碍胎,救急权衡)、小活络丹、玉真散、活血调经诸方。
- **可用而有法度**:桂枝茯苓丸治妊娠癥块漏下——"有故无殒",但**只可渐消缓散,量小而频,不可峻攻**;胶艾汤、泰山磐石散为胎漏滑胎正治方。
- 黄芪妊娠后期连服致过期妊娠/难产的报道(见当归补血汤类方使用背景),孕晚期大量黄芪慎。

### 服法即安全(方剂学通则)

- 峻药缓用:丸以缓之(小活络丹、桂枝茯苓丸、滚痰丸)、枣汤送服(十枣汤)、糜粥自养(十枣汤、瓜蒂散后)、临卧咽膈徐下(滚痰丸)。
- 反佐防格拒:真寒假热热药冷服(四逆辈),真热假寒寒药热服;白通加猪胆汁人尿、通脉四逆加猪胆汁。
- 中病即止方:发汗剂(麻黄汤"可一而不可再")、泻下剂、涌吐剂、固涩剂(涩敛留邪)、消食剂(攻伐)、开窍剂(辛香耗气)。
