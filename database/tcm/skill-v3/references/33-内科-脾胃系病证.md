## 总说

脾主运化升清,胃主受纳通降;脾喜燥恶湿、病多虚多寒,胃喜润恶燥、病多实多热("实则阳明,虚则太阴")。治疗要点:①遵生理特性——治胃常用甘凉滋润,慎辛香燥热;治脾多醒脾化湿,少甘润滋腻;健脾常配升提,和胃多用降逆;②**六腑以通为用**——胃肠病以通为主法;③久病久痛入络,注意活血通络;④肝木疏土——肝郁乘脾犯胃者疏肝为先。

## 胃痛

**定义**:上腹胃脘近心窝处疼痛(俗称心口痛)。对应急慢性胃炎、消化性溃疡、功能性消化不良等。

**鉴别(要紧)**:
- 与**真心痛**:真心痛在左胸膺、剧痛短暂发作性、伴胸憋汗出心悸气短、危急;胃痛在心下脘部、多隐痛胀痛持续、伴胃肠症状。**中老年上腹痛伴胸闷汗出者先排心梗(心电图)。**
- 与胁痛(以胁肋为主)、腹痛(胃脘以下)。

**辨证要点**:虚实——实者痛剧拒按、食后痛甚、新病;虚者痛缓喜按、**饥而痛甚**、久病。寒热——寒痛遇寒甚得温减;热痛灼痛急迫、得寒减、烦渴便秘。气血——在气者胀痛无定处、与情志相关、嗳气频;在血者刺痛定处、食后入夜甚、可见呕血便血、舌紫暗。

**治则**:理气和胃止痛,立足于"通"。**忌滥用解热镇痛类西药刺激胃黏膜;饮食伤胃者忌补益药**。

**分证论治**:

| 证 | 治法 | 代表方 |
|---|---|---|
| 寒邪客胃 | 温胃散寒行气止痛 | 香苏散合良附丸 |
| 饮食伤胃 | 消食导滞和胃 | 保和丸;胀而便秘合小承气汤/枳实导滞丸 |
| 肝气犯胃 | 疏肝解郁理气止痛 | 柴胡疏肝散;泛酸加乌贼骨煅瓦楞 |
| 肝胃郁热 | 泄肝清热 | 化肝煎或丹栀逍遥散合左金丸 |
| 湿热中阻 | 清化湿热理气和胃 | 清中汤;备选黄连温胆汤 |
| 瘀血停胃 | 化瘀通络理气和胃 | 失笑散合丹参饮 |
| 胃阴亏虚 | 养阴益胃和中止痛 | 一贯煎合芍药甘草汤 |
| 脾胃虚寒 | 温中健脾和胃止痛 | 黄芪健中汤;虚寒甚合理中汤/附子理中汤 |

寒热错杂者半夏泻心汤辛开苦降。

**危重变证(红线)**:①**呕血、黑便**——出血,大量出血气随血脱;②**腹痛剧烈拒按、大汗淋漓、四肢厥冷**——疑穿孔/厥脱;二者立即急诊。久痛不愈、消瘦、痛规律改变——胃镜排查恶变。

**调护(A级)**:规律饮食忌暴饮暴食;发作时流质/半流质、少食多餐;忌浓茶咖啡辛辣、细嚼慢咽;调情志;慎用阿司匹林类与激素类药物(遵医嘱)。

**附·吐酸**:泛酸——肝胃郁热者左金丸加味(泄肝和胃);脾胃虚寒者吴茱萸汤合香砂六君子汤;湿阻者越鞠丸。**附·嘈杂**:似饥非饥、莫可名状——胃热温胆汤加芩连;胃虚四君子汤;血虚归脾汤。

## 痞满

**定义**:心下痞塞、胸膈胀满,**触之无形、按之柔软、压之不痛**(此三点与胃痛、结胸、鼓胀分界)。《伤寒论》"满而不痛者,此为痞"。对应慢性胃炎、功能性消化不良、胃下垂等。

**鉴别**:胃痛有痛压之可痛;鼓胀腹大如鼓在大腹、皮急;胸痹以胸闷痛为主;结胸硬满而痛拒按。

**辨证**:实痞——能食、食后尤甚、便秘苔厚腻脉实;虚痞——饥饱均满、食少便溏、舌淡脉虚。

**分证论治**:实痞——饮食内停(保和丸)、痰湿中阻(二陈平胃汤;化热黄连温胆汤)、湿热阻胃(泻心汤合连朴饮;寒热错杂半夏泻心汤)、肝胃不和(越鞠丸合枳术丸);虚痞——脾胃虚弱(补中益气汤;胃下垂多此型)、胃阴不足(益胃汤)。

**调护(A级)**:节制饮食忌暴食生冷肥甘;情志舒畅;腹部保暖;适当锻炼。

## 呕吐

**定义**:胃失和降、气逆于上,胃中之物从口吐出。有物有声为呕,有物无声为吐,有声无物为干呕。

**先排它病(红线线索,教材原载)**:**喷射状呕吐**→查颅脑;呕吐不止伴尿少浮肿→查肾功能;伴剧烈腹痛便闭→肠梗阻;伴右上腹痛发热→肝胆胰;**育龄女性先验妊娠**;食物中毒→催吐/洗胃并就医。

**鉴别**:反胃——朝食暮吐、暮食朝吐、宿谷不化(脾胃虚寒);噎膈——进食梗阻食不得入(病在食道,病重)。呕吐物辨证:酸腐量多——食积;苦水黄水——胆热;酸水绿水——肝热;痰涎清水——痰饮;少量黏沫——胃阴虚。

**治则**:和胃降逆为总则,但**勿见呕止呕**——胃有痈脓、痰饮、食滞、毒物时呕吐是排邪反应,当因势利导;误吞毒物当探吐。呕吐不止防津伤气脱。

**分证论治**:

| 证 | 治法 | 代表方 |
|---|---|---|
| 外邪犯胃 | 疏邪解表化浊和中 | 藿香正气散;风热银翘散加味;暑湿新加香薷饮 |
| 食滞内停 | 消食化滞和胃 | 保和丸 |
| 痰饮内阻 | 温中化饮和胃 | 小半夏汤合苓桂术甘汤;化热温胆汤 |
| 肝气犯胃 | 疏肝理气降逆 | 四七汤;备选四逆散合半夏厚朴汤 |
| 脾胃气虚 | 健脾益气和胃 | 香砂六君子汤 |
| 脾胃阳虚 | 温中健脾降逆 | 理中汤 |
| 胃阴不足 | 滋养胃阴降逆 | 麦门冬汤 |

生姜为"呕家圣药"。服药法(A级实用):**少量频服、选气味小者、以热饮为宜、可加少量姜汁防格拒**。

**附·反胃**:朝食暮吐、宿谷不化、吐后舒适——脾胃虚寒,丁香透膈散(温中健脾降逆)。**须先胃镜排除幽门梗阻及占位。**

## 噎膈

> **红线定位:进行性吞咽困难=食管癌的典型表现,必须立即胃镜检查。中医治疗为确诊后的辅助/姑息层面。**

**定义**:吞咽梗噎不顺、饮食难下、纳而复出。噎为膈之始,膈乃噎之渐。病机:痰、气、瘀交阻食道,后期津枯阳衰。对应食管癌、贲门癌及贲门失弛缓症、食管炎等良性病变。

**鉴别**:与反胃(食能下咽,朝食暮吐);与**梅核气**(咽中如有物、吐之不出咽之不下,但**饮食下咽顺利**——无形痰气,多功能性)。

**分证论治**:痰气交阻(初期,情志舒畅减轻)——启膈散;瘀血内结(饮食难下、胸膈痛定、吐物如赤豆汁)——通幽汤;津亏热结(水饮可下食物难进、便干如羊矢)——沙参麦冬汤(备选五汁安中饮);气虚阳微(晚期水饮不下、面浮足肿)——补气运脾汤。治标勿过辛散香燥,治本勿过甘酸滋腻——"存得一分津液,留得一分胃气"。

教材所载单方多含**马钱子、雄黄、硇砂、朱砂**等剧毒品——**一律不得自行使用**(见安全表)。

**调护(A级预防价值)**:忌过烫过快饮食、忌霉变食物(霉花生霉玉米)、少食酸菜泡菜、戒烟酒;食管慢性病及时治;进食后少量温水冲洗食道。

## 呃逆

**定义**:胃气上逆动膈,喉间呃呃连声、声短而频、难以自制(打嗝)。**一时性呃逆无兼证者属生理现象,无须治疗**。

**鉴别**:干呕(有声无物之呕)、嗳气(声沉缓伴酸腐,"饱食之息")。

**危重提示**:**重病后期呃逆不止、呃声低微、气不得续、饮食不进**——"病深者,其声哕",元气衰败之危候,就医。顽固性呃逆持续不止也应查因(脑血管病、尿毒症、胸腹腔病变)。

**分证论治**:胃中寒冷——丁香散(丁香柿蒂类;丁香燥热不可过用);胃火上逆——竹叶石膏汤(腑实便秘小承气加竹茹柿蒂);气机郁滞——五磨饮子(备选旋覆代赭汤);脾胃阳虚——理中丸加丁香柿蒂;胃阴不足——益胃汤合橘皮竹茹汤。降逆平呃通用药:旋覆花、代赭石、柿蒂、刀豆子。

**简易止呃法(A级,教材原载)**:①含水屏气吞咽法(口含温水、掩耳鼻、吞下后片刻放开,可重复2~3次);②取嚏法(纸捻轻探鼻取嚏);③转移注意/惊转法;④指压内关穴约10分钟。

## 腹痛

**定义**:胃脘以下、耻骨毛际以上疼痛。

**三科鉴别(红线判断核心,教材原表)**:
- **内科腹痛**:多先发热后腹痛,痛无定处不甚剧,**腹软、压痛不明显**;
- **外科急腹症**:**先腹痛后发热、痛剧有定处、腹肌紧张、反跳痛**——立即外科急诊(阑尾炎、穿孔、胰腺炎、肠梗阻等);
- **妇科腹痛**:痛在小腹,与经、带、胎、产相关(宫外孕破裂大出血为最急红线)。

**危重指征**:腹痛剧烈、拒按、冷汗淋漓、四肢不温、呕吐不止——警惕厥脱,**立即急救**。

**辨证**:寒痛拘急遇冷剧得热减;热痛有热感伴便秘得凉减;气滞痛时轻时重攻窜、嗳气矢气减;血瘀刺痛不移拒按夜甚;伤食胀痛嗳腐、痛甚欲便便后减。部位:胁腹少腹多属肝,脐上大腹属脾胃,脐下小腹属膀胱大小肠。

**分证论治**:寒邪内阻——良附丸合正气天香散;湿热壅滞——大承气汤(胆囊炎样痛用大柴胡汤、阑尾炎样痛大黄牡丹皮汤——**此二者本属就医病**);饮食积滞——枳实导滞丸(轻者保和丸);肝郁气滞——柴胡疏肝散(痛引睾丸加橘核荔枝核);瘀血内停——少腹逐瘀汤;中虚脏寒——小建中汤(备选大建中/附子理中/温脾汤/当归四逆汤)。

**调护(A级)**:虚寒者进热食,热证进温食,食积者暂禁食或少食;饭后勿立即运动。

## 泄泻

**定义**:排便次数增多、粪质稀溏甚至如水样。基本病机**脾虚湿盛**("无湿不成泄")。急性以湿盛为主,慢性以脾虚为主。

**鉴别**:与**痢疾**——痢有赤白脓血、里急后重,泻则粪稀无脓血、泻下爽利;与**霍乱**——吐泻交作、转筋、目眶凹陷、津竭阳衰之象,凶险。

**危重指征**:暴泻不止伴高热呕吐——防津竭致痉厥闭脱(**尤其小儿老人,口服补液并及时就医**);泻而不能食、形瘦、滑脱不禁为重症。

**治则**:运脾化湿。暴泻**不可骤用补涩**(闭门留寇);久泻**不可分利太过**(伤阴);"利小便实大便"仅适于暴泻水湿。健脾(虚致泻)与运脾(湿困脾——苍术厚朴藿香蔻仁)分用;久泻酌加升阳风药(升麻柴胡葛根防风,少少与之)。

**分证论治**:

| 证 | 特征 | 治法 | 代表方 |
|---|---|---|---|
| 寒湿(暴泻) | 便清稀如水,恶寒身痛,苔白腻 | 散寒化湿 | 藿香正气散;湿重胃苓汤 |
| 湿热(暴泻) | 泻下急迫、肛门灼热、粪黄褐臭,苔黄腻 | 清热利湿 | 葛根芩连汤 |
| 食滞(暴泻) | 泻下臭如败卵、嗳腐、泻后痛减 | 消食导滞 | 保和丸;重者枳实导滞丸(通因通用) |
| 脾胃虚弱(久泻) | 时溏时泻、稍进油腻便次即增、面萎神疲 | 健脾益气化湿 | 参苓白术散;中气下陷补中益气汤 |
| 肾阳虚衰(久泻) | **五更泄**、完谷不化、形寒腰酸 | 温肾健脾固涩 | 四神丸;滑脱真人养脏汤;寒热错杂乌梅丸 |
| 肝气乘脾(久泻) | 情绪诱发腹痛即泻、胸胁胀闷 | 抑肝扶脾 | 痛泻要方 |

**调护(A级,教材原载)**:急性泄泻予流质/半流质,忌辛辣油腻荤腥;耗伤胃气者以**淡盐汤、米粥饭汤养胃气**(即家庭口服补液思路);虚寒者淡姜汤;平时可服山药、莲子、白扁豆、芡实类助运化;牛奶面筋不耐受者避食。

## 痢疾(据五/六版系教材补编)

> **定位:法定肠道传染病(细菌性/阿米巴痢疾),确诊治疗以现代医学为主,注意隔离;疑似疫毒痢(儿童多见)为红线急症。**

**定义**:外感时行疫毒、内伤饮食,邪蕴肠腑、气血壅滞,以**腹痛、里急后重、下痢赤白脓血**为主症的传染性外感病,夏秋多发。

**与泄泻鉴别**:痢疾便次多而量少、赤白脓血、里急后重、腹痛持续便后不止;泄泻粪稀无脓血、泻下爽利、痛随泻减。先泻后痢为病进,先痢后泻为病退。

**辨证**:实痢起病急、痛而拒按、便后里急后重暂减;虚痢起病缓、痛而喜按、便后坠胀不减。热痢脓血鲜红赤多、黏稠臭秽、身热口渴;寒痢白多赤少、清稀不臭、面白肢冷。

**治则**:祛邪导滞+调气和血("调气则后重自除,行血则便脓自愈")+顾护胃气。**忌过早补涩**(关门留寇)、忌峻下攻伐、忌分利小便。热痢清之,寒痢温之,初痢通之,久痢虚则补之。

**分证论治**:湿热痢——芍药汤(成药香连丸;热重加白头翁黄柏秦皮);**疫毒痢**(急骤高热、痢下鲜紫脓血、神昏惊厥,儿童可高热惊厥先于泻痢——**立即急救**)——白头翁汤合芍药汤,营血热盛合犀角地黄汤类(住院范畴);寒湿痢——不换金正气散;虚寒痢——桃花汤合真人养脏汤;休息痢(时发时止)——连理汤,寒热错杂乌梅丸;阴虚痢——驻车丸;噤口痢(下痢不能食)——实者开噤散,虚者六君子汤加菖蒲姜汁。

**危候**:痢下如猪肝/屋漏水/纯血、高热神昏、四肢厥逆、噤口不食——预后差,急救。

**调护**:床旁隔离;管水管粪灭蝇;流行季可食蒜瓣、马齿苋绿豆汤预防;食物须熟烂,忌荤腥油腻。

## 便秘(据五/六版系教材补编)

**定义**:大肠传导失常,大便排出困难、排便时间或间隔延长(常三五日以上一行,或粪质不干但排出无力)。

**鉴别**:与积聚——便秘可在左下腹扪及条索状包块,**排便后消失**;积聚包块与排便无关。**警示:中老年新发便秘、进行性加重、便血消瘦者,肠镜排查肿瘤。**

**辨证**:热秘——粪干坚、肛门灼热、口臭面赤苔黄燥;气秘——欲便不出、便而不畅、嗳气胁胀;冷秘——便艰涩、腹痛拘急、手足不温;虚秘——气虚(粪不干硬但努挣乏力汗出)、血虚(便干面白心悸)、阴虚(干如羊屎、潮热盗汗)、阳虚(便或干或不干、小便清长、四肢不温腰膝冷)。

**治则**:实证泻热/温散/理气辅以导滞;虚证滋阴养血、益气温阳,**以润下为基础缓下为宜**。**"不得一见便秘,便用大黄、芒硝、巴豆、牵牛之属"——愈下愈结,老人虚人尤忌**(景岳、东垣再三告诫)。

**分证论治**:

| 证 | 治法 | 代表方 |
|---|---|---|
| 肠胃积热(热秘) | 泻热导滞润肠 | 麻子仁丸 |
| 气机郁滞(气秘) | 顺气导滞 | 六磨汤 |
| 阴寒积滞(冷秘) | 温里散寒通便 | 大黄附子汤 |
| 气虚 | 补气润肠 | 黄芪汤;下陷脱肛补中益气汤 |
| 血虚 | 养血润肠 | 润肠丸;五仁丸 |
| 阴虚 | 滋阴润肠 | 增液汤;肾阴虚六味地黄丸 |
| 阳虚 | 温阳润肠 | 济川煎;肾阳虚金匮肾气丸 |

外导法:蜜煎导(《伤寒论》)可配合。番泻叶3~9g 泡服可用于热秘应急(**短期,不可依赖**)。

**调护(A级,首选)**:多食富含纤维的粗粮蔬果、避辛辣燥火;**增加活动、锻炼腹肌、避免久坐**;养成**定时排便**习惯;调畅情志。老人习惯性便秘食疗可用决明子煎水或蜂蜜(教材经验)。
