# 针灸 · 灸法、按摩与外治法(附非药物手段安全分级)

> 来源:《针灸学》教材(现行本)第四章刺灸法、第五章其他疗法。穴位定位见 `38`/`39`,病证配穴见 `41`。
> 本文件是本 Skill **非药物干预的操作层**:A 级手段教到"能安全上手",B 级手段只讲"是什么、为什么禁自行、就医时会遇到什么"。
> **经典出处补强**:施术时机禁忌(醉/饱/饥/渴/怒/劳/惊、房事前后不施术)源自灵枢·终始"十二刺禁",极虚者(形肉已夺/大失血/大汗/大泄/新产)禁攻泻源自灵枢"五夺"——见 [[43-经典-灵枢与难经要义]] 第九节;**热证阴虚禁灸**的经典原文("微数之脉,慎不可灸""脉浮热甚而反灸之……必咽燥吐血")与火逆诸变见 [[44-经典-伤寒论六经方证条文]] 第三节;"针能杀生人"(灵枢·玉版)为针刺禁自行的最强经典出处。

## 一、非药物手段安全分级(本 Skill 正式分级)

| 级 | 含义 | 手段 |
|---|---|---|
| **A(零门槛,可直接建议)** | 无创、风险极低,按要领即可 | 按穴/自我按摩、热熨、泡脚、熏洗(非眼部)、敷贴(无刺激性药物)、起居食疗情志调摄 |
| **A⁻(须教全要领与禁忌后方可建议)** | 低风险但有烫伤/过敏/禁忌人群问题 | **艾条悬灸(温和灸/雀啄灸)**、耳穴压豆、刮痧(本书未详,待补)、眼部熏洗 |
| **B(禁自行,须专业人员操作;本 Skill 只解释不教学)** | 有创或有实质伤害风险 | **针刺(毫针/三棱针点刺放血/温针)——绝对禁自行**、艾炷直接灸(瘢痕灸/化脓灸)、拔罐(火罐;真空罐亦保守归此)、穴位注射、头针、耳针(刺)、发泡疗法、正骨推拿(专业整复类) |

三条通则:①任何手段**不替代红线排查**——应急流程(SKILL)先行;②孕妇、婴幼儿、年老体弱、出血性疾病、皮肤破损感染部位,每项手段单独核对禁忌;③做了 A/A⁻ 级处置而症状不减或加重,按"升级指征"就医,不加量重复。

## 二、灸法(A⁻ 级核心通道)★

灸法用艾绒制品点燃熏灼体表,温热刺激防治疾病。"针所不为,灸之所宜"(《灵枢·官能》);尤宜**慢性虚弱性病证及风寒湿邪为患**。

### 家庭适用:艾条悬灸

- **温和灸**:点燃艾条一端,对准穴位,距皮肤 **0.5~1 寸**(约 2~3cm)熏烤,以**温热而无灼痛**为度;每穴 **3~5 分钟,皮肤稍起红晕即止**。
- **雀啄灸**:艾条端与皮肤距离不固定,如雀啄食般上下移动或左右回旋,刺激略强。
- 操作要领:①给他人施灸时,另一手食中二指分置穴位两旁感知温度,防烫伤(老人、糖尿病患者、感觉迟钝者尤须);②**先上后下**——先灸上部、背部,后灸下部、腹部;先头身后四肢;③灸毕彻底掐灭艾条(入小口瓶隔氧),防复燃。

### 灸法的作用(选灸依据,配穴回 `39`/`41`)

①温经通络、行气活血、祛湿散寒——风寒湿痹、气血虚之眩晕、乳少、闭经;②温补中气、回阳固脱——久泄久痢、遗尿、崩漏、脱肛、阴挺、寒厥;③消瘀散结——乳痈初起、瘰疬未化脓者;④**保健灸**:常灸**足三里、关元、气海、大椎(教材原文"大信"系OCR误,应为大椎)**可鼓舞正气、防病保健。

### 灸法禁忌(A⁻ 的"⁻"所在)

1. **实证、热证、阴虚发热者一般不宜灸**(先辨寒热虚实,拿不准不灸——与 `13`/`14` 辨证互参)。
2. **孕妇腹部与腰骶部不灸**。
3. 颜面五官、大血管部位不施瘢痕灸(家庭悬灸此类部位也从严:面部谨慎、勿近眼)。
4. `39` 各穴注意栏标"禁灸/不灸"者从其标注(如睛明、攒竹、太阳、头维、人迎、迎香、素髎、水沟、丝竹空、鱼腰;翳风勿直接灸)。

### 灸后处理

微红灼热属正常,自行消退。**小水泡勿擦破,任其自然吸收**;大水泡属烫伤,勿自行挑破,消毒处理不当会感染——保守做法:保护创面,就医处理。(教材载消毒针刺破放液涂龙胆紫,系医护操作。)

### 艾炷灸与温针灸(存目,B 级)

艾炷灸分直接灸(**瘢痕灸/化脓灸**:灸后化脓留瘢,剧痛,**绝对禁自行**;无瘢痕灸也须专业把握)与间接灸(**隔姜灸**——解表散寒温中止呕;**隔盐灸神阙**——回阳固脱,治虚脱急症属医师抢救手段;隔蒜灸——解毒;隔附子饼灸——温肾壮阳)。间接灸虽传统上家庭亦用,本 Skill 保守定位:**在专业人员示教过要领后**方可作 A⁻ 对待,否则以艾条悬灸代替。温针灸(针柄置艾)随针刺归 B 级。

## 三、按摩疗法(A 级素材)★

散寒止痛、健脾和胃、消积导滞、疏通经络、滑利关节。适应证:发热畏寒头痛、咳喘、腹胀泄泻、脘痛纳呆、痿痹、跌打损伤筋骨不利等(**急症先过红线排查**)。

### 十种基本手法

| 手法 | 要点 | 适用 |
|---|---|---|
| 推法 | 拇指指腹/掌根着力,有节律单向或回旋推动 | 全身 |
| 拿法 | 拇指与余指对称用力拿捏肌肉筋腱,**强刺激**,一处 1~3 次即止 | 四肢、肩颈 |
| 按法 | 拇指/掌根按压并**稍留片刻** | 全身 |
| 摩法 | 指腹/掌根/鱼际贴附作盘旋摩动(快 120 次/分,慢 50 次/分) | 全身(腹部保健常用) |
| 揉法 | 鱼际/掌根/拇指着力,带动皮下组织左右回旋 | 全身 |
| 摇法 | 两手托握关节上下,缓缓左右旋转摇动 | 四肢、颈、腰关节(幅度小、缓慢,**颈部摇法慎**) |
| 滚法 | 手背指掌关节接触患部,前臂旋转带动滚动 | 颈腰背臀四肢 |
| 搓法 | 双掌夹住患处快速搓揉并上下移动,轻→重→轻 | 四肢、腰背、胸腹 |
| 掐法 | 拇食指对称掐取穴位用力内收,**开窍提神**(如掐水沟、十宣) | 四肢、头面,急救 |
| 捏法 | 拇食指(或五指)提捏皮肤肌肉沿经络推行(小儿"捏脊"即此) | 全身 |

### 要领与禁忌

- 手法求**柔和、有力、持久、均匀**,力达组织深部;每次 15~30 分钟。可蘸水、滑石粉、石蜡油、姜汁减阻。
- 腰腹部施术前先排尿;注意保暖遮盖。
- **禁忌:严重心脏病、结核病、出血性疾病、癌症(肿块局部)、急性炎症、急性传染病、皮肤破损处禁按摩;孕妇腰腹部禁按摩**。
- 家庭加严三条(承 `39` 头注):颈前动脉区不按揉;眼区不按压;骨突急性红肿热痛(疑感染/骨折)不按。

## 四、热熨法(A 级)★

吸热物体(可拌药)加热后熨敷局部或穴位,行气活血、散寒定痛、祛瘀消肿。适应证:痹证、腰腿痛、脘腹痛、泄泻(**虚寒性**者)。

- 家庭做法:盐、麸皮(或市售坎离砂)炒热至 60~70℃装布袋,熨敷患处,每次 15~30 分钟;温度过高垫布,先在自己前臂内侧试温。
- **禁忌:热证、实证、局部无知觉者禁用**(感觉障碍者极易烫伤);随时观察皮肤,潮红起泡即停。

## 五、熏洗疗法(A 级;眼部 A⁻)

药液趁热先熏后洗。适应证:筋骨疼痛、皮肤病、阴痒带下、肛门疾病、目赤肿痛。

- 四肢熏洗:患肢架于药液盆上覆巾熏蒸,待不烫时浸洗。
- 坐浴:药液盆加有孔木盖,先熏后坐浴。**孕妇及月经期禁坐浴**。
- 眼部熏洗(A⁻):碗盛药液覆纱布留小孔,患眼对孔熏蒸,不烫后蘸洗——**药液务必过滤干净、温度严控**,眼部拿不准就医。
- 通则:避风保温;严控温度防烫伤;所用药物过 `安全表`。

## 六、敷药法与贴药法(A 级,药物过安全表)

- **敷药法**:鲜药捣烂或药膏摊于棉垫敷贴患处,厚 0.2~0.5cm,围敷超出肿块 2cm;水调剂易干须保湿。适应证:痈疽疮疖、跌打损伤等。**皮肤过敏(红痒疹泡)立即洗去**。
- **贴药法**:膏药文火烘软化后趁热贴敷(以能揭开为度,防烫伤);鲜叶(苦瓜叶等)洗净即贴。皮肤过敏即停。
- 注意:敷贴药物含毒性药(如生南星、生半夏外用等)须核对 `安全表`;破损皮肤不敷刺激性药。

## 七、耳穴(压豆=A⁻;耳针刺=B 禁自行)

耳穴分布如**倒置胎儿**:头面部对应穴在**耳垂**,上肢在**耳舟**,躯干下肢在**对耳轮及其上下脚**,内脏集中在**耳甲艇、耳甲腔**。

### 常用耳穴速查(压豆定位用)

| 区 | 要穴 | 主治 |
|---|---|---|
| 三角窝 | **神门** | 失眠多梦、烦躁(最常用) |
| 三角窝 | 子宫(精宫) | 痛经、带下 |
| 耳轮 | 耳尖 | 目赤肿痛、发热、高血压 |
| 耳屏 | 肾上腺 | 低血压、昏厥、咳喘 |
| 对耳屏 | 平喘 | 哮喘、咳嗽 |
| 对耳屏 | 皮质下、枕 | 失眠、痛症、神经系统 |
| 屏间切迹 | 内分泌 | 月经不调、妇科 |
| 耳轮脚周围 | 胃、大肠、小肠 | 胃痛呃逆、腹泻便秘 |
| 耳甲艇 | 肾、膀胱 | 腰痛耳鸣、遗尿癃闭 |
| 耳甲腔 | 心、肺 | 心悸、咳喘、皮肤病 |
| 耳垂 | 眼、内耳、面颊 | 目疾、耳鸣、面瘫 |
| 耳背 | 降压沟 | 高血压辅助 |

选穴思路:①按病位(胃痛选胃);②按脏腑理论(皮肤病选肺——肺主皮毛;目赤选肝——肝开窍于目);③每次 2~3 穴,先找**压痛敏感点**(火柴头轻探,最痛处即是)。

### 压豆法(A⁻ 要领)

王不留行籽(或磁珠)置小方胶布中央,贴压敏感点,每日自行按压 3~5 次、每次每穴约 1 分钟(酸胀微痛为度),3~5 天更换一侧耳交替。**禁忌:耳廓冻伤、炎症部位禁用;有习惯性流产史的孕妇禁用**;胶布过敏改用低敏胶布或停用;贴处发红胀痛疑感染即除去就医。

### 耳针刺(B 级存目)

毫针刺耳穴须严格消毒(耳软骨感染后果重),属专业操作。就诊时遇之属常规疗法,不必疑虑。

## 八、针刺与拔罐等(B 级——只说"是什么、为何禁自行")

### 毫针刺法存目

进针(指切/夹持/舒张/提捏)、角度(直刺 90°/斜刺 45°/平刺 15°)、行针得气(酸麻胀重感,"气至而有效")、补泻、留针出针——皆专业技能,须练针与解剖知识。**禁自行的核心理由=教材自载的事故谱**:

- **晕针**:突然头晕面白、恶心多汗、四肢冷,重者昏仆。(此机制同样适用于**灸与按摩过强**:出现先兆立即停止,平卧头低位,保暖,饮温糖水;掐水沟、按内关;不缓解就医。)
- **滞针、弯针、断针**:断端留体内须 X 线定位手术取出。
- **血肿**:误伤血管。
- **创伤性气胸(最重)**:胸骨上窝、锁骨中线第 6 肋间以上前胸、腋中线第 8 肋间以上侧胸、第 11 胸椎以上背部诸穴刺法不当可刺破肺。表现:胸痛胸闷心慌、呼吸不畅,重者呼吸困难、紫绀、血压下降。**任何人针刺后出现进行性胸闷胸痛呼吸困难=红线,立即急诊**。
- **刺伤延髓/脊髓**:项部(哑门、风府)与背部正中线第 1 腰椎以上穴位深刺可致严重后果。
- 教材针刺禁忌(就诊前可自查告知医师):过饥过劳、精神高度紧张不刺;**孕 3 个月内下腹禁针,3 个月以上上下腹、腰骶及合谷、三阴交、昆仑、至阴禁针**;正常月经期一般不针;小儿囟门未闭头顶不针且不留针;自发出血倾向者不针;皮肤感染溃疡瘢痕肿瘤部位不针。

### 拔罐法存目(B 级)

投火法/闪火法造负压吸附,留罐约 10 分钟,紫红瘀血为度。适应证:风寒湿痹肩背腰腿痛、胃肠病、咳喘、急性扭伤瘀血(刺血拔罐)。**禁自行理由**:明火烫伤风险、留罐过久起泡溃破。禁忌:皮肤溃疡、水肿、大血管部位、高热抽搐、孕妇腹部腰骶、出血性疾病。真空抽气罐虽无明火,负压强度与时长仍难自控,保守同归 B。

### 穴位注射、头针存目(B 级)

穴位注射=穴内注射药液(维生素、当归川芎注射液等),剂量部位皆专业决定;误入关节腔/脊髓腔/血管有实害;孕妇下腹腰骶及合谷三阴交不作穴注。头针=按大脑皮层功能投射区(运动区/感觉区/晕听区等)刺头皮,主治脑源性疾病(中风瘫痪、失语等),脑出血急性期血压未稳不作——皆医院疗法,知其存在即可。

### 发泡疗法(B 级,不推荐自行)

以毛茛、大蒜等刺激性药物敷穴致皮肤起泡("天灸"类)。有创、易感染、疤痕风险,本 Skill 不作建议;遇正规医疗机构开展的"三伏贴"类属其变体,由机构把控。

### 吹药法(存目)

药末吹布口腔咽喉耳鼻患处(冰硼散类)。所涉药末须核对 `安全表`(部分含朱砂、硼砂);吹药后半小时勿饮食吞咽。建议在医师药师指导下使用成品制剂。
