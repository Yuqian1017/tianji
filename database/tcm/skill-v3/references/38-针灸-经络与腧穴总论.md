# 针灸 · 经络与腧穴总论(定位方法是家庭按穴的根基)

> 来源:《针灸学》教材(现行本)绪言、第一章经络总论、第二章腧穴总论。
> 经络系统的基础理论(经络生理功能、与体质的关系)已详于 `08-经络与体质`,本文件只收针灸临床直接用到的部分;十二经逐经循行与病候详表在 `39`,`08` 所缺的循行专表自本文件与 `39` 起补齐。
> **本 Skill 非药物手段分级:按穴按摩=A 级,艾灸悬灸=A⁻级(须教要领禁忌),针刺=B 级绝对禁自行**——分级细则见 `40-针灸-灸法按摩与外治法`。本文件的"刺灸法"信息仅为知识背景与就医参考,不构成自行针刺的依据。

## 一、经络系统速览(针灸视角)

- **十四经** = 十二经脉 + 任脉、督脉(奇经八脉中唯此二脉有专属腧穴)。十四经是循经取穴的基础。
- 十二经脉分布规律:阴经行四肢**内侧**(太阴在前、厥阴在中、少阴在后),阳经行四肢**外侧**(阳明在前、少阳在中、太阳在后);表里相配(肺—大肠、心包—三焦、心—小肠、脾—胃、肝—胆、肾—膀胱)。
- 走向:手三阴从胸走手,手三阳从手走头,足三阳从头走足,足三阴从足走腹(《灵枢·逆顺肥瘦》)。
- 流注次序:肺→大肠→胃→脾→心→小肠→膀胱→肾→心包→三焦→胆→肝→复回肺,如环无端。
- 奇经八脉:任脉"阴脉之海"、督脉"阳脉之海"、冲脉"十二经之海/血海"、带脉约束诸经、阴阳维分调六阴六阳、阴阳跷主肢体运动与眼睑开合。
- 络脉:十五络(十二经+任督各一+脾之大络大包),十二经络穴皆在肘膝以下,别走相表里之经——这是"原络配穴"的结构基础。络穴一览见 `39` 第十六节。

### 经络的临床应用(对家庭参谋直接有用的两条)

1. **分经诊断——头痛按部位归经**(与 `34-肝胆病证·头痛` 引经药条互参):前额痛→阳明;两侧痛→少阳;后枕痛→太阳;巅顶痛→厥阴。针灸按经取穴,用药按引经药。
2. **反应点辅助判断**:脏腑病可在相应经脉部位出现压痛、结节、条索(如阑尾炎多在阑尾穴压痛)。只作就医线索,不作确诊依据。

## 二、腧穴分类与作用

| 类别 | 定义 | 数目 |
|---|---|---|
| 十四经穴 | 有定名定位、归属十四经 | 361(清《针灸逢源》定数沿用至今;本教材计 362,系教材自增 1 穴) |
| 经外奇穴 | 有定名定位、未归经,主治较单纯而特效 | 国标《经穴部位》定 48 穴 |
| 阿是穴 | 无定名定位,以压痛点/反应点为穴("以痛为输") | 不定 |

腧穴作用三层:**近治**(一切腧穴皆治所在局部——阿是穴按压的依据)、**远治**(肘膝以下经穴治本经远隔部位病,如合谷治头面、足三里调胃肠)、**特殊作用**(①双向良性调整:天枢既治泄泻又治便秘,内关对心率快慢双向调节;②相对特异:大椎退热、至阴矫正胎位)。

主治总规律:本经穴治本经病,表里经穴互治表里两经病,邻近经穴配合治局部病。

## 三、特定穴十类(选穴的骨架)

| 类 | 含义 | 要点 |
|---|---|---|
| 五输穴 | 井荥输经合,肘膝以下,经气由浅入深 | 井在指趾末端,合在肘膝附近;配五行称"五行输" |
| 原穴 | 脏腑原气留止处,腕踝附近 | 阴经以输为原(同穴);阳经另置一原 |
| 络穴 | 十五络分出处 | 原络配穴治表里两经病 |
| 郄穴 | 经气深聚处,共 16(十二经+阴阳跷维) | 除梁丘外皆在肘膝以下;多用于急症痛症 |
| 背俞穴 | 脏腑之气输注背部,足太阳经第一侧线 | 依脏腑高低排列,共 12 |
| 募穴 | 脏腑之气汇聚胸腹部,共 12 | 与所属脏腑位置相近;俞募配穴 |
| 下合穴 | 六腑之气下合足三阳,共 6 | 胃胆膀胱在本经;大肠小肠在胃经;三焦在膀胱经 |
| 八会穴 | 脏腑气血筋脉骨髓之会 | 见 `39` 第十七节专表 |
| 八脉交会穴 | 十二经通奇经八脉的 8 穴,腕踝上下 | 见 `39` 第十七节专表 |
| 交会穴 | 两经以上相交会之穴 | 多在头面躯干(如三阴交=足三阴之会) |

(标本、根结、气街、四海为经气分布的理论补充,支撑"上病下取、下病上取",家庭应用无需展开,存目。)

## 四、腧穴定位四法 ★家庭取穴核心技能

取穴准确与效果直接相关。四法常结合使用:先用骨度分寸定大致位置,再以解剖标志校正,按压有酸胀感处即是。

### 1. 骨度分寸法(把身体自己折成"寸",不因高矮胖瘦而误差)

| 部位 | 起止 | 折作 | 备注 |
|---|---|---|---|
| 头部 | 前发际→后发际 | 12寸 | 发际不清:眉心→大椎作 18 寸(眉心→前发际 3 寸,大椎→后发际 3 寸) |
| 胸腹 | 两乳头之间 | 8寸 | 横量;胸胁直寸按肋骨,每肋作 1.6 寸 |
| 胸腹 | 胸剑联合(胸骨下端)→脐中 | 8寸 | 直量 |
| 胸腹 | 脐中→耻骨联合上缘 | 5寸 | 直量 |
| 背腰 | 大椎以下→尾骶 | 21椎 | 按棘突定穴:肩胛下角平第 7 胸椎,髂嵴平第 4 腰椎棘突;背部横寸以两肩胛骨内缘作 6 寸 |
| 上肢 | 腋前纹头→肘横纹 | 9寸 | 手三阴三阳通用 |
| 上肢 | 肘横纹→腕横纹 | 12寸 | 同上 |
| 下肢 | 耻骨上缘→股骨内上髁上缘 | 18寸 | 足三阴用 |
| 下肢 | 胫骨内侧髁下缘→内踝尖 | 13寸 | 足三阴用 |
| 下肢 | 股骨大转子→膝中 | 19寸 | 足三阳用;"膝中"前平犊鼻、后平委中;臀横纹→膝中作 14 寸 |
| 下肢 | 膝中→外踝尖 | 16寸 | 足三阳用 |

### 2. 解剖标志法

- **固定标志**:五官、乳头、脐、骨节突起凹陷——如两眉间取印堂,两乳间取膻中。
- **动作标志**:须做动作才出现——如张口耳屏前凹陷取听宫,握拳掌横纹头取后溪。

### 3. 手指同身寸(以**被取穴者本人**手指为尺)

- 中指同身寸:中指中节屈曲,两端横纹头间距=1 寸(四肢直寸、背部横寸)。
- 拇指同身寸:拇指指间关节横度=1 寸。
- **横指同身寸("一夫法")**:食中环小四指并拢,平中指中节横纹处四指总宽=3 寸——家庭最常用(如足三里"犊鼻下 3 寸"即四横指)。

### 4. 简便取穴法

两耳尖直上连线中点取百会;两手虎口交叉、食指端到达处取列缺;立正垂手中指端取风市。
