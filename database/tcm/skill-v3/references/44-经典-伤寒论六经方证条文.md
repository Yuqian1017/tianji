# 经典 · 《伤寒论》六经方证条文(方证条文层)

> 来源:《伤寒论》宋本全本(高保衡、孙奇、林亿校定十卷二十二篇,397 法、112 方)。
> **定位:方证条文层**——记"六经提纲+主要方证的经典条文+方名",锚定 [[14-六经卫气营血三焦经络辨证]] 六经辨证与方剂通道。**古方组成、剂量、煎服法一律不作用药依据**:本库已收之方(桂枝汤、麻黄汤、小柴胡汤、白虎汤、三承气、四逆汤、理中丸、真武汤、五苓散、炙甘草汤等)以 [[25-解表泻下和解剂]] [[26-清热祛暑温里剂]] [[27-补益固涩剂]] [[28-安神开窍理气理血剂]] [[29-治风治燥祛湿剂]] [[30-祛痰消食驱虫涌吐剂]] 现代条目为准;条文引用时只标方名。**方药建议永远 B 级(须医师把关)**。
> 服药调护类原文(啜粥、温覆、禁口、中病即止)属护理智慧,可作 A 级生活层引用。

## 二、六经提纲与病程通则

| 经 | 提纲条文(原文) |
|---|---|
| 太阳 | "太阳之为病,**脉浮,头项强痛而恶寒**。"中风:"发热,汗出,恶风,脉缓";伤寒:"必恶寒,体痛,呕逆,脉阴阳俱紧" |
| 阳明 | "阳明之为病,**胃家实**是也。"外证:"身热,汗自出,**不恶寒,反恶热**" |
| 少阳 | "少阳之为病,**口苦、咽干、目眩**也。" |
| 太阴 | "太阴之为病,**腹满而吐,食不下,自利益甚,时腹自痛**。若下之,必胸下结鞕。" |
| 少阴 | "少阴之为病,**脉微细,但欲寐**也。" |
| 厥阴 | "厥阴之为病,**消渴,气上撞心,心中疼热,饥而不欲食,食则吐蚘**,下之利不止。" |

- **六经病欲解时**(原文):太阳"从巳至未上"(约 9~15 时)、阳明"从申至戌上"(15~21 时)、少阳"从寅至辰上"(3~9 时)、太阴"从亥至丑上"(21~3 时)、少阴"从子至寅上"(23~5 时)、厥阴"从丑至卯上"(1~7 时)——各经得其主气时段病势易缓之说;今作病情昼夜观察的参考坐标(与 [[43-经典-灵枢与难经要义]]"旦慧昼安夕加夜甚"节律互参),不作预测依据。
- **温病界定**(太阳上):"太阳病,**发热而渴,不恶寒者,为温病**"——伤寒温病分野的第一出处(温病三书接口,配难经五十八)。
- 传变判断:"伤寒一日,太阳受之,**脉若静者为不传;颇欲吐,若躁烦,脉数急者,为传也**";"伤寒二三日,阳明少阳证不见者,为不传也"(以证候变化判进退,家庭观察可用)。
- 寒热真假(太阳上):"病人**身大热,反欲得近衣**者,热在皮肤,寒在骨髓也;**身大寒,反不欲近衣**者,寒在皮肤,热在骨髓也"——盖被恶寒的取向是辨真假寒热的经典指标([[11-八纲辨证]] 互参)。

## 三、太阳病方证(表证与误治救逆)

- **桂枝汤**("群方之魁"):"太阳中风,阳浮而阴弱……啬啬恶寒,淅淅恶风,翕翕发热,鼻鸣干呕者,桂枝汤主之。"**服法原文即服药调护总范**:"服已须臾,**啜热稀粥一升余,以助药力。温覆令一时许,遍身漐漐微似有汗者益佳,不可令如水流漓**,病必不除。若一服汗出病差,**停后服,不必尽剂**……**禁生冷、粘滑、肉面、五辛、酒酪、臭恶等物**"——发汗适度、中病即止、服药禁口三原则出处([[25-解表泻下和解剂]] 服法节之祖)。**禁例**:"桂枝本为解肌,若其人**脉浮紧,发热汗不出者,不可与之**";"酒客病,不可与桂枝汤";"凡服桂枝汤吐者,其后必吐脓血"(内热者禁辛甘温)。
- **麻黄汤**:"头痛发热,身疼腰痛,骨节疼痛,恶风,**无汗而喘**者。"**葛根汤**:"项背强几几,无汗恶风";合病下利亦主之。**葛根黄芩黄连汤**:误下"利遂不止……喘而汗出者"(表里两解,热利之祖)。
- **大青龙汤**:"脉浮紧……不汗出而**烦躁**者"——**禁例原文**:"若脉微弱,汗出恶风者,**不可服之,服之则厥逆,筋惕肉瞤,此为逆也**"(峻汗剂之戒)。**小青龙汤**:"伤寒表不解,**心下有水气**,干呕发热而咳"(外寒内饮,[[31-内科-肺系病证]] 咳喘互参)。
- **五苓散**:"脉浮,小便不利,微热**消渴**者";"渴欲饮水,**水入则吐**者,名曰**水逆**"。饮水通则:"大汗出,胃中干,烦躁不得眠,欲得饮水者,**少少与饮之**,令胃气和则愈"(★病中饮水宜少量频饮,不可暴饮——家庭护理 A 级)。
- **苓桂朮甘汤**:"心下逆满,气上冲胸,**起则头眩**,脉沉紧"(痰饮眩晕之祖)。**真武汤**(太阳篇):"汗出不解,其人仍发热,心下悸,**头眩,身瞤动,振振欲擗地**"(阳虚水泛)。
- **栀子豉汤类**:"虚烦不得眠,若剧者,必反复颠倒,**心中懊憹**";禁例:"病人**旧微溏者,不可与服之**"。
- **五泻心汤痞证鉴别**(★但满不痛为痞):误下"但满而不痛者,此为痞……宜**半夏泻心汤**"(呕+肠鸣+痞);"干噫食臭,胁下有水气,腹中雷鸣下利"用**生姜泻心汤**;"下利日数十行,谷不化……其痞益甚"(胃虚客逆)用**甘草泻心汤**;"心下痞,按之濡,其脉关上浮"用**大黄黄连泻心汤**;"心下痞而复**恶寒汗出**"用**附子泻心汤**([[33-内科-脾胃系病证]] 痞满之祖方群)。
- **结胸与峻下剂**(存目为主):"按之痛,寸脉浮,关脉沉,名曰结胸";大陷胸汤(大黄芒消甘遂)、十枣汤(芫花甘遂大戟)、瓜蒂散皆**含毒剧药之峻剂,今属 B 级中的最高警戒层**(见 [[安全-配伍妊娠禁忌与毒性药]]);"结胸证,其脉浮大者,不可下,下之则死";"结胸证悉具,烦躁者亦死"。十枣汤服法"强人服一钱匕,羸人服半钱,平旦服……得快下利后,**糜粥自养**"。
- **蓄血证**:"太阳病不解,热结膀胱,**其人如狂**,血自下,下者愈……但少腹急结者,乃可攻之,宜**桃核承气汤**";"少腹鞕满,小便自利,其人如狂者,血证谛也,**抵当汤**主之"(水蛭虻虫,毒剧存目);"阳明证,其人**喜忘**者,必有蓄血……屎虽鞕,大便反易,**其色必黑**"([[28-安神开窍理气理血剂]] 活血法源头之一)。
- **炙甘草汤**(复脉汤):"伤寒**脉结代,心动悸**,炙甘草汤主之"([[32-内科-心系病证]] 心悸主方之祖;结代脉=脉搏不齐,今可由手表心律提醒佐证,见 SKILL 问诊步3)。
- **热入血室**:"妇人中风……经水适断者,此为热入血室,其血必结,故使如疟状,发作有时,**小柴胡汤**主之";"昼日明了,**暮则讝语,如见鬼状**者……无犯胃气及上二焦,必自愈"(经期外感高热谵语的经典解释)。
- 柴胡加龙骨牡蛎汤(胸满烦惊):**原方含铅丹,有毒——今用皆去铅丹或以生铁落等代,绝不可按原方**(安全表汞砷铅警示互参)。
- 火逆诸条(温针、烧针、熏、灸误用):"脉浮热甚,而反灸之……必咽燥吐血";"**微数之脉,慎不可灸**,因火为邪……焦骨伤筋,血难复也"——**热证、阴虚数脉禁灸的经典出处**(已立于 [[40-针灸-灸法按摩与外治法]] 灸法禁忌,此为原文层)。

## 四、阳明病方证(里热里实)

- **白虎汤**:"伤寒脉浮滑""三阳合病,腹满身重……口不仁面垢,讝语遗尿……**自汗出**者";**白虎加人参汤**:"大烦渴不解,脉洪大""大渴,舌上干燥而烦,欲饮水数升";"**背微恶寒**""时时恶风"为加参之征。禁例:"伤寒脉浮,发热无汗,其表不解,**不可与白虎汤**"(表未解禁清)。
- **三承气汤使用法度**(★攻下三戒,[[25-解表泻下和解剂]] 承气类互参):
  ①**试探法**:"若不大便六七日,恐有燥屎。欲知之法,**少与小承气汤,汤入腹中,转失气者,此有燥屎也,乃可攻之;若不转失气者,此但初头鞕,后必溏,不可攻之**";
  ②**中病即止**:"得下,余勿服""若一服讝语止者,更莫复服""一服利,则止后服";
  ③**辨鞕溏**:"须小便利,屎定鞕,乃可攻之"。
  调胃承气(燥热初结,"不吐不下,心烦")、小承气(痞满,"微和胃气,勿令至大泄下")、大承气(痞满燥实,"痞、满、燥、实"俱备/潮热/手足濈然汗出/绕脐痛/喘冒不能卧)。
- **急下三证**(阳明):"目中不了了,睛不和""发热汗多者""发汗不解,腹满痛者"——急下存阴;少阴亦有急下三证(见第六节)。
- **导法**(★家庭可知的外导思想,[[33-内科-脾胃系病证]] 便秘互参):"津液内竭,虽鞕,不可攻之,当须自欲大便,宜**蜜煎导**而通之。若土瓜根及大猪胆汁,皆可为导"——蜜煎肛门栓与胆汁灌肠为最早的通便外治记载;今日等价物=开塞露、甘油栓,原则相同:**津亏便结者外导优于强攻**。
- **麻子仁丸**(脾约):"趺阳脉浮而涩……大便则鞕,其脾为约"(习惯性便秘润下之祖)。
- **茵陈蒿汤**:"但头汗出,身无汗,剂颈而还,小便不利,渴引水浆者,此为瘀热在里,身必发黄""身黄如橘子色"(阳黄主方,[[34-内科-肝胆病证]] 黄疸);栀子蘖皮汤、麻黄连轺赤小豆汤分主黄疸轻证与兼表。
- **吴茱萸汤**:"**食谷欲呕**,属阳明也"(胃寒呕吐;又主少阴吐利烦躁、厥阴头痛干呕吐涎沫——一方三经,[[33-内科-脾胃系病证]] 呕吐/[[34-内科-肝胆病证]] 头痛互参)。
- 谵语郑声之辨:"**实则讝语,虚则郑声**。郑声者,重语也";死候:"直视讝语,喘满者死,下利者亦死"。

## 五、少阳病方证

- **小柴胡汤主证**:"往来寒热,**胸胁苦满,默默不欲饮食,心烦喜呕**"+或然七证;**"但见一证便是,不必悉具"**(★柴胡证识别总诀);"上焦得通,津液得下,胃气因和,**身濈然汗出而解**"。
- 少阳三禁:"少阳不可发汗""不可吐下,吐下则悸而惊"(治以和解,[[25-解表泻下和解剂]] 和解剂)。
- **小建中汤**:"伤寒二三日,**心中悸而烦**者""腹中急痛,先与小建中汤"(甘温建中;"**呕家不可用建中汤,以甜故也**")。**大柴胡汤**:"呕不止,心下急,郁郁微烦"(少阳阳明合治)。**柴胡桂枝干姜汤**:"胸胁满微结,小便不利,渴而不呕,但头汗出,往来寒热"。

## 六、三阴病方证(虚寒与危重)

### 太阴
- "自利不渴者,属太阴,以其脏有寒故也,当温之,**宜服四逆辈**";腹满时痛用桂枝加芍药汤,大实痛加大黄;**"太阴为病,脉弱,其人续自便利,设当行大黄、芍药者,宜减之,以其人胃气弱,易动故也"**(★体弱者攻药减量的经典依据)。

### 少阴(心肾虚衰,本篇死证最密)
- **麻黄细辛附子汤**:"少阴病,始得之,**反发热,脉沉**者"(太少两感)。
- **黄连阿胶汤**(※锚点方,→ [[28-安神开窍理气理血剂]] 已补条目):"少阴病,得之二三日以上,**心中烦,不得卧**"——阴虚火旺不寐之祖方([[36-内科-气血津液病证]] 不寐)。
- **附子汤**:"口中和,**其背恶寒**者,当灸之,附子汤主之";"身体痛,手足寒,骨节痛,脉沉者"(阳虚身痛)。
- **桃花汤**:"下利便脓血"(虚寒滑脱之痢,与白头翁汤热痢相对,[[33-内科-脾胃系病证]] 痢疾寒热两纲)。
- **咽痛诸方**:猪肤汤(下利咽痛胸满心烦)、甘草汤→桔梗汤(咽痛轻重递进)、苦酒汤(咽中伤生疮声不出)、半夏散及汤——**原文自注:"半夏有毒,不当散服"**(★仲景原书即标注药物毒性,经典层的用药安全意识;半夏炮制要求见 [[安全-配伍妊娠禁忌与毒性药]])。
- **白通汤/白通加猪胆汁汤**:"利不止,厥逆无脉,干呕烦者";"服汤,**脉暴出者死,微续者生**"(回阳后脉象判生死)。**通脉四逆汤**:"下利清谷,里寒外热,手足厥逆,脉微欲绝,身反不恶寒,**其人面色赤**"(阴盛格阳戴阳证)。**真武汤**(少阴本篇):"腹痛,小便不利,四肢沉重疼痛,自下利者,此为有水气"。
- **四逆散**:"少阴病,四逆,其人或咳,或悸……或泄利下重"(阳郁致厥,非寒厥——手足冷不必皆是阳虚,[[28-安神开窍理气理血剂]] 疏肝之祖)。
- 少阴**急下三证**:"口燥咽干者""自利清水,色纯青,心下必痛,口干燥者""六七日,腹胀不大便者"——宜大承气;**急温**:"少阴病,脉沉者,急温之,宜四逆汤"。
- **少阴死证六条**(★红线佐证,配 [[32-内科-心系病证]] [[31-内科-肺系病证]] 危重指征):"恶寒身蜷而利,**手足逆冷者,不治**";"吐利,躁烦,四逆者死";"下利止而**头眩,时时自冒**者死";"四逆恶寒而身蜷,脉不至,**不烦而躁**者死";"**六七日,息高者死**"(呼吸表浅上提=呼吸衰竭画像);"脉微细沉……复烦躁,不得卧寐者死"。可治之候:"手足温者可治""时自烦,欲去衣被者可治"(**四肢回温、由静转烦=阳气来复**——家庭观察危重转归的经典指标)。

### 厥阴(寒热错杂、厥热胜复)
- 厥之总义:"凡厥者,**阴阳气不相顺接,便为厥。厥者,手足逆冷是也**";"**厥深者热亦深,厥微者热亦微**"——热厥用白虎汤("脉滑而厥者,里有热也"),**手足冷≠必是寒证**(★误温热厥之戒);"诸四逆厥者,不可下之,虚家亦然"。
- **乌梅丸**:"蚘厥者……得食而呕,又烦者,蚘闻食臭出,其人常自吐蚘"+"**又主久利**"(寒热错杂久利,[[33-内科-脾胃系病证]] 久泻互参;含附子细辛蜀椒,B 级)。
- **当归四逆汤**:"**手足厥寒,脉细欲绝**者"(血虚寒厥——冻疮、雷诺样表现的经典方,[[37-内科-肢体经络病证]] 互参);内有久寒加吴茱萸生姜。
- **白头翁汤**:"**热利下重**者""下利,欲饮水者,以有热故也"([[33-内科-脾胃系病证]] 痢疾湿热主方)。
- **除中**(危候):"厥利者,当不能食,今反能食者,恐为**除中**……食以索饼,不发热者,知胃气尚在,必愈;恐暴热来出而复去也"——垂危之人突然暴食暴清醒须警惕回光返照,经典早有记载。
- 厥阴死证:"发热,下利厥逆,**躁不得卧者死**";"发热,下利至甚,厥不止者死";"六七日不利,便发热而利,其人**汗出不止者死,有阴无阳故也**";"下利后脉绝,手足厥冷,晬时脉还,手足温者生,脉不还者死"。
- 干姜黄芩黄连人参汤:"**食入口即吐**"(寒格吐逆);麻黄升麻汤存目(疑非仲景);"呕家有痈脓者,**不可治呕,脓尽自愈**"(见脓血呕吐今日即就医)。

## 七、霍乱与差后劳复(病后调理层)

- **霍乱**(吐利并作,今之急性胃肠炎类):"热多欲饮水者,**五苓散**主之;寒多不用水者,**理中丸**主之"。理中丸服法:"以沸汤数合,和一丸,研碎,温服之……**腹中未热,益至三四丸**,然不及汤"+加减法+"服汤后如食顷,**饮热粥一升许,微自温,勿发揭衣被**"([[26-清热祛暑温里剂]] 理中条目之源)。吐利肢冷脉微用四逆汤/通脉四逆加猪胆汁汤。**呕吐腹泻不止+四肢冷+脉微=红线**,经典与今识一致([[33-内科-脾胃系病证]] 危重指征互参)。
- **差后(病愈后)诸方**:劳复用枳实栀子豉汤;"更发热"用小柴胡;腰以下水气用牡蛎泽泻散;"喜唾,久不了了,胸上有寒"用理中丸;"**虚羸少气,气逆欲吐**"用**竹叶石膏汤**(★热病后期气阴两伤调理名方,[[26-清热祛暑温里剂]] 互参)。
- **食复通则**(★病后饮食铁律):"病人脉已解,而日暮微烦,以病新差,**人强与谷,脾胃气尚弱,不能消谷,故令微烦,损谷则愈**"——大病初愈减食即安,与 42 素问热论"病热少愈,食肉则复,多食则遗"同源,家庭护理 A 级要点。
- 烧裈散(阴阳易)存目:时代产物,无现代应用。

## 八、治则治法通则(条文名句)

- **辨证总纲**:"观其脉证,**知犯何逆,随证治之**"(坏病处理原则=一切误治变局的方法论)。
- **表里先后**:"伤寒,医下之,续得下利清谷不止,身疼痛者,**急当救里**;后身疼痛,清便自调者,急当救表。救里宜四逆汤,救表宜桂枝汤"(里虚寒危及生命时先救里——急则治本的实例);"下利腹胀满,身体疼痛者,**先温其里,乃攻其表**"。
- **自愈机转**:"凡病若发汗、若吐、若下、若亡血、亡津液,**阴阳自和者,必自愈**";"大下之后,复发汗,小便不利者,亡津液故也,**勿治之**,得小便利,必自愈"(★不过度治疗、留人体自复余地的经典表达)。
- **汗法法度**:"凡发汗,欲令手足俱周,时出似漐漐然……**不可令如水流离**";"凡服汤发汗,**中病便止,不必尽剂**也"。
- 战汗:"必先振栗汗出而解"(正邪剧争后的解期,发抖后大汗热退可为向愈)。
- 欲愈之候:"凡得病,反能饮水,此为欲愈之病"(但"小渴者乃强与饮之,因成其祸"——能饮是好兆,强灌是祸端)。

## 九、不可汗吐下禁例汇总(★安全层,与 [[安全-配伍妊娠禁忌与毒性药]] [[40-针灸-灸法按摩与外治法]] 互参)

- **发汗七禁**(不可发汗篇+太阳篇):"**咽喉干燥者**,不可发汗;**淋家**发汗必便血;**疮家**虽身疼痛,汗出则痓;**衄家**汗出必额上陷、直视不得眠;**亡血家**发汗则寒栗而振;**汗家**重发汗必恍惚心乱;尺中迟(**营气不足血少**)者不可发汗"+"下利不可发汗,汗出必胀满"——阴血亏虚者禁辛温发散,今日感冒药选用同理(与灵枢五夺同义)。
- **不可下**:"诸四逆厥者,不可下之,**虚家亦然**";"结胸证,其脉浮大者,不可下,下之则死";"伤寒**呕多**,虽有阳明证,不可攻之";"太阳病,外证未解,不可下,下之为逆"(表未解禁攻里)。
- **不可吐**:"诸四逆厥者,不可吐之,虚家亦然"。
- 动气(脐周搏动)四禁存目:动气在右/左/上/下,皆不可汗下(虚象之征)。
- 峻剂个别禁例已随方记于第三节(大青龙、白虎、瓜蒂散"诸亡血虚家不可与")。
- **误治谱系**(火逆、误汗、误下诸变条文密布全书):其总训即"**虚虚实实,医杀之耳**"(难经语)的伤寒版——本 Skill"涉药四关"核验流程之经典根基。
