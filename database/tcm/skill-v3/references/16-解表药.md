# 解表药(第八章)

> 提炼自《中药学》七版第八章。29 味:发散风寒 17 + 发散风热 12。
> **本章读法**:表证(恶寒发热、头身痛、无汗或汗出不畅、脉浮)先入此章;先辨风寒/风热定节,再按兼症(咳喘、鼻塞、头痛部位、气滞、水肿、疹痒)选药。
> 所引成方全条目见 `24`~`30`;证型→方路径见 [[00-证型直达选方总索引]]。

## 章纲

- **机理**:多辛散轻扬,主入肺、膀胱经(肺合皮毛、太阳主一身之表),偏行肌表,开腠理、促发汗,使表邪随汗外解——《内经》"其在皮者,汗而发之"。防止表邪内传是解表的另一层意义。
- **兼效**:部分药兼利水消肿、止咳平喘、透疹、止痛、消疮,故水肿、咳喘、麻疹、风疹、痹痛、疮疡初起**兼有表证**者也用本章药。
- **配伍**:四时之邪不同——夏配祛暑化湿,秋配润燥;虚人外感须扶正祛邪(气虚配益气、阳虚配助阳、阴虚配养阴、血虚配养血);温病初起卫分证配清热解毒。
- **禁忌与用法**:①发汗力强者量勿过大,防**过汗亡阳伤阴**;②汗为津液、血汗同源——**表虚自汗、阴虚盗汗、疮疡日久、淋证、失血者慎用**;③因时因地:春夏腠理疏、南方炎热量宜轻,冬季及北方严寒量宜重;④辛散轻扬,**入汤剂不宜久煎**。

## 第一节 发散风寒药(辛温解表)

节纲:辛温,发散肌表风寒。主治**风寒表证**——恶寒发热、无汗或汗出不畅、头身疼痛、鼻塞流涕、口不渴、苔薄白、**脉浮紧**。

### ★ 麻黄 辛微苦温│肺、膀胱│发汗解表,宣肺平喘,利水消肿
三效同出一源——**宣肺**:外开皮毛之郁闭则汗出表解,内降上逆之气则喘平,上宣肺气、通调水道、下输膀胱则水消。
1. **发汗解表要药**:辛开腠、温散寒,发汗力强,主**风寒表实无汗**,配桂枝相须(麻黄汤《伤寒论》)。
2. **平喘要药**:风寒束肺配杏仁、甘草(三拗汤);寒饮伏肺痰多清稀配细辛、干姜、半夏(小青龙汤);**肺热壅盛配石膏**(麻杏甘石汤)——寒热喘咳皆可随配伍取用。
3. 风水水肿兼表证(甘草麻黄汤、越婢加术汤《金匮》)。此外散寒通滞,治风寒痹证、阴疽痰核。
- 2~9g;**发汗生用,平喘蜜炙**。表虚自汗、阴虚盗汗、肺肾虚喘慎用。《纲目》:"张仲景治伤寒,无汗用麻黄,有汗用桂枝。"

### ★ 桂枝 辛甘温│心、肺、膀胱│发汗解肌,温通经脉,助阳化气
辛甘温煦,**甘温助卫阳、畅营血**,开腠发汗较麻黄温和——故风寒表证**无论表实无汗(配麻黄)、表虚有汗(配白芍,调和营卫,桂枝汤)、阳虚受寒(配麻黄附子细辛)均宜**。
2. **寒凝血滞诸痛**:胸痹(枳实薤白桂枝汤)、中焦虚寒脘腹冷痛(小建中汤)、经寒痛经(温经汤)、风寒湿痹肩臂痛(桂枝附子汤)——温通经脉、散寒止痛。
3. **痰饮、蓄水**:温脾阳助运水、温肾阳助膀胱气化(苓桂术甘汤、五苓散)。
4. **心悸**:助心阳、通血脉、止悸动(炙甘草汤);阴寒冲气上凌之奔豚重用之(桂枝加桂汤)。
- 3~9g。辛温助热、易伤阴动血:**外感热病、阴虚火旺、血热妄行忌用;孕妇及月经过多者慎用** → [[安全-配伍妊娠禁忌与毒性药]]。

### 荆芥 与 防风(风药之通用对)

- **荆芥** 辛微温│肺肝│祛风解表,透疹消疮,止血——**发散风寒药中药性最平和**,风寒(荆防败毒散)风热(银翘散)皆可;麻疹不透、风疹瘙痒;疮疡初起兼表;**炒炭止血**(吐衄下血崩漏)。4.5~9g 不久煎;发表透疹生用、止血炒炭,荆芥穗更长于祛风。
- **防风** 辛甘微温│膀胱肝脾│祛风解表,胜湿止痛,止痉——**"风药之润剂""治风通用药"**(甘缓微温不峻烈),风寒、风湿(羌活胜湿汤)、风热皆可配;卫虚感冒与黄芪白术相反相成(玉屏风散:固表不留邪,祛邪不伤正);风疹瘙痒(消风散);风湿痹痛(蠲痹汤);**破伤风**(玉真散);土虚木乘之痛泻(痛泻要方)。4.5~9g。阴血亏虚、热病动风不宜。
- **鉴别**:皆微温不燥、发表散风,风寒风热两可,同治风疹。荆芥质轻透散,**发汗力强**,兼透疹、消疮、止血;防风**祛风力强**,兼胜湿、止痛、止痉。

### 止痛通窍组(羌活·白芷·细辛·藁本)——按头痛分经选药

> 头痛引经:**太阳**羌活/藁本(后头连项、巅顶),**阳明**白芷(前额眉棱骨),**少阴**细辛;另少阳柴胡、厥阴吴茱萸(见各章)。

- **羌活** 辛苦温│膀胱肾│解表散寒,祛风胜湿,止痛——气味雄烈,善治**风寒夹湿之表证**(恶寒无汗+头痛项强+肢体酸重,九味羌活汤);**上半身风寒湿痹、肩背肢节痛**尤宜(蠲痹汤)。3~9g。辛燥,阴血亏虚慎;量大易呕,脾胃虚弱不宜。
- **★ 白芷** 辛温│肺胃大肠│解表散寒,祛风止痛,通鼻窍,燥湿止带,消肿排脓——**阳明头额痛、眉棱骨痛、牙痛要药**(都梁丸、川芎茶调散);**鼻渊**(苍耳子散);寒湿带下(配鹿角霜白术,湿热带下配黄柏车前子);疮痈**初起散结、脓成排脓**(仙方活命饮、托里透脓散)。3~9g。辛温香燥,阴虚血热忌。
- **★ 细辛** 辛温,**有小毒**│肺肾心│解表散寒,祛风止痛,通窍,温肺化饮——既散在表风寒,又除在里寒邪:**阳虚外感**(麻黄附子细辛汤);**少阴头痛、风冷牙痛、痹痛**(止痛力颇强);鼻渊;**寒饮咳喘**(小青龙汤、苓甘五味姜辛汤)。
  - **1~3g,散剂每次 0.5~1g**(《本草别说》"若单用末,不可过半钱匕"——"细辛不过钱"指散剂);**反藜芦**;阴虚阳亢头痛、肺燥干咳忌。过量致呼吸麻痹 → [[安全-配伍妊娠禁忌与毒性药]]。
  - **麻黄·桂枝·细辛鉴别**:皆辛温发散风寒。麻黄发汗最强主表实重证;桂枝和缓,表实表虚皆宜;细辛发汗不及而**散寒力胜**,达表入里,配伍治阳虚外感。
- **藁本** 辛温│膀胱│祛风散寒,除湿止痛——性味俱升,**专达巅顶**,治太阳风寒**巅顶头痛**(神术散);风湿一身尽痛(羌活胜湿汤)。3~9g。血虚、肝阳、火热头痛忌。

### 通鼻窍组(苍耳子·辛夷)

- **苍耳子** 辛苦温,**有毒**│肺│发散风寒,通鼻窍,祛风湿,止痛——**治鼻渊良药**(头痛、不闻香臭、流浊涕,一药数效,苍耳子散);风湿痹痛;疥癣风疹。3~9g,炒去硬刺。**一次超 30g 或炮制不当可致肝肾损害、痉挛死亡**;血虚头痛不宜 → [[安全-配伍妊娠禁忌与毒性药]]。附苍耳草(小毒,祛风清热解毒)。
- **辛夷** 辛温│肺胃│发散风寒,通鼻窍——芳香上达,**鼻渊头痛鼻塞流涕要药**,风寒配白芷细辛,风热配薄荷黄芩。3~9g,**有毛刺激咽喉,须纱布包煎**。阴虚火旺忌。

## 第二节 发散风热药(辛凉解表)

节纲:辛苦偏寒凉,发汗力较缓。主治**风热感冒、温病初起邪在卫分**——发热、微恶风寒、咽干口渴、头痛目赤、舌边尖红、苔薄黄、**脉浮数**。部分兼清头目、利咽喉、透疹、止痒、止咳。

### 疏散风热三要药:薄荷 · 牛蒡子 · 蝉蜕

- **★ 薄荷** 辛凉│肺肝│疏散风热,清利头目,利咽透疹,**疏肝行气**——辛凉药中**宣散力最强、有一定发汗作用**者,风热感冒温病卫分首选之列(银翘散);风热上攻之头痛眩晕、目赤多泪、咽喉肿痛(六味汤);麻疹风疹;**肝郁气滞**(逍遥散);兼芳香辟秽化湿和中(夏令暑湿秽浊)。3~6g **后下**;叶长发汗、梗偏行气。发汗耗气,体虚多汗不宜。
- **★ 牛蒡子** 辛苦寒│肺胃│疏散风热,**宣肺祛痰**,利咽透疹,**解毒消肿**——升散之中具清降之性:风热兼**咽喉红肿、咳痰不利**者最宜(银翘散);麻疹不透(竹叶柳蒡汤)、风疹;**痈肿疮毒、丹毒、痄腮喉痹**(普济消毒饮),性偏滑利兼通便,热毒兼便秘尤宜。6~12g,炒用寒滑之性略减。气虚便溏慎。
- **蝉蜕** 甘寒│肺肝│疏散风热,**利咽开音**,透疹,**明目退翳,息风止痉**——质轻上浮:风热**咽痛音哑**尤宜;麻疹风疹;肝经风热之**目赤翳障**(蝉花散);**急慢惊风、破伤风**(五虎追风散)、小儿夜啼。3~10g,止痉需大量。孕妇慎(《别录》"主妇人生子不下")。
- **三药鉴别**(皆疏散风热、透疹、利咽):薄荷**发汗力强**兼清利头目、疏肝;牛蒡子兼**宣肺祛痰、解毒散肿、滑肠**;蝉蜕兼**开音、退翳、止痉**。
> 张锡纯:薄荷为"温病之麻黄";连翘单用一两可使周身微汗终宵不竭;蝉蜕单用三钱即得微汗,"以皮达皮"善托疹托毒。详 [[48-医家-衷中参西录要义]]。

### 桑叶 与 菊花(轻清疏散对)

- **★ 桑叶** 甘苦寒│肺肝│疏散风热,**清肺润燥**,平抑肝阳,清肝明目——风热咳嗽(配菊花,桑菊饮);**肺热燥咳**(轻者桑杏汤,重者清燥救肺汤);肝阳眩晕;目赤昏花且**甘润益阴**(精血不足配黑芝麻,扶桑至宝丹);兼凉血止血。5~9g;**蜜炙增强润肺止咳**。
- **★ 菊花** 辛甘苦微寒│肺肝│疏散风热,平抑肝阳,清肝明目,清热解毒——风热感冒(桑菊饮);**肝阳上亢头痛眩晕**(配石决明白芍;热极动风,羚角钩藤汤);目赤昏花(风热配蝉蜕木贼,肝火配决明子夏枯草,**肝肾不足配枸杞地黄——杞菊地黄丸**);疮痈(力不及野菊花)。5~9g。**黄菊花疏散风热,白菊花平肝明目**。
- **鉴别**:功效几同。桑叶**疏散力强**,兼清肺润燥、凉血止血;菊花**平肝清肝明目力强**,兼清热解毒。

### 蔓荆子 辛苦微寒│膀胱肝胃│疏散风热,清利头目
解表之力较弱,**偏于清利头目、疏散头面之邪**:①风热**头昏头痛**多用(配薄荷菊花);风邪上攻之偏头痛配川芎、白芷、细辛;②目赤肿痛、目昏多泪(配菊花蝉蜕白蒺藜);药性升发,配黄芪人参升麻葛根,治**中气不足、清阳不升之耳鸣耳聋**(益气聪明汤);③兼祛风止痛,治风湿痹痛(羌活胜湿汤)。5~9g。

### 升散清阳三药:柴胡 · 升麻 · 葛根

- **★ 柴胡** 苦辛微寒│肝胆│**解表退热,疏肝解郁,升举阳气**——①**少阳证要药**:往来寒热、胸胁苦满、口苦咽干目眩,配黄芩清半表半里(小柴胡汤);表证发热风寒风热皆可(正柴胡饮、柴葛解肌汤);②**肝郁气滞**:胁痛、月经不调、痛经(柴胡疏肝散;肝郁血虚,逍遥散);③气虚下陷、脏器脱垂(补中益气汤);兼退热截疟。3~9g;解表生用量稍重,疏肝醋炙、升阳生用量轻。**"柴胡劫肝阴"**——阴虚阳亢、肝风内动、阴虚火旺、气逆者忌慎。
- **★ 升麻** 辛微甘微寒│肺脾胃大肠│解表透疹,**清热解毒**,升举阳气——风热风寒表证皆可配;**麻疹不透**(升麻葛根汤、宣毒发表汤);**尤善清阳明热毒**:牙龈肿痛口疮(清胃散)、大头瘟(普济消毒饮)、痄腮、温毒发斑;**升提之力强于柴胡**(补中益气汤、举元煎、升陷汤)。3~9g;发表透疹解毒生用,升阳炙用。麻疹已透、阴虚火旺、阴虚阳亢忌。
- **★ 葛根** 甘辛凉│脾胃│**解肌退热,透疹,生津止渴,升阳止泻**——①表证发热**项背强痛**专长(无汗葛根汤,有汗桂枝加葛根汤;郁热柴葛解肌汤);②麻疹不透(升麻葛根汤);③**热病口渴、消渴**(鼓舞胃气上行生津,玉泉丸);④**热泻热痢**(葛根芩连汤)、**脾虚泄泻**(七味白术散)。现代:扩血管降压,善解高血压**项紧**。9~15g;生津退热透疹生用,**止泻煨用**。
- **三药鉴别**(皆发表、升阳):柴胡升**肝胆**之气,善疏散少阳、疏肝解郁;升麻升**脾胃**清阳(升提最强),善清热解毒;葛根升脾胃清阳而**生津、止泻**,善解肌治项背强。柴胡+升麻→升阳举陷(脏器脱垂);升麻+葛根→透疹。附葛花:解酒毒醒脾(3~15g)。
