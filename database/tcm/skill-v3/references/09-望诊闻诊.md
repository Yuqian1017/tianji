# 四诊(一):望诊、闻诊

> 提炼自《中医诊断学》第二章第一、二节。诊断原理:**司外揣内**("有诸内者,必形诸外");原则:审察内外整体察病、辨证求因、**四诊合参**("望闻问切,医之不可缺一")。
> **温病诊法补强**(v2.2《温病三书》):叶天士察舌六系(白/黄/绛/紫/淡红/黑,逐象立法,含"黑苔未必皆热""绛而光亮胃阴亡")、**验齿**(齿为肾之余,龈为胃之络)、辨斑疹(斑大片属肌肉、疹小粒属血络;**阴斑当温**)、白㾦(水晶色气液未竭、枯白如骨凶)与薛生白验舌投剂两条,见 [[47-温病-卫气营血三焦方证条文]] 第一、二部分;本节教材舌诊体系为准,47 为温病动态舌象专档。

## 一、望神

| 状态 | 表现 | 意义 |
|---|---|---|
| **得神** | 神志清楚、目光明亮精彩内含、面色荣润、反应灵敏、呼吸平稳 | 精充气足;病亦轻,预后好 |
| **失神** | 精神萎靡、神昏谵语、**循衣摸床撮空理线**、目暗睛迷、大肉已脱 | 精损气亏,病重预后不良 |
| **假神** | 久病重病本已失神,突然精神转佳、颧赤如妆、言语不休、食欲骤增 | **回光返照**:阴不敛阳、虚阳外越,临危预兆。鉴别:假神出现突然、与整体病情不符 |
| 神气不足 | 精神不振、健忘困倦、声低懒言 | 轻度失神,虚证(心脾两亏、肾阳不足) |
| 神志异常 | 烦躁(心火痰火阴虚火旺);**癫**(淡漠痴呆自语,痰气郁结,阴证"文痴");**狂**(骂詈妄动登高而歌,痰火扰心,阳证"武痴");**痫**(突然昏倒吐沫抽搐醒后如常,肝风挟痰) | |

## 二、望色(五色主病)

常色:明润含蓄(主色终生不变,客色随四时环境)。病色:

| 色 | 主病 | 细辨 |
|---|---|---|
| **青** | 寒、痛、瘀血、惊风、肝病 | 青黑/苍白淡青=阴寒;青灰唇青紫=心血瘀阻;小儿眉间鼻柱唇周青紫=惊风先兆 |
| **黄** | 湿、虚(脾) | **萎黄**(淡黄憔悴)=脾胃气虚;**黄胖**(黄而虚浮)=脾虚湿停;**阳黄**(鲜明如橘)=湿热;**阴黄**(晦暗如烟熏)=寒湿 |
| **赤** | 热 | 满面通红=实热;两颧嫩红=虚热;危重面红如妆=**戴阳**(虚阳上越) |
| **白** | 虚寒、血虚 | 㿠白虚浮=阳虚;淡白消瘦=营血亏;苍白=阳气暴脱或失血 |
| **黑** | 肾虚、水饮、寒、痛、瘀 | 黑而焦干=肾精久耗;目眶黑=肾虚水泛/寒湿带下;青黑剧痛=寒凝瘀阻 |

## 三、望形体姿态

- 形体:强弱胖瘦。**"肥人多湿"**(形盛气虚痰湿)、**"瘦人多火"**(阴血不足虚火);大肉脱失=脏腑精气衰竭危象。
- 姿态:睑面唇指颤动=外感发痉之兆/血虚阴亏;四肢抽搐角弓反张=痉病(热极生风、惊风、破伤风);战栗=疟疾/战汗之兆;痿(软弱无力)、痹(关节痛)、瘫痪、厥(卒倒呼吸自续)。
- 体位辨证:畏缩多衣=寒;揭衣被=热;卧喜向外身轻自转=阳证热证实证,卧喜向里身重难转=阴证寒证虚证;**蜷卧成团**=阳虚寒盛或剧痛,仰面伸足=阳热证;**但坐不得卧、卧则气逆**=咳喘肺胀水饮;但卧不耐坐、坐则昏眩=气血双亏。

## 四、局部望诊(要点)

- **头囟**:囟陷(津伤髓亏虚)、囟填(热盛)、解颅(肾气不足);头摇不能自主=肝风。发:稀疏干枯=精血不足;突然片状脱发=血虚受风;小儿发结如穗=疳积。
- **目**:目眦赤=心火;白睛赤=肺火;胞睑红肿=脾火;全目赤肿=肝经风热;白睛黄=黄疸;目窠微肿如卧蚕=水肿初起;**目窝凹陷**=阴液耗损;**瞳仁扩大**=肾精耗竭濒危;戴眼反折(目睛上视)=惊风痉厥;横目斜视=肝风;睑废(睑下垂)=脾肾亏/脾气虚。
- **鼻**:鼻头色赤=肺热,白=气虚血少,黄=里湿热,青=腹中痛,微黑=水气;**鼻煽**——新病肺热、久病肺肾精气虚衰危证;清涕风寒、浊涕风热、浊涕腥臭=鼻渊。
- **耳**:耳轮焦黑干枯=肾精亏极;耳轮甲错=久病血瘀;耳廓萎缩=肾气竭绝。
- **唇口齿龈咽喉**:唇深红=实热,淡红=虚寒,深红干焦=热极伤津,淡白=气血两虚,青紫=阳虚血瘀;口噤(痉病惊风)、口撮(脐风破伤风)、口僻(中风)、口张气但出不返(肺气将绝)。齿燥=胃津伤,**齿燥如枯骨**=肾精枯竭;龈红肿出血=胃火,微红微肿齿缝出血=肾阴虚火,淡白不肿出血=脾不摄血;睡中齘齿=胃热虫积。咽喉:红肿溃烂=肺胃积热/热毒;鲜红娇嫩肿痛不甚=阴虚火旺;**乳蛾**=肺胃热盛;灰白假膜擦之不去重擦出血=**白喉**(疫喉)。
- **躯体前后阴**:颈瘿(大脖子)、瘰疬(串珠);鸡胸佝偻=先天不足;桶状胸咳喘=痰热壅肺;胁间饱胀咳引痛=悬饮;**臌胀**——气臌(立卧均高)、水臌(立位胀卧位平摊);腹凹如舟=精气大亏;角弓反张;阴囊水疝/狐疝;阴挺(中气不足);脱肛、痔瘘。
- **皮肤**:丹毒(赤如涂丹);黄疸分阳黄阴黄;**肌肤甲错**=瘀血;**斑**(点大成片平铺,摸不应手,分阳斑阴斑)vs**疹**(如粟高起,摸之碍手:麻疹风疹隐疹);白㾦水泡;疮疡四辨——**痈**(红肿热痛根束)、**疽**(漫肿无头不热少痛)、**疔**(形小根深坚硬麻痒)、**疖**(浅小易脓易愈)。

## 五、舌诊(核心)

**原理**:舌为心之苗、脾之外候,五脏六腑之外候。分部:**舌尖候心肺、舌中候脾胃、舌根候肾、舌边候肝胆**(外感病亦可按三焦:尖上中中根下)。
**正常舌象:淡红舌、薄白苔**(柔软灵活,苔薄白润、揩之不去有根)。察舌质重在辨正气虚实,察舌苔重在辨邪气浅深与性质、胃气存亡。

### 舌质
- **舌神**:荣润灵动为有神(善候),枯晦板滞为无神(凶候)。
- **舌色**:淡白=虚寒/气血两亏;红=热(实热或虚热);**绛**=外感热入营血、内伤阴虚火旺;紫=血瘀(绛紫干枯=热盛,淡紫青紫湿润=寒凝阳虚);青=寒凝阳郁/瘀血。
- **舌形**:苍老(纹理粗糙坚敛)=实证;娇嫩浮胖=虚证;**胖大**=水饮痰湿(常兼齿痕),肿胀满口=热毒酒毒;瘦薄=气血两虚/阴虚火旺;**芒刺**=热盛(尖=心火,边=肝胆火,中=胃肠热);**裂纹**=精血亏损津伤(先天舌裂有苔覆盖者除外);**齿痕**=脾虚湿盛。
- **舌态**:强硬(热入心包、高热伤津、痰阻、**中风先兆**);痿软(气血虚极、阴亏已极);短缩(**危重**:寒凝/痰阻/热盛伤津/气血虚);颤动(血虚生风、热极生风);**歪斜**=中风或先兆;吐弄=心脾有热、小儿智力发育不全;麻痹=血虚/肝风。

### 舌苔
- **苔质**:厚薄(见底/不见底)辨邪之浅深——由薄转厚病进,由厚转薄病退;润燥辨津液——滑苔=寒湿痰饮,燥苔=热盛伤津/阳不化津;**腐苔**(如豆腐渣,揩之可去)=食积痰浊胃肠郁热,**腻苔**(细腻致密刮之不脱)=湿浊痰饮;**剥落**——镜面舌(光滑无苔)=胃阴枯竭胃气大伤(危),花剥苔=胃之气阴两伤;有根(真苔)=胃气未衰,无根(浮涂可刮)=胃气已衰。舌苔骤长骤退=病情暴变。
- **苔色**:白=表证寒证(特殊:**积粉苔**=瘟疫/内痈毒热;糙裂苔=燥热暴伤津);黄=里证热证(淡黄轻、深黄重、焦黄结;白转黄=表邪入里化热;黄滑润=阳虚水湿);灰=里证(干=热伤津,润=痰饮寒湿);黑=危重(黑燥裂芒刺=热极津枯,黑滑润舌淡=阴寒盛水湿不化)。
- **综合诊察**:舌质舌苔一般一致(红舌黄干苔=里实热,淡舌白润苔=里虚寒);不一致时四诊合参(如红绛舌+白干苔=燥热伤津苔未及转黄)。注意**染苔**(饮食药物假象)。

## 六、望排出物与小儿指纹

- 总纲:**清白稀=寒虚;黄赤稠浊=热实;黑而有块=瘀**。
- 痰:黄稠成块=热痰;白清稀=寒痰;白滑量多易咯=湿痰;少而粘难咯=燥痰;带血=热伤肺络。
- 呕吐物:清稀无臭=寒呕;酸臭秽浊=热呕;清水痰涎量多=痰饮;酸腐夹食=食积;黄绿苦水=肝胆郁热/湿热。
- 大便:清稀完谷不化=寒泻;黄糜恶臭=热泻;白=脾虚/黄疸;干结如羊屎=阴血亏;粘冻脓血里急后重=痢疾;柏油便=胃络出血;先血后便色鲜=近血(痔),先便后血色暗=远血。
- 小便:清长量多=寒;短赤灼痛=热;膏脂=膏淋;砂石=石淋;血尿灼痛=血淋;米泔浊尿形瘦=脾肾虚损。
- **小儿指纹**(三岁以下,代寸口):三关=风关(掌侧第一节)→气关→命关。要诀:**"浮沉分表里,红紫辨寒热,淡滞定虚实,三关测轻重"**;鲜红=外感风寒,紫红=热,青=风证痛证,紫黑=血络闭郁,淡白=脾虚;**透关射甲**=病情危重。

## 七、闻诊

### 听声音
- 发声:高亢多言躁动=实热;低微沉静=虚寒。**音哑失音**:新病=实("金实不鸣":外感风寒风热袭肺、痰浊壅肺);久病=虚("金破不鸣":肺肾阴虚虚火灼金)。鼾声不绝昏睡不醒=高热神昏/中风入脏。小儿阵发惊呼=惊风。
- 语言:**谵语**(神昏胡言、声高有力)=实热扰心;**郑声**(语言重复、低微断续)=心气大伤,虚证;狂言=痰火扰心(阳证);癫语=痰气郁结(阴证);独语(喃喃自语见人便止)=心气血虚/痰蒙;错语(言后自知)=虚证或肝郁痰阻。
- 呼吸:**实喘**(发病急、气粗声高、唯以呼出为快、脉数有力)vs**虚喘**(病缓、气怯声低、得引长息为快、动则更甚)——外邪痰浊 vs 肺肾气阴虚。**哮**(喉中痰鸣如哨,反复发作):冷哮(冬春遇冷,寒饮阻肺)、热哮(夏秋燥热,痰热阻肺)。上气(呼多吸少)、短气(呼吸短促似喘不抬肩)=肺气不足/饮停胸中;少气(语声低微)=诸虚;气粗=实,气微=虚。
- 咳嗽:外感急短属实、内伤缓长属虚;咳声紧闷=寒湿,清脆=燥热;昼甚=热燥,夜甚=肺肾阴亏;无力低微=肺气虚。**顿咳**(百日咳:阵咳鹭鸶鸣,小儿);**犬吠样咳**=白喉。
- 呕吐嗳气呃逆(皆胃气上逆):吐势徐缓声弱=虚寒,急而响亮=实热;嗳气低弱=脾胃虚,高亢嗳后腹减=食滞肝郁;呃声高亢有力=实热,低沉无力=虚寒(久病呃逆不止为胃气衰败之兆)。太息=肝郁气机不畅。

### 嗅气味
口臭=胃火/宿食/口腔病;汗有酸腐气=实热/阴虚火旺;鼻臭=鼻渊;**呼气烂苹果味=消渴重症;尿臊气=水肿病危**(病室同);嗳气酸腐=食滞化热;大便恶臭=湿热,腥=脾胃虚寒;矢气败卵味=宿食;带下臭秽色黄=湿热,腥而色白=寒湿;病室血腥味=失血,尸臭=脏腑败坏。

## 经典补强(v2.0 经典五书反哺;详 [[42-经典-素问要义]] 第四节、[[43-经典-灵枢与难经要义]] 第八节、[[45-经典-金匮要略杂病方证条文]] 第一节)

- **金匮首篇望闻三条**:鼻头色青=腹中痛("苦冷者死");**语声寂然喜惊呼=骨节间病、喑喑然不彻=心膈间病、啾啾然细而长=头中病**(闻声定病位);**息摇肩者心中坚、息张口短气者肺痿唾沫、"呼吸动摇振振者,不治"**(呼吸形态的危重分级——耸肩呼吸、张口短气、全身随呼吸摇动逐级加重)。
- **五府败象**(素问·脉要精微论):头倾视深(精神将夺)、背曲肩随(府将坏)、腰转摇不能(肾将惫)、膝屈伸不能行则偻附(筋将惫)——"**得强则生,失强则死**",老人体态即脏腑盛衰之镜。
- 面部分部与病色动态(灵枢·五色):明堂=鼻,阙中候肺、下极候心、鼻端候脾;"青黑为痛,黄赤为热,白为寒";"其色**上行者病益甚,下行如云彻散者病方已**"。
