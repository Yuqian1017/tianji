# 中药总论(药性理论与用药法度)

> 提炼自《中药学》七版总论(第一~七章+命名分类)。这是各论所有单味药的"读法总纲"——先掌握五维药性如何叠加推导功效,再看各论才不会死记。

## 一、药性理论:一味药的五个投影

任何一味中药,本草都从**四气、五味、升降浮沉、归经、有毒无毒**五个维度标定其性能。这五者不是孤立标签,而是同一药物作用的不同侧面,**必须综合叠加**才能准确判断功效与适应证。药物治病的总机理:以药物的**偏性**纠正人体的阴阳偏盛偏衰(扶正祛邪、调理脏腑)。

### 1. 四气(四性):寒、热、温、凉

- 反映药物对人体**寒热倾向**的作用:寒凉属阴,温热属阳;寒与凉、温与热只是程度差(凉次于寒,温次于热),另有"大寒/微温"等更细区分。本质只有**寒热两性**。
- **平性**:寒热不显、药性平和者(党参、山药、甘草、天麻)。仍归入四气(实则偏温偏凉),故称"四气"不称"五气"。
- **判定依据**:由药物作用于人体后的疗效反推——能缓解阳热证(高热烦渴、面红、脉洪数)者为寒凉(石膏、知母、栀子);能缓解阴寒证(肢厥、面白、脉微)者为温热(附子、肉桂、干姜)。
- **用药总则**(《素问》"寒者热之,热者寒之"):寒凉药治阳热证(实热、温毒发斑、火毒疮疡、热结便秘、痰热喘咳等),温热药治阴寒证(中寒腹痛、寒疝、阳痿宫冷、寒痹、亡阳虚脱等)。用反则病重(桂枝下咽阳盛则毙,承气入胃阴盛以亡)。程度也需对应:当热用温、当寒用凉则药轻病重;当温用热、当凉用寒则反伤阴阳。
- 寒热错杂→寒热并用;阴阳格拒→寒热反佐(以热治寒而寒拒热,反佐寒药引入)。

### 2. 五味:辛、甘、酸、苦、咸(+淡、涩)

味既是口尝的真实滋味,更是**对功效的高度概括**(故本草标味常与口感不符)。《素问》"辛散、酸收、甘缓、苦坚、咸软"为纲:

| 味 | 作用纲领 | 主治 | 代表 |
|---|---|---|---|
| **辛** | 能散能行(发散、行气、行血) | 表证、气血阻滞 | 苏叶散寒、木香行气、川芎活血。另"辛润"(款冬润肺、菟丝养肾)属少数 |
| **甘** | 能补能和能缓(补益、和中、缓急、调和药性) | 虚证、拘急疼痛、中毒解救、调和诸药 | 人参补气、熟地补血、饴糖缓急、甘草和药解毒 |
| **酸** | 能收能涩(收敛固涩) | 虚汗、久咳、久泻、遗精、遗尿、崩带 | 五味子敛汗、乌梅敛肺、山茱萸涩精 |
| **苦** | 能泄能燥能坚(清泄火热/泄降气逆/通泄大便;燥湿;坚阴即泻火存阴) | 热证、喘咳、呕逆、便秘、湿证、阴虚火旺 | 黄芩泻火、杏仁降气、大黄通便、黄连燥湿、知母黄柏泻火存阴 |
| **咸** | 能下能软(泻下、软坚散结) | 便秘、痰核瘿瘤、癥瘕痞块 | 芒硝泻下、海藻牡蛎软坚、鳖甲消癥。另"咸走血"入肾(玄参、紫草凉血;紫河车、龟板补肾) |
| **淡** | 能渗能利(渗湿利尿) | 水肿、小便不利 | 茯苓、猪苓、泽泻、薏苡仁。"淡附于甘",故仍称五味 |
| **涩** | 同酸(收敛固涩) | 虚汗、泄泻、遗精、出血 | 莲子、禹余粮、乌贼骨。本草常以酸代涩 |

**气味综合原则**:气味相同则功近(辛温多发散风寒,甘温多补气助阳);气同味异/味同气异则功别(同温性:麻黄辛温散寒、杏仁苦温降气、乌梅酸温敛肺、肉苁蓉咸温补肾;同辛味:桂枝辛温、薄荷辛凉、附子辛热、石膏辛寒)。一药兼数味则治疗范围扩大(当归辛甘温→补血活血行气散寒)。复方中有时"用其气"有时"用其味"(升麻配黄芪取甘升阳、配葛根取辛透疹、配石膏取寒清热)。

> **芳香药性**(五味的补充):辟秽防疫、解表散邪(薄荷香薷)、悦脾开胃(木香佛手)、化湿去浊(苍术藿香佩兰)、通窍止痛(辛夷白芷细辛)、行气活血(香附乳香没药)、开窍醒神(麝香冰片苏合香)。

### 3. 升降浮沉:药物作用的四种趋向

针对病势(呕吐呃逆喘=上逆,脱肛遗尿崩漏=下陷,自汗盗汗=向外,表邪入里=向内)与病位(表里上下)选向:**升浮属阳**(上行向外:发表、透疹、升阳举陷、开窍、涌吐),**沉降属阴**(下行向内:清热、泻下、利水、潜阳、降逆、收敛)。
- 影响因素:①**气味**——辛甘温热多升浮,苦酸咸寒凉多沉降;②**质地**——花叶皮枝质轻多升浮,子实矿石贝壳质重多沉降("诸花皆升,旋覆独降;诸子皆降,苍耳独升"为例外);③**炮制**——酒制升、姜炒散、醋炒收、盐炒下行;④**配伍**——少量升浮药入大队沉降药中随之降,反之亦然。
- 病在上在表宜升浮,在下在里宜沉降;病势上逆宜降,下陷宜升。可升降并用(石膏配麻黄宣降肺气;黄连配肉桂交通心肾)。

### 4. 归经:药物作用的定位

某药对某脏腑经络有**选择性亲和**,是定位概念(四气五味只定性定向,归经才定位)。以脏腑经络学说为基础、以所治病证为依据总结(朱砂远志治心悸失眠故归心经;桔梗苏子治喘咳故归肺经)。
- **意义**:①辨证用药有的放矢(肺热用桑白皮、胃火用石膏、心火用朱砂丹参、肝热用夏枯草龙胆草);②区别相似药(同治头痛:羌活太阳、葛根阳明、柴胡少阳、吴茱萸厥阴、细辛少阴);③须结合脏腑相关(滋水涵木、培土生金),勿单纯见肺治肺;④须与四气五味升降浮沉合参(同归肺经:紫苏温散、薄荷凉散、干姜温肺化饮、黄芩清肺泻火)。

### 5. 毒性(有毒无毒)

- **古义**:毒=药物偏性(毒药为一切药总称);又指毒副作用大小(大毒/常毒/小毒/无毒——大毒治病十去其六,无毒十去其九)。
- **今义**:毒性=对机体的损害性(急慢性毒性、致癌致畸致突变、成瘾);毒药=能致功能障碍/死亡之物;剧毒=治疗量与中毒量接近。**副作用**(常用量下与治疗无关的轻微不适)有别于毒性,且有相对性(常山解疟时催吐为副作用)。
- **《药典》毒性分级**:大毒、有毒、小毒三类。
- **法定毒性中药**(《医疗用毒性药品管理办法》28种):全名单以 [[安全-配伍妊娠禁忌与毒性药]] 第四节为唯一权威源,此处不重复罗列。
- **中毒五因**:剂量过大、误服伪品、炮制不当(生附子生乌头)、制剂服法不当(乌附煎煮太短/服后受寒进冷)、配伍不当(甘遂配甘草、乌头配瓜蒌)。
- **毒性成分类别**(便于识别中毒):生物碱类(乌头附子唇舌麻木→心律失常;马钱子士的宁惊厥角弓反张;曼陀罗莨菪抗胆碱)、毒苷类(强心苷夹竹桃万年青;氰苷苦杏仁桃仁枇杷仁→组织缺氧;皂苷天南星商陆黄药子)、毒蛋白类(巴豆相思子→溶血肝肾损害)、萜与内酯类(苦楝马桑莽草子)、矿物类(砒霜三氧化二砷"七窍流血";朱砂硫化汞;铅丹)。

## 二、配伍:七情

| 七情 | 含义 | 意义 | 例 |
|---|---|---|---|
| 单行 | 单味药治单纯病 | — | 独参汤、清金散(单黄芩) |
| **相须** | 两同类药合用增效 | 协同,复方核心 | 麻黄+桂枝、知母+贝母、全蝎+蜈蚣 |
| **相使** | 一主一辅,辅助提高主药功效(不必同类) | 协同 | 黄芪+茯苓、枸杞+菊花、黄连+木香 |
| **相畏** | 一药毒副作用被另一药抑制 | 制毒(与相杀同一关系两说法) | 半夏畏生姜、甘遂畏大枣 |
| **相杀** | 一药能消除另一药毒副作用 | 制毒 | 生姜杀半夏毒、绿豆杀巴豆毒、防风杀砒霜毒 |
| **相恶** | 一药破坏(削弱)另一药功效 | **配伍禁忌** | 人参恶莱菔子、生姜恶黄芩 |
| **相反** | 两药合用产生剧烈毒副作用 | **配伍禁忌** | 甘草反甘遂、贝母反乌头(见十八反十九畏) |

相须相使增效(常用),相畏相杀制毒(用毒药与解救),相恶相反为禁忌。另有"药对"经验(超出七情):桂枝配芍药调和营卫、枳实配白术消补兼施、干姜配五味子开合宣降、黄连配肉桂交通心肾等。

## 三、用药禁忌

1. **配伍禁忌——十八反、十九畏**(硬约束) → 全文与安全细则见 [[安全-配伍妊娠禁忌与毒性药]]
2. **证候禁忌**:据药性各有适应范围(麻黄发汗宣肺利尿,故表虚自汗、阴虚盗汗、肺肾虚喘禁用;黄精滋腻,脾虚有湿痰多、中寒便溏不宜)。详见各论每药"使用注意"。
3. **妊娠禁忌**:禁用(毒烈:巴豆、牵牛、大戟、商陆、麝香、三棱、莪术、水蛭、斑蝥、雄黄、砒霜)、慎用(通经祛瘀行气破滞辛热滑利:桃仁、红花、牛膝、大黄、枳实、附子、肉桂、干姜、木通、冬葵子、瞿麦)。慎用者可据病情斟酌("有故无殒")。→ [[安全-配伍妊娠禁忌与毒性药]]
4. **服药饮食禁忌(忌口)**:一般忌生冷、油腻、腥膻、辛辣刺激;分证:热病忌辛辣煎炸、寒病忌生冷、肝阳上亢忌辛热助阳、疮疡皮肤病忌鱼虾发物。

## 四、剂量与用法

- **剂量**:每味药成人一日干品汤剂用量。换算:1两(旧16进制)=30g,1钱=3g,1分=0.3g。一般常用内服 5~10g;较大量 15~30g;鲜品 30~60g(约干品4倍)。
- 剂量考量:剧毒/峻烈药严控(从小量渐增,中病即止);花叶皮枝质轻量小,矿石贝壳质重量大;贵重药(犀角羚羊角麝香牛黄鹿茸珍珠)尽量减量;老幼孕产体弱减量(5岁以下用成人1/4,5岁以上减半);病重势急量大。
- **特殊煎法**(处方须注明):
  - **先煎**:矿石贝壳类(磁石、代赭石、石膏、龙骨、牡蛎、石决明、龟板、鳖甲)打碎先煎20~30分钟;**附子、乌头等毒药先煎45~60分钟**降毒。
  - **后下**:芳香易挥发(薄荷、青蒿、香薷、木香、砂仁、沉香、豆蔻);久煎破坏成分者(钩藤、大黄、番泻叶),煎沸5~10分钟前放入。
  - **包煎**:粘性/粉末/绒毛类(蛤粉、滑石、青黛、旋覆花、车前子、蒲黄、灶心土)。
  - **另煎(炖)**:贵重药单煎2~3小时(人参、西洋参、羚羊角、鹿茸)。
  - **烊化(溶化)**:胶类(阿胶、鹿角胶、龟板胶、饴糖、蜂蜜)。
  - **泡服(焗服)**:藏红花、番泻叶、胖大海。
  - **冲服**:贵重/量轻/难溶/高温易坏(麝香、牛黄、珍珠、三七、川贝、雷丸、鹤草芽、朱砂;竹沥姜汁等液体药)。
  - **煎汤代水**:灶心土;质轻量大者(玉米须、丝瓜络、金钱草)。
- **服法**:一般温服;解表药偏热服并温覆取汗,寒证热药热服,热证寒药冷服;真热假寒寒药温服,真寒假热热药冷服。病在膈上饭后服,膈下饭前服;补益药空腹,安神药睡前,治疟药发作前2小时。发汗泻下清热药得效即止,不必尽剂。

## 五、道地药材与采集(简)

- **道地药材**:特定产地质优效佳(甘肃当归、宁夏枸杞、青海大黄、内蒙黄芪、东北人参细辛五味子、河南地黄牛膝山药、云南三七茯苓、四川黄连川芎贝母乌头、山东阿胶、浙江贝母)。
- **采集时节**:全草花初开时、叶花蕾盛开时、花采未放花蕾、果实种子多在成熟时(青皮枳实乌梅未熟采)、根及根茎秋末春初、树皮根皮春夏(牡丹皮地骨皮秋后)、动物按活动季(鹿茸清明前后未骨化)。
- **炮制目的**(八项):纯净分级、切制饮片、干燥贮藏、矫味矫臭、**降低毒副作用**、增强疗效、改变药性扩大应用(生地→熟地、生首乌→制首乌、天南星→胆南星)、引药入经(盐炒入肾、醋炒入肝)。方法:修治、水制、火制(炒炙烫煅煨)、水火共制(蒸煮炖潬淬)、其他(制霜、发酵、精制、水飞)。

## 经典补强(v2.0 经典五书反哺)

- **体质与耐药**(灵枢·论痛):"胃厚、色黑、大骨及肥者,皆**胜毒**;其瘦而薄胃者,皆**不胜毒**"——同药同量,瘦弱者更易受药害;剂量因人(体质因素)的经典出处。
- **用药之度**(素问·五常政大论):"大毒治病,十去其六……无毒治病,十去其九;**谷肉果菜,食养尽之**"——中病即止、食养收尾(与 `07` 治则层同源)。
- **服药调护范式之祖**(伤寒论桂枝汤方后注,全文见 [[44-经典-伤寒论六经方证条文]] 第三节):啜热稀粥以助药力、温覆取**微似汗**("不可令如水流漓")、**一服病差停后服不必尽剂**、禁生冷粘滑肉面五辛酒酪臭恶——服法与食忌是方义的一部分,后世一切"将息法"由此出。
