# 经络与体质

> 提炼自《中医基础理论》第五、六章。经络=运行气血、联络脏腑肢节、沟通内外上下、调节人体功能的通路系统。
> 注:十二经逐经循行、腧穴与病候专表已建于 `38-针灸-经络与腧穴总论`(定位方法/特定穴)与 `39-针灸-十四经穴与奇穴`(逐经循行+361 穴),来源为《针灸学》现行教材;本文件保留经络系统纲要与体质部分,循行细节以 `38`/`39` 为准。

## 一、经络系统组成

- **十二经脉(正经)**:直属脏腑、有表里配合。附属:**十二经别**(离入出合,加强表里两经联系、补正经所未及)、**十二经筋**(联缀百骸、主司关节运动)、**十二皮部**(体表反应区)。
- **奇经八脉**:督、任、冲、带、阴跷、阳跷、阴维、阳维——不直属脏腑、无表里配合、除带脉外皆自下而上行、上肢无分布。
- **络脉**:**十五别络**(十二经+任督之络+脾之大络,沟通表里两经)、孙络(最细)、浮络(浅表)。

## 二、十二经脉

**命名**:手/足+三阴(太阴、厥阴、少阴,行内侧属脏)三阳(阳明、少阳、太阳,行外侧属腑)+脏腑名。四肢内侧前中后=太阴、厥阴、少阴;外侧前中后=阳明、少阳、太阳。

**流注次序**(营气循行序,肝终复注于肺,如环无端):
肺(手太阴)→大肠(手阳明)→胃(足阳明)→脾(足太阴)→心(手少阴)→小肠(手太阳)→膀胱(足太阳)→肾(足少阴)→心包(手厥阴)→三焦(手少阳)→胆(足少阳)→肝(足厥阴)→肺。

**走向与交接**:"手之三阴,从胸走手;手之三阳,从手走头;足之三阳,从头走足;足之三阴,从足走腹"。阴经与阳经在**四肢末端**交接(表里经)、同名阳经在**头面**交接("头为诸阳之会")、阴经与阴经在**胸腹**交接。

**分布规律**:头面与四肢——**阳明在前、少阳在侧、太阳在后**;躯干——足三阴与足阳明行胸腹(胸部由内向外:肾经距中线2寸、胃经4寸、脾经6寸),手三阳与足太阳行肩背腰,手三阴、足少阳、足厥阴行腋胁侧腹。

**表里配合**(经脉络属+脏腑相合):肺—大肠、脾—胃、心—小肠、肾—膀胱、心包—三焦、肝—胆。

## 三、奇经八脉

| 脉 | 功能 | 要点 |
|---|---|---|
| **督脉** | **阳脉之海**(总督一身阳经,六阳经会于大椎) | 行背正中;属脑络肾,反映脑肾脊髓功能;主生殖 |
| **任脉** | **阴脉之海**(总任一身阴经) | 行腹正中;起于胞中,**任主胞胎**(调月经、妊养胎儿) |
| **冲脉** | **十二经脉之海、五脏六腑之海、血海**(涵蓄灌注诸经气血) | 起于胞中,挟脐上行;"太冲脉盛,月事以时下" |
| **带脉** | 约束纵行诸脉、固摄胞胎 | 环腰一周,唯一横行 |
| 阴阳跷脉 | 主宰一身左右之阴阳,司下肢矫健与眼睑开合 | 与寤寐相关 |
| 阴阳维脉 | 维络一身表里之阴阳 | 阳维主表、阴维主里 |

督、任、冲皆起于胞中,同出会阴——**"一源三歧"**;奇经与肝肾及女子胞、脑、髓等奇恒之府关系密切。奇经的溢蓄调节:十二经气血有余则蓄于奇经,不足则由奇经溢出补充。

## 四、经络的生理功能与应用

沟通内外上下、联络脏腑肢节(整体性之结构基础);运行气血、濡养全身;感应传导(针感"得气");调节机能平衡。
应用:阐释病理传变(外邪由表入里的途径、脏腑病变反映于体表——"肺心有邪,其气留于两肘");指导诊断(**辨证归经**:循经部位判断病属何经)与治疗(**循经取穴**、药物归经)。

## 五、体质

**概念**:先天禀赋+后天获得所形成的形态结构与功能活动上**相对稳定的固有特征**。注意:体质名词(阴虚质、阳虚质等)≠辨证之证候名——体质反映**非疾病状态下**即存在的个体特异性。

**分类(阴阳分类法)**:

| 类型 | 特征 | 易感与演变 |
|---|---|---|
| **阴阳平和质** | 强壮胖瘦适度、面色明润、性格随和、耐受力强 | 不易感邪少病,易长寿 |
| **偏阳质**(亢奋、偏热、多动) | 形偏瘦而结实、面色偏红、性外向急躁、畏热喜冷、精力旺盛 | 易感**风暑热**邪,受邪多从**阳化热**,病多热证实证、易化燥伤阴;易演化为**阳亢、阴虚、痰火**病理体质 |
| **偏阴质**(不足、偏寒、多静) | 形偏胖而弱、面白欠华、性内向喜静、畏寒喜热、精力偏弱 | 易感**寒湿**邪,受邪多从**阴化寒**,易传里直中;易演化为**阳虚、痰湿、痰饮**病理体质 |

**体质学说的应用**(辨证与治疗的底层变量):
1. **与病因**:"同气相求"——决定易感性(阳虚体易感寒、阴虚体不耐温热、湿盛体易感湿;"肥人多痰湿,善病中风;瘦人多火,易得痨嗽;年老肾衰,多病痰饮咳喘")。
2. **与发病**:体质强弱决定正气虚实、是否发病及发病类型(脾阳素虚者稍进生冷即泻)。
3. **与病机**:决定**从化**(同感风寒,阳热体从阳化热、阴寒体从阴化寒;同感湿邪,或成湿热或成寒湿)与**传变**(伤寒从少阴寒化者素体阳虚,从厥阴热化者素体阴虚)。
4. **与辨证**:体质是"证"形成的物质基础——**同病异证、异病同证**的决定因素主要在体质而不在病因。
5. **与治疗**:
   - **因人论治**:阳虚寒湿之体感湿热,须通阳化湿,药过凉则湿闭阳更虚;阴虚之体湿从热化,治法迥异。"阳盛阴虚之体慎温热,阳虚阴盛之体慎寒凉"。
   - **用药宜忌**:阴虚宜甘寒酸寒咸寒清润、忌辛热苦寒沉降;阳虚宜益火温补、忌苦寒泄火;体壮量宜大、体弱量宜小。
   - **善后调理**:视体质定饮食宜忌(燥红质热病初愈慎辛温之品;腻滞质病愈慎滋腻酸涩)。
   - 兼顾年龄(小儿稚阴稚阳中病即止、老人审体质保真气慎劫夺——叶天士)、性别(女子以肝为先天,注意肝郁血虚)、生活条件(膏粱厚味多痰湿湿热)。
