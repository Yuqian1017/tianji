# 病机

> 提炼自《中医基础理论》第八章。病机=疾病发生、发展、变化的机理("病之机要")。"谨守病机,各司其属"——病机十九条出《素问·至真要大论》(教材未列全文,待经典轮读《素问》时补入专表)。
> 基本病机四纲:**邪正盛衰、阴阳失调、气血失常、津液失常**;再加内生五气与传变转归。脏腑病机细目由诊断轮"脏腑辨证"承载。

## 一、发病机理

- **邪正斗争决定发病**:"正气存内,邪不可干""邪之所凑,其气必虚"。正气不足是发病的内在根据(决定是否发病、发病部位——何处正虚邪即客之、病情轻重);邪气是重要条件,特定情况下起决定作用(疠气、高温电击毒伤等——故须"避其毒气")。
- **影响因素**:外环境(季节气候——"四时之气,更伤五脏";地理;工作生活;社会);内环境——**体质**(决定易感性与**从化**:阳盛之体从阳化热、阴盛之体从阴化寒;"所受之邪,每从其人之脏气而化")、精神状态(兴奋态多致实证、抑郁态易致虚证)、遗传(先天禀赋,病机为肾之精气阴阳亏虚)。
- **发病类型**:卒发(感邪甚/情志遽变/疫气/毒伤/外伤)、**伏发**(伏而后发:破伤风、狂犬病、伏气温病——初起即见里热无表证)、徐发(寒湿粘滞、年高体弱、内伤积渐)、继发(以原发病为前提:肝病→症积臌胀、疟母、小儿疳积)、**合病**(两经三经同时受邪)与**并病**(一经未罢又传一经,有先后次第)、**复发**(条件=余邪未尽+正虚未复+诱因;类型:少愈即复、休止与复发交替[哮喘宿根]、急性发作与缓解交替[胆石])。

## 二、邪正盛衰(虚实)

**"邪气盛则实,精气夺则虚"**——虚实是邪正力量对比的概括。

- **实**:邪盛而正未衰,多见于外感初中期、痰食水血滞留。例:阳明经证(**大热、大汗、大渴、脉洪大**"四大")、阳明腑证(潮热谵语、腹满硬痛拒按、便秘、苔黄燥、脉沉数有力)。
- **虚**:正气不足、脏腑机能衰退,多见于病后期与慢性病。例:脾不统血之崩漏(出血+面白神疲心悸气短舌淡脉细)。
- **虚实错杂**:虚中夹实(脾阳不振之水肿——虚为主)、实中夹虚(实热伤津——实为主)。辨主次缓急。
- **虚实转化**:由实转虚(失治误治、正气日损);**因虚致实**(正虚运化无力→气滞瘀血痰饮水湿停积,如肾阳虚水泛——实为虚实错杂)。
- **虚实真假**(现象与本质不一致):
  - **真虚假实**("至虚有盛候"):腹满但时减、腹痛但**喜按**,兼纳呆神疲舌胖嫩脉虚。
  - **真实假虚**("大实有羸状"):热结肠胃反见精神萎靡、肢体倦怠,但语声高亢气粗、痛而**拒按**、得泄反快。

## 三、阴阳失调

| 病机 | 机制 | 特征 |
|---|---|---|
| 阳盛则热 | 阳邪亢盛(实热),"阳盛则阴病" | 热象为主,虚象不显;须辨阴是相对不足还是已成阴虚 |
| 阴盛则寒 | 阴邪内盛(实寒),"阴盛则阳病" | 寒象为主;实际多伴不同程度阳伤 |
| 阳虚则寒 | 阳气虚损失温煦(虚寒),脾肾为主**尤以肾阳为关键** | **虚而有寒**:畏寒肢冷+喜静蜷卧、小便清长、下利清谷 |
| 阴虚则热 | 阴液亏耗虚性亢奋(虚热),肝肾为主**尤以肾阴为本** | **虚而有热**:五心烦热、骨蒸潮热、颧红盗汗、舌红少苔脉细数 |
| 阴阳互损 | 阴损及阳、阳损及阴→阴阳两虚 | 多在损及肾之阴阳时发生 |
| **阴阳格拒** | 阴盛格阳(**真寒假热**:四肢厥逆下利清谷脉微欲绝+身反不恶寒面颊泛红;格阳=假热于体表,**戴阳**=假热于头面"面赤如妆");阳盛格阴(**真热假寒**:热厥,"热深厥亦深") | 危重阶段的假象辨识关键 |
| 阴阳转化 | 重阳必阴、重阴必阳(皆为真转化,区别于格拒之假象) | 高热突转肢厥脉微=由阳转阴 |
| **阴阳亡失** | **亡阳**:大汗淋漓(汗清稀而凉)、手足逆冷、神情淡漠、脉微欲绝;**亡阴**:汗热而粘、渴喜冷饮、身干瘪眶陷、烦躁、脉细数疾无力 | 亡阴可迅速导致亡阳,终至"阴阳离决,精气乃绝" |

## 四、气血失常

**气失调**:
- **气虚**(少气懒言、疲倦乏力、脉细软无力;多责脾肺)——与阳虚之别:气虚是虚而**无寒象**,阳虚是虚而有寒象。
- **气陷**(气虚升举无力:内脏下垂、脱肛、崩漏带下,兼腰腹坠胀)。
- **气脱**(气虚之极:虚脱缓、暴脱急;阳气暴脱冷汗如珠肢厥)。
- **气滞**(闷胀、疼痛为特点;与肝肺脾胃关系密切,尤以肝郁为先;气滞则血瘀、水停)。
- **气逆**(肺逆咳喘、胃逆呕呃嗳气、肝逆头痛眩晕面红易怒,甚则**血随气逆**咯吐血昏厥;多实,亦有因虚而逆)。
- **气闭**(浊邪闭窍:突然昏厥;心闭神昏最重)。

**血失调**:
- **血虚**(眩晕、面唇舌甲淡白无华;责心肝脾肾)——与阴虚之别:血虚是虚而无热象,阴虚有热象。
- **血瘀**(见 05-病因·瘀血);与气虚气滞血寒血热互为因果。
- **血热**(热象+耗血动血伤阴:出血、心烦、舌绛)。
- **血寒**(肢体冷痛麻木、得温则减、月经不调)。
- **出血**(火气上逆/热迫血行/气不摄血/瘀血阻络/外伤)。

**气血同病**:气滞血瘀、气虚血瘀(重者肢体瘫软不用、肌肤甲错)、气不摄血(下部出血为多)、**气随血脱**(大出血时)、气血两虚(肌肤干燥、肢体麻木)。

## 五、津液失常

- **津液不足**:伤津(轻,清稀易补——汗吐泻后口渴、目陷螺瘪转筋)vs 伤阴/脱液(重,稠厚难复——舌光红无苔、唇舌干而不引饮、形瘦肉脱、手足蠕动)。"伤津并不一定兼伤阴,伤阴则必兼伤津"。津亏→**津枯血燥**(虚热内生、血燥生风)、**津亏血瘀**(舌紫绛瘀斑)。
- **水湿停聚**:输布障碍(关键在**脾运**)与排泄障碍(汗责肺、尿责**肾**)→湿浊困阻、痰饮凝聚、水液潴留(水肿、腹水)。**水停气阻**(水饮阻肺则咳喘不得卧、凌心则心悸心痛、停中焦则脘胀纳呆、停四肢则沉重胀痛);**气随液脱**(大汗剧吐泻)。

## 六、内生五气(脏腑失调的病理结果,非外来病因)

| 内邪 | 核心病机 | 类型与特征 |
|---|---|---|
| **风气内动**(肝风) | 体内阳气亢逆变动——眩晕、肢麻、震颤、抽搐 | **热极生风**(实:高热神昏抽搐角弓反张);**肝阳化风**(水不涵木、上盛下虚:眩晕欲仆、口眼喎斜、半身不遂、卒然仆倒——内伤杂病);**阴虚风动**(温病后期真阴亏:手足蠕动、瘛疭、神倦舌绛);**血虚生风**(麻木不仁、筋肉跳动、拘挛不伸);血燥生风(皮肤干燥甲错瘙痒落屑) |
| **寒从中生**(内寒) | 阳虚温煦气化失职,责脾肾**尤肾阳** | 特点:**冷、白、稀、润、静**;温煦失职(形寒肢冷)+水液不化(尿清痰稀便溏水肿)。与外寒之别:内寒虚而有寒、虚象更著;外寒以寒为主 |
| **湿浊内生**(内湿) | **脾虚生湿**(脾失健运为关键,与肾阳相关) | 湿犯上焦胸闷咳,中焦脘痞纳呆口腻苔厚腻,下焦便溏尿不利;泛溢肌肤为肿。可寒化热化 |
| **津伤化燥**(内燥) | 津液不足,失濡而"干" | 多见肺胃肾大肠:肌肤干糙脱屑、口燥咽干、舌光红龟裂、爪脆发枯、便秘尿短 |
| **火热内生**(内火) | 阳盛/邪郁化火/**五志化火**/阴虚火旺 | **实火**:病急程短,壮热面赤口渴喜冷、便秘尿赤、狂躁舌红苔黄燥脉洪数("气有余便是火");**虚火**:病缓程长,五心烦热、午后颧红、盗汗咽干、舌红少苔脉细数(火象常集中一处:虚火牙痛咽痛)。共同特点:**热、赤、稠、燥、动** |

## 七、疾病的传变

传变=**病位传变**+**病性转化**。外感病基本形式是表里传变,内伤病基本形式是脏腑传变。意义:见微知著,把握病势,早期截断。

### 表里出入
- 表邪入里(正不胜邪,病进):皮毛→络脉→经脉→脏腑。
- 里病出表(正胜邪却,向愈):汗出热解、疹透于外。
- **半表半里**:邪入少阳,枢机不利——往来寒热、胸胁苦满、口苦咽干、目眩、默默不欲饮食、心烦喜呕。

### 外感病传变(一般规律;条文细则待经典轮/温病轮)
- **六经传变**:太阳→阳明→少阳→太阴→少阴→厥阴(由表入里、由阳入阴、由轻而重)。三阳以热证实证为主(太阳:发热恶寒头项强痛脉浮;阳明:但热不寒、腹满硬痛便秘谵语;少阳:半表半里证);三阴正衰(太阴:腹满吐利食不下、脉缓弱;少阴:"脉微细,但欲寐",分**寒化**[四肢厥逆下利清谷]/**热化**[心烦不得卧];厥阴:寒热错杂,消渴、气上撞心、心中疼热、饥不欲食、食则吐蛔)。特殊:越经传、表里传、**直中**(不经三阳径现三阴证,素体阳虚者多直中太阴)、合病、并病。
- **卫气营血传变**:**顺传**卫→气→营→血(卫分:发热恶寒,病在肺卫;气分:但热不恶寒;营分:舌红绛、心烦不寐,病在心与心包;血分:耗血动血、伤阴动风,最重)。**逆传**:肺卫之邪径陷心包,病势凶险。变式:卫气同病、气血两燔、直入营血、湿温传变无定。
- **三焦传变**:上焦手太阴肺始→中焦脾胃(顺)→下焦肝肾;肺卫径传心包为逆传。

### 内伤杂病传变
- 经络之间(肝火循经犯肺=木火刑金)、经络与脏腑之间(皮毛→孙脉→络脉→经脉→内连脏腑)。
- **五脏生克传变**:《难经》五邪——虚邪(从后/生我者来=母病及子)、实邪(从前/我生者来=子病及母)、贼邪(从所不胜来=相乘)、微邪(从所胜来=相侮)、正邪(本脏自病)。
  - **顺传**(病虽进而易退):母病及子(水不涵木)、相乘("见肝之病,知肝传脾")。
  - **逆传**(病重难复):子盗母气(久咳损脾,"脾胃一虚,肺气先绝")、相侮(水寒侮土之五更泄/脾肾泄)。
- 脏腑表里:由腑及脏病较重,由脏及腑病较轻。

### 病性转化
- 寒热转化:**由寒化热**(表寒证转阳明里热;哮喘痰稀白转黄稠——多为正气来复,较易治);**由热转寒**(便血鲜红转紫暗兼畏寒肢冷——正气损伤,多难愈)。
- 虚实转化:由实转虚、因虚致实(见上)。

### 影响传变的因素
体质(强者传变少;决定从化方向)、病邪性质(伤寒循六经,温病循卫气营血/三焦)、地域气候、生活状况(情志饮食劳逸房事)、**治疗当否**(误治致变证坏证:风寒误下→协热下利;风温误汗→斑疹神昏)。

## 八、疾病的转归

- **痊愈**(正胜邪退)与**死亡**(邪胜正衰、阴阳离决;中医以**亡神**为判断死亡的重要标志——"得神者昌,失神者亡")。
- **缠绵**:邪正相持、正虚邪恋→慢性迁延(是疾病的延续)。
- **后遗**:病理过程已终结,遗留形态功能异常(偏瘫失语痴呆震颤;伤残)——是疾病的结局。
- **复发**:诱因有**食复**(病后脾胃尚虚,纳谷太骤,"轻者损谷自愈,重者消导方瘥")、**劳复**(劳力/劳神/**房劳复**最重)、**情志复**(怒复)、**重感复**、自复。防复发:除邪务尽+病后调摄(饮食渐进、充分休息、节欲、调情志)。
