## 二、养生与治未病(素问开卷即养生)

- **总纲**(上古天真论):"**法于阴阳,和于术数,食饮有节,起居有常,不妄作劳**,故能形与神俱,而尽终其天年,度百岁乃去";反面即今人写照——"以酒为浆,以妄为常,醉以入房……故半百而衰"。精神养生:"**恬惔虚无,真气从之,精神内守,病安从来**"。
- **生命节律**(上古天真论):女子以七为节(二七天癸至→七七天癸竭),男子以八为节(二八精气溢泻→八八齿发去);"肾者主水,受五藏六府之精而藏之"——肾精为生长衰老之本。
- **四季起居**(四气调神大论):春三月夜卧早起、广步于庭、使志生;夏三月夜卧早起、无厌于日、使志无怒;秋三月早卧早起、与鸡俱兴、使志安宁;冬三月**早卧晚起、必待日光**、去寒就温。**"春夏养阳,秋冬养阴"**。逆之则"逆春气则伤肝,夏为寒变"等按季递损。
- **治未病名句**(四气调神大论):"**圣人不治已病治未病**,不治已乱治未乱……夫病已成而后药之,譬犹渴而穿井,斗而铸锥,不亦晚乎!"
- **阳气论**(生气通天论):"**阳气者,若天与日,失其所则折寿而不彰**";阳气一日节律"平旦人气生,日中而阳气隆,日西而阳气已虚"——故"暮而收拒,无扰筋骨,无见雾露"(晚间不宜剧烈运动之经典依据);烦劳则张为煎厥、大怒形气绝为薄厥(情志红线);"**阴平阳秘,精神乃治;阴阳离决,精气乃绝**"。
- **五味与饮食**(生气通天论、藏气法时论、五藏生成篇):"阴之所生,本在五味;阴之五宫,伤在五味"——味过于酸/咸/甘/苦/辛各有所伤;"**谨和五味,骨正筋柔……长有天命**"。配膳总纲:"**五谷为养,五果为助,五畜为益,五菜为充**,气味合而服之,以补精益气"(药攻食养分工的源头,与 [[23-病证用药索引]] 药食同源思想互参)。五味所禁(宣明五气):辛走气、咸走血、苦走骨、甘走肉、酸走筋,病各无多食。
- **五劳所伤**(宣明五气):"久视伤血,久卧伤气,久坐伤肉,久立伤骨,久行伤筋"(久坐办公者的经典处方即"起而行之")。
- **四时伏邪链**(生气通天/阴阳应象):"冬伤于寒,春必温病;春伤于风,夏生飧泄;夏伤于暑,秋必痎疟;秋伤于湿,冬生咳嗽"。
- **因地制宜**(异法方宜论):东砭石、西毒药、北灸焫、南微针、中导引按蹻,"圣人杂合以治,各得其所宜"——多手段并用、按条件选法,正是资源受限适配思路的经典源头。
- **神为治本**(汤液醪醴论):"精神不进,志意不治,故病不可愈"——患者的求治意愿与信心是疗效前提;"病为本,工为标"。水肿治则名句"**开鬼门,洁净府**"(发汗、利小便,[[35-内科-肾系病证]] 水肿治法之祖)。

## 三、阴阳五行与藏象的经典源头(关键句备查)

- 阴阳总纲(阴阳应象大论):"**阴阳者,天地之道也……治病必求于本**";"阳化气,阴成形";"重寒则热,重热则寒";"**壮火食气,少火生气**"(温补不可过之理);"气味辛甘发散为阳,酸苦涌泄为阴";"风胜则动,热胜则肿,燥胜则干,寒胜则浮,湿胜则濡泻"(六淫致病速记,配 [[05-病因]]);"**善诊者,察色按脉,先别阴阳**"。
- 治法层次(阴阳应象):"其高者,因而越之;其下者,引而竭之;中满者,泻之于内;其在皮者,汗而发之";"形不足者,温之以气;精不足者,补之以味";"阳病治阴,阴病治阳"。"善治者治皮毛"——病浅早治。
- 十二官(灵兰秘典论):"心者,君主之官,神明出焉;肺者,相傅之官;肝者,将军之官;胆者,中正之官;膻中者,臣使之官;脾胃者,仓廪之官;大肠传道、小肠受盛;肾者,作强之官;三焦者,决渎之官;膀胱者,州都之官,气化则能出矣"。"主明则下安,以此养生则寿"。
- 藏象定义(六节藏象论):心为"生之本,神之变",肺为"气之本,魄之处",肾为"**封藏之本,精之处**",肝为"罢极之本,魂之居",脾胃大小肠三焦膀胱为"仓廪之本"。"凡十一藏,取决于胆也"。
- 藏泻之分(五藏别论):"五藏者,藏精气而不泻也,故满而不能实;六府者,传化物而不藏,故实而不能满"。"气口何以独为五藏主——五藏六府之气味皆出于胃,变见于气口"(独取寸口之理论根基,与 [[10-问诊切诊]] 互参)。医患条件:"拘于鬼神者,不可与言至德;**病不许治者,病必不治,治之无功矣**"。
- 气血劳逸(经脉别论):"**生病起于过用**"(过劳致病总纲);"食气入胃,散精于肝……饮入于胃,游溢精气,上输于脾,脾气散精,上归于肺,通调水道,下输膀胱"(水谷代谢全链,[[04-精气血津液]] 之源)。

## 四、诊法源头(关键句备查)

- 诊病时机与总纲(脉要精微论):"诊法常以平旦"(今义:安静休息态下诊察最准);"切脉动静而视精明,察五色,观五藏有余不足……以此参伍,决死生之分"。
- 望神败象(脉要精微):"头者,精明之府,头倾视深,精神将夺矣;背者,胸中之府,背曲肩随,府将坏矣;腰者,肾之府,转摇不能,肾将惫矣;膝者,筋之府,屈伸不能,行则偻附,筋将惫矣"——**得强则生,失强则死**(老人体态即脏腑盛衰之镜,家庭观察可用,配 [[09-望诊闻诊]])。"衣被不敛,言语善恶不避亲疏者,此神明之乱也"。
- 胃气为本(平人气象论):"平人之常气禀于胃,**人无胃气曰逆,逆者死**";"人以水谷为本,故人绝水谷则死"——食欲存亡是判断病势的家庭级指标。"妇人手少阴脉动甚者,妊子也"(存目)。
- 虚实死候(玉机真藏论):"**五实死,五虚死**"——五实:脉盛、皮热、腹胀、前后不通、闷瞀;五虚:脉细、皮寒、气少、泄利前后、饮食不入。**转机**:"浆粥入胃,泄注止,则虚者活;身汗得后利,则实者活"(能进食、二便通=生机,危重判断经典依据,与 [[31-内科-肺系病证]]~[[37-内科-肢体经络病证]] 各危重指征互参)。"邪气盛则实,精气夺则虚"(通评虚实论,虚实辨证总定义,[[11-八纲辨证]] 之源)。
- 问诊人事(疏五过论、徵四失论):"必问尝贵后贱""必问饮食居处,暴乐暴苦"——境遇剧变致病(脱营、失精);四失之一即"诊病不问其始,忧患饮食之失节,卒持寸口,何病能中"(先问病史再谈脉证,问诊优先原则的经典出处,配 [[10-问诊切诊]] 十问)。
- 梦象与脏气(方盛衰论、脉要精微):肺虚梦白物兵战、肾虚梦舟船溺人、肝虚梦菌香生草、心虚梦救火、脾虚梦饮食不足(参考性意象,不作诊断依据)。

## 六、治则治法总汇(至真要大论为主,★本文件核心)

### 病机十九条(至真要大论,全录)

> "审察病机,无失气宜……谨守病机,各司其属,有者求之,无者求之,盛者责之,虚者责之。"

| 归属 | 条文 |
|---|---|
| 五脏 | 诸**风掉眩**,皆属于**肝**;诸**寒收引**,皆属于**肾**;诸**气膹郁**,皆属于**肺**;诸**湿肿满**,皆属于**脾**;诸**痛痒疮**,皆属于**心** |
| 上下 | 诸**厥固泄**,皆属于**下**;诸**痿喘呕**,皆属于**上** |
| 火(五条) | 诸热瞀瘛;诸禁鼓栗,如丧神守;诸逆冲上;诸躁狂越;诸病胕肿,疼酸惊骇——皆属于**火** |
| 热(四条) | 诸胀腹大;诸病有声,鼓之如鼓;诸转反戾,水液浑浊;诸呕吐酸,暴注下迫——皆属于**热** |
| 寒 | 诸病水液,**澄澈清冷**,皆属于**寒**(分泌物排泄物清稀=寒、浑浊=热,家庭辨寒热利器,配 [[11-八纲辨证]]) |
| 风/湿 | 诸**暴强直**,皆属于**风**;诸**痉项强**,皆属于**湿** |

### 治则名句(备查引用)

- 正治:"**寒者热之,热者寒之**,微者逆之";反治:"甚者从之——**热因寒用,寒因热用,塞因塞用,通因通用**,必伏其所主,而先其所因"(至真要)。
- 治法长卷(至真要):"坚者削之,客者除之,**劳者温之**,结者散之,留者攻之,燥者濡之,急者缓之,散者收之,损者温之,惊者平之,上之下之,摩之浴之……适事为故"。"疏其血气,令其调达,而致和平"。
- 求属之治(至真要):"**诸寒之而热者取之阴,热之而寒者取之阳**"(王冰注即"壮水之主以制阳光,益火之源以消阴翳",阴虚发热不可再清、阳虚生寒不可再散之理)。
- 君臣佐使(至真要):"**主病之谓君,佐君之谓臣,应臣之谓使**"(方剂结构之祖,[[24-方剂总论]] 源头);奇偶大小方:"君一臣二,奇之制也……近者奇之,远者偶之"。
- 标本(至真要/标本病传65):"**急则治标,缓则治本**"之经典表述——"小大不利治其标,小大利治其本"(二便不通先通之);"间者并行,甚者独行"。"知标与本,用之不殆"。
- 用药度(五常政大论70):"**大毒治病,十去其六;常毒治病,十去其七;小毒治病,十去其八;无毒治病,十去其九。谷肉果菜,食养尽之,无使过之,伤其正也**"——中病即止、食养收尾,本 Skill"能食养不用药"层级的经典依据(配 [[安全-配伍妊娠禁忌与毒性药]])。"无盛盛,无虚虚,而遗人夭殃"。
- 孕妇用药(六元正纪大论71):"妇人重身,毒之何如?**有故无殒,亦无殒也**……衰其太半而止,过者死"——有是病用是药,但须中病即减;实际操作仍以 [[安全-配伍妊娠禁忌与毒性药]] 妊娠禁忌表为准,孕期用药一律 B 级。
- 五郁治法(六元正纪71):"**木郁达之,火郁发之,土郁夺之,金郁泄之,水郁折之**"([[36-内科-气血津液病证]] 郁证治法之源)。
- 制衡观(六微旨大论68):"**亢则害,承乃制,制则生化**"。
- 藏气法时论22:五脏苦欲补泻(肝苦急急食甘以缓之、心苦缓急食酸以收之、脾苦湿急食苦以燥之、肺苦气上逆急食苦以泄之、肾苦燥急食辛以润之)——遣药配伍的经典依据之一。

## 七、刺法与运气(存目)

- **刺法篇群存目**:宝命全形25(凡刺之真,必先治神)、八正神明26、离合真邪27、刺要50/刺齐51/刺志53/针解54/长刺节55、缪刺论63、四时刺逆从64、气穴58/气府59(365 穴统计)、骨空论60(灸寒热法及任督冲循行——任督冲病候已入 [[14-六经卫气营血三焦经络辨证]] 奇经部分)、水热穴61、调经论62(有余泻之不足补之/神气血形志五虚五实)。取穴定位主治以 [[38-针灸-经络与腧穴总论]] [[39-针灸-十四经穴与奇穴]] 为准。
