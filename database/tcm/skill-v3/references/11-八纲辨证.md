# 八纲辨证

> 提炼自《中医诊断学》第三章。八纲=阴阳、表里、寒热、虚实,是**各种辨证的总纲**(脏腑、气血津液、卫气营血辨证皆在此基础上分支)。阴阳为总纲("二纲六要"):表、热、实属阳;里、寒、虚属阴。辨证须注意八纲间的**相兼、错杂、转化、真假**。

## 一、表里(病位与病势深浅)

| | 表证 | 里证 |
|---|---|---|
| 成因 | 六淫疫疠经皮毛口鼻入侵,外感初期 | 表邪内传/外邪直中脏腑/内伤(七情饮食劳逸) |
| 特征 | **恶寒发热同见**、头身痛、苔薄白(无变化)、**脉浮**;兼鼻塞流涕咳嗽咽痒 | **但热不寒或但寒不热**、脏腑症状(腹痛呕恶、二便异常)、**苔厚、脉沉** |
| 轻重 | 病浅而轻 | 病深而重 |

- **半表半里证**:寒热往来、胸胁苦满、心烦喜呕、默默不欲饮食、口苦咽干目眩、脉弦(≈少阳病证)。
- 表里同病(新感加旧病、外感兼伤食等);**表邪入里=病进**(恶寒罢、反恶热、渴饮舌红苔黄尿赤),**里邪出表=病退**(内热烦躁转发热汗出、斑疹白㾦外透)。

## 二、寒热(病性)

| | 寒证 | 热证 |
|---|---|---|
| 共性表现 | 恶寒喜暖、面白、肢冷蜷卧、**口淡不渴**、痰涕清稀、**尿清长便稀溏**、舌淡苔白润、**脉迟或紧** | 恶热喜冷、**口渴喜冷饮**、面红目赤、烦躁、痰涕黄稠、吐衄、**尿短赤便干结**、舌红苔黄干、**脉数** |

鉴别眼目:寒热喜恶、渴与不渴、面色赤白、四肢温凉、二便、舌象、脉象。

**细分四证**:
- 表寒:恶寒重发热轻、无汗、脉浮紧;表热:发热重恶寒轻、微渴或有汗、舌边尖红、脉浮数。
- 里寒(虚寒):形寒肢冷、口淡渴喜热饮、静而少言、尿清便溏、脉沉迟;里热(实热):面红身热、渴喜冷饮、烦躁多言、尿赤便结、脉数。
- 实寒(寒实):畏寒喜暖、腹痛**拒按**、肠鸣腹泻或痰鸣喘嗽、苔白润脉迟紧;虚寒(阳虚):畏寒肢冷、腹痛**喜温喜按**、便溏尿清、少气乏力、舌淡嫩、脉沉迟**无力**。
- 实热:壮热、腹胀满痛拒按、便秘尿赤、神昏谵语、舌红苔黄干、脉洪滑数实;虚热(阴虚):颧红消瘦、**潮热盗汗五心烦热**、咽干、舌红少苔、脉细数。

**寒热错杂**:上寒下热(胃脘冷痛呕清涎+尿频尿痛)、上热下寒(胸中烦热咽痛+腹痛喜暖便溏)、表寒里热(恶寒无汗+烦躁口渴脉浮紧)、表热里寒(发热咽痛+便溏尿清肢凉)。治时分寒热多少定主次。

**寒热转化**:寒化热=正气尚盛(表寒证转壮热心烦苔黄);热转寒=邪盛正虚(热痢日久转虚寒痢;高热大汗吐泻后骤见肢厥脉微=亡阳,急转)。

**寒热真假**(危重关头,细察勿误):
- **真寒假热**(阴盛格阳):身热但**反欲盖衣被**、面红如妆**时隐时显**、渴欲**热饮而不多**、脉大**按之无力**;兼四肢厥冷下利清谷尿清舌淡。
- **真热假寒**(阳盛格阴,热厥):手足冷、脉沉,但**身热不恶寒反恶热**、脉沉数**有力**、烦渴冷饮口臭谵语尿赤便结舌红苔黄干;"热深厥亦深"、**胸腹灼手**。
- 辨要:假象多现于**四肢、皮肤、面色**;以**里证、舌象、脉象(按之有力无力)**为本质依据。

## 三、虚实(邪正盛衰)

| | 虚证 | 实证 |
|---|---|---|
| 表现 | 阳虚:面淡白、形寒肢冷、神疲气短、自汗、便滑尿失禁、舌淡胖嫩、脉虚沉迟;阴虚:五心烦热、消瘦颧红、盗汗潮热、舌红少苔、脉细数 | 发热、腹胀痛**拒按**、胸闷烦躁甚神昏谵语、呼吸气粗痰盛、便秘或下利里急后重、尿涩痛、**舌苍老苔厚腻、脉实有力** |
| 鉴别 | 久病、体弱、声息低微、舌淡嫩、脉无力 | 暴病、体壮、声高息粗、舌苍老、脉有力 |

**虚实错杂**(攻补兼施,分主次):实证夹虚(白虎加人参汤例:阳明热盛兼气阴伤)、虚证夹实(温病晚期肾阴亏兼余热)、虚实并重(小儿疳积:食积+脾虚)。

**虚实转化**:由实转虚(实热日久津气耗伤);因虚致实(心脾气虚→心脉瘀阻猝痛)。

**虚实真假**:
- 真实假虚("大实有羸状"):神情沉静但**语出声高气粗**、脉沉伏迟涩但**按之有力**、形寒肢冷但**胸腹久按灼手**——治仍攻邪。
- 真虚假实("至虚有盛候"):腹胀满**时减**、腹痛**喜按**、脉弦**重按无力**——治用补法。
- 鉴别四要点:①脉之有力无力有神无神(浮沉候对比);②舌质胖嫩与苍老;③言语发声亢亮与低怯;④体质、病因、病之新久与治疗经过。

## 四、阴阳(总纲)

- **阴证**=里+寒+虚:面暗淡、萎靡蜷卧、声低少言、口淡不渴、便溏尿清、腹痛**喜按**、身寒足冷、舌淡胖嫩苔润、脉沉迟微细弱无力。
- **阳证**=表+热+实:面红、灼热烦躁、声粗多言甚骂詈、口渴引饮、便秘奇臭尿短赤、腹痛**拒按**、身热足暖、舌红绛苔黄黑生芒刺、脉浮洪数大滑实有力。

**真阴不足**(肾阴虚):颧赤唇红、口燥咽干、五心烦热、头晕耳鸣、腰腿酸软、骨蒸盗汗、梦遗、便结尿少、舌红干少苔、脉细数无力。
**真阳不足**(肾阳虚):面㿠白、形寒肢冷、口淡多涎、喘咳身肿、腹大胫肿、便溏或**五更泄**、阳痿精冷宫寒不孕、舌淡胖苔白滑、脉沉迟无力。

**亡阴与亡阳鉴别**(危候,救治刻不容缓):

| | 亡阴 | 亡阳 |
|---|---|---|
| 汗 | **汗热、味咸、不粘** | **汗凉、味淡、微粘** |
| 四肢 | 温和 | 厥冷 |
| 舌 | 红干 | 淡白润 |
| 脉 | 细数无力 | 微细欲绝 |
| 其他 | 身热烦躁、口渴喜冷饮、肌肤皱瘪尿极少 | 身冷蜷卧神疲、口淡喜热饮 |

亡阴可导致亡阳,亡阳亦耗阴液;临床分主次及时救治。

## 经典补强(v2.0 经典五书反哺;详 [[43-经典-灵枢与难经要义]] 第十二节、[[44-经典-伤寒论六经方证条文]] 第二节)

- **真假寒热的经典试金石**(伤寒论·太阳上):"病人**身大热,反欲得近衣**者,热在皮肤,寒在骨髓也;**身大寒,反不欲近衣**者,寒在皮肤,热在骨髓也"——盖被取向是家人可直接观察的真假辨。
- **脏病腑病观察三条**(难经五十一/五十二难):病欲得寒、欲见人=病在**腑**(阳);病欲得温、不欲见人、恶闻人声=病在**脏**(阴);痛处**止而不移**=脏病,**上下行流居处无常**=腑病。
- **虚实之诊**(难经四十八难):"**痒者为虚,痛者为实**;出者为虚,入者为实;言者为虚,不言者为实;缓者为虚,急者为实"。总定义:"**邪气盛则实,精气夺则虚**"(素问·通评虚实论)。
