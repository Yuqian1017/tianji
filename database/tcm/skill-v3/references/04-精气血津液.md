# 精、气、血、津液

> 提炼自《中医基础理论》第四章。生命的基本物质,皆本源于一气("人有精、气、津、液、血、脉,余意以为一气耳"——《灵枢·决气》)。
> 总纲:气属阳主动,宜运行不息而不宜郁滞;精血津液属阴主静,宜宁谧秘藏而不宜妄泄。**"气为主,血为辅;气为重,血为轻"**,"治病以气为首务"——气有不调之处,即病本所在之处(《景岳全书》)。

## 一、精

- 含义分层:①广义——构成人体、维持生命的一切精微物质(含精血津液);②狭义——禀受父母、藏于肾的**生殖之精**;③又可指脏腑之精、正气("精气夺则虚")。
- 来源:**先天之精**(父母,构成胚胎的原始物质)+**后天之精**(水谷,脾胃化生灌溉五脏六腑,盈者藏肾)。"先天生后天,后天养先天","精藏于肾而非生于肾"——譬诸钱粮,储于库(肾)而源于化(脾胃)。
- 功能与临床锚点:
  - **繁衍生殖**(精盈天癸至)→ 不育不孕从补肾填精论治
  - **生长发育**(生长壮老已随肾精盛衰)→ 五迟五软、早衰从补肾论治
  - **生髓化血**(精生髓、髓化血,脑为髓海)→ 血虚可补肾益髓(血肉有情之品);老年痴呆从补肾益髓论治
  - **濡润脏腑**→"久病必穷肾",疾病末期补肾之阴精

## 二、气

**生成**:先天精气(肾)+水谷精气(脾胃)+自然清气(肺)。三脏定位:**肺为气之主、脾胃为气血生化之源、肾为生气之源**(论呼吸则"肺为气之主,肾为气之根")。临证补气主要补脾肺;气虚之极从补肾入手。

**六大功能及失常**:

| 功能 | 内容 | 失常表现 |
|---|---|---|
| 推动 | 激发生长发育、脏腑经络活动、血行津布 | 生长迟缓早衰、血瘀水停 |
| 温煦 | "气主煦之",阳气为体温与代谢之源("少火生气") | 畏寒肢冷、血津运行迟缓(气虚为阳虚之渐,阳虚为气虚之极) |
| 防御 | 护卫肌表、正邪交争驱邪、自我修复 | 易感冒(玉屏风散益气固表)、病难愈 |
| 固摄 | 摄血、摄津(汗尿唾胃肠液)、固精、固脏腑之气 | 出血、自汗多尿、遗精滑泄、脏器下垂;轻为散泄重为脱 |
| 营养 | 水谷精气化气血,卫气温养肌腠,营气化血灌溉 | 形瘦神疲 |
| 气化 | 精气血津液的生成、转化、代谢("味归形,形归气,气归精,精归化") | 消化吸收、汗尿排泄等全程障碍 |

**气机(升降出入)**:"出入废则神机化灭,升降息则气立孤危"。脏腑规律:心肺在上宜降,肝肾在下宜升,**脾胃居中为升降之枢**;肝从左升、肺从右降为气机之翼;六腑以降为顺而降中寓升。

**分类**:

| 气 | 来源 | 分布 | 功能与候察 |
|---|---|---|---|
| **元气**(原气/真气) | 肾中精气化生,赖后天充养("非先天之气不能化,非后天之气不能生") | 发于肾间命门,**经三焦**循行全身 | 生命活动的原动力;推动生长发育、温煦激发五脏六腑("五脏之阴气非此不能滋,五脏之阳气非此不能发") |
| **宗气**(大气) | 肺吸清气+脾升水谷精气,**合营卫而成** | 积于胸中膻中(上气海),上走息道、下蓄丹田、注气街 | **走息道司呼吸**(言语声音呼吸强弱系之)、**贯心脉行气血**(调心率心律)——以**虚里**(左乳下)搏动与脉象候之;"宗气不下,脉中之血,凝而留止" |
| **营气**(营阴) | 水谷精气之**精专**部分 | 行**脉中**,十二经+任督十四经流注,昼夜五十周 | 化生血液、营养全身("泌其津液,注之于脉,化以为血") |
| **卫气**(卫阳) | 水谷之**悍气**,慓疾滑利 | 行**脉外**,昼行阳二十五周、夜行阴二十五周 | 护卫肌表御邪、温养脏腑肌肉皮毛、司腠理开阖调汗。卫气不足→自汗易感;循行异常→寤寐异常(行阳久则少寐,行阴久则多寐) |

营卫关系:营主内守属阴、卫主外卫属阳,"营在脉中,卫在脉外,营周不休,五十而复大会,阴阳相贯,如环无端";病则营卫不和(桂枝汤调和营卫)。

## 三、血

- **生成**:物质基础=水谷精微+营气+津液+精髓;"中焦受气取汁,变化而赤,是谓血"。五脏协作:**生化于脾、总统于心(奉心化赤)、藏受于肝、宣布于肺、施泄于肾**(《景岳全书》)——故血虚可从补心脾、滋肝血、补肾益髓多路论治。
- **循行**:心肺与脉构成循环系统;需**推动力**(心主血脉为基本动力+肺朝百脉+肝疏泄)与**固摄力**(脾统血+肝藏血)平衡——推动不足则迟涩血瘀,固摄不足则出血。
- **失常两端与治法**:出血——重在辨因(清热止血/益气止血/平肝止血/祛瘀止血),非见血止血;血瘀——活血祛瘀,多在和血基础上进行,不宜猛峻。
- **功能**:营养滋润("血主濡之"——目视耳听掌握足步皆赖之);**神志活动的物质基础**("血者,神气也"——血虚则惊悸失眠多梦,失血甚则烦躁恍惚癫狂昏迷)。

## 四、津液

- **津 vs 液**:清稀流动大者为津(布体表孔窍血脉,滋润,易耗易补);稠厚流动小者为液(灌骨节脏腑脑髓,濡养,难耗难复)。
- **代谢链**:"饮入于胃,游溢精气,上输于脾,脾气散精,上归于肺,通调水道,下输膀胱,水精四布,五经并行"(《素问·经脉别论》)。生成在脾胃+小肠主液+大肠主津;输布赖脾散精、肺行水、**肾主津液(蒸腾气化贯穿始终)**、肝疏泄、三焦决渎;排泄为汗与呼气(肺)、尿(肾与膀胱,最关键)、粪(大肠)。三脏定位:**"其本在肾,其标在肺,其制在脾"**(《景岳全书·肿胀》)。
- **功能**:滋润濡养、化生血液(渗入孙络)、**调节阴阳**(天寒衣薄则为溺与气,天热衣厚则为汗)、排泄废物。
- **失常**:不足——轻为**伤津**(多汗吐泻后口渴尿少、目陷螺瘪转筋),重为**伤阴/脱液**(舌光红无苔、形瘦肉脱、手足蠕动);"伤津乃伤阴之渐,伤阴乃津枯之甚"。停聚——湿浊困阻、痰饮、水肿,治以化湿利湿逐水祛痰,配行气。

**五脏化液**:汗为心液(心气虚自汗、心阴虚盗汗;大汗亡阳)、涕为肺液(风寒清涕/风热浊涕/肺燥干)、涎为脾液、泪为肝液(肝血虚目涩、湿热眵多)、唾为肾液(多唾久唾耗肾精)。

## 五、相互关系(病机与治法的推理依据)

- **气与血——"气为血之帅,血为气之母"**:气能生血(补气生血,如当归补血汤思路)、气能行血(气行则血行,气滞则血瘀→调气为上,调血次之)、气能摄血(补气摄血;"血脱者固气",独参汤);血能生气、血能载气(大出血→**气随血脱**)。
- **气与精**:气化为精(气聚精盈)、精化为气(失精家少气)。
- **气与津液**:气能生津、行津("气行水亦行"——治水肿行气利水并用)、摄津;津能载气(汗吐太过→**气随液脱**,"吐下之余,定无完气")。
- **精与血**:精血同源、互生(血虚补肾填精,如再障之治)。
- **血与津液**:津血同源、汗血同源——**"夺血者无汗,夺汗者无血"**;衄家、亡血家不可发汗;多汗夺津者不可破血逐瘀。血水互病:瘀血可致水肿,水肿可致血瘀→水病治血、血病治水、水血兼顾。

## 经典补强(v2.0 经典五书反哺;详 [[43-经典-灵枢与难经要义]] 第二节)

- **六脱证候**(灵枢·决气):**精脱者耳聋;气脱者目不明;津脱者腠理开、汗大泄;液脱者骨属屈伸不利、色夭、脑髓消、胫酸、耳数鸣;血脱者色白、夭然不泽、脉空虚**——大汗、大泻、失血之后判断"脱在哪一层"的经典指标。
- "**夺血者无汗,夺汗者无血**"(灵枢·营卫生会):失血者不可发汗、大汗者慎再耗血——气血津液同源互限的安全推论(与 `安全-配伍妊娠禁忌与毒性药` 及伤寒"衄家亡血家不可发汗"互证,见 [[44-经典-伤寒论六经方证条文]] 第九节)。
- "谷不入,半日则气衰,一日则气少"(灵枢·五味);"平人不食饮七日而死"(灵枢·平人绝谷)——进食状况是判断气之盛衰的第一观察项,不能进食属硬红线。
