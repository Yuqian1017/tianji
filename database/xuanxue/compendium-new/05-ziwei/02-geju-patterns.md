## 一、十四主星同宫组合

| **紫微·天机** | 卯/酉 | 善谋略·多变·聪明但犹豫 |
| **紫微·太阳** | — | 不同宫 |

## 二、经典吉格

### 紫府同宫格

**条件**：紫微与天府同在寅宫或申宫。

**含义**：两颗最贵之星同宫，如两位帝王联手，格局极高。

**特点**：
- 领导力极强
- 有贵气·有器量
- 适合从政或经商（大企业）
- 缺点：可能过于保守或自负

### 紫杀同宫格

**条件**：紫微与七杀同宫。

**含义**：帝王驾驭将军，霸气与执行力并存。

**特点**：
- 敢想敢做·魄力十足
- 适合创业·军事·竞争性行业
- 有掌控欲

### 机月同梁格

**条件**：天机·太阴·天同·天梁四星分布在三方四正（寅申巳亥四宫）。

**含义**：四颗文星交辉，适合从事公职、学术。

**特点**：
- 擅长机关事务·文书工作
- 学识好·有条理
- 缺乏开创性
- 经典「公务员格局」

### 府相朝垣格

**条件**：天府在命宫三方，天相在命宫三方，形成夹辅。

**含义**：库星和印星同护命宫，物质与精神双丰。

### 日月并明格

**条件**：太阳在巳或午宫（旺位），太阴在亥或子宫（旺位），分布在命宫三方。

**含义**：日月各居旺位，光明通达。

**特点**：
- 名利双收
- 男性太阳旺 → 事业好
- 女性太阴旺 → 气质佳

### 日照雷门格

**条件**：太阳在卯宫坐命或在卯宫三方。

**含义**：旭日东升（卯=东方），光芒万丈。

### 月朗天门格

**条件**：太阴在亥宫坐命。

**含义**：月亮在最旺之位，清辉照命。

### 武曲守财格

**条件**：武曲坐财帛宫，且不逢煞星。

**含义**：财星入财宫，天生善理财。

### 文桂文华格

**条件**：文昌和文曲同时在命宫三方。

**含义**：才华横溢·学业有成。

### 左右同宫格

**条件**：左辅右弼同宫或夹命。

**含义**：有强大助力·贵人多。

### 魁钺夹命格

**条件**：天魁天钺分居命宫两侧。

**含义**：贵人护驾·逢凶化吉。

## 三、经典凶格

### 命无主星

**条件**：命宫中没有十四主星。

**影响**：性格不够鲜明，需看对宫主星和三方四正弥补。不一定凶，但格局偏低。

### 火铃夹命

**条件**：火星和铃星分居命宫两侧。

**影响**：情绪不稳·容易冲动·多灾。

### 羊陀夹命

**条件**：擎羊和陀罗分居命宫两侧。

**影响**：进退两难·多阻碍·易有伤灾。

### 空劫夹命

**条件**：地空和地劫分居命宫两侧或同在命宫。

**影响**：
- 物质层面：破财·投资失利
- 精神层面：反而有利——地空地劫赋予超脱和创意，适合宗教、哲学、艺术

### 巨门化忌

**条件**：巨门星化忌。

**影响**：口舌是非极重·易招诽谤·暗箭难防。在夫妻宫尤忌（婚姻多争吵）。

### 廉贞化忌

**条件**：廉贞化忌。

**影响**：感情纠缠·法律纠纷·血光之灾。遇煞星加重。

### 武曲化忌

**条件**：武曲化忌。

**影响**：财务损失·投资失败·理财不当。在财帛宫尤凶。

## 四、星曜庙旺利陷

十四主星在不同地支宫位有不同的亮度（力量）：

| 主星 | 庙(最旺) | 旺(次旺) | 利(中等) | 陷(最弱) |
|------|---------|---------|---------|---------|
| 紫微 | 子午 | 寅申辰戌 | 卯酉巳亥 | 丑未 |
| 天机 | 子午卯酉 | 寅申 | 辰戌 | 巳亥丑未 |
| 太阳 | 巳午 | 辰卯 | 寅 | 亥子丑（日落之象）|
| 武曲 | 辰戌丑未 | 子午 | 寅申 | 卯酉巳亥 |
| 天同 | 子午卯酉 | 寅申 | 辰戌 | 巳亥丑未 |
| 廉贞 | 寅申 | 子午 | 辰戌丑未 | 卯酉巳亥 |
| 天府 | 全宫不陷 | — | — | — |
| 太阴 | 亥子 | 酉戌 | 申 | 寅卯辰巳午（日出后月隐）|
| 贪狼 | 子午卯酉 | 辰戌丑未 | 寅申 | 巳亥 |
| 巨门 | 子午卯酉 | 辰戌 | 寅申 | 丑未巳亥 |
| 天相 | 子午卯酉 | 寅申 | 辰戌丑未 | 巳亥 |
| 天梁 | 子午卯酉 | 寅申 | 辰戌丑未 | 巳亥 |
| 七杀 | 子午寅申 | 辰戌丑未 | 卯酉 | 巳亥 |
| 破军 | 子午 | 辰戌 | 寅申丑未 | 卯酉巳亥 |

> ⚠️ 庙旺利陷的分级标准在不同流派中有差异。以上为中州派/三合派常用标准。

```javascript
const STAR_BRIGHTNESS = {
  '紫微': {
    '庙': ['子','午'],
    '旺': ['寅','申','辰','戌'],
    '利': ['卯','酉','巳','亥'],
    '陷': ['丑','未']
  },
  '太阳': {
    '庙': ['巳','午'],
    '旺': ['辰','卯'],
    '利': ['寅'],
    '陷': ['亥','子','丑','未','申','酉','戌']
  },
  '太阴': {
    '庙': ['亥','子'],
    '旺': ['酉','戌'],
    '利': ['申'],
    '陷': ['寅','卯','辰','巳','午','未','丑']
  }
  // ... 其余主星同理
};

function getStarBrightness(starName, palace) {
  const data = STAR_BRIGHTNESS[starName];
  if (!data) return '未知';
  if (data['庙'].includes(palace)) return '庙';
  if (data['旺'].includes(palace)) return '旺';
  if (data['利'].includes(palace)) return '利';
  if (data['陷'].includes(palace)) return '陷';
  return '平';
}
```
