## 逐星详解

### 紫微星

**本质**：帝座·尊星·领导力·面子·主观·自尊心强

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 天生有领导气质·自尊心极强·喜欢被尊重·不愿屈居人下。庙旺：一方之主；落陷+煞：徒有其表·眼高手低 |
| 兄弟宫 | 兄弟中有出色者·或朋友中有权势人物 |
| 夫妻宫 | 配偶条件不错·但可能强势·婚姻中谁做主是问题 |
| 子女宫 | 子女有出息·但可能管不住·或子女叛逆 |
| 财帛宫 | 财运不错·花钱大方·注重排场·理财能力看会什么星 |
| 疾厄宫 | 脾胃系统·注意消化和皮肤。庙旺则体质不差 |
| 迁移宫 | 在外有面子·受人尊重·适合外出发展 |
| 交友宫 | 下属中有能人·但可能功高震主 |
| 官禄宫 | 事业心强·适合管理岗·大公司·政府机关 |
| 田宅宫 | 住宅讲究·家境不差·有继承运 |
| 福德宫 | 精神层面要求高·追求质量·内心有骄傲 |
| 父母宫 | 父母有一定社会地位·或对自己期望很高 |

### 天机星

**本质**：智慧·谋略·变化·犹豫·善计划·敏感多思

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 聪明善思·善于策划·心思细腻但犹豫。庙旺：军师型；陷+煞：小聪明·想太多不行动 |
| 夫妻宫 | 配偶聪明·但感情多变·可能想太多导致纠结 |
| 财帛宫 | 靠脑力赚钱·技术流·理财善变·钱来得快去得也快 |
| 官禄宫 | 适合技术·策划·研究·顾问类工作。忌做一成不变的事 |
| 疾厄宫 | 肝胆·神经系统·失眠·焦虑 |
| 福德宫 | 脑子停不下来·想太多·容易焦虑·需要学会放空 |

### 太阳星

**本质**：光明·付出·名声·男性·政治·博爱·不求回报

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 热情开朗·乐于助人·但常燃烧自己照亮别人。庙旺（日出）：光芒四射；落陷（日落）：付出无回报·消耗 |
| 夫妻宫 | 男命：自己重事业忽略伴侣。女命：丈夫能力不差但在外忙碌 |
| 财帛宫 | 赚钱大方·花钱也大方·不善守财·适合做需要曝光的事业 |
| 父母宫 | 父亲的代表。庙旺：父有能力；落陷：父缘薄或父辛苦 |
| 疾厄宫 | 眼睛·心脏·血压·头部 |
| 官禄宫 | 适合公众事业·政治·教育·传媒·需要抛头露面的工作 |

### 武曲星

**本质**：财星·决断·刚毅·孤刚·行动力·军人气质

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 性格刚直·重视金钱·行动力强。庙旺：武职显贵·财运好；陷+煞：刚而无谋·财来财去 |
| 夫妻宫 | 配偶务实·可能刚硬·婚姻中钱是重要话题 |
| 财帛宫 | 最佳财星位置。正财运佳·理财能力强·适合金融投资 |
| 官禄宫 | 适合金融·财务·军警·体育·需要决断力的行业 |
| 疾厄宫 | 呼吸系统·鼻·肺·骨折·外伤 |

### 天同星

**本质**：福星·温和·享受·懒散·感性·知足常乐

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 温和善良·人缘好·但缺乏冲劲。庙旺：有福气·晚年好；陷+煞：懒·安于现状·错失机会 |
| 夫妻宫 | 配偶温和体贴·感情融洽·但可能缺少激情 |
| 财帛宫 | 赚钱轻松但不大·小富则安·不喜冒险 |
| 福德宫 | 最佳位置。心态好·知足·精神世界丰富·有福 |

### 廉贞星

**本质**：复杂·政治·桃花·囚禁·极端·感情强烈

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 复杂多面·政治头脑·感情强烈。庙旺：从政经商皆可；陷+煞：桃花劫·官非·牢狱 |
| 夫妻宫 | 感情复杂·爱恨分明·配偶可能有故事 |
| 官禄宫 | 适合复杂环境·政治·法律·公关·需要手段的工作 |
| 疾厄宫 | 心脏·血液·免疫系统·精神类疾病 |

### 天府星

**本质**：财库·稳重·保守·面善·好面子·重实际

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 为人稳重·面善·爱享受·保守。财库之星，一生不缺吃穿 |
| 财帛宫 | 守财能力强·会理财·善于积累·但保守错过大机会 |
| 田宅宫 | 不动产运佳·家底厚·适合投资房产 |
| 官禄宫 | 适合大机构·体制内·银行·稳定型职业 |

### 太阴星

**本质**：阴柔·细腻·地产·母性·女性·艺术·夜间

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 细腻敏感·爱好艺术·重感情。庙旺（夜生）：富贵；陷（日生）：内心脆弱·感情波折 |
| 夫妻宫 | 女命：自己温柔；男命：妻子温柔漂亮。陷则感情不稳 |
| 财帛宫 | 不动产·女性相关行业·夜间经济·慢慢积累的财 |
| 田宅宫 | 不动产极佳位置。庙旺：豪宅；陷：房产纠纷 |

### 贪狼星

**本质**：桃花·才艺·欲望·应酬·多才多艺·好奇心

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 多才多艺·社交达人·桃花旺。庙旺：才艺出众·人缘极佳；陷+煞：酒色财气·沉迷享乐 |
| 夫妻宫 | 配偶长相不差·但桃花多·感情生活丰富 |
| 子女宫 | 桃花星在此=子女缘或本人桃花。子女聪明有才艺 |
| 官禄宫 | 适合娱乐·社交·销售·需要人脉的行业 |

### 巨门星

**本质**：口才·是非·怀疑·分析·暗蚀·争议

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 口才极佳·善分析·但疑心重。庙旺：名嘴·律师·分析师；陷：是非不断·小人多 |
| 夫妻宫 | 婚姻多口舌·配偶直言·争吵多但不一定分 |
| 官禄宫 | 适合需要口才的行业：律师·教师·主播·销售·医生 |
| 疾厄宫 | 口腔·食道·消化道·也主暗疾难查 |

### 天相星

**本质**：印星·辅佐·衣食·正派·文书·依附

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 正派温文·适合辅佐·衣食不缺。但独立性差·需要好的主星相配 |
| 官禄宫 | 适合秘书·助理·幕僚·行政·需要配合他人的工作 |
| 注意 | 天相两侧的星（夹宫）非常关键。财荫夹印=大吉；刑囚夹印=大凶 |

### 天梁星

**本质**：荫星·正义·化煞·清高·孤傲·老成

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 正义感强·好为人师·化解灾厄能力强。庙旺：贵人运极佳；陷：孤高不合群·逢灾虽化但辛苦 |
| 疾厄宫 | 化煞最佳位置。大病化小·有贵人医生 |
| 父母宫 | 父母有荫·长辈缘好·得祖荫 |
| 官禄宫 | 适合监察·审计·医疗·教育·宗教·公益 |

### 七杀星

**本质**：将星·果断·孤独·冲劲·变动·开创

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 独立果断·开创力强·但孤独。庙旺：将帅之才·独当一面；陷+煞：冲动妄为·伤痕累累 |
| 夫妻宫 | 配偶个性强·婚姻有波折·聚少离多 |
| 官禄宫 | 适合开创型事业·军警·外科·创业·竞技 |
| 迁移宫 | 在外闯荡·变动多·适合离乡发展 |

### 破军星

**本质**：耗星·破坏·开创·变化·不安定·革命

| 所在宫 | 断语 |
|--------|------|
| 命宫 | 喜新厌旧·不安现状·人生波折多但有开创力。庙旺：先破后立·成大事；陷：破而不立·一事无成 |
| 夫妻宫 | 婚姻变动大·可能离婚·或配偶工作性质多变 |
| 财帛宫 | 赚钱方式多变·大起大落·不善守财 |
| 子女宫 | 子女叛逆·个性独立·或头胎不顺 |
