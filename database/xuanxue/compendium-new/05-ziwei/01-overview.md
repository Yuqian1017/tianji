## 一、紫微斗数概述

### 基本框架

紫微斗数将一个人的命运投射到**十二宫**中，每个宫代表人生的一个面向。宫中的**星曜**决定了该面向的特质和走势。

### 与八字的对比

| 维度 | 八字 | 紫微斗数 |
|------|------|---------|
| 核心元素 | 五行·天干地支 | 星曜·宫位 |
| 分析单位 | 十神 | 星曜(100+颗) |
| 信息密度 | 8个字 | 12宫×多颗星 |
| 看人性格 | 日干+格局 | 命宫主星 |
| 看具体事 | 用神+大运 | 对应宫位+四化飞星 |
| 擅长领域 | 五行平衡·大势判断 | 具体事件·性格细节 |

### 宫位排列

十二宫按**逆时针**排列在十二地支位上，命宫位置由出生月和时辰确定。

```
🔮 实战：安命宫

// 命宫位置 = 寅宫 + 生月数 - 生时数
// 生月：正月=1, 二月=2, ...
// 生时：子时=1, 丑时=2, 寅时=3, ...

function findMingGong(lunarMonth, hourBranchIdx) {
  // hourBranchIdx: 子=0, 丑=1, 寅=2, ..., 亥=11
  // 命宫地支 = 寅(2) + 生月 - 生时
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  
  // 公式：从寅起正月顺数到出生月，再从该宫起子时逆数到出生时辰
  const monthPos = (2 + lunarMonth - 1) % 12; // 正月在寅
  const mingGongIdx = (monthPos - hourBranchIdx + 12) % 12;
  
  return branches[mingGongIdx];
}

// 身宫位置 = 寅宫 + 生月数 + 生时数 - 2
function findShenGong(lunarMonth, hourBranchIdx) {
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  const monthPos = (2 + lunarMonth - 1) % 12;
  const shenGongIdx = (monthPos + hourBranchIdx) % 12;
  return branches[shenGongIdx];
}
```

### 星曜组合

主星不是单独发挥作用的，它们按固定搭配出现在宫位中：

**紫微星系**（紫微永远跟这些星同宫或对宫）：
紫微·天机·太阳·武曲·天同·廉贞

**天府星系**（天府永远跟这些星同宫或对宫）：
天府·太阴·贪狼·巨门·天相·天梁·七杀·破军

> 🔮 **关键规则**：紫微和天府的位置一旦确定，其他十二颗主星的位置就自动确定了。这就是为什么排紫微盘的第一步是「安紫微天府」。

### 辅星与煞星

除了十四主星，还有数十颗辅星和煞星：

**六吉星**（有利）：
左辅·右弼·天魁·天钺·文昌·文曲

**六煞星**（不利，但可以激发潜力）：
擎羊·陀罗·火星·铃星·地空·地劫

**四化星**（最重要的动态元素）：
化禄·化权·化科·化忌

## 四、四化飞星

> *"四化是紫微斗数的精髓。主星是静态的性格底色，四化是动态的际遇变化。"——师父*

### 四化含义

| 化星 | 含义 | 效果 |
|------|------|------|
| **化禄** | 财·享受·顺利·桃花 | 让星曜变得更好·更顺 |
| **化权** | 权力·掌控·刚硬·竞争 | 增强星曜的主导力 |
| **化科** | 名声·学术·贵人·曝光 | 提升星曜的声誉 |

3. **流年四化**：以流年干飞四化 → 当年运势
4. **宫干飞四化**：各宫天干飞四化 → 宫与宫之间的联动（高阶）

## 五、命盘解读流程

### 第一步：看命宫主星

命宫主星决定了命主的基本性格和人生格局。

| 主星在命宫 | 大致性格 |
|-----------|---------|
| 紫微 | 有领导力·自尊心强·有贵气 |
| 天机 | 聪明善变·多谋略·不安分 |
| 太阳 | 热情开朗·乐于助人·好面子 |
| 武曲 | 刚毅果断·重财·不擅表达 |
| 天同 | 温和懒散·有福气·缺行动力 |
| 廉贞 | 复杂多面·有魅力·感情丰富 |
| 天府 | 稳重保守·有安全感·善理财 |
| 太阴 | 内敛细腻·重感情·有洁癖 |
| 贪狼 | 多才多艺·有欲望·桃花旺 |
| 巨门 | 口才好·多疑·善辩·是非多 |
| 天相 | 稳重公正·善辅佐·看环境 |
| 天梁 | 正直清高·有正义感·逢凶化吉 |
| 七杀 | 刚强孤傲·有冲劲·不怕困难 |
| 破军 | 叛逆革新·先破后立·变化大 |

### 第二步：看三方四正

**三方**：命宫的三合宫位（命宫、事业宫、财帛宫构成三角）  
**四正**：三方 + 对宫（迁移宫）

三方四正的星曜组合决定了命主的核心格局。

### 第三步：看四化落宫

生年四化落在哪个宫，就影响那个宫的主题：
- 化禄在财帛宫 → 天生财运好
- 化忌在夫妻宫 → 婚姻容易有波折
- 化权在官禄宫 → 事业上有掌控欲和成就

### 第四步：看大运流年

大运（十年一步）和流年叠加，判断各阶段的运势起伏。

## 六、紫微排盘核心算法概要

```
🔮 排盘步骤（简化）

1. 输入：农历出生年月日时 + 性别
2. 安命宫身宫 → 由月和时确定
3. 安十二宫 → 从命宫逆时针排列
4. 起紫微 → 由农历生日和五行局确定紫微在哪个宫
5. 安天府 → 紫微位置确定后自动确定
6. 安其余主星 → 按固定规则从紫微和天府出发排布
7. 安辅星煞星 → 各有各的安星规则
8. 飞四化 → 年干决定四化飞入哪些星
9. 安大运 → 从命宫或身宫起步
```

> ⚠️ **紫微排盘极其复杂**，安星规则超过50条。完整算法见 `reference/ziwei-algorithm.md`（后续补充）。强烈建议开发时采用逐步验证策略，每安一组星就与已知正确的命盘对照。
