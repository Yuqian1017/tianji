## 一、宫干飞四化（飞星派核心）

### 原理

每个宫位都有一个天干（由年干起五虎遁排出）。这个天干可以飞出四化（禄权科忌），落到其他宫位，形成宫与宫之间的**因果链**。

```
命宫天干 → 飞四化 → 落入某宫 = 命宫"主动"影响某宫
某宫天干 → 飞四化 → 落入命宫 = 某宫"主动"影响命宫
```

### 四化飞入的含义

| 四化 | 飞入某宫的含义 |
|------|-------------|
| **化禄** | 我（起飞宫）给那个宫带来好处·缘分·投入·喜欢 |
| **化权** | 我对那个宫有掌控力·在意·执着·付出努力 |
| **化科** | 我给那个宫带来名声·贵人·理性·小确幸 |
| **化忌** | 我给那个宫带来执念·亏欠·纠缠·问题 |

### 飞四化操作流程

```javascript
/**
 * 从某个宫位飞出四化
 * @param palaceStem 该宫的天干
 * @param starPositions 全盘各星所在宫位 { '紫微':'寅', '天机':'丑', ... }
 * @returns 四化落入的宫位
 */
function flyFromPalace(palaceStem, starPositions) {
  const sihua = SIHUA_TABLE[palaceStem];
  
  return {
    禄: { star: sihua.禄, targetPalace: starPositions[sihua.禄] },
    权: { star: sihua.权, targetPalace: starPositions[sihua.权] },
    科: { star: sihua.科, targetPalace: starPositions[sihua.科] },
    忌: { star: sihua.忌, targetPalace: starPositions[sihua.忌] }
  };
}
```

### 自化

当一个宫飞出的四化落在**自己宫内**的星上 = **自化**。

```
自化禄：本宫的好处留不住，向外散发（慷慨/流失）
自化权：本宫自我控制强，但过于执著于本宫事务
自化科：本宫自带光环，名声自来
自化忌：本宫自我消耗·自找麻烦·内耗
```

### 经典飞化解读模式

#### 命宫飞忌入某宫

| 忌入宫位 | 解读 |
|---------|------|
| 兄弟宫 | 对手足有亏欠·不善交际·兄弟关系有压力 |
| 夫妻宫 | 对配偶执念深·婚姻有波折·太在意感情 |
| 子女宫 | 对子女操心·子女缘或桃花有困扰 |
| 财帛宫 | 赚钱辛苦·钱财留不住·对金钱焦虑 |
| 疾厄宫 | 身体是弱点·容易生病·需注意健康 |
| 迁移宫 | 出外不顺·外在环境有压力·人际关系困难 |
| 交友宫 | 交友不慎·下属给你麻烦·合伙有风险 |
| 官禄宫 | 事业压力大·工作不顺·职场纠纷 |
| 田宅宫 | 家庭有烦恼·不动产有问题·家运不稳 |
| 福德宫 | 内心不安·精神压力·想太多·睡不好 |
| 父母宫 | 与长辈关系有压力·文书（考试/合同）不顺 |

#### 化忌追踪法

```
找化忌落宫 → 该宫再飞忌 → 追踪最终去向

这叫「忌追忌」，可以追查问题的根源。

例：命宫飞忌入夫妻宫 → 夫妻宫再飞忌入财帛宫
= 婚姻问题的根源在于金钱
```

#### 禄忌对看

```
禄 = 给出（付出·投入·喜欢）
忌 = 收回（执念·讨债·纠缠）

禄忌同宫：又爱又恨·给了又后悔·矛盾纠结
禄忌对冲：给出去的和纠结的在对立面，内心撕裂
命宫飞禄入A宫 + A宫飞忌入命宫：双向纠缠·缘分最深
```

## 二、四化的四层结构

### 层级体系

| 层级 | 来源 | 时间跨度 | 重要度 |
|------|------|---------|--------|
| **本命四化** | 年干飞四化 | 一生 | ★★★★★ 基本格局 |
| **大运四化** | 大运天干飞四化 | 10年 | ★★★★ 十年主题 |
| **流年四化** | 流年天干飞四化 | 1年 | ★★★ 年度变化 |
| **流月四化** | 流月天干飞四化 | 1月 | ★★ 精细推断 |

### 叠加判断法

```
本命盘定「体质」— 你是什么样的人
大运盘定「阶段」— 这十年的主题
流年盘定「事件」— 今年会怎样
流月盘定「时间」— 哪个月应验

当多层四化落入同一宫位时，力量叠加：
- 本命忌 + 大运忌 → 叠忌 = 该宫事务在这十年非常困难
- 本命禄 + 流年禄 → 双禄 = 今年该宫大吉
- 大运忌 + 流年忌 → 双忌流年 = 今年该宫有大事（凶）
```

## 三、小限

### 定义

小限 = 每年一个宫位，从命宫开始轮转。比大运更精细，是逐年的「焦点宫」。

### 小限起法

```javascript
/**
 * 小限排法
 * 男命：阳年顺行，阴年逆行
 * 女命：阳年逆行，阴年顺行
 * 起始宫位按出生年支定：
 */
const XIAOXIAN_START = {
  // 出生年支 → 小限1岁起宫
  '子': '辰', '丑': '巳', '寅': '午', '卯': '未',
  '辰': '申', '巳': '酉', '午': '戌', '未': '亥',
  '申': '子', '酉': '丑', '戌': '寅', '亥': '卯'
};

function getXiaoxian(yearBranch, yearStem, gender, age) {
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  
  const isYangYear = stems.indexOf(yearStem) % 2 === 0;
  const isMale = gender === 'male';
  
  // 顺逆判断
  const forward = (isYangYear && isMale) || (!isYangYear && !isMale);
  
  const startBranch = XIAOXIAN_START[yearBranch];
  const startIdx = branches.indexOf(startBranch);
  
  const step = forward ? 1 : -1;
  // 1岁在起始宫，每年移动一宫
  const targetIdx = (startIdx + (age - 1) * step + 120) % 12;
  
  return branches[targetIdx];
}
```

### 小限断法

```
小限落入某宫 → 该宫就是今年的「主战场」

看什么：
1. 该宫的主星组合 → 今年的基调
2. 该宫有无煞星 → 今年的阻力
3. 该宫的四化（本命+大运+流年）→ 今年的核心事件
4. 该宫的三方四正 → 外部环境

小限重叠大运宫 → 该年事件应验力最强
小限宫有本命化忌 → 该年触发本命弱点
小限宫有流年化禄 → 该年有好事临门
```

## 四、流年断法完整流程

### 第一步：确定流年盘

```
流年天干 = 当年干支的天干
流年地支 → 决定流年命宫位置

流年命宫 = 太岁（流年地支）所在宫位
流年其余十一宫从流年命宫逆排
```

### 第二步：四化叠加

```
在本命盘的基础上叠加：
1. 本命四化（已有）
2. 大运四化（大运天干飞入本命盘）
3. 流年四化（流年天干飞入本命盘）
4. 小限位置
```

### 第三步：交叉比对


### 第四步：重点事件判断

| 事件类型 | 看什么 |
|---------|--------|
| 财运 | 流年禄忌与财帛宫·田宅宫的关系 |
| 感情 | 流年禄忌与夫妻宫·子女宫的关系 |
| 事业 | 流年禄忌与官禄宫的关系 |
| 健康 | 流年忌入疾厄宫·或小限逢煞 |
| 搬家/出行 | 流年化与迁移宫·田宅宫的关系 |
| 学业 | 流年化与父母宫（文书）·官禄宫的关系 |

### 应验时间

```
流年事件通常在以下时间应验：
1. 流月四化与流年四化同向的月份
2. 小限与大运宫重合的年份
3. 太岁冲本命宫的年份

精确到月：
流月命宫 = 斗君起法（从流年命宫起正月，顺数到生月，再顺数到生时）
```

### 斗君（流月命宫起点）

```javascript
function getDouJun(yearBranch, lunarMonth, hourBranch) {
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  
  // 从太岁（流年地支）所在宫起正月，逆数到生月
  const yearIdx = branches.indexOf(yearBranch);
  const monthOffset = lunarMonth - 1;
  const baseIdx = (yearIdx + monthOffset) % 12;
  
  // 从该宫起子时，顺数到生时
  const hourIdx = branches.indexOf(hourBranch);
  const doujunIdx = (baseIdx + hourIdx) % 12;
  
  return branches[doujunIdx];
}

// 斗君确定后，正月流月命宫=斗君所在宫
// 二月=顺行一宫，三月再顺行一宫...
```
