## 一、四柱八字的结构

```
        天干    地支
年柱：   甲      辰      ← 出生年（以立春分界）
月柱：   丁      卯      ← 出生月（以节气分界）
日柱：   壬      午      ← 出生日（以子时分界）
时柱：   丁      未      ← 出生时辰
```

每柱一干一支，四柱共八字。其中**日干**（日柱的天干）是整个命盘的核心，称为**日主**或**日元**，代表命主本人。

---

## 二、年柱

### 规则

- 以**立春**为年分界（不是公历1月1日，也不是农历正月初一）
- 立春前出生 → 用上一年的干支
- 立春后出生 → 用当年的干支

### 算法

```
🔮 实战：年柱计算

function yearPillar(year) {
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  return {
    stem: stems[(year - 4) % 10],
    branch: branches[(year - 4) % 12]
  };
}

// ⚠️ 必须结合立春精确时间判断是否已过年界
// 立春时间每年不同，需查表或用天文算法
```

## 三、月柱

### 规则

- 以**节气**分界（不是农历初一），每月以「节」开始：

| 月份 | 地支 | 起始节气 | 大约公历 |
|------|------|---------|---------|
| 正月 | 寅 | 立春 | 2月4日前后 |
| 二月 | 卯 | 惊蛰 | 3月6日前后 |
| 三月 | 辰 | 清明 | 4月5日前后 |
| 四月 | 巳 | 立夏 | 5月6日前后 |
| 五月 | 午 | 芒种 | 6月6日前后 |
| 六月 | 未 | 小暑 | 7月7日前后 |
| 七月 | 申 | 立秋 | 8月8日前后 |
| 八月 | 酉 | 白露 | 9月8日前后 |
| 九月 | 戌 | 寒露 | 10月8日前后 |
| 十月 | 亥 | 立冬 | 11月7日前后 |
| 十一月 | 子 | 大雪 | 12月7日前后 |
| 十二月 | 丑 | 小寒 | 1月6日前后 |

- 月支固定，月干由年干推算

### 五虎遁月口诀

| 年干 | 正月(寅月)天干 | 规律 |
|------|--------------|------|
| 甲/己 | 丙寅 | 年干×2+2（取模） |
| 乙/庚 | 戊寅 | |
| 丙/辛 | 庚寅 | |
| 丁/壬 | 壬寅 | |
| 戊/癸 | 甲寅 | |

```
🔮 实战：月柱计算

function monthPillar(yearStem, monthBranchIdx) {
  // monthBranchIdx: 0=寅月(正月), 1=卯月, ..., 11=丑月
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const branches = ['寅','卯','辰','巳','午','未','申','酉','戌','亥','子','丑'];
  
  const yearStemIdx = stems.indexOf(yearStem);
  const startStemIdx = (yearStemIdx % 5) * 2 + 2; // 正月天干起点
  
  return {
    stem: stems[(startStemIdx + monthBranchIdx) % 10],
    branch: branches[monthBranchIdx]
  };
}
```

> ⚠️ **节气精度**：节气时间必须精确到分钟级。在节气交接日出生的人，差几个小时可能月柱完全不同，导致整个命盘变化。生产环境务必使用精确的天文历算数据。

## 四、日柱

### 规则

- 以**子时（23:00）**为日分界
- 23:00之后出生 → 算下一天
- 日柱**必须查万年历**，无可靠的简易公式

### 常用近似算法（仅供参考，不可用于生产）

```
🔮 参考：日柱近似算法（需验证）

// 以2000年1月1日（庚辰年·丙子月·甲子日）为基准点
function approxDayPillar(year, month, day) {
  // 这是简化算法，可能有1-2天误差
  // 生产环境必须查万年历数据库
  const baseDate = new Date(2000, 0, 1); // 2000-01-01 = 甲子日
  const targetDate = new Date(year, month - 1, day);
  const diffDays = Math.floor((targetDate - baseDate) / 86400000);
  
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  
  const idx = ((diffDays % 60) + 60) % 60;
  return {
    stem: stems[idx % 10],
    branch: branches[idx % 12]
  };
}
```

## 五、时柱

### 规则

十二时辰对应地支（固定），时干由日干推算。

| 时辰 | 地支 | 时间范围 |
|------|------|---------|
| 子时 | 子 | 23:00-00:59 |
| 丑时 | 丑 | 01:00-02:59 |
| 寅时 | 寅 | 03:00-04:59 |
| 卯时 | 卯 | 05:00-06:59 |
| 辰时 | 辰 | 07:00-08:59 |
| 巳时 | 巳 | 09:00-10:59 |
| 午时 | 午 | 11:00-12:59 |
| 未时 | 未 | 13:00-14:59 |
| 申时 | 申 | 15:00-16:59 |
| 酉时 | 酉 | 17:00-18:59 |
| 戌时 | 戌 | 19:00-20:59 |
| 亥时 | 亥 | 21:00-22:59 |

### 五鼠遁时口诀

| 日干 | 子时天干 | 口诀 |
|------|---------|------|
| 甲/己 | 甲子 | 甲己还加甲 |
| 乙/庚 | 丙子 | 乙庚丙作初 |
| 丙/辛 | 戊子 | 丙辛从戊起 |
| 丁/壬 | 庚子 | 丁壬庚子居 |
| 戊/癸 | 壬子 | 戊癸起壬子 |

```
🔮 实战：时柱计算

function hourPillar(dayStem, hour) {
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  
  const branchIdx = hour === 23 ? 0 : Math.floor((hour + 1) / 2);
  const dayStemIdx = stems.indexOf(dayStem);
  const startStem = (dayStemIdx % 5) * 2;
  
  return {
    stem: stems[(startStem + branchIdx) % 10],
    branch: branches[branchIdx]
  };
}
```

> ⚠️ **真太阳时**：严格的八字排盘需要使用出生地的**真太阳时**，而非北京时间。中国横跨五个时区，西部地区（如新疆）可能需要修正1-2小时。建议程序提供真太阳时校正选项。

## 六、完整排盘示例

**出生信息**：1990年4月15日 14:30 北京

1. **年柱**：1990年立春后 → (1990-4)%10=6→庚, (1990-4)%12=6→午 → **庚午**
2. **月柱**：4月15日在清明(4月5日)后、立夏(5月6日)前 → 三月辰月 → 庚年起丙寅→丁卯→戊辰→己巳→庚午...不对，重算：庚年正月=戊寅，三月=庚辰 → **庚辰**
3. **日柱**：查万年历 → **丙辰**（需查表确认）
4. **时柱**：14:30=未时，丙日起子时=戊子，数到未：戊子己丑庚寅辛卯壬辰癸巳甲午乙未 → **乙未**

```
年柱    月柱    日柱    时柱
庚午    庚辰    丙辰    乙未
```

## 七、本课心法

1. **立春分年，节气分月**：这是最容易出错的地方
2. **日柱必查表**：不要信任何简易公式
3. **日主是核心**：日干代表命主本人，一切分析以它为中心
4. **真太阳时**：精确排盘的必要条件
