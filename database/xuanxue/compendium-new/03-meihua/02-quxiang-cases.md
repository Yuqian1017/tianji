## 一、取象原则

### 黄金三则

1. **第一印象法则**：看到卦象的第一个念头往往最准
2. **关联法则**：象必须与所问之事有逻辑关联
3. **情境法则**：同一个卦在不同情境下取不同的象

### 取象的层级

```
第一层：自然象（水火风雷山泽天地）→ 最基础
第二层：人事象（父母·子女·官鬼…）→ 结合六亲
第三层：引申象（说卦传万物类象）→ 需要积累
第四层：灵感象（超越常规的直觉联想）→ 高手境界
```

## 三、实战取象案例

### 案例一：测来人何事

**场景**：某人来找你，你不知道他要问什么。以来人方位和时间起卦。

来人从**南方**来 → 离卦(先天数3)  
时辰**午时** → 午=7(序数) → 7÷8余7=艮卦  
动爻 → (3+7)÷6=1余4 → 四爻动

得卦：**火山旅**，四爻动在上卦，上卦离为用，下卦艮为体。

**取象**：
- 离(用)：文书·光明·发现·附着
- 艮(体)：停止·房屋·仓库
- 用(火)克体(土)？不对——火生土 → **用生体** → 有利

**读象**：来人可能与文书（离）和房屋（艮）有关。用生体 → 对来人有利。

**推断**：此人来问房产或文书手续的事，结果对他有利（用生体）。

**验证**（假设）：来人果然是来问房产证能否办下来。

### 案例二：测天气

**场景**：朋友问明天是否会下雨，可以去野餐吗。

随机得数：5和3  
上卦 = 5÷8余5 = 巽  
下卦 = 3÷8余3 = 离  
动爻 = (5+3)÷6=1余2 → 二爻动（下卦）

得卦：**风火家人**，二爻动在下卦，下卦离为用，上卦巽为体。

**取象**：
- 巽(体)：风
- 离(用)：火·太阳·晴朗
- 用(火)生体(木)→ 用生体 → 有利 → 天气对「出行」有利

**读象**：风和日丽——有风（巽）有阳光（离），火生木，天气好。

**结论**：明天天气晴好，适合野餐。

### 案例三：测感情

**场景**：女生问她暗恋的男生是否喜欢她。以时间起卦。

得卦：**泽山咸** → 动爻在三爻

- 咸 = 感 → 卦名本身就是「感应」！
- 上兑（少女）下艮（少男）→ 少男少女相感
- 体用：三爻动在下卦 → 下卦艮为用，上卦兑为体
- 用(土)生体(金) → **用生体** → 对方对自己有好感！

**加外应**：起卦时听到窗外有鸟叫两声 → 兑(2)→ 喜悦  
再看：兑为口、为笑 → 有好消息

**结论**：对方对你有好感，且卦象「咸」本身就是感应之卦。可以主动一些。

### 案例四：测生意

**场景**：某人想开一家餐厅，问能否成功。

报数起卦：88和16  
上卦 = 88÷8=11余0 → 坤(8)  
下卦 = 16÷8=2余0 → 坤(8)  
动爻 = (88+16)÷6=17余2 → 二爻动

得卦：**坤为地**，二爻动 → 下卦坤为用，上卦坤为体。

**取象**：
- 坤+坤 = 纯坤 → 承载·大众·食物·包容
- 体用比和（都是土）→ 平稳，但缺乏生机
- 动爻在下 → 底部有变化

**深层分析**：
- 坤为地，重坤 = 厚德载物，但也 = 过于保守/缺乏特色
- 做餐厅看子孙爻（经营能力、食物）→ 坤卦中子孙为金 → 金在坤土中被生 → 经营能力可以
- 看妻财爻（利润）→ 妻财为水 → 水被土克 → 利润被控制 → 赚钱不多但稳定

**外应**：起卦时看到一辆满载货物的大卡车 → 坤象（大车·承载·运输）→ 印证坤象，量大但利薄

**结论**：餐厅可以开，走大众路线（坤=大众），但利润偏薄。建议走量不走价。

### 案例五：外应断事（高阶）

> *"这个案例不是正式起卦的，是我在路上自然看到的。"——师姐*

**场景**：师姐正在想一个项目能否谈成，走在路上突然看到：

1. 一棵大树（巽/震 → 木）
2. 树旁有个水龙头在滴水（坎 → 水）
3. 一个小孩跑过来关了水龙头（艮 → 止）

**读象**：
- 木得水生（水生木）→ 项目有滋养，条件好
- 但水被关了（艮=止）→ 资源供给会中断
- 小孩关水 → 子辈/下属的行为导致中断

**判断**：项目初期有利，但中途会因为团队中某个人的操作导致资源断裂。需提前防范。

## 四、取象练习法

> *"怎么练取象？每天做三件事：一观、二联、三验。"——师姐*

### 一观：观察练习

每天选一个场景，用八卦描述你看到的一切：
- 早上上班路上看到什么 → 用八卦语言描述
- 午饭时的环境 → 用八卦语言描述

### 二联：联想练习

给自己一个卦，用30秒说出尽可能多的象：
- 今天的练习卦：坎
- 30秒内：水·暗·险·流动·肾·耳·黑色·冬天·北方·酒·墨·弯曲·隐藏·智慧·盗贼·陷阱……

### 三验：验证练习

每周用梅花起三个卦，记录判断，一周后验证：
- 准了 → 分析为什么准
- 不准 → 分析哪里取象偏了
