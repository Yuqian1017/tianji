> **难度标签**：🟢必修：分类取用 | 🟡进阶：各类断法要点

> *"理论学完了，现在实战。"师姐拿出一叠卦例，"人问得最多的就那几类——感情、事业、财运、疾病、出行。每一类的断法有共性，也有特殊点。"*

## 一、断卦通用流程

无论问什么事，都先走这个流程：

```
1. 明确问题 → 确定用神
2. 找用神 → 在卦中定位（或找伏神）
3. 判旺衰 → 月建日辰对用神的作用
4. 看动态 → 动爻与用神的生克关系
5. 查特殊 → 空亡·入墓·合冲·进退·回头生克
6. 看世应 → 世为我，应为对方
7. 综合判断 → 吉凶 + 应期 + 建议
```

## 二、感情婚姻

### 用神

| 性别 | 用神 | 辅助参考 |
|------|------|---------|
| **男问女** | 妻财爻 | 世爻=自己，应爻=对方 |
| **女问男** | 官鬼爻 | 世爻=自己，应爻=对方 |
| **问婚姻关系** | 世应关系 | 世应相生→和睦，世应相克→矛盾 |

### 关键判断

- **世应相生** → 两人有感情基础
- **世应相克** → 矛盾多，需看谁克谁
- **世应相合** → 关系紧密（但也可能是牵制）
- **世应相冲** → 意见不合或分离
- **用神旺相** → 对方条件好/感情深
- **用神衰空** → 对方无心/条件差/没有这个人
- **间爻（世应之间的爻）动** → 有第三者或阻碍

### 六神辅助判断

| 六神临用神 | 含义 |
|-----------|------|
| 青龙 | 对方相貌好/有才华 |
| 朱雀 | 对方健谈/多口舌 |
| 勾陈 | 对方稳重/老实/迟缓 |
| 螣蛇 | 对方心思复杂/虚幻 |
| 白虎 | 对方性格刚烈/有暴力倾向 |
| 玄武 | 对方有隐瞒/不忠 |

## 三、事业工作

### 用神

| 问题 | 用神 |
|------|------|
| 问工作运 | 官鬼爻（官=职位·工作） |
| 问升职 | 官鬼爻 + 父母爻（印=权柄·文书） |
| 问考试 | 父母爻（文书=成绩·文凭） |
| 问面试 | 官鬼爻(工作) + 世爻(表现) |
| 问跳槽 | 世爻(自己)·官鬼爻(新工作)·应爻(新单位) |

### 关键判断

- **官鬼旺相生世** → 有好工作机会，且对自己有利
- **官鬼克世** → 工作压力大/领导给压力
- **子孙持世** → 自由但不稳定（子孙克官鬼=制约工作）
- **兄弟持世** → 竞争激烈/有人抢

## 四、财运求财

### 用神

| 问题 | 用神 |
|------|------|
| 求财/投资 | 妻财爻 |
| 问工资 | 妻财爻 + 官鬼爻（有工作才有工资） |
| 问借钱能否收回 | 妻财爻 + 应爻（对方） |

### 关键判断

- **妻财旺相** → 有财可得
- **妻财衰空** → 无财/财不实
- **兄弟持世** → 破财卦！兄弟=劫财（比劫夺财）
- **兄弟动** → 有人跟你争财/破财事件
- **子孙旺相** → 财源之根（子孙生妻财），持久生财
- **子孙持世** → 有生财之道，但不是正财（偏门/副业）

> *"最怕兄弟持世动爻又多——这基本就是'钱留不住'的格局。"——师姐*

## 五、疾病健康

### 用神

| 情况 | 用神 |
|------|------|
| 自己问病 | 世爻（自己的身体） |
| 问他人之病 | 对应六亲爻（如问父亲→父母爻） |
| 看病因 | 官鬼爻（官鬼=病源·病魔） |
| 看治疗 | 子孙爻（子孙克官鬼=制病） |
| 看医药 | 子孙爻 + 父母爻 |

### 关键判断

- **世爻旺相** → 身体底子好，可恢复
- **世爻衰弱被克** → 病情严重
- **子孙旺相** → 有救（子孙=医药·抗病力）
- **官鬼旺相克世** → 病势凶猛
- **官鬼空亡** → 不是大病/虚惊
- **官鬼入墓** → 病被封住，可能暂时缓解但根因未除

> ⚠️ **免责声明**：六爻断病仅作参考，绝不能替代正规医疗诊断和治疗。如有健康问题请务必就医。

## 六、出行行事

### 用神

| 问题 | 用神/关注点 |
|------|------------|
| 出行是否顺利 | 世爻=自己，应爻=目的地 |
| 路上安全 | 看官鬼（灾祸）是否克世 |
| 能否成行 | 世爻是否空/墓/被合住 |

### 关键判断

- **世爻旺相，应爻生世** → 出行顺利，目的地有利
- **世爻被克/空亡** → 不宜出行
- **官鬼临白虎动克世** → 有安全风险
- **官鬼临朱雀** → 路上有口舌/纠纷
- **官鬼临玄武** → 小心被骗/丢东西

## 七、失物寻找

### 用神

- 失物 → 妻财爻
- 失物方向 → 妻财爻所临地支的方位

### 关键判断

- **妻财旺相不空** → 能找回
- **妻财空亡** → 东西不在了/很难找回
- **妻财入墓** → 东西被藏在某处（墓库对应的方位/地点）
- **妻财被兄弟持** → 被人拿了

## 八、断卦示例（完整）

> *师姐写了一个完整的断卦过程在黑板上：*

**问题**：某人问能否升职。  
**时间**：卯月 丙辰日  
**得卦**：雷火丰 → 变 泽火革（三爻动）

```
离宫·火

六神  六亲   爻     纳甲     动变
玄武  官鬼  上爻  庚戌土 ⚋  
白虎  父母  五爻  庚申金 ⚋  应
螣蛇  妻财  四爻  庚午火 ⚊  
勾陈  子孙  三爻  己亥水 ⚊  动 → 丁丑土 (变)
朱雀  妻财  二爻  己丑土 ⚋  世 → ？不，二爻是丑，世在二爻
      ... （简化示意）
```

**分析思路**：
1. **用神**：问升职 → 官鬼爻（上爻庚戌土）
2. **月建**：卯月 → 木旺，木克土 → 官鬼受月克，**偏衰**
3. **日辰**：辰日 → 辰土，官鬼得日辰比和，有所补救
4. **动爻**：三爻子孙亥水动 → 子孙克官鬼！不利
5. **变爻**：亥水变丑土 → 回头克？不，水变土不是回头克的标准判断…

> *（此处为教学示意，实际断卦需要更精确的装卦。重点是展示分析思路。）*

## 九、本课心法

> *"分类占断的要诀就是——用神选对，忌神看清，世应分明。记住这三点，什么问题都有章法。"——师姐*

1. **用神因事而异**：问钱看财，问官看官，问病看鬼
2. **世应定立场**：世永远是自己，应永远是对方
3. **六神助解读**：六神不定吉凶，但提供细节信息
4. **兄弟是破财杀手**：凡问财，最怕兄弟旺动
5. **子孙是万能解药**：子孙生财克鬼，多数时候是好的（但问官运除外——子孙克官）
