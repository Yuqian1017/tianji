# 金丹期 · 六爻篇 · 第二课 · 装卦全流程

> **难度标签**：🟢必修：纳甲·世应·六亲 | 🟡进阶：六神

> *师姐在纸上画了一个空白的六爻框架。*
> *"起卦只是第一步——你得到了六个爻。现在要做的，是给每个爻赋予身份。就像你有六个人，但你还不知道谁是谁、谁跟谁是什么关系。"*
> *"装卦，就是查户口。"*

---

## 一、装卦的完整步骤

```
起卦（得到六个爻）
  ↓
第一步：确定卦名 → 查上下卦得六十四卦名
  ↓
第二步：确定卦宫 → 查京房八宫，确定该卦属于哪个宫
  ↓
第三步：纳甲 → 给每个爻配上天干地支
  ↓
第四步：装世应 → 确定世爻和应爻的位置
  ↓
第五步：配六亲 → 根据卦宫五行和各爻地支五行，配父母/兄弟/子孙/妻财/官鬼
  ↓
第六步：安六神 → 根据日干，安青龙/朱雀/勾陈/螣蛇/白虎/玄武
  ↓
完成！可以开始断卦
```

## 二、第一步：确定卦名

根据上卦（外卦）和下卦（内卦）查表得卦名。

查表方法见练气期第四课的六十四卦速查表。

**命名规则**：「上卦名 + 下卦名」或传统卦名。
- 上坎下震 → 水雷屯
- 上乾下乾 → 乾为天
- 上坤下坤 → 坤为地

## 四、第三步：纳甲（最关键）

> *"纳甲就是给每个爻一个地址——天干地支。有了地址，才能论五行生克。"——师姐*

### 纳甲规则

纳甲是按八卦来的——上卦纳上卦的八卦对应的干支，下卦纳下卦的八卦对应的干支。

**八卦纳甲对照表**：

| 八卦 | 纳天干 | 纳地支（内卦/下卦） | 纳地支（外卦/上卦） |
|------|--------|-------------------|-------------------|
| **乾** | 甲 | 子寅辰 | 午申戌 |
| **坤** | 乙 | 未巳卯 | 丑亥酉 |
| **震** | 庚 | 子寅辰 | 午申戌 |
| **巽** | 辛 | 丑亥酉 | 未巳卯 |
| **坎** | 戊 | 寅辰午 | 申戌子 |
| **离** | 己 | 卯丑亥 | 酉未巳 |
| **艮** | 丙 | 辰午申 | 戌子寅 |
| **兑** | 丁 | 巳卯丑 | 亥酉未 |

**读法**：从初爻到三爻（内卦）/ 从四爻到上爻（外卦），依次对应。

**示例——水雷屯（上坎下震）**：

| 爻位 | 阴阳 | 八卦 | 纳甲 |
|------|------|------|------|
| 上爻 | ⚋ 阴 | 坎(外) | 戊子 |
| 五爻 | ⚊ 阳 | 坎(外) | 戊戌 |
| 四爻 | ⚋ 阴 | 坎(外) | 戊申 |
| 三爻 | ⚋ 阴 | 震(内) | 庚辰 |
| 二爻 | ⚋ 阴 | 震(内) | 庚寅 |
| 初爻 | ⚊ 阳 | 震(内) | 庚子 |

```
const NAJIA = {
  // [内卦初爻, 内卦二爻, 内卦三爻, 外卦四爻, 外卦五爻, 外卦上爻]
  '乾': { stem: '甲', inner: ['子','寅','辰'], outer: ['午','申','戌'] },
  '坤': { stem: '乙', inner: ['未','巳','卯'], outer: ['丑','亥','酉'] },
  '震': { stem: '庚', inner: ['子','寅','辰'], outer: ['午','申','戌'] },
  '巽': { stem: '辛', inner: ['丑','亥','酉'], outer: ['未','巳','卯'] },
  '坎': { stem: '戊', inner: ['寅','辰','午'], outer: ['申','戌','子'] },
  '离': { stem: '己', inner: ['卯','丑','亥'], outer: ['酉','未','巳'] },
  '艮': { stem: '丙', inner: ['辰','午','申'], outer: ['戌','子','寅'] },
  '兑': { stem: '丁', inner: ['巳','卯','丑'], outer: ['亥','酉','未'] }
};

function applyNajia(lowerGua, upperGua) {
  const lower = NAJIA[lowerGua];
  const upper = NAJIA[upperGua];
  
  return [
    { stem: lower.stem, branch: lower.inner[0] },  // 初爻
    { stem: lower.stem, branch: lower.inner[1] },  // 二爻
    { stem: lower.stem, branch: lower.inner[2] },  // 三爻
    { stem: upper.stem, branch: upper.outer[0] },  // 四爻
    { stem: upper.stem, branch: upper.outer[1] },  // 五爻
    { stem: upper.stem, branch: upper.outer[2] },  // 上爻
  ];
}
```

> ⚠️ **特别注意**：
> - 乾和震的纳支相同（子寅辰/午申戌），但纳干不同（甲vs庚）
> - 坤和巽都是逆序排列（从大到小）
> - 纳甲必须严格查表，这是六爻系统的硬编码数据

## 五、第四步：装世应

世爻代表自己（问事者），应爻代表对方（问事的对象/外部环境）。

### 世应位置规则

| 卦在八宫中的位置 | 世爻 | 应爻 |
|----------------|------|------|
| 本宫卦（八纯卦） | 六爻(上爻) | 三爻 |
| 一世卦 | 初爻 | 四爻 |
| 二世卦 | 二爻 | 五爻 |
| 三世卦 | 三爻 | 上爻 |
| 四世卦 | 四爻 | 初爻 |
| 五世卦 | 五爻 | 二爻 |
| 游魂卦 | 四爻 | 初爻 |
| 归魂卦 | 三爻 | 上爻 |

**规律**：世应永远间隔两个爻位。世在N → 应在(N+3-1)%6+1。

## 六、第五步：配六亲

> *"六亲是六爻断卦的灵魂。每个爻的六亲身份决定了它代表什么人、什么事。"——师姐*

### 六亲定义

以**卦宫五行**为「我」，与各爻地支五行对比：

| 关系 | 六亲名 | 含义 | 代表 |
|------|--------|------|------|
| 同我者 | **兄弟** | 与我平级 | 兄弟姐妹·朋友·竞争者·阻碍·破财 |
| 生我者 | **父母** | 滋养我的 | 父母·长辈·文书·房屋·庇护 |
| 我生者 | **子孙** | 我产出的 | 子女·晚辈·下属·自由·医药·制官鬼 |
| 克我者 | **官鬼** | 制约我的 | 官府·上司·丈夫·疾病·灾祸·压力 |
| 我克者 | **妻财** | 我所控的 | 妻子·财产·天气·有形资产 |

### 配六亲算法

```
卦宫五行 → 我
各爻地支五行 → 对方

我 vs 对方:
  同五行 → 兄弟
  对方生我 → 父母
  我生对方 → 子孙
  对方克我 → 官鬼
  我克对方 → 妻财
```

**示例——水雷屯（坎宫·水）**：

| 爻位 | 纳甲 | 地支五行 | vs 宫(水) | 六亲 |
|------|------|---------|----------|------|
| 上爻 | 戊子 | 子=水 | 同我 | 兄弟 |
| 五爻 | 戊戌 | 戌=土 | 土克水 | 官鬼 |
| 四爻 | 戊申 | 申=金 | 金生水 | 父母 |
| 三爻 | 庚辰 | 辰=土 | 土克水 | 官鬼 |
| 二爻 | 庚寅 | 寅=木 | 水生木 | 子孙 |
| 初爻 | 庚子 | 子=水 | 同我 | 兄弟 |

```
🔮 实战：六亲配置

const BRANCH_WUXING = {
  '子': 'water', '丑': 'earth', '寅': 'wood', '卯': 'wood',
  '辰': 'earth', '巳': 'fire', '午': 'fire', '未': 'earth',
  '申': 'metal', '酉': 'metal', '戌': 'earth', '亥': 'water'
};

function getSixRelation(palaceWuxing, branchWuxing) {
  const relations = {
    // [我, 对方] → 六亲
    'same': '兄弟',
    'generates_me': '父母',    // 对方生我
    'i_generate': '子孙',      // 我生对方
    'restricts_me': '官鬼',    // 对方克我
    'i_restrict': '妻财'       // 我克对方
  };
  
  if (palaceWuxing === branchWuxing) return relations.same;
  
  const order = ['wood', 'fire', 'earth', 'metal', 'water'];
  const myIdx = order.indexOf(palaceWuxing);
  const tgtIdx = order.indexOf(branchWuxing);
  const diff = (tgtIdx - myIdx + 5) % 5;
  
  switch(diff) {
    case 1: return relations.i_generate;     // 子孙
    case 2: return relations.i_restrict;     // 妻财
    case 3: return relations.restricts_me;   // 官鬼
    case 4: return relations.generates_me;   // 父母
    default: return relations.same;
  }
}
```

```
function assignSixSpirits(dayStem) {
  const spirits = ['青龙','朱雀','勾陈','螣蛇','白虎','玄武'];
  const stemStartMap = {
```

## 九、本课心法

1. **装卦是机械操作**：查表就行，不需要理解也不需要灵感，严格按规则来
2. **卦宫决定六亲**：这是最容易出错的地方——六亲是按卦宫五行定的，不是按卦名
3. **纳甲必须查表**：不要试图推算，直接用数据表
4. **六神是辅助**：六神提供额外的象义信息，但不改变吉凶判断
