## 卦例一：问面试能否通过（入门级）

### 背景
某人即将参加一家公司的面试，问能否被录用。

**起卦时间**：甲辰年 丙寅月 庚午日 （铜钱法）  
**得卦**：天火同人 → 变 乾为天（五爻动）

### 装卦

```
离宫·火

六神   六亲    爻位    纳甲      动静
玄武   子孙    上爻    壬戌土    ⚊  应
白虎   妻财    五爻    壬申金    ⚊  动 → 壬午火（变）
螣蛇   兄弟    四爻    壬午火    ⚊
勾陈   官鬼    三爻    己亥水    ⚊
朱雀   子孙    二爻    己丑土    ⚋  世
青龙   父母    初爻    己卯木    ⚊
```

### 分析思路

**第一步：确定用神**
- 问面试/工作 → 用神取**官鬼爻**
- 卦中官鬼在三爻：己亥水

**第二步：判断旺衰**
- 月建：寅月（木旺）→ 木不直接生水，寅中藏甲丙戊，与水关系为「水生木→泄气」→ 官鬼在寅月偏泄
- 日辰：午日（火旺）→ 水克火，但水去克火是耗力 → 日辰耗官鬼
- 初步判断：官鬼**偏弱**

**第三步：看动爻影响**
- 五爻壬申金动 → 金生水 → **动爻生官鬼**！这是好消息
- 五爻是妻财爻（申金），动来生官鬼亥水 → 有人帮忙/有资源支持
- 但五爻动变午火 → **回头克**（火克金）→ 动爻自身受伤，生官鬼的力量打了折扣

**第四步：看世应**
- 世在二爻（己丑土）→ 自己
- 应在上爻（壬戌土）→ 面试官/公司
- 世应皆土，**比和** → 双方没有明显冲突，但也没有特别吸引力

**第五步：看空亡**
- 庚午日 → 甲戌旬 → 空亡：**申、酉**
- 五爻壬申金空亡！→ 动爻空亡 = **动而无功**
- 申金本来生官鬼的，但空了，力量大减

**第六步：综合判断**
- 官鬼偏弱（月泄日耗）
- 有动爻来生，但动爻空亡+回头克 → 帮助力量很弱
- 世应比和 → 不突出

**结论**：面试通过的可能性**偏低**。官鬼无力，且唯一的助力（申金）空亡。建议另寻机会，或等申金出空（过旬后）再争取。

**应期推断**：如果最终能成，应在申金出空或逢申日（申金填实）时有消息。

> *"看到了吗？关键转折点在'申金空亡'。如果申金不空，这个卦的结论可能完全不同——有力的财爻动来生官，面试大概率能过。一个空亡，翻盘了。"——师姐*

## 卦例二：问感情发展（中级）

### 背景
女性问与男友的感情能否长久。

**起卦时间**：甲辰年 丁卯月 壬子日  
**得卦**：泽水困 → 变 泽地萃（二爻动）

### 装卦

```
兑宫·金

六神   六亲    爻位    纳甲      动静
青龙   父母    上爻    丁未土    ⚋
朱雀   兄弟    五爻    丁酉金    ⚊  世
勾陈   子孙    四爻    丁亥水    ⚊
螣蛇   官鬼    三爻    戊午火    ⚋
白虎   父母    二爻    戊辰土    ⚊  动 → 乙丑土（变）  应
玄武   妻财    初爻    戊寅木    ⚋
```

### 分析

**用神**：女问男友 → 取**官鬼爻**  
官鬼在三爻：戊午火

**月建**：卯月（木旺）→ 木生火 → 官鬼得月**生**，偏旺  
**日辰**：子日（水旺）→ 水克火 → 官鬼受日**克**

月生日克，力量拉扯，官鬼**中等偏上**。

**动爻**：二爻戊辰土动  
- 辰土（父母爻）→ 火生土 → 官鬼生父母爻 → 这是官鬼在**泄气**给父母
- 但辰土动不直接克官鬼，只是在泄官鬼的力
- 变爻丑土 → 辰变丑，土变土，**伏吟**（同五行）→ 停滞

**世应关系**：
- 世在五爻（酉金）→ 自己
- 应在二爻（辰土）→ 对方/这段关系
- 辰土生酉金 → **应生世** → 对方对自己有情意 ✓
- 但应爻动了 → 应爻不安分，变化中

**关键发现**：
- 官鬼午火在卯月得生，基本旺 → 男方条件/感情基础不差
- 但日辰子水克午火 → 当下有压力/矛盾（子午冲！）
- 二爻辰土动化伏吟 → 关系停滞不前，陷入反复
- 世爻酉金在卯月受克（卯酉冲！）→ 自己状态不佳

**空亡**：壬子日 → 甲戌旬 → 空亡：申、酉  
→ **世爻酉金空亡！** → 自己心不在焉/对这段关系心里打鼓

**综合判断**：
- 男方有感情（官鬼月令得生），但当下两人有冲突（子午冲）
- 自己内心犹豫（世爻空亡）
- 关系走向停滞（应爻动化伏吟）
- 短期内难以有突破，但不是彻底无望

**结论**：感情基础在，但当前阶段矛盾明显、进展停滞。自己需要先想清楚（世爻出空后），双方才能进入下一阶段。不建议此时做重大决定。

## 卦例三：问财运（中级）

### 背景
某人计划投资一个项目，问能否获利。

**起卦时间**：甲辰年 丙寅月 戊辰日  
**得卦**：水风井 → 变 水雷屯（初爻动）

### 装卦

```
震宫·木

六神   六亲    爻位    纳甲      动静
勾陈   父母    上爻    戊子水    ⚋
螣蛇   妻财    五爻    戊戌土    ⚊  应
白虎   官鬼    四爻    戊申金    ⚋
玄武   官鬼    三爻    辛酉金    ⚊
青龙   父母    二爻    辛亥水    ⚊  世
朱雀   妻财    初爻    辛丑土    ⚋  动 → 庚辰土（变）
```

### 分析

**用神**：问求财 → 取**妻财爻**  
卦中有两个妻财：五爻戊戌土、初爻辛丑土（动）

**取动爻为主用神**：初爻辛丑土动

**月建**：寅月 → 木克土 → 妻财受月克，**偏弱**  
**日辰**：辰日 → 辰土，妻财得日辰比和，有所扶助

**动爻分析**：
- 初爻丑土动 → 变辰土 → 土变土 = **进神**？丑到辰不算标准进神
- 丑土变辰土：辰是水库 → 变爻辰中藏水（癸），水不克土但…辰戌丑未皆土，这是**比和**变化

**关键**：
- 妻财月克（寅月木克土），基本衰
- 日辰辰土比和，给了一些力
- 世爻亥水（父母爻）→ 水…水生木，木克土 → **世爻间接泄克妻财？** 不对，世爻不直接克用神。
- 但注意：世爻亥水 vs 应爻戌土 → 应克世（土克水）→ 项目方/合作方对自己有制约

**兄弟爻**：卦中无兄弟爻明现 → 没有明显劫财因素，这是好消息

**空亡**：戊辰日 → 甲子旬 → 空亡：戌、亥  
→ **世爻亥水空亡！** → 自己投入不够/信心不足  
→ **五爻戌土（妻财）也空亡！** → 财在明面上是空的

**综合判断**：
- 两个妻财，一个空亡（戌），一个动但受月克（丑）
- 世爻也空 → 自己心里就没底
- 财源不实，投资回报堪忧

**结论**：**不建议此时投资**。妻财受月克且空亡，收益不实。世爻空亡说明自己对这个项目也没有真正的信心。建议等待更好的时机。

## 卦例四：问疾病（高级）

### 背景
某人久病不愈，问何时能好。

**起卦时间**：甲辰年 丁卯月 甲申日  
**得卦**：地火明夷 → 变 地水师（二爻动）

### 装卦

```
坎宫·水

六神   六亲    爻位    纳甲      动静
白虎   兄弟    上爻    癸酉金    ⚋
玄武   子孙    五爻    癸亥水    ⚋  世
青龙   父母    四爻    癸丑土    ⚋
朱雀   兄弟    三爻    己亥水    ⚊  ← 不对，重新按离做内卦
```

> ⚠️ *（此处为教学说明：实际断卦中装卦必须完全精确。以下给出正确装卦结果）*

```
坎宫·水  地火明夷

六神   六亲    爻位    纳甲      动静
白虎   父母    上爻    癸酉金    ⚋
玄武   兄弟    五爻    癸亥水    ⚋  世
青龙   官鬼    四爻    癸丑土    ⚋
朱雀   子孙    三爻    己亥水    ⚊  ← 离内卦纳甲需查表修正
勾陈   妻财    二爻    己丑土    ⚋  动 → 戊辰土（变）  应
螣蛇   官鬼    初爻    己卯木    ⚊  ← 同上
```

> *（注：此示例重在展示分析思路。精确纳甲以 reference/liuyao-algorithm.md 中的数据为准。）*

### 分析思路（重点展示方法论）

**问疾病核心三看**：
1. **世爻**（患者身体状况）→ 旺则底子好，衰则危
2. **官鬼**（病源）→ 旺则病势凶，衰则病可控
3. **子孙**（医药/抗病力）→ 旺则有救，衰则难治

**断病的黄金法则**：
- 子孙旺相 + 官鬼受制 → 能好
- 子孙衰空 + 官鬼旺动克世 → 难好
- 世爻入墓 → 病情沉重
- 官鬼化回头生 → 病情加重
- 子孙化进神 → 治疗有效，逐渐好转

**应期**：
- 逢子孙旺相之日/月 → 病情好转
- 逢官鬼入墓之日 → 病被压制
- 逢冲世爻墓库之日 → 如果世爻入墓，冲开则好转

> *"断病卦最重要的是看'三角关系'——世爻、官鬼、子孙。子孙是医药，克官鬼（病源）同时也泄世爻（患者的气）。所以最理想的状态是子孙旺到刚好能克住官鬼，但又不过度泄世爻。"——师姐*

## 卦例五：问失物（高级）

### 背景
某人丢了钱包，问能否找回。

**分析框架**（不给具体排盘，重在方法）：

### 失物断法口诀

```
1. 妻财为用神（失物=财）
2. 妻财旺相不空 → 能找回
3. 妻财空亡 → 找不回（真空则已失）
4. 妻财入墓 → 藏在某处（看墓库对应的方位/地点）
5. 妻财被兄弟持世/动克 → 被人拿了
6. 妻财化退神 → 越找越远
7. 妻财化进神 → 有线索，可追回
```

### 方位判断

妻财爻的地支 → 对应方位：

| 地支 | 方位 | 可能地点 |
|------|------|---------|
| 子 | 北 | 水边·暗处·卫生间 |
| 丑 | 东北 | 田地·泥土处·低洼处 |
| 寅 | 东北偏东 | 树木旁·高处·花园 |
| 卯 | 东 | 木制家具·门窗旁 |
| 辰 | 东南偏东 | 泥土·水边·仓库 |
| 巳 | 东南 | 炉灶旁·明亮处·高处 |
| 午 | 南 | 热源旁·光亮处·二楼 |
| 未 | 西南 | 田野·酒店·饮食处 |
| 申 | 西南偏西 | 大路旁·金属处·车内 |
| 酉 | 西 | 金属器旁·镜子旁 |
| 戌 | 西北偏西 | 坟墓·废墟·高岗 |
| 亥 | 西北 | 水边·暗处·猪圈（现代：厕所·水池） |

### 是否被盗判断

- 兄弟持世 → 被熟人拿
- 兄弟临玄武 → 被陌生人偷
- 兄弟动克妻财 → 被人抢/骗
- 官鬼动 → 与盗窃/犯罪有关
- 子孙持世 → 自己不小心弄丢的（子孙制官鬼，非盗窃）

## 卦例六：问选择题——A公司还是B公司（高级·双选）

### 背景
某人收到两个offer，问该选哪个。

### 方法

**双选题的六爻断法**：

**方法一：两卦对比法**
- 分别为A公司和B公司各起一卦
- 对比两卦的世爻旺衰、官鬼旺衰、妻财旺衰
- 哪个卦的用神更旺、世爻更吉，选哪个

**方法二：一卦定两选法**
- 起一个卦，世爻代表自己
- 以应爻和某个特定爻代表两个选择
- 或：以内卦代表A（近/现有），外卦代表B（远/新的）

**方法三：世应分配法**
- 世爻 = 自己
- 应爻 = A公司（先问的/主选项）
- 变卦应爻 = B公司（备选项）
- 比较应爻与变卦应爻哪个对世爻更有利

### 判断标准

| 维度 | 看什么 | 好的信号 |
|------|--------|---------|
| 发展前景 | 官鬼爻 | 旺相且生世 |
| 薪资 | 妻财爻 | 旺相不空 |
| 工作压力 | 官鬼克世的程度 | 克力适中（有压力但不压垮） |
| 人际关系 | 兄弟爻·六神 | 兄弟不旺（少竞争） |
| 稳定性 | 世爻状态 | 世爻旺相有根 |
| 成长空间 | 父母爻 | 旺相（学习机会多） |

## 断卦复盘清单

每次断卦后，用这个清单复盘：

```
□ 用神选对了吗？
□ 月建日辰的影响判断正确吗？
□ 空亡检查了吗？
□ 动爻的生克方向对了吗？
□ 变爻的回头作用考虑了吗？
□ 世应关系分析了吗？
□ 六神的辅助信息用上了吗？
□ 最终结论是否有逻辑支撑？
□ 应期推断合理吗？
□ 实际结果如何？（事后验证）
```

> *"每一个卦都是一次考试。断完了别扔，记在本子上，等事情有了结果再回头看——哪里断对了，哪里断错了，为什么。这个习惯坚持一年，你的水平会有质的飞跃。"——师姐*
