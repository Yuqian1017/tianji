# 附录 · 择日基础

> *"择日是所有术数最直接的实战应用——不是算过去，不是看性格，而是选未来。"——师父*

---

## 一、择日的核心逻辑

择日 = 选择一个好的时空节点去做某件事。

**核心原则**：所选之日的干支五行配置，应当与事主命局的用神相合、忌神相避。

### 择日三要素

```
天时 → 选年月日时的干支组合
地利 → 选方位（奇门遁甲的应用领域）
人和 → 与事主命局匹配
```

---

## 二、基础择日法：十二建除

十二建除（十二值日）是最基础的择日体系：

| 值日 | 含义 | 宜 | 忌 |
|------|------|-----|-----|
| **建** | 建立·创始 | 创业·上任·动工 | 破土·安葬 |
| **除** | 除旧·清理 | 就医·打扫·解除 | 开业·嫁娶 |
| **满** | 圆满·丰盈 | 嫁娶·开业·祈福 | 动土·开仓 |
| **平** | 平常·持平 | 修造·布道 | 嫁娶·出行 |
| **定** | 安定·确立 | 嫁娶·开市·签约 | 出行·诉讼 |
| **执** | 执守·坚持 | 建造·种植·捕捉 | 搬家·开市 |
| **破** | 破裂·冲突 | 拆卸·求医 | 嫁娶·签约·一切吉事 |
| **危** | 危险·警惕 | 祭祀·安床 | 出行·登高·一切大事 |
| **成** | 成就·完成 | 嫁娶·开市·签约·搬家 | 诉讼 |
| **收** | 收获·归纳 | 收账·收割·嫁娶 | 动土·开渠 |
| **开** | 开放·开通 | 开业·出行·动工·嫁娶 | 安葬 |
| **闭** | 关闭·终止 | 安葬·修墙·闭户 | 开市·出行·嫁娶 |

### 十二建除的排法

以月建（月支）为「建」日起点，地支顺排：

```
例：寅月（正月）
寅日 = 建    卯日 = 除    辰日 = 满    巳日 = 平
午日 = 定    未日 = 执    申日 = 破    酉日 = 危
戌日 = 成    亥日 = 收    子日 = 开    丑日 = 闭
```

```javascript
function getTwelveDay(monthBranch, dayBranch) {
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  const labels = ['建','除','满','平','定','执','破','危','成','收','开','闭'];
  
  const monthIdx = branches.indexOf(monthBranch);
  const dayIdx = branches.indexOf(dayBranch);
  const offset = (dayIdx - monthIdx + 12) % 12;
  
  return labels[offset];
}
```

## 三、天干地支择日要点

### 天德·月德

天德月德为两颗贵人星，所临之日适合行事。

**月德**：

| 月支 | 月德干 |
|------|--------|
| 寅午戌月 | 丙 |
| 申子辰月 | 壬 |
| 亥卯未月 | 甲 |
| 巳酉丑月 | 庚 |

**天德**：

| 月支 | 天德干 |

| 四月(巳) | 辛 |
| 五月(午) | 亥 |

### 择日忌讳

| 凶日 | 说明 | 忌 |
|------|------|-----|
| **四离日** | 春分·秋分·夏至·冬至前一天 | 百事忌 |
| **四绝日** | 立春·立夏·立秋·立冬前一天 | 百事忌 |
| **杨公忌日** | 每月固定几天（正月十三等） | 民间忌 |
| **月破日** | 日支与月支相冲 | 百事忌 |
| **岁破日** | 日支与年支相冲 | 重大事忌 |
| **三煞日** | 与年三煞方位对应之日 | 动土忌 |

### 与命主八字配合

```
择日黄金法则：
1. 所选日的日干应为命主的喜用神
2. 所选日不冲命主年柱和日柱
3. 所选日不在命主空亡之内
4. 日课（所选时间的四柱）本身要协调
```

## 四、嫁娶择日

嫁娶是最常见的择日需求：

### 核心原则

1. **女命为主**：以新娘八字为主选日
2. **大利月**：以女方属相定大利月和小利月

| 女方属相 | 大利月 | 小利月 |
|---------|--------|--------|
| 鼠 | 六月·十二月 | 正月·七月 |
| 牛 | 五月·十一月 | 四月·十月 |
| 虎 | 二月·八月 | 三月·九月 |
| 兔 | 正月·七月 | 六月·十二月 |
| 龙 | 四月·十月 | 五月·十一月 |
| 蛇 | 三月·九月 | 二月·八月 |
| 马 | 六月·十二月 | 正月·七月 |
| 羊 | 五月·十一月 | 四月·十月 |
| 猴 | 二月·八月 | 三月·九月 |
| 鸡 | 正月·七月 | 六月·十二月 |
| 狗 | 四月·十月 | 五月·十一月 |
| 猪 | 三月·九月 | 二月·八月 |

3. **忌冲新娘日柱**
4. **宜用建·满·定·成·开日**
5. **忌破日·四离四绝**

## 五、开业/搬家/动工择日要点

### 开业

- 宜：建·满·成·开日
- 忌：破·闭·收日
- 日干宜为命主的财星或食伤（生财之意）
- 方位宜合奇门生门·开门方位

### 搬家

- 宜：成·开·满·定日
- 忌：破·闭·平日
- 不冲新居坐向
- 天德月德入日最佳

### 动工（建造/装修）

- 宜：定·执·成·开日
- 忌：破·危日
- 不动太岁方（年支对应的方位当年不宜动土）
- 查三煞方位

## 六、本课心法

> *"择日的精髓不在于找一个完美的日子——完美的日子不存在。而在于找一个对你来说最不坏的日子，同时避开所有大忌。"——师父*

1. **避凶优先**：先排除所有凶日，再从剩余中选吉日
2. **因人而异**：同一天对不同人可能吉凶不同
3. **因事而异**：嫁娶和开业的择日标准不同
4. **简则灵**：过度追求完美反而选不出日子，抓主要矛盾即可
