# 奇门遁甲排盘算法规格

> **验证状态（2026-07-10）**
> 确定性排盘：`accepted_declared_school`
> 传统格局、用神、吉凶与事件预测：`not_validated`
> 规范数据：`database/qimen/qimen-core.json`

本规格只定义一套可复算口径：**时家奇门、拆补法、转盘法、中五寄坤二、天禽随天芮**。置闰、茅山、飞盘及其他寄宫规则必须另建版本，不能静默混入。

## 二、节气、三元与局数

### 精确节气边界

奇门使用二十四节气的**精确交节时刻**。交节日不能从 00:00 整天切换，也不能用固定月日近似。

```javascript
const solar = Solar.fromYmdHms(year, month, day, hour, minute, 0);
const currentJieqi = solar.getLunar().getPrevJieQi(false);
```

例如本项目历法适配器给出的 2024 年边界：

- 芒种：`2024-06-05 12:09:54`。12:09 仍属小满，12:10 起属芒种。
- 夏至：`2024-06-21 04:51:00`。04:50 仍为阳遁芒种局，04:51 起为阴遁夏至局。

### 拆补法定三元

每一天向前找到最近的甲日或己日，称为符头。按符头地支定元：

```javascript
const YUAN_BY_FUTOU_BRANCH = {
  子:'上元', 午:'上元', 卯:'上元', 酉:'上元',
  寅:'中元', 申:'中元', 巳:'中元', 亥:'中元',
  辰:'下元', 戌:'下元', 丑:'下元', 未:'下元'
};
```

这与《奇门遁甲统宗》《奇门法窍》所述“甲己为符头，子午卯酉上元、寅申巳亥中元、辰戌丑未下元”的规则一致。旧稿“从节气日算天数再除以 5”不成立，已废止。

## 三、地盘

地盘包含中五宫，按九宫数字顺序排三奇六仪：

```javascript
function layoutDipan(juNum, dunType) {
  return Object.fromEntries(SANQI_LIUYI_ORDER.map((stem, offset) => {
    const delta = dunType === 'yang' ? offset : -offset;
    const palace = ((juNum - 1 + delta + 18) % 9) + 1;
    return [palace, stem];
  }));
}
```

- 阳遁 N 局：戊从 N 宫起，按 `1→2→...→9→1` 顺排。
- 阴遁 N 局：戊从 N 宫起，按 `9→8→...→1→9` 逆排。
- 乙不固定在中五宫；它所在宫位由局数和阴阳遁共同决定。

## 四、旬首、值符和值使

### 旬首

把时干支放入六十甲子，十个一旬。旬首和遁干为：

| 旬首 | 遁干 |
|---|---|
| 甲子 | 戊 |
| 甲戌 | 己 |
| 甲申 | 庚 |
| 甲午 | 辛 |
| 甲辰 | 壬 |
| 甲寅 | 癸 |

甲时必须按完整时干支取对应遁干，不能把所有甲时都当作甲子戊。

### 值符和值使原位

1. 找旬首遁干在地盘的宫位。
2. 该宫本位九星为值符。
3. 该宫本位八门为值使；旬首遁干在中五时，本口径以坤二死门为值使原位。

### 值符落宫

1. 非甲时，以时干在地盘的宫位为值符落宫。
2. 甲时，以本旬遁干在地盘的宫位为值符落宫。
3. 若落中五，本口径寄坤二。

### 值使落宫

值使不直接跟随值符落宫。先计算时干支在本旬内距旬首的步数：

1. 从值使原位起数；中五原位先寄坤二。
2. 阳遁按九宫数字顺数，阴遁逆数。
3. 若结果落中五，阳遁寄艮八、阴遁寄坤二。
4. 再以值使门为起点，沿八宫几何顺序布其余七门。

## 五、三盘转动

### 天盘九星与天盘干

值符星从原位转到值符落宫，其余七颗外围星保持相对位置同步转动；各星携带原宫地盘干形成天盘干。

本口径中：

- 天禽保留中五身份，同时随天芮同宫。
- 天禽携带中五地盘干；天芮携带坤二地盘干。
- 中五不参与八宫几何旋转。

### 人盘八门

```javascript
const GATE_SEQUENCE = ['休门','生门','伤门','杜门','景门','死门','惊门','开门'];
```

从值使落宫起，沿八宫顺时针几何顺序布门。这里的顺时针是盘面几何顺序，不是九宫数字顺逆。

### 神盘八神

- 值符神从值符星落宫起。
- 阳遁沿八宫顺时针排 `值符→螣蛇→太阴→六合→白虎→玄武→九地→九天`。
- 阴遁沿八宫逆时针排同一序列。

## 七、验证证据与边界

- 项目内独立公式：2020-2024 共 23,751 张盘，1,900,230 项检查，0 fail。
- `3meta@2.6.0`：23,751 张盘的起局、值符值使、地/天/人/神盘核心字段 0 mismatch；其立春日年柱按整日提前切换的 38 个差异另行记录，不影响奇门核心。
- `kinqimen@0.3.1`：2024 年共享口径字段 189,789 项 0 mismatch；其值使门算法不同，相关字段被明确排除，不能作为本口径八门证据。

通过排盘不等于通过断事。以下仍为 `not_validated`：

- 星、门、神的吉凶和人物、疾病、职业映射。
- 格局名称、克应、用神与应期。
- 方位、出行、投资、诉讼、医疗和关系建议。
- 任何具体事件预测。
