## 二、三大主线

### 生命线（地纹）

```
起点：拇指和食指之间
走向：围绕拇指丘向手腕方向弧形延伸

看什么：
  长度 → 不代表寿命长短！代表生命力·精力·体质
  深浅 → 深而清晰 = 精力充沛·体质好
          浅而模糊 = 体质偏弱·需注意健康
  弧度 → 弧度大（远离拇指）= 精力旺盛·活动范围大
          弧度小（贴近拇指）= 宅·精力一般
  分支 → 向上分支 = 积极向上的变化
          向下分支 = 精力分散
  断裂 → 不代表早亡！代表生活重大变化（换工作/搬家/大病后恢复）
  岛纹 → 该阶段身体虚弱或困顿
  链状 → 体质差·从小多病
```

### 智慧线（人纹·头脑线）

```
起点：食指下方（与生命线同起点或略上）
走向：横向或斜向延伸至手掌中部或对侧

看什么：
  长度 → 长 = 思维缜密·善于分析
          短 = 直觉型·行动派·不爱深想
  深浅 → 深 = 专注力强·意志坚定
          浅 = 注意力分散·想法多但不持久
  走向 → 直线（横穿）= 理性·逻辑·适合理工商
          下弯（向月丘）= 感性·想象力·适合艺术创意
          大幅下弯 = 极度浪漫·脱离现实倾向
  起点 → 与生命线相连 = 谨慎·依赖家庭
          与生命线分离 = 独立·大胆·自主
  分叉 → 尾端分叉（作家叉）= 善于表达·多才多艺
  岛纹 → 该阶段精神压力大·注意力下降
  断裂 → 思维方式重大转变
```

### 感情线（天纹）

```
起点：小指下方
走向：横向延伸至食指或中指方向

看什么：
  长度 → 延伸到食指下 = 重感情·理想主义·完美主义
          止于中指下 = 以自我为中心的感情观
          在食指中指间 = 平衡型·最理想
  深浅 → 深而清晰 = 感情深厚·重感情
          浅而细 = 感情淡薄·理性对待感情
  走向 → 平直 = 理性看待感情·不太浪漫
          上弯 = 热情·主动·感情丰富
          下弯 = 被动·压抑感情
  分支 → 尾端多分支 = 多情·桃花多
          无分支 = 专一
  链状 → 感情经历多波折
  断裂 → 感情重大创伤或观念巨变
  双重感情线 → 感情强烈·精力旺盛
```

## 三、重要辅助线

### 命运线（事业线·玉柱纹）

```
位置：掌心纵向，从手腕向中指方向延伸
含义：事业·命运走向·社会地位

有且清晰 → 事业目标明确·有方向感
没有 → 不代表没事业！代表自由职业型·不走传统路线
起于手腕中央 → 白手起家·靠自己
起于生命线 → 家族帮助·受家庭影响
起于月丘（小指侧掌底）→ 贵人运·适合公众事业
起于智慧线 → 中年（35左右）才找到方向
起于感情线 → 晚成型·50后才稳定
断断续续 → 事业多变·跳槽多
```

### 太阳线（成功线）

```
位置：无名指下方的纵向线
含义：名声·成就·满足感·艺术才华

有且清晰 → 有名望·事业有成就感·不一定有钱但有满足
没有 → 默默耕耘型·不求名
从感情线起 → 中晚年成名
从智慧线起 → 靠才华成名
越长越好
```

### 婚姻线（结合线）

```
位置：小指根部与感情线之间的短横线
含义：重要感情关系（不一定是法律婚姻）

一条深而清晰 → 一段重要感情·稳定
两条或以上 → 多段重要感情经历
长 → 感情持久深厚
短 → 短暂但深刻的关系
上翘 → 对感情要求高·可能不婚
下弯 → 对方可能先离开
分叉 → 有分离可能
```

### 健康线

```
位置：从小指下方斜向生命线方向
含义：健康状况（反向指标——没有最好）

没有 → 好事！代表健康状况佳
有且波浪/断续 → 消化系统·肝胆问题
有且清晰 → 需注意健康·神经紧张型
```

## 五、指节与指形

### 手指长度

```
食指长（接近中指）→ 领导力强·野心大
食指短 → 缺自信·服从型
无名指长（超过中指第一节中线）→ 冒险·赌性·艺术
无名指短 → 保守·稳健
小指长（超过无名指第一关节）→ 口才好·善沟通
小指短 → 表达能力弱·但可能很真诚
拇指大而粗 → 意志力强·有主见
拇指小而细 → 优柔寡断·感性
```

### 指缝

```
手指并拢时：
  无缝隙 → 善于理财·不浪费
  有缝隙 → 花钱大方·存不住钱
  小指特别歪/远离无名指 → 独立·与众不同
```

## 六、左右手

```
传统说法：男左女右
现代手相学：以常用手（惯用手）为主

常用手（右手居多）→ 后天发展·当前状态·自己创造的命运
非常用手 → 先天禀赋·潜意识·遗传特质

两手差异大 → 后天经历与先天设定差异大·自我改变能力强
两手相似 → 按先天轨迹发展·稳定型
```

## 七、AI手相分析关键点

```
AI通过摄像头或照片分析时，重点看：
1. 手型判断（五行归类）
2. 三大主线（生命线·智慧线·感情线的长度·深浅·走向·特征）
3. 命运线有无及状态
4. 掌丘隆起/凹陷情况
5. 手指比例
6. 特殊纹路（岛纹·链纹·星纹·十字纹等）
7. 综合给出性格·事业·感情·健康·财运简评

拍摄要求：
- 掌心朝上·手指自然张开
- 光线充足均匀·避免阴影
- 尽量垂直拍摄（减少透视变形）
- 拍常用手（惯用手）为主
```

> ⚠️ **重要声明**：手相分析基于传统文化经验，不是科学诊断。掌纹会随年龄和经历变化。AI分析受拍摄质量影响，仅供趣味参考。
