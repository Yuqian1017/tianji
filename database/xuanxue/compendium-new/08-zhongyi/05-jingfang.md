# 中医篇 · 第五课 · 方剂基础与十大经方

> **难度标签**：🟡进阶：君臣佐使 | 🔴专研：十大经方·方剂五行

> *"开方就像排兵布阵——君臣佐使，各司其职。一个方剂就是一支小部队，每味药都有明确的任务。"——师父*

## 一、方剂组织原则：君臣佐使

| 角色 | 功能 | 用量 | 比喻 |
|------|------|------|------|
| **君药** | 针对主证·主要治疗作用 | 最大 | 主帅 |
| **臣药** | 辅助君药·加强疗效 | 次之 | 副将 |
| **佐药** | 佐助（协助）·佐制（制约毒性）·反佐 | 较小 | 参谋 |
| **使药** | 引经药（引导药力到达病所）·调和药 | 最小 | 传令兵 |

## 三、十大经方详解

### 1. 四君子汤 — 补气基础方

```
【主治】脾胃气虚：面色萎白·气短乏力·食少便溏·舌淡苔白
【组成】
  君：人参 10g  — 大补元气·健脾
  臣：白术 10g  — 健脾燥湿
  佐：茯苓 10g  — 渗湿健脾
  使：甘草  5g  — 调和诸药·益气

【五行解读】全方入脾（土），甘味为主，专补后天之本
【衍生方】六君子汤（+陈皮半夏）→ 香砂六君子汤（+木香砂仁）
```

### 2. 四物汤 — 补血基础方

```
【主治】血虚：面色苍白·头晕眼花·心悸·月经量少色淡
【组成】
  君：熟地 15g — 滋阴补血·填精
  臣：当归 10g — 补血活血·调经
  佐：白芍 10g — 养血柔肝·敛阴
  使：川芎  5g — 活血行气·引药入血分

【五行解读】入肝（木）·心（火），养血为主
【名言】"四物汤为妇科第一方"
```

### 3. 逍遥散 — 疏肝名方

```
【主治】肝气郁结·脾虚：胁肋胀痛·情绪抑郁·食少倦怠·月经不调
【组成】
  君：柴胡 10g  — 疏肝解郁
  臣：当归 10g  — 养血补肝
  臣：白芍 10g  — 柔肝缓急
  佐：白术 10g  — 健脾
  佐：茯苓 10g  — 渗湿健脾
  使：甘草  5g  — 调和
  佐：薄荷  3g  — 助柴胡疏肝
  佐：生姜  3片 — 温中和胃

【五行解读】木（肝）郁克土（脾）→ 疏木（柴胡）+ 补土（白术茯苓）+ 养木本体（当归白芍）
【衍生方】加味逍遥散（+丹皮栀子 → 清肝热）
```

### 4. 六味地黄丸 — 滋阴代表

```
【主治】肾阴虚：腰膝酸软·头晕耳鸣·盗汗·五心烦热·舌红少苔
【组成】三补三泻
  君：熟地 24g — 补肾阴填精（补肾）
  臣：山萸肉12g — 补肝涩精（补肝）
  臣：山药 12g — 补脾固精（补脾）
  佐：泽泻  9g — 泻肾火利湿（泻肾浊）
  佐：丹皮  9g — 泻肝火（泻肝热）
  佐：茯苓  9g — 渗脾湿（泻脾湿）

【五行解读】水（肾）不足 → 补水（熟地）+ 补生水之源（山萸补肝·肝木含水）+ 补制水之土（山药健脾·使土不过克水）
【衍生方】
  知柏地黄丸（+知母黄柏·加强清虚热）
  杞菊地黄丸（+枸杞菊花·明目）
  金匮肾气丸（+附子肉桂·阴中求阳）
```

### 5. 桂枝汤 — 万方之祖

```
【主治】外感风寒表虚：恶风·发热·汗出·头痛·鼻塞·脉浮缓
【组成】
  君：桂枝  9g — 解肌发表·温通经脉
  臣：白芍  9g — 敛阴和营
  佐：生姜  9g — 助桂枝散寒
  佐：大枣  3枚— 助白芍养营
  使：甘草  6g — 调和营卫

【经典意义】桂枝辛温走表·白芍酸收敛内 → 一散一收·调和营卫
【名言】仲景第一方，"群方之冠"
```

### 6. 麻黄汤 — 发汗峻剂

```
【主治】外感风寒表实：恶寒·发热·无汗·身痛·脉浮紧
【组成】
  君：麻黄  9g — 发汗解表·宣肺
  臣：桂枝  6g — 助麻黄发汗·温经
  佐：杏仁  6g — 降肺气·止咳
  使：甘草  3g — 调和·缓麻黄峻烈

【与桂枝汤区别】
  桂枝汤：有汗（表虚）→ 调和营卫
  麻黄汤：无汗（表实）→ 发汗解表
```

### 7. 小柴胡汤 — 和解少阳

```
【主治】少阳证：往来寒热·胸胁苦满·口苦咽干·目眩·脉弦
【组成】
  君：柴胡 24g — 疏散少阳邪气
  臣：黄芩  9g — 清少阳郁热
  佐：半夏  9g — 和胃降逆
  佐：人参  6g — 补正气防邪深入
  佐：生姜  9g — 助半夏和胃
  佐：大枣  4枚— 补中益气
  使：甘草  6g — 调和诸药

【经典意义】半表半里之方，一散（柴胡）一清（黄芩）一补（人参）一降（半夏）
```

### 8. 血府逐瘀汤 — 活血化瘀代表

```
【主治】胸中血瘀：胸痛如刺·头痛日久不愈·心悸失眠·急躁
【组成】
  君：桃仁 12g·红花 9g — 活血化瘀
  臣：当归 9g·川芎 5g·赤芍 6g — 养血活血
  佐：柴胡 3g·枳壳 6g — 疏肝理气（气行血行）
  佐：牛膝 9g — 引血下行
  佐：生地 9g — 凉血养阴
  使：桔梗 5g·甘草 3g — 引药入胸·调和

【核心思路】活血（桃红） + 养血（归芎芍） + 行气（柴枳） = 化瘀不伤正
```

### 9. 补中益气汤 — 升阳举陷

```
【主治】脾胃气虚·气陷：食少体倦·气短懒言·面色萎黄·内脏下垂
【组成】
  君：黄芪 15g — 补中益气·升阳固表
  臣：人参  5g · 白术 10g · 甘草 5g — 补脾益气
  佐：当归 10g — 养血
  佐：陈皮  6g — 理气（补而不滞）
  使：升麻  3g · 柴胡  3g — 升举阳气

【名方名言】李东垣名方，"甘温除大热"（用补气药治发烧）
```

### 10. 温胆汤 — 理气化痰

```
【主治】痰热内扰：胆怯易惊·虚烦不眠·恶心·口苦·舌苔腻
【组成】
  君：半夏  6g — 燥湿化痰·降逆和胃
  臣：竹茹  6g — 清热化痰
  臣：枳实  6g — 行气消痰
  佐：陈皮  9g — 理气化痰
  佐：茯苓  5g — 健脾渗湿
  使：甘草  3g — 调和
  引：生姜  5片·大枣 1枚 — 调和脾胃

【应用广泛】失眠·焦虑·梅核气·痰多·眩晕皆可加减使用
```

## 四、方剂加减的五行逻辑

```
基本原则：
- 虚则补其母（如肝虚→补肾水，水生木）
- 实则泻其子（如肝火旺→泻心火，木之子为火）
- 抑强扶弱（如肝木太旺克脾土→疏肝+健脾）

加减举例：
逍遥散（疏肝健脾基础方）
  + 肝火旺 → 加丹皮·栀子（泻肝火）= 加味逍遥散
  + 血虚重 → 加熟地·首乌（补肝血）
  + 脾虚重 → 加黄芪·党参（加强补土）
  + 有瘀血 → 加丹参·桃仁（活血化瘀）
```
