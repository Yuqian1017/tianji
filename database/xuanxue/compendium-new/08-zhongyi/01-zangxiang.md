# 中医篇 · 第一课 · 藏象与五行医学

> **难度标签**：🟢必修：五脏五行对应 | 🟡进阶：气血·病因·辨证 | 🔴专研：四诊详细

> *师父从柜子里翻出一本泛黄的《黄帝内经》。*
> *"你学了这么多术数，有没有想过——古人为什么要发明这些系统？"*
> *"为了预测未来？"*
> *"不，为了治病。阴阳五行最早的应用不是算命，而是医学。《黄帝内经》成书比任何算命书都早。中医是五行学说最严密、最系统的实践场。"*
> *"所以学中医是为了回到源头？"*
> *"是为了让你真正理解五行不是抽象概念，而是活生生的身体感受。你说'肝属木'的时候，不应该只是背下来——你应该知道为什么是木，木的生发条达和肝的疏泄功能是怎么对应的。"*

## 一、中医的哲学基础

### 整体观

中医把人体看作一个**整体系统**——不是零件的组合，而是一个动态平衡的有机体。

```
核心观点：
1. 人体是一个整体（脏腑经络互相关联）
2. 人与自然是一个整体（天人合一）
3. 人与社会是一个整体（情志影响身体）
```

### 辨证论治

中医不是「头痛医头脚痛医脚」，而是**辨证**——辨别疾病的「证型」（本质），再针对证型治疗。

```
同病异治：同样是头痛，风寒头痛用辛温药，风热头痛用辛凉药
异病同治：不同的病，如果证型相同，可以用同一个方
```

## 二、脏腑理论（藏象学说）

### 五脏

| 脏 | 五行 | 主要功能 | 开窍 | 主体 | 志 | 液 | 华 |
|----|------|---------|------|------|-----|-----|-----|
| **肝** | 木 | 主疏泄·藏血·主筋 | 目 | 筋 | 怒 | 泪 | 爪(指甲) |
| **心** | 火 | 主血脉·藏神·主神志 | 舌 | 脉 | 喜 | 汗 | 面 |
| **脾** | 土 | 主运化·统血·主肌肉 | 口 | 肉 | 思 | 涎 | 唇 |
| **肺** | 金 | 主气·主宣降·主皮毛 | 鼻 | 皮 | 悲 | 涕 | 毛 |
| **肾** | 水 | 藏精·主水·主骨·纳气 | 耳/二阴 | 骨 | 恐 | 唾 | 发 |

### 六腑

| 腑 | 配脏 | 功能 |
|----|------|------|
| 胆 | 肝 | 贮藏排泄胆汁·主决断 |
| 小肠 | 心 | 受盛化物·分清泌浊 |
| 胃 | 脾 | 受纳腐熟水谷·喜润恶燥 |
| 大肠 | 肺 | 传导糟粕·主津 |
| 膀胱 | 肾 | 贮尿排尿·气化津液 |
| 三焦 | 心包 | 通行元气·运行水液 |

### 五脏之间的五行关系

```
         肝(木)
        ↗    ↘
    肾(水)    心(火)
        ↑      ↓
    肺(金) ← 脾(土)

生：肾水→肝木→心火→脾土→肺金→肾水
克：肾水→心火→肺金→肝木→脾土→肾水
```

**临床意义**：

| 关系 | 生理 | 病理 |
|------|------|------|
| 木生火 | 肝血养心，肝气助心行血 | 肝火上炎→心火亢盛 |
| 火生土 | 心阳温脾，助脾运化 | 心火不足→脾胃虚寒 |
| 土生金 | 脾运化精微，上输于肺 | 脾虚→肺气不足（培土生金） |
| 金生水 | 肺气肃降，通调水道助肾 | 肺虚→肾水不足 |
| 水生木 | 肾精滋养肝血 | 肾阴不足→肝阳上亢 |
| 木克土 | 肝气疏泄助脾运化 | 肝气犯胃（木克土太过） |
| 土克水 | 脾运化水湿防肾水泛滥 | 脾虚→水湿泛滥 |
| 水克火 | 肾水上济心火防亢 | 肾水不足→心火亢盛（心肾不交） |
| 火克金 | 心火制约肺金防过敛 | 心火→灼肺（咳血） |
| 金克木 | 肺金肃降制约肝木升发太过 | 肺虚→肝气上逆 |

## 三、气血津液

### 气

| 气的分类 | 来源 | 功能 |
|---------|------|------|
| **元气** | 先天之精（肾）+ 后天水谷 | 推动生长发育·生命根本动力 |
| **宗气** | 天气（吸入）+ 谷气（脾运化） | 贯心脉行气血·走息道司呼吸 |
| **营气** | 水谷精微之精华 | 营养全身·化生血液 |
| **卫气** | 水谷精微之悍烈部分 | 护卫肌表·温养脏腑·开合汗孔 |

### 气的功能

```
推动：推动血液运行、津液输布
温煦：维持体温
防御：抵御外邪（卫气）
固摄：统摄血液（脾统血）、固摄津液（防汗尿失禁）
气化：物质转化（气→血→津液等的相互转化）
```

### 血

- 由营气和津液在心肺的作用下化生
- 心主血脉（推动）、肝藏血（贮藏）、脾统血（防止出血）
- 气为血之帅（气推动血行）、血为气之母（血载气运行）

### 津液

- 津：清稀·流动性大·布于体表皮肤肌肉
- 液：浓稠·流动性小·灌注于关节脑髓

## 四、病因学

### 六淫（外感病因）

| 邪气 | 性质 | 好发季节 | 侵犯脏腑 | 典型症状 |
|------|------|---------|---------|---------|
| **风** | 善行数变·为百病之长 | 春 | 肝·肌表 | 头痛·鼻塞·游走性疼痛 |
| **寒** | 凝滞·收引·伤阳 | 冬 | 肾·脾 | 恶寒·疼痛（得温则减）·腹泻 |
| **暑** | 炎热·升散·多挟湿 | 夏 | 心 | 高热·汗多·口渴·乏力 |
| **湿** | 重浊·黏滞·趋下 | 长夏 | 脾 | 身重·腹胀·大便黏·苔腻 |
| **燥** | 干涩·伤津 | 秋 | 肺 | 口干·鼻干·皮肤干·干咳 |
| **火(热)** | 炎上·伤津·动血 | 夏 | 心 | 高热·烦躁·出血·疮疡 |

### 七情（内伤病因）

| 情志 | 对应脏 | 气机变化 | 过度后果 |
|------|--------|---------|---------|
| **喜** | 心 | 气缓 | 心神涣散·注意力差 |
| **怒** | 肝 | 气上 | 头痛·眩晕·吐血·中风 |
| **忧/悲** | 肺 | 气消 | 气短·乏力·意志消沉 |
| **思** | 脾 | 气结 | 食欲不振·腹胀·消瘦 |
| **恐** | 肾 | 气下 | 二便失禁·腰膝酸软 |
| **惊** | 心 | 气乱 | 心悸·失眠·精神恍惚 |

## 五、四诊概要

### 望诊

| 望什么 | 看什么 | 意义 |
|--------|--------|------|
| 望神 | 两目·精神·反应 | 有神=正气未衰；无神=正气大虚 |
| 望面色 | 赤白青黄黑 | 赤=热·白=寒/虚·青=痛/瘀·黄=湿/脾虚·黑=肾虚/寒 |
| 望舌 | 舌质+舌苔 | 淡红舌薄白苔=正常 |
| 望形体 | 胖瘦·姿态 | 形盛气虚=痰湿体质 |

### 舌诊速查

| 舌象 | 意义 |
|------|------|
| 淡白舌 | 气血不足·阳虚 |
| 红舌 | 热证（实热或阴虚） |
| 绛舌 | 热入营血·重症 |
| 紫舌 | 瘀血·寒凝 |
| 白苔 | 表证·寒证 |
| 黄苔 | 里证·热证 |
| 腻苔 | 湿浊·痰饮 |
| 剥苔 | 胃阴不足 |

### 闻诊

```
听声音：声高气粗=实证，声低气怯=虚证
嗅气味：口臭=胃热，体臭=湿热
```

### 问诊（十问歌）

```
一问寒热二问汗，三问头身四问便，
五问饮食六问胸，七聋八渴俱当辨，
九问旧病十问因，再兼服药参机变，
妇人尤必问经期，迟速闭崩皆可见。
```

### 切诊（脉诊基础）

| 常见脉象 | 特征 | 意义 |
|---------|------|------|
| **浮脉** | 轻取即得 | 表证·风邪 |
| **沉脉** | 重按才得 | 里证 |
| **迟脉** | 一息不足四至(＜60次/分) | 寒证 |
| **数脉** | 一息五至以上(＞90次/分) | 热证 |
| **虚脉** | 三部脉举按皆无力 | 虚证 |
| **实脉** | 三部脉举按皆有力 | 实证 |
| **滑脉** | 往来流利如珠 | 痰饮·食积·妊娠 |
| **涩脉** | 往来艰涩不畅 | 瘀血·精血不足 |
| **弦脉** | 端直以长如按琴弦 | 肝病·痛证·痰饮 |
| **细脉** | 脉细如线 | 气血不足·湿证 |

## 六、八纲辨证

```
         ┌─ 表证（病在肌表·六淫初侵）
阴阳 ──→ ├─ 里证（病在脏腑·较深）
  │     ├─ 寒证（阳虚或寒邪·恶寒·面白·脉迟）
  │     ├─ 热证（阴虚或热邪·发热·面赤·脉数）
  │     ├─ 虚证（正气不足·乏力·脉虚）
  └──→  └─ 实证（邪气盛·腹胀拒按·脉实）

阳证 = 表·热·实
阴证 = 里·寒·虚
```

### 常见证型速查

| 证型 | 症状 | 治法 | 代表方 |
|------|------|------|--------|
| 风寒感冒 | 恶寒重·发热轻·无汗·鼻塞流清涕 | 辛温解表 | 麻黄汤/桂枝汤 |
| 风热感冒 | 发热重·恶寒轻·咽痛·黄涕 | 辛凉解表 | 银翘散 |
| 肝气郁结 | 胁肋胀痛·情绪抑郁·叹气·月经不调 | 疏肝理气 | 逍遥散 |
| 脾胃虚弱 | 食少腹胀·便溏·乏力·面黄 | 健脾益气 | 四君子汤 |
| 肾阳虚 | 腰膝冷痛·畏寒·夜尿多·阳痿 | 温补肾阳 | 金匮肾气丸 |
| 肾阴虚 | 腰膝酸软·五心烦热·盗汗·口干 | 滋补肾阴 | 六味地黄丸 |
| 心肾不交 | 失眠·心烦·健忘·腰酸 | 交通心肾 | 交泰丸 |
| 血瘀证 | 固定刺痛·舌暗有瘀斑·肿块 | 活血化瘀 | 血府逐瘀汤 |

## 七、本课心法

> *"中医和术数用的是同一套语言——阴阳五行。但中医是最'接地气'的应用。你说肝木克脾土，在八字里是抽象的十神关系，在中医里就是一个病人情绪不好（肝气郁结）导致吃不下饭（脾胃失调）。"——师父*

1. **整体观**：不看局部看整体，不看症状看证型
2. **五行是活的**：脏腑之间的生克不是机械的，是动态平衡
3. **治未病**：最高明的中医是在你还没生病的时候就调理好
