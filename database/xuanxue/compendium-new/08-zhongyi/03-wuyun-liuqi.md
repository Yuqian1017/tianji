## 一、基本框架

五运六气 = **五运**（天干主运）+ **六气**（地支主气）

```
五运 → 天干 → 管全年的五行偏性（旺什么·衰什么）
六气 → 地支 → 管全年分六段的气候变化（风寒暑湿燥火）
两者叠加 → 该年的综合气候特征和易发疾病
```

## 二、五运

### 天干化运

| 天干 | 化运 | 五行 | 太过/不及 |
|------|------|------|----------|
| 甲·己 | 土运 | 土 | 甲=太过·己=不及 |
| 乙·庚 | 金运 | 金 | 庚=太过·乙=不及 |
| 丙·辛 | 水运 | 水 | 丙=太过·辛=不及 |
| 丁·壬 | 木运 | 木 | 壬=太过·丁=不及 |
| 戊·癸 | 火运 | 火 | 戊=太过·癸=不及 |

**规律**：阳干=太过，阴干=不及。

```javascript
const TIANGAN_YUN = {
  '甲': { yun:'土', excess:true },  '己': { yun:'土', excess:false },
  '乙': { yun:'金', excess:false }, '庚': { yun:'金', excess:true },
  '丙': { yun:'水', excess:true },  '辛': { yun:'水', excess:false },
  '丁': { yun:'木', excess:false }, '壬': { yun:'木', excess:true },
  '戊': { yun:'火', excess:true },  '癸': { yun:'火', excess:false }
};
```

### 太过与不及的影响

| 状态 | 含义 | 对应疾病特点 |
|------|------|-------------|
| **太过** | 该运五行偏旺 | 本脏功能亢进 + 所克之脏受损 |
| **不及** | 该运五行偏弱 | 本脏功能不足 + 所不胜（克我者）乘之 |

**示例**：
- 2024甲辰年 → 甲=土运太过 → 脾胃系统敏感，湿气重，被克的肾水受影响
- 2025乙巳年 → 乙=金运不及 → 肺系统偏弱，呼吸道疾病多，被火（克金者）乘袭

### 主运（固定不变）

每年固定分五段，按五行相生顺序：

| 运段 | 时间（约） | 主运 | 含义 |
|------|-----------|------|------|
| 初运 | 大寒→春分 | 木 | 春·风 |
| 二运 | 春分→芒种 | 火 | 夏·热 |
| 三运 | 芒种→处暑 | 土 | 长夏·湿 |
| 四运 | 处暑→立冬 | 金 | 秋·燥 |
| 终运 | 立冬→大寒 | 水 | 冬·寒 |

### 客运（每年变化）

以该年天干所化之运为初运，按五行相生顺序排。

```javascript
function getKeYun(yearStem) {
  const wuxingOrder = ['木','火','土','金','水'];
  const yunData = TIANGAN_YUN[yearStem];
  const startIdx = wuxingOrder.indexOf(yunData.yun);
  
  // ⚠️ 客运以角徵宫商羽（木火土金水）为基础
  // 初运=该年化运，依次相生
  const keYun = [];
  for (let i = 0; i < 5; i++) {
    keYun.push(wuxingOrder[(startIdx + i) % 5]);
  }
  return keYun;
}

// 2024甲辰年：甲=土运
// 客运：土→金→水→木→火
```

### 司天在泉的含义

```
司天 → 管上半年（大寒→大暑）的气候主调
在泉 → 管下半年（大暑→大寒）的气候主调

司天·在泉永远是对冲关系（阴阳互根）
```

### 客气（每年变化）

以司天之气为三气（上半年中段），其他按三阴三阳顺序排列。

```javascript
const SANYINSANYANG_ORDER = ['厥阴风木','少阴君火','太阴湿土','少阳相火','阳明燥金','太阳寒水'];

function getKeQi(yearBranch) {
  const sitianName = DIZHI_QI[yearBranch].sitian;
  const sitianIdx = SANYINSANYANG_ORDER.indexOf(sitianName);
  
  // 司天排在第三气位置，其他按顺序
  const keQi = [];
  for (let i = 0; i < 6; i++) {
    // 从司天前3位开始（即初气=司天-2位）
    keQi.push(SANYINSANYANG_ORDER[(sitianIdx - 2 + i + 6) % 6]);
  }
  return keQi;
}
```

## 五、五运六气在八字中的应用

### 先天体质分析

```
根据出生年的运气特征，推断先天体质倾向：

1. 出生年五运太过→该五行对应脏腑先天偏亢
   例：甲年出生（土运太过）→ 脾胃系统偏旺→可能体型偏胖/湿气重

2. 出生年五运不及→该五行对应脏腑先天偏弱
   例：乙年出生（金运不及）→ 肺系统偏弱→易感冒/过敏/皮肤敏感

3. 司天在泉→出生时的气候环境影响先天体质
   例：子午年出生（少阴君火司天）→ 带有火热体质特征
```

### 流年健康预测框架

```
将八字用神与当年运气结合：

1. 当年五运克命局用神 → 该年健康需注意
2. 当年司天在泉克命局某五行 → 对应脏腑当年脆弱
3. 大运五行与流年运气共振 → 影响放大

例：命局木弱之人 + 2025乙巳年（金运不及→木被释放→风木司天）
→ 肝木被激发但无约束 → 肝阳上亢 → 头痛·眩晕·高血压风险
```

## 六、体质九分法

中医将人分为九种基本体质，与五行直接对应：

| 体质 | 五行偏性 | 特征 | 易感疾病 | 调理方向 |
|------|---------|------|---------|---------|
| 平和质 | 平衡 | 体态适中·精力充沛 | 少病 | 维持 |
| 气虚质 | 土/气不足 | 疲乏·气短·易感冒 | 感冒·内脏下垂 | 补气健脾 |
| 阳虚质 | 火不足 | 畏寒·手脚冷·精神差 | 水肿·腹泻 | 温阳补火 |
| 阴虚质 | 水不足 | 口干·手足心热·盗汗 | 失眠·便秘 | 滋阴清热 |
| 痰湿质 | 土/水过剩 | 体胖·腹胀·痰多 | 高血脂·糖尿病 | 化痰祛湿 |
| 湿热质 | 土+火过剩 | 面垢油光·口苦·大便黏 | 痤疮·泌尿感染 | 清热利湿 |
| 血瘀质 | 血行不畅 | 面色晦暗·唇暗·舌有瘀斑 | 心脑血管·肿瘤 | 活血化瘀 |
| 气郁质 | 木气郁结 | 情绪低落·叹气·胁肋胀 | 抑郁·乳腺增生 | 疏肝理气 |
| 特禀质 | 先天不足 | 过敏·遗传病 | 哮喘·荨麻疹 | 益气固表 |
