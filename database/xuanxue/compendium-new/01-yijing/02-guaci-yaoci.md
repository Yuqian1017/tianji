## 一、卦辞

### 什么是卦辞

卦辞是对整个卦的**总体判断**，通常只有一两句话。

**示例——乾卦卦辞**：
> 乾：元亨利贞。

**示例——屯卦卦辞**：
> 屯：元亨，利贞。勿用有攸往，利建侯。

### 卦辞常见术语

> 「卦辞」与解释它的《彖传》不是同一文本层。下表的「在断卦中」是现代教学性类比，不是古汉语词义的唯一定论。

| 术语 | 含义 | 在断卦中 |
|------|------|----------|
| **元** | 大·始·善之长 | 有创始之力，事情有根基 |
| **亨** | 通达·顺利 | 事情能推进，不阻滞 |
| **利** | 有利·适宜 | 做某事有利，条件具备 |
| **贞** | 正固·坚守·占问 | 需要守正道，或指占问的结果 |
| **吉** | 好结果 | 事情向好发展 |
| **凶** | 坏结果 | 事情有危险 |
| **悔** | 后悔·忧虑 | 有困难但在改善中 |
| **吝** | 困难·羞吝 | 有困扰但不至于凶 |
| **咎** | 过错·灾祸 | 有过失 |
| **无咎** | 无过错 | 虽有波折但不会有大问题 |
| **厉** | 危险 | 有风险，需警惕 |
| **有攸往** | 有所前往·采取行动 | 指是否适合行动 |
| **利涉大川** | 利于渡大河 | 适合做大事、冒险 |
| **利建侯** | 利于建立基业 | 适合创业、拓展 |
| **利见大人** | 利于见贵人 | 适合求助、拜访权贵 |

### 教学用风险梯度（非经文固定等级）

> 下列顺序只用于初学者辨别词语语气。「悔、吝、厉、咎」的具体作用依赖句法和卦爻语境，不能跨文本机械排成预测分数。

```
吉 > 无咎 > 悔 > 吝 > 厉 > 咎 > 凶

吉：好事
无咎：没坏事（不等于好事）
悔：有忧虑但在转好
吝：有困扰，进退两难
厉：危险信号
咎：出了错
凶：坏事
```

> *"注意'无咎'——不是说没事，是说'虽然有问题但你处理对了所以不算过错'。周易里最多的判断就是无咎，因为人生大部分时候就是这样：不完美，但过得去。"——师父*

## 二、爻辞

### 爻位含义

每个六爻卦有六个爻位，从下到上编号为初、二、三、四、五、上。阳爻称「九」，阴爻称「六」。

| 爻位 | 称呼规则 | 社会隐喻 | 含义层级 |
|------|---------|---------|---------|
| **上爻** | 上九/上六 | 退休元老·化外之人 | 事物终结·过度·超脱 |
| **五爻** | 九五/六五 | 君位·领袖 | 尊位·核心决策者 |
| **四爻** | 九四/六四 | 近臣·辅佐 | 接近权力中心·多惧 |
| **三爻** | 九三/六三 | 中层·过渡 | 内卦之极·多凶（位不正） |
| **二爻** | 九二/六二 | 内臣·贤者 | 内卦中位·多吉（中正） |
| **初爻** | 初九/初六 | 平民·初学者 | 事物起始·潜伏 |

### 爻位的核心规则

**1. 当位与不当位**

奇数爻位（初、三、五）适合阳爻，偶数爻位（二、四、上）适合阴爻。

- 阳爻在奇数位 → 当位（得正）→ 有利
- 阴爻在偶数位 → 当位（得正）→ 有利
- 反之 → 不当位 → 不利（但不绝对）

**2. 中位**

二爻和五爻是下卦和上卦的「中位」。居中意味着中庸、适度、不偏不倚。

- **中且正**（如六二：阴爻在偶数中位）→ 最佳状态
- **中而不正**（如九二：阳爻在偶数中位）→ 虽不正但居中，仍不错

**3. 承与乘**

- **承**：阴爻在阳爻之下 → 承顺，有依托
- **乘**：阴爻在阳爻之上 → 乘凌，不安分

**4. 应与比**

- **应**：初与四应，二与五应，三与上应（间隔三位）。一阴一阳为正应（有助力），同性则不应。
- **比**：相邻两爻的关系。一阴一阳为亲比，同性为敌比。

```
🔮 实战：爻位关系判定

function analyzeYaoPosition(hexagram) {
  // hexagram: [初,二,三,四,五,上], 1=阳, 0=阴
  const results = [];
  
  for (let i = 0; i < 6; i++) {
    const yao = hexagram[i];
    const position = i + 1; // 1-6
    const isYang = yao === 1;
    const isOddPosition = position % 2 === 1;
    
    // 当位判断
    const isProper = (isYang && isOddPosition) || (!isYang && !isOddPosition);
    
    // 中位判断
    const isCenter = (position === 2 || position === 5);
    
    // 应位
    const correspondIdx = i < 3 ? i + 3 : i - 3;
    const hasCorrespond = hexagram[correspondIdx] !== hexagram[i]; // 阴阳不同则有应
    
    results.push({
      position,
      name: getYaoName(isYang, position), // 如"九五""六二"
      proper: isProper,
      center: isCenter,
      correspond: hasCorrespond,
      correspondWith: correspondIdx + 1
    });
  }
  return results;
}

function getYaoName(isYang, position) {
  const posNames = ['初', '二', '三', '四', '五', '上'];
  const prefix = position === 1 ? '初' : position === 6 ? '上' : posNames[position-1];
  const yinyang = isYang ? '九' : '六';
  if (position === 1) return `初${yinyang}`;
  if (position === 6) return `上${yinyang}`;
  return `${yinyang}${prefix}`;
}
```

## 三、读卦辞爻辞的方法

### 实际断卦时怎么用

> *"起了一个卦出来，你不是六十四条卦辞六条爻辞都要看的。你只看跟你相关的。"——师父*

**一种传统经文占的教学取法**：

> 不同筮法与流派对多动爻有不同取法。下列只是入门概括，不是京房六爻纳甲的通用算法。

1. 无动爻 → 以卦辞为主
2. 一个动爻 → 重点看该动爻的爻辞
3. 多个动爻 → 综合看，侧重变卦
4. 六爻皆动 → 看变卦卦辞

**梅花易数中的用法**：

梅花易数不太依赖爻辞，而是看卦象（体用关系）。但卦辞可以作为辅助参考。

> ⚠️ **注意**：六爻和梅花虽然都用六十四卦，但断卦方法完全不同。六爻重「六亲生克」，梅花重「体用五行」。卦辞爻辞在两个体系中的权重也不同。

## 四、《十翼》简介

《易传》共有十篇，传统称为「十翼」并旧题孔子；本课不把这一旧题当作已确证的一人作者结论：

| 篇名 | 内容 | 学习价值 |
|------|------|---------|
| **彖传(上下)** | 解释卦辞 | 理解每卦核心含义 |
| **象传(上下)** | 大象传解释卦象，小象传解释爻辞 | 学习如何「取象」 |
| **文言传** | 专论乾坤两卦 | 深入理解阴阳之道 |
| **系辞传(上下)** | 易经哲学总论 | 「一阴一阳之谓道」等核心命题 |
| **说卦传** | 八卦取象大全 | 最实用——列出了八卦的各种象义 |
| **序卦传** | 六十四卦排列的逻辑 | 理解卦序 |
| **杂卦传** | 用最精炼的语言概括每对卦 | 速记利器 |

### 说卦传取象（节选）

## 五、本课心法

> *"读周易，记住三个层次：辞、象、理。"*
> *"辞是表面文字，象是画面联想，理是背后逻辑。能同时看到三层的人，才算真正读懂了。"*
> *"那我现在在哪层？"*
> *"你连辞都没读完，你在地下室。"*

1. **辞不离象**：卦辞爻辞都是基于卦象写的，看辞要联想象
2. **元亨利贞是常客**：这四个字出现频率最高，要烂熟于心
3. **无咎 ≠ 吉**：无咎只是及格线，别高兴太早
4. **爻位定层级**：同样是「吉」，在二五爻和在三四爻，含金量完全不同
