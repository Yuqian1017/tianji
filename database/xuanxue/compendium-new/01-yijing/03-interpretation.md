## 一、象数理占：四层解读法

> *"一个卦摆在面前，你怎么看？这里有个万能框架：象→数→理→占。"——师父*

### 第一层：象（Symbol）

「象」是直觉层——看到卦象，直接联想画面。

**取象的方法**：

1. **卦象取象**：上下卦各是什么八卦？对应什么自然现象？
   - 水雷屯 = 坎(水)上 + 震(雷)下 = 乌云密布，雷声隐隐 → 暴风雨前
   
2. **爻象取象**：看爻的位置和阴阳形态
   - 一阳陷于二阴中 → 被困、被包围
   - 阳爻在最上方 → 到顶了、即将转变

3. **互卦取象**：取二三四五爻的互卦，看隐藏的象
   - 互卦是「卦中卦」，代表事物内部的隐藏因素

4. **说卦传取象**：按八卦的万物类象扩展联想

> *"取象这个东西，说白了就是看图说话。但高手和菜鸟的区别在于——菜鸟看到什么说什么，高手知道该看什么。"——师姐*

### 第二层：数（Number）

「数」是逻辑层——涉及具体的数字运算。

- **先天数**：乾1兑2离3震4巽5坎6艮7坤8 → 梅花易数起卦核心
- **后天数（洛书数）**：坎1坤2震3巽4中5乾6兑7艮8离9 → 奇门九宫
- **爻数**：6789 → 六爻起卦判动静
- **卦序数**：文王序1-64 → 某些数字占用
- **五行生克数**：旺相休囚死的力量评估

### 第三层：理（Principle）

「理」是推演层——根据卦象和数字推导出逻辑结论。

- 五行生克关系
- 爻位的当位、得中、有应
- 动爻变卦的逻辑推演
- 时间因素（月建日辰对旺衰的影响）

### 第四层：占（Divination）

「占」是应用层——把前三层的分析落实为具体的判断。

- 问什么事 → 取什么用神
- 整合象数理的信息 → 给出判断
- 判断时间（应期）
- 给出建议

```
🔮 解卦流程模板

1. 明确问题 → 确定用神
2. 看象 → 上下卦象、互卦象、动爻象
3. 查数 → 旺衰、生克力量计算
4. 推理 → 用神旺/衰？世应关系？动静变化？
5. 断占 → 吉凶判断 + 时间推断 + 具体建议
```

## 二、互卦、变卦、错卦、综卦

这四种卦变换是解卦的核心工具：

### 互卦 (Nuclear Hexagram)

> *"互卦是事物的内在状态——表面看着好，互卦一看可能里面全是问题。"——师姐*

**取法**：取原卦二、三、四爻为下卦，三、四、五爻为上卦。

```
原卦：  上 五 四 三 二 初
互卦上卦：  五 四 三
互卦下卦：     四 三 二
```

**用途**：反映事物发展的中间过程、内在状态。

### 变卦 (Changed Hexagram)

**取法**：将所有动爻（老阳→阴，老阴→阳）变化后得到的卦。

**用途**：反映事物的发展趋势、最终结果。

- 原卦（本卦）→ 现在的状态
- 变卦（之卦）→ 将来的状态

### 错卦 (Inverted Hexagram)

**取法**：所有爻阴阳翻转（阳变阴，阴变阳）。

**用途**：从对立面看问题——我看到的是这样，对方看到的是错卦的样子。

### 综卦 (Reversed Hexagram)

**取法**：整个卦上下颠倒。

**用途**：换个角度看同一件事——同一个局面，从另一个立场出发会怎么看。

### 四卦的关系总结

| 变换 | 操作 | 视角 |
|------|------|------|
| 互卦 | 取中间四爻 | 内部/过程 |
| 变卦 | 动爻翻转 | 未来/结果 |
| 错卦 | 全部翻转 | 对面/对方 |
| 综卦 | 上下颠倒 | 换位/反面 |

```
🔮 实战：四种卦变换

function allTransforms(hexagram, movingLines = []) {
  // hexagram: [初,二,三,四,五,上], 1=阳, 0=阴
  
  // 互卦
  const huGua = {
    lower: [hexagram[1], hexagram[2], hexagram[3]],
    upper: [hexagram[2], hexagram[3], hexagram[4]]
  };
  
  // 变卦
  const bianGua = hexagram.map((yao, i) => 
    movingLines.includes(i) ? (yao === 1 ? 0 : 1) : yao
  );
  
  // 错卦
  const cuoGua = hexagram.map(yao => yao === 1 ? 0 : 1);
  
  // 综卦
  const zongGua = [...hexagram].reverse();
  
  return { huGua, bianGua, cuoGua, zongGua };
}
```

## 三、体用关系

> *"体用之分，是所有占卜的第一步。你分不清体用，后面全白搭。"——师姐*

### 定义

- **体**：问事者本身，被动方，主体
- **用**：问的事/对方，主动方，客体

### 在不同术数中

| 术数 | 体 | 用 |
|------|----|----|
| **梅花易数** | 不动的那个卦（上卦或下卦） | 有动爻的那个卦 |
| **六爻** | 世爻 | 应爻（或用神） |
| **八字** | 日主（日干） | 其他七个字 |
| **紫微** | 命宫 | 其他十一宫 |
| **奇门** | 日干落宫 | 时干落宫/用神落宫 |

### 体用生克判断

```
用生体 → 体方得利 → 吉（对问事者有利）
体生用 → 体方泄气 → 不利（精力/资源外流）
用克体 → 体方受制 → 凶（受到压制/伤害）
体克用 → 体方有力 → 吉（能掌控局面）
体用比和 → 力量均衡 → 平（维持现状）
```

## 四、断卦的常见误区

> *"我见过太多人学了点皮毛就到处给人算卦，算出来的东西牛头不对马嘴。"——师父*

### 误区一：死套吉凶

「得了泰卦就是好，得了否卦就是坏。」

**纠正**：看动爻、看用神旺衰、看问题类型。泰卦世爻入墓也可以不吉。

### 误区二：忽略时间

「这个卦是吉卦，所以随时去都行。」

**纠正**：月建日辰决定了五行旺衰。同一个卦在不同月份问，结果可能完全不同。

### 误区三：过度解读

「初爻代表脚，动爻在初爻说明脚会受伤！」

**纠正**：取象要结合问题。问事业时初爻代表起步阶段，不代表脚。取象必须联系具体情境。

### 误区四：信息过载

「这个卦有三个动爻，变卦又变了，互卦也要看，还有错卦综卦……」

**纠正**：信息要有主次。先抓主要矛盾（用神旺衰、世应关系），再看辅助信息。信息太多反而找不到重点。
