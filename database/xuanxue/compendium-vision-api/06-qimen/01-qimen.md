## 一、奇门遁甲概述

### 什么是奇门遁甲

奇门遁甲是古代最高级的预测和决策系统，以**九宫**为框架，将**天盘（九星）、地盘（八门）、人盘（八神）、神盘（三奇六仪）**四层信息叠加在一起，形成一个时空全息图。

**「奇」**= 三奇：乙（日奇）、丙（月奇）、丁（星奇）  
**「门」**= 八门：开、休、生、伤、杜、景、死、惊  
**「遁」**= 隐遁：甲（六甲隐于六仪之下）  
**「甲」**= 十天干之首

### 与其他术数的关系

| 维度 | 奇门遁甲 | 六爻 | 八字 | 紫微 |
|------|---------|------|------|------|
| **时间维度** | 当下时空全息 | 起卦时间 | 出生时间 | 出生时间 |
| **空间维度** | 九宫方位 | 无明确空间 | 无明确空间 | 十二宫 |
| **信息层数** | 4层（天地人神） | 1层（六爻） | 1层（四柱） | 2层（宫+星） |
| **适用范围** | 择时·方位·行军·商战 | 具体事件预测 | 命运全局 | 命运全局 |

## 四、排盘基础

### 阳遁与阴遁

- **冬至后** → **阳遁**（1-9局，顺排）
- **夏至后** → **阴遁**（9-1局，逆排）

具体哪一局，由节气和上中下三元决定（每个节气分上中下三元，共72候对应18局）。

### 排盘简要步骤

```
1. 确定阴遁/阳遁 → 看冬至还是夏至后
2. 确定局数 → 查节气+三元表
3. 布地盘 → 值符（甲遁于六仪）按局数排入九宫
4. 布天盘（九星）→ 时干决定值符星，随之旋转
5. 布人盘（八门）→ 时干决定值使门，随之旋转
6. 布神盘（八神）→ 阳遁顺排，阴遁逆排
7. 四层叠加 → 完成
```

> ⚠️ **奇门排盘的流派差异极大**——拆补局vs置闰局vs茅山法vs飞盘法。本 compendium 以最通用的「拆补法·转盘」为基础说明。开发时建议提供流派选项。

```
🔮 实战：判断阴遁阳遁

function getYinYangDun(solarMonth, solarDay) {
  // 简化判断：冬至后阳遁，夏至后阴遁
  // 精确判断需要查节气精确时间
  // 冬至大约12月21日，夏至大约6月21日
  
  if ((solarMonth === 12 && solarDay >= 21) || 
      solarMonth <= 5 || 
      (solarMonth === 6 && solarDay < 21)) {
    return 'yang'; // 阳遁
  }
  return 'yin'; // 阴遁
}
```

## 五、奇门断事要点

### 用神取法

| 问事 | 用神 | 看什么 |
|------|------|--------|
| 自己 | 日干落宫 | 日干在哪个宫 |
| 求财 | 生门·戊 | 生门落宫状态 |
| 婚姻 | 六合·乙(日奇)+庚 | 乙庚合 |
| 出行 | 开门 | 开门落宫的吉凶 |
| 疾病 | 天芮星·死门 | 芮星/死门落宫 |
| 官司 | 开门(原告)·值符(法官) | 落宫生克 |
| 方位 | 直接看各宫吉凶 | 吉门吉星之方可往 |

### 奇门格局

奇门遁甲有一系列固定的「格局」，由天盘地盘叠加组合判定：

**吉格（部分）**：
- **乙+丙** → 奇仪相佐，万事大吉
- **丙+丁** → 星月相照，百事通达
- **三奇得使** → 三奇(乙丙丁)临开休生三吉门，大吉

**凶格（部分）**：
- **庚+庚** → 太白入庚（白虎猖狂），大凶
- **庚+癸** → 大格，行动受阻
- **庚+丙** → 荧入太白，火入金乡，凶

## 六、大六壬入门

> *"六壬是三式之一，比奇门更古老。它的核心是十二天将和十二地支的关系。"——大师兄*

### 基本框架

- **四课**：由日干、日支推算出四组干支
- **三传**：从四课中取出三个地支作为事情发展的过程（初传·中传·末传）
- **十二天将**：贵人·螣蛇·朱雀·六合·勾陈·青龙·天空·白虎·太常·玄武·太阴·天后

### 大六壬的特色

- 以**日干**为中心（类似八字）
- 十二天将类似六爻的六神，但更细致
- 擅长预测**具体事件**和**来意**

> 大六壬体系庞大，完整内容可独立成册。本 compendium 提供入门框架，深入学习需专门典籍。

## 七、太乙神数概览

太乙神数是三式中最宏观的一个，主要用于预测**国运和大势**：

- 以**太乙**（北极星）为核心
- 推算周期以**年**为单位（而非时辰）
- 主要应用于历史趋势和大局预测
- 个人命理应用相对较少

> 太乙神数在当代应用最少，此处仅作体系介绍。

## 八、本课心法

> *大师兄收起地上的图。*
> *"三式——奇门遁甲、大六壬、太乙神数——是中国术数的最高层。奇门择时择方，六壬占事占人，太乙观国观势。"*
> *"我能学到什么程度？"*
> *"奇门入门可以日常用——出行、面试、谈判，选个好方位好时间。精通？那是一辈子的事。"*
> *"那大六壬和太乙呢？"*
> *"大六壬建议以后专门深入。太乙……先了解它存在就好。你又不需要推算国运。"*

1. **奇门重时空**：什么时间往什么方向做什么事
2. **四层叠加**：天盘·地盘·人盘·神盘，综合判断
3. **格局定吉凶**：固定的奇仪组合有固定的吉凶意义
4. **三式各有所长**：奇门择时，六壬占事，太乙观势
