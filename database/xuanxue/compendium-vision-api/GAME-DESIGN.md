# GAME-DESIGN.md — 修仙游戏化系统设计

> 将整个学习和使用过程包装为一场修仙之旅。用户不是在"学玄学"，而是在"修炼"。

---

## 一、核心理念

```
传统教学：学习 → 做题 → 及格
修仙体系：修炼 → 历劫 → 突破 → 升阶

用户不是"学生"，是"修士"
不是"做对一道题"，是"悟了一层天机"
不是"学完八字"，是"八字之道·元婴境小成"
```

### 境界突破（渡劫）

```
当灵力达到某境界门槛时，不是自动升级，而是触发"渡劫"事件：

渡劫 = 一组该境界的综合测试题（5-10题）
通过 → 境界突破 + 动画 + 奖励
失败 → 灵力不扣，但需要重新渡劫

渡劫动画：
- 练气→筑基：雷劫（屏幕闪电+震动）
- 筑基→金丹：丹成（金光聚顶）
- 金丹→元婴：元婴出窍（小人影浮现）
- 元婴→化神：星河灌顶
- 化神→合体：天地交融
- 合体→渡劫：九天雷劫（最华丽）
- 渡劫→大乘：功德圆满（佛光普照）
- 大乘→飞升：飞升动画（极少数人能到）
```

## 四B、道侣系统

### 核心设计

```
道侣不是"攻略游戏"——是在修炼过程中自然发展的关系线。
好感积累到一定程度，NPC会主动触发对话。
玩家不选"追谁"，而是自然地在某个领域投入更多 → 与该领域NPC关系加深 → 剧情自然展开。

可攻略角色：
  · 沈星河（师姐）— 六爻/梅花线
  · 钱多余（二师兄）— 八字线
  · 大师兄（真名：陆渊）— 奇门线

师父不可攻略。
（真的吗？）
```

> **CC开发注意**：师父线是**隐藏真线**。游戏UI中显示"师父不可攻略"。
> 但满足特定隐藏条件后，会解锁第四条线。这条线的存在本身就是剧透，
> 所以在代码中不要有任何提示，直到条件触发。详见下方「师父·隐藏真线」章节。
```

### 好感阶段与解锁

| 好感度 | 阶段 | 关系称谓 | 解锁内容 |
|--------|------|---------|---------|
| 0-20 | **初识** | 同门 | 基础教学对话 |
| 20-40 | **相识** | 师兄弟/师姐弟 | 课后闲聊·NPC偶尔主动说话 |
| 40-60 | **相知** | 好友 | 专属支线剧情·NPC送你小礼物（灵力/道具） |
| 60-80 | **心动** | 知己 | 暧昧对话·双人任务·NPC表露心迹 |
| 80-90 | **道侣试炼** | — | 触发双人渡劫（特殊考试） |
| 90-100 | **道侣** | 道侣 | 结为道侣·专属称号·双修加成·结局剧情 |

### 沈星河线（主线·最完整）

**人设深化**：
```
表面：冷面·严谨·不苟言笑·对六爻有近乎偏执的追求
内里：其实很在意别人的看法·小时候被说"女孩子学什么占卜"·
      用冷漠掩盖不安·对真正认可她的人会格外珍惜
特征：永远扎高马尾·喜欢在月下起卦·有一套祖传的铜钱
弱点：不会表达感情·一紧张就更冷·送东西会装作"顺便的"
```

#### 师父线的特殊设计

```
与其他三条线的区别：

1. 【隐藏性】其他线明确告诉你可以攻略。师父线游戏写着"不可攻略"。
   玩家必须自己发现。这本身就是一个"解卦"过程。

2. 【被动性】其他线是你主动推好感。师父线大部分好感来自
   你做其他事情时师父的观察——他在暗中注意你。
   你学得越好·对其他人越温柔·他的好感越高。

3. 【门槛最高】需要渡劫期+完成全部教学+其他三条线都推到相知。
   意思是：你必须足够了解这个世界和这些人，才能理解师父。

4. 【没有甜言蜜语】师父不会说"我喜欢你"。
   他的表白是"收好这枚铜钱"。
   他的情话是"你让我想起她——不对，你是你"。
   他的承诺是"一起走"。
