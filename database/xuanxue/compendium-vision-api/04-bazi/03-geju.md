## 一、日主强弱判断

### 三步判断法

**第一步：看月令**（权重最大，约占40%）

月令（月支）是日主的「生存环境」。日主在月令中处于什么状态？

| 月令与日主关系 | 状态 | 判断 |
|--------------|------|------|
| 月令与日主同五行 | 得令 | 强 |
| 月令生日主 | 得令（印月） | 偏强 |
| 月令被日主所生 | 泄气 | 偏弱 |
| 月令克日主 | 受克 | 弱 |
| 月令被日主所克 | 耗力 | 偏弱 |

**第二步：看四柱干支（除月令外）对日主的助力**（约占40%）

- 天干有比肩/劫财/正印/偏印 → 帮身
- 天干有食伤/财星/官杀 → 耗泄克身
- 地支藏干同理（但力量弱于天干明透）

**第三步：看地支根**（约占20%）

日主在地支中是否有「根」（日干的五行在地支藏干中出现）：

- 有根 → 日主有支撑，不至于太弱
- 无根 → 日主虚浮，即使天干有帮也不稳

```
🔮 实战：日主强弱评估（简化版）

function assessDayMaster(dayMaster, pillars) {
  let helpScore = 0;
  let drainScore = 0;
  
  const dayElement = getStemElement(dayMaster);
  
  // 1. 月令权重
  const monthBranch = pillars.month.branch;
  const monthElement = getBranchElement(monthBranch);
  const monthRelation = wuxingRelation(monthElement, dayElement);
  if (monthRelation === 'same' || monthRelation === 'generate') helpScore += 40;
  else drainScore += 40;
  
  // 2. 其他干支
  const allStems = [pillars.year.stem, pillars.month.stem, pillars.hour.stem];
  allStems.forEach(stem => {
    const shishen = getShishen(dayMaster, stem);
    if (['比肩','劫财','正印','偏印'].includes(shishen)) helpScore += 10;
    else drainScore += 10;
  });
  
  // 3. 地支根
  const allBranches = [pillars.year.branch, pillars.month.branch, 
                       pillars.day.branch, pillars.hour.branch];
  const hasRoot = allBranches.some(br => {
    const hidden = getHiddenStems(br);
    return hidden.some(h => getStemElement(h) === dayElement);
  });
  if (hasRoot) helpScore += 15;
  
  const total = helpScore - drainScore;
  return {
    score: total,
    strength: total > 20 ? '身强' : total > -20 ? '中和' : '身弱',
    needHelp: total < 0, // 身弱需要印比帮身
    needDrain: total > 0  // 身强需要食伤财官泄耗克
  };
}
```

## 二、用神与喜忌

> *"用神是八字的药方——日主缺什么，用神就补什么。"——二师兄*

### 取用神原则

| 日主状态 | 用神 | 喜神 | 忌神 |
|---------|------|------|------|
| **身强** | 食伤(泄)·财星(耗)·官杀(克) | 用神的同类/生助者 | 印星·比劫 |
| **身弱** | 印星(生)·比劫(帮) | 用神的同类/生助者 | 食伤·财星·官杀 |
| **从强** | 印星·比劫（顺其势） | 同上 | 食伤·财·官杀 |
| **从弱** | 食伤·财·官杀（顺其势） | 同上 | 印·比劫 |

**从格**：当日主极弱无根、四柱全是克泄耗时，不再扶助，而是「从势」——跟随最强的五行走。从格是特殊格局。

### 喜忌判断的应用

- **大运流年走喜用神** → 运势好
- **大运流年走忌神** → 运势差
- **用神被克/受伤** → 该阶段不利
- **用神得生/得助** → 该阶段有利

## 三、格局论

### 正格八格

以月令中**透出**的十神定格局（月支藏干透到天干上的十神）：

| 格局 | 条件 | 特点 |
|------|------|------|
| **正官格** | 月令透正官 | 端正·重名誉·适合仕途 |
| **七杀格** | 月令透七杀 | 霸气·有魄力·需制化 |
| **正印格** | 月令透正印 | 学识·温和·有贵人 |
| **偏印格** | 月令透偏印 | 聪明·另类·多变 |
| **正财格** | 月令透正财 | 务实·稳定·勤劳致富 |
| **偏财格** | 月令透偏财 | 大方·善交际·财运活 |
| **食神格** | 月令透食神 | 有才华·好享受·福气 |
| **伤官格** | 月令透伤官 | 才华出众·叛逆·不安分 |

> 比肩和劫财不成格（因为是同类，不产生作用力）。

### 特殊格局

| 格局 | 条件 | 特点 |
|------|------|------|
| **从财格** | 日主极弱，财星极旺无帮身 | 以财为主导 |
| **从官杀格** | 日主极弱，官杀极旺 | 以官杀为主导 |
| **从儿格** | 日主极弱，食伤极旺 | 以才华为主导 |
| **专旺格(曲直/炎上/稼穑/从革/润下)** | 日主极旺，同类遍布 | 一气专旺 |
| **化气格** | 日干合化成功 | 以化出的五行为主 |

### 格局成败

格局成立还需要看**是否被破**：

- 正官格 → 怕伤官来克（伤官见官）
- 食神格 → 怕偏印来夺（枭神夺食）
- 财格 → 怕比劫来抢（群劫争财）
- 七杀格 → 需食神制或印化（无制则凶）

## 四、大运与流年

### 大运排法

大运从**月柱**开始排，根据性别和年干的阴阳决定顺逆：

| 条件 | 排法 |
|------|------|
| 阳年(甲丙戊庚壬年)男 / 阴年(乙丁己辛癸年)女 | **顺排**（月柱之后的干支依次排） |
| 阴年男 / 阳年女 | **逆排**（月柱之前的干支依次排） |

### 起运年龄

从出生日到下一个（顺排）或上一个（逆排）**节气**的天数，除以3，即为起运岁数（余1天=4个月，余2天=8个月）。

```
🔮 实战：大运排列

function getDayun(monthStem, monthBranch, isForward) {
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  
  const stemIdx = stems.indexOf(monthStem);
  const branchIdx = branches.indexOf(monthBranch);
  const step = isForward ? 1 : -1;
  
  const dayun = [];
  for (let i = 1; i <= 8; i++) { // 通常排8步大运
    const s = stems[((stemIdx + i * step) % 10 + 10) % 10];
    const b = branches[((branchIdx + i * step) % 12 + 12) % 12];
    dayun.push({ stem: s, branch: b, decade: i });
  }
  return dayun;
}
```

### 流年

流年就是当年的干支。每年都在变，与大运叠加影响命运。

**大运看十年趋势，流年看当年具体事件。**

### 大运流年的分析

```
大运干支的十神 + 流年干支的十神
↓
与命局用神/忌神对比
↓
走用神运 → 好运
走忌神运 → 坏运
用神被冲克 → 出事
忌神被制化 → 转运
```

## 五、合婚与择日基础

### 八字合婚

> *"合婚不是看属相合不合那么简单。属相只占5%的权重。"——二师兄*

合婚看的要素（按重要性排序）：

1. **用神互补**：两人的用神最好互相补益
2. **日干关系**：日干相合（天干五合）为佳
3. **日支关系**：日支六合 > 三合 > 无关系 > 六冲
4. **财官配置**：男命财星旺配女命官星旺
5. **大运同步**：两人运势节奏是否同步

### 择日基础

择日的核心原则：

1. 选与命主用神相合的日子
2. 避开命主忌神当令的日子
3. 日课四柱本身也要组合协调
4. 不冲命主的年柱和日柱

## 六、本课心法

> *"八字的核心逻辑就三步：排盘→定强弱→找用神。用神找对了，后面大运流年的吉凶就是顺推。找错了？全盘皆输。"——二师兄*

1. **月令为王**：月令决定了日主的基本强弱
2. **用神定一生**：用神是命局的药方，大运流年走到用神就是好运
3. **格局看层次**：格局高低决定了人生成就的天花板
4. **动态看运**：八字是静态的底盘，大运流年是动态的道路
