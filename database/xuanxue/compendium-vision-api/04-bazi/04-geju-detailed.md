## 一、正格八格详解

### 正官格

**成格条件**：月令中正官透干，且未被合化克破。

```
成格标志：正官一位为贵（不宜多）
相配：官印相生（正官+正印）→ 大吉，有权有识
      财生官（正财+正官）→ 有名有利
破格：伤官见官 → 大忌！冲突激烈
      官杀混杂（正官+七杀同现）→ 不清不楚
```

**正官格的人**：规矩·守法·重名誉·适合公务员/管理者。

**经典八字示例框架**：
```
日主：甲木
月令：酉月（辛金正官当令）
配局：年或时有壬癸水（印星）→ 官印相生格
忌讳：卯木不透（伤官卯克官酉）
```

### 七杀格

**成格条件**：月令中七杀透干，且被制或化。

```
制杀三法：
1. 食神制杀（食神克七杀）→ 最佳，威而不猛
2. 印化杀（偏印泄杀生身）→ 杀印相生，有文武兼备之象
3. 合杀（日干合杀或他干合杀）→ 化敌为友

无制则凶：七杀无食神制、无印化、无合 → 灾难不断
```

**七杀格的人**：霸气·有魄力·适合创业/军警/竞争性行业。

### 食神格

**成格条件**：月令中食神透干。

```
食神制杀 → 大吉：有才华且能化解压力
食神生财 → 吉：才华变现
食神配印 → 中：印克食，才华被约束（但也代表有规矩）
枭神夺食 → 大凶！偏印克食神，断送才华和财路

判断枭印夺食：
- 偏印透干且旺
- 食神透干但弱
- 无财星解救（财克偏印可解）
```

**食神格的人**：温和·有口福·有才华·享受型。

### 伤官格

**成格条件**：月令中伤官透干。

```
伤官见官 → 大凶（除非特殊格局"伤官伤尽"）
伤官配印 → 吉：印制伤，叛逆被约束，反成大器
伤官生财 → 吉：才华变成钱
伤官伤尽 → 吉：命局中无一点官星，伤官独大反而成格

伤官伤尽的条件：
- 伤官旺透
- 八字中无正官（天干地支藏干都没有）
- 无七杀混入
```

**伤官格的人**：聪明·叛逆·口才好·适合艺术/法律/自由职业。

### 正财格/偏财格

```
财格共性：
- 忌比劫（群劫争财 → 破财）
- 喜官杀（财生官 → 有财有势）
- 喜食伤（食伤生财 → 财源不断）

正财格特点：稳定·勤劳·一分耕耘一分收获
偏财格特点：大方·投机·意外之财·人缘好

身强用财 → 能担财，越多越好
身弱用财 → 财多身弱，反而是压力（钱赚到了但享不了）
```

### 正印格/偏印格

```
印格共性：
- 喜官杀（官杀生印 → 权力带来智慧）
- 忌财星（财克印 → 世俗冲击精神）
- 喜比劫（印生比劫 → 有帮手）

正印格：温和·有贵人·学历高·有庇护
偏印格：孤僻·另类·多变·有偏门才华

印旺身弱 → 好事（有靠山）
印旺身旺 → 过度保护，缺乏行动力（印太多 → 懒）
```

## 二、特殊格局

### 专旺格（五种）

| 格名 | 日主 | 条件 | 特点 |
|------|------|------|------|
| **曲直格** | 甲乙木 | 木极旺，地支寅卯辰全，无金克 | 仁慈·生发·文学 |
| **炎上格** | 丙丁火 | 火极旺，地支巳午未全，无水克 | 热情·礼仪·光明 |
| **稼穑格** | 戊己土 | 土极旺，地支辰戌丑未多，无木克 | 厚重·信用·稳定 |
| **从革格** | 庚辛金 | 金极旺，地支申酉戌全，无火克 | 果断·义气·刚硬 |
| **润下格** | 壬癸水 | 水极旺，地支亥子丑全，无土克 | 智慧·流通·变化 |

### 从格

| 格名 | 条件 | 用神 | 忌神 |
|------|------|------|------|
| **从财格** | 日主无根极弱，财星极旺 | 食伤·财星 | 印·比劫 |
| **从官杀格** | 日主无根极弱，官杀极旺 | 官杀·财星 | 印·比劫·食伤 |
| **从儿格** | 日主弱，食伤极旺 | 食伤·财星 | 印·比劫 |
| **从势格** | 日主无根，财官食伤混旺 | 顺势走 | 印·比劫 |

> ⚠️ **从格判断要极其谨慎**：日主只要有一点根（地支藏干有同类），就可能不是从格。误判从格会导致用神完全反转。

### 化气格

天干五合化气：

| 合 | 化 | 条件 | 格名 |
|----|------|------|------|
| 甲己合 | 化土 | 月令土旺 | 化土格 |
| 乙庚合 | 化金 | 月令金旺 | 化金格 |
| 丙辛合 | 化水 | 月令水旺 | 化水格 |
| 丁壬合 | 化木 | 月令木旺 | 化木格 |
| 戊癸合 | 化火 | 月令火旺 | 化火格 |

化气格极为罕见，条件：合化成功 + 化出的五行在月令得令 + 无破。
