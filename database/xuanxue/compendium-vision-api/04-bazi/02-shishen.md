> **难度标签**：🟢必修：十神含义 | 🟡进阶：十神组合

> *"十神是八字的灵魂。学六爻你得记六亲，学八字你得记十神。原理一样——都是五行生克——但八字把它细分成了十个。"*
> *"为什么是十个不是五个？"*
> *"因为每种关系还分阴阳。同性为偏，异性为正。比如克我的：阳克阳、阴克阴叫七杀（偏官），阴克阳、阳克阴叫正官。性格完全不一样。"*

## 一、十神定义

以**日干**（日主）为「我」，看其他七个字的天干/地支藏干与日干的关系：

| 关系 | 同性（偏） | 异性（正） |
|------|-----------|-----------|
| **同我者** | 比肩 | 劫财 |
| **生我者** | 偏印（枭神） | 正印 |
| **我生者** | 食神 | 伤官 |
| **克我者** | 七杀（偏官） | 正官 |
| **我克者** | 偏财 | 正财 |

### 阴阳判定规则

「同性」= 日干与该字的天干阴阳相同  
「异性」= 日干与该字的天干阴阳不同

| 日干 | 阳干(甲丙戊庚壬) | 阴干(乙丁己辛癸) |
|------|-----------------|-----------------|
| 阳干 | 同性→偏 | 异性→正 |
| 阴干 | 异性→正 | 同性→偏 |

```
🔮 实战：十神计算

function getShishen(dayStem, otherStem) {
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const dayIdx = stems.indexOf(dayStem);
  const otherIdx = stems.indexOf(otherStem);
  
  const dayYin = dayIdx % 2; // 0=阳, 1=阴
  const otherYin = otherIdx % 2;
  const samePolarity = dayYin === otherYin;
  
  // 五行关系
  const dayElement = Math.floor(dayIdx / 2); // 0木1火2土3金4水
  const otherElement = Math.floor(otherIdx / 2);
  const diff = (otherElement - dayElement + 5) % 5;
  
  const shishenMap = {
    0: samePolarity ? '比肩' : '劫财',
    1: samePolarity ? '食神' : '伤官',
    2: samePolarity ? '偏财' : '正财',
    3: samePolarity ? '七杀' : '正官',
    4: samePolarity ? '偏印' : '正印'
  };
  
  return shishenMap[diff];
}
```

## 二、十神性格与象义

### 比肩

| 维度 | 含义 |
|------|------|
| 人物 | 兄弟·朋友·同辈·竞争者 |
| 性格 | 独立·自我·固执·不愿妥协 |
| 正面 | 自主·坚强·有主见 |
| 负面 | 刚愎自用·不合群·争夺 |
| 身体 | 肌肉·体力 |

### 劫财

| 维度 | 含义 |
|------|------|
| 人物 | 异性兄弟姐妹·合伙人·对手 |
| 性格 | 果断·野心·敢冒险·不甘人后 |
| 正面 | 行动力强·善于社交 |
| 负面 | 争财·破财·冲动·投机 |

### 食神

| 维度 | 含义 |
|------|------|
| 人物 | 晚辈·学生·下属（温和型） |
| 性格 | 温和·乐观·享受·才华·好口福 |
| 正面 | 有才华·有口福·有创造力·厚道 |
| 负面 | 懒散·贪图享乐·不思进取 |
| 特殊 | 食神制杀——食神克制七杀，化解压力 |

### 伤官

| 维度 | 含义 |
|------|------|
| 人物 | 晚辈·学生（叛逆型） |
| 性格 | 聪明·叛逆·口才好·不服管·追求自由 |
| 正面 | 才华横溢·敢于创新·表达力强 |
| 负面 | 目中无人·克官（影响仕途/婚姻）·口舌是非 |
| 特殊 | 伤官见官——伤官与正官同现，冲突激烈 |

### 偏财

| 维度 | 含义 |
|------|------|
| 人物 | 父亲·情人·意外之财 |
| 性格 | 慷慨·大方·重义气·善交际 |
| 正面 | 人缘好·财运好·善于经商 |
| 负面 | 花钱大手大脚·感情复杂 |

### 正财

| 维度 | 含义 |
|------|------|
| 人物 | 妻子（男命）·正当收入 |
| 性格 | 务实·节俭·稳重·脚踏实地 |
| 正面 | 踏实赚钱·持家有方 |
| 负面 | 小气·过于保守·执着于钱 |

### 七杀（偏官）

| 维度 | 含义 |
|------|------|
| 人物 | 小人·压力源·强势上司·情人（女命） |
| 性格 | 霸气·威严·有魄力·不服输 |
| 正面 | 领导力·决断力·敢于担当 |
| 负面 | 暴躁·独裁·多灾多难（无制化时） |
| 特殊 | 七杀必须被制或化（食神制杀、印化杀），否则为凶 |

### 正官

| 维度 | 含义 |
|------|------|
| 人物 | 上司·丈夫（女命）·权威·法律 |
| 性格 | 正派·守规矩·有责任感·重名誉 |
| 正面 | 仕途顺利·受人尊重·有管理能力 |
| 负面 | 拘谨·过于在意他人看法·压力大 |

### 偏印（枭神）

| 维度 | 含义 |
|------|------|
| 人物 | 继母·偏门学问·小众领域 |
| 性格 | 聪明·多疑·孤独·思想另类 |
| 正面 | 学术能力强·善于研究·有独特见解 |
| 负面 | 不合群·反复无常·枭神夺食（克食神→断财路） |

### 正印

| 维度 | 含义 |
|------|------|
| 人物 | 母亲·老师·贵人·学历 |
| 性格 | 仁慈·包容·好学·重视精神世界 |
| 正面 | 有贵人相助·学历高·受庇护 |
| 负面 | 依赖性强·缺乏行动力·想太多做太少 |

## 三、十神之间的关系链

```
正印/偏印 → 生 → 日主 → 生 → 食神/伤官 → 生 → 正财/偏财 → 生 → 正官/七杀 → 生 → 正印/偏印
                    ↑                                         |
                    └────────────── 克 ──────────────────────┘
```

**核心制化关系**：
- 食神制杀：食神 → 克 → 七杀（化解压力，吉）
- 伤官见官：伤官 → 克 → 正官（破坏秩序，凶）
- 枭神夺食：偏印 → 克 → 食神（断送才华和财路，凶）
- 官印相生：正官 → 生 → 正印（权力+智慧，大吉）
- 财生官：正财 → 生 → 正官（以财求官，有名有利）
- 印克食伤：正印 → 克 → 伤官（约束叛逆，有利有弊）

## 四、本课心法

> *"十神就是十种角色。你的八字里谁最强势，谁就是你人生的主旋律。"——二师兄*

1. **日主为我**：一切十神关系以日干为中心
2. **正偏有别**：同一种关系，正和偏性格截然不同
3. **制化为用**：凶神（七杀·伤官）被制化后反而成格局
4. **藏干也算**：地支藏干里的十神同样重要，只是力量较弱
