## 一、实战案例

### 案例 1：从旺之命（身旺用食伤泄秀）

**命主**：男，1988年农历三月初十 卯时

```
年柱：戊辰（土·土）
月柱：丙辰（火·土）
日柱：甲寅（木·木）← 日主甲木
时柱：丁卯（火·木）

十神：
戊=偏财  辰=偏财
丙=食神  辰=偏财
甲=日主  寅=比肩
丁=伤官  卯=劫财
```

**分析步骤**：

1. **日主强弱**：甲木生于辰月（暮春），辰中有余气乙木。寅卯两根本气在日时，甲木坐寅比肩帮身。身旺无疑。

2. **用神确定**：身旺喜泄耗克——食伤（火）泄木生财最好。丙丁火在月时透出，食伤泄秀为用。

3. **格局**：食伤生财格。食神丙火泄甲木，生辰中戊土偏财。才华→转化为财富。

4. **大运走向**：
   - 早年行东方木运（帮身太旺，少年不顺）
   - 中年行南方火运（食伤泄秀，事业爆发期）
   - 晚年行西方金运（官杀克身，压力大但有地位）

5. **性格断语**：食伤旺者聪明有才华，善于表达，创意型人才。偏财透出，对钱有感觉，适合做生意。

**结论**：适合创业型、创意型职业。中年30-50岁是黄金期。

### 案例 2：从弱之命（身弱用印比）

**命主**：女，1995年农历八月十五 午时

```
年柱：乙亥（木·水）
月柱：乙酉（木·金）
日柱：己未（土·土）← 日主己土
时柱：庚午（金·火）

十神：
乙=七杀  亥=正财
乙=七杀  酉=食神
己=日主  未=比肩
庚=伤官  午=正印
```

**分析步骤**：

1. **日主强弱**：己土生于酉月（金旺泄土），月干年干双乙木克土。坐未有根但不强。身弱。

2. **用神确定**：身弱喜生扶——印星（火）生土最好，比劫（土）帮身次之。午火正印在时柱为用。

3. **格局**：七杀格带印。双乙七杀透出，需要印星化杀生身。杀印相生——适合有权威的工作。

4. **大运关键**：
   - 行火运（印星）：贵人多、学业好、身体好
   - 行土运（比劫）：朋友帮助、自信增强
   - 行水运（财星）：财耗身（身弱不担财），花钱多或为钱操心
   - 行木运（官杀）：压力最大，需注意健康

5. **性格断语**：七杀带印，外柔内刚。食伤酉金透庚，有艺术细胞和表达能力。

### 案例 3：特殊格局（从财格）

**命主**：男，1990年农历六月初八 申时

```
年柱：庚午（金·火）
月柱：癸未（水·土）
日柱：乙丑（木·土）← 日主乙木
时柱：甲申（木·金）

分析要点：
- 乙木生于未月（土旺·木死地）
- 年庚克乙，月癸虽为正印但癸水在未月无根
- 丑未冲动，申金克木
- 只有时干甲木帮身，但甲坐申被截脚
→ 日主极弱，从格？
```

**从财格判定**：乙木几乎无根无助，满盘财官旺盛。若从财格成立，则以财星（土）为用，忌见印比（水木来帮身反而不好）。

**从格要诀**：
- 从格怕见帮身的大运流年（破格）
- 从财格顺着财走，适合经商、金融
- 一旦行木运（帮身破格），反而是最差的运

## 二、八字合婚

### 基本原则

```
合婚看什么：
1. 日柱天合地合 → 最佳（如甲子配己丑=天合地合）
2. 年柱相合或相生 → 大环境和谐
3. 用神互补 → A需要的五行正好是B旺的五行
4. 忌日柱天克地冲 → 矛盾最大
```

### 合婚评分算法

```javascript
function marriageScore(baziA, baziB) {
  let score = 60;
  
  // 1. 日柱关系（权重最大）
  const dayRelation = getDayPillarRelation(baziA.day, baziB.day);
  if (dayRelation === '天合地合') score += 25;
  else if (dayRelation === '天合') score += 15;
  else if (dayRelation === '地合') score += 10;
  else if (dayRelation === '天克地冲') score -= 25;
  else if (dayRelation === '天克') score -= 10;
  
  // 2. 年柱关系
  const yearRelation = getBranchRelation(baziA.yearBranch, baziB.yearBranch);
  if (yearRelation === '六合') score += 10;
  if (yearRelation === '三合') score += 8;
  if (yearRelation === '六冲') score -= 10;
  if (yearRelation === '相刑') score -= 5;
  
  // 3. 用神互补
  if (baziA.xiyong === baziB.strongElement) score += 15;
  if (baziB.xiyong === baziA.strongElement) score += 15;
  
  // 4. 男命财星与女命日主的关系
  // 男命看财（妻星），女命看官（夫星）
  const maleWife = baziA.gender === 'male' ? checkSpouseStar(baziA) : checkSpouseStar(baziB);
  if (maleWife === 'strong') score += 5;
  
  return { score: Math.min(100, Math.max(0, score)), details: [] };
}
```
