## 一、风水核心概念

### 气

风水的核心是**气**——看不见摸不着，但可以感知的环境能量场。

```
气的运动规律：
- 气乘风则散 → 风太大的地方气散，不聚财
- 界水则止 → 水能让气停下来，所以近水为佳
- 藏风聚气 → 好风水的核心：微风环绕，气流聚而不散
```

### 形与理

| 派别 | 核心 | 方法 |
|------|------|------|
| **形势派** | 看山水形状·环境格局 | 观山形·看水势·辨龙穴砂水 |
| **理气派** | 看方位·罗盘·飞星 | 八宅·玄空飞星·三合 |

现代风水通常**形理兼用**。

## 二、形势派基础

### 四象格局

理想的风水格局是**四象齐全**：

```
              朱雀（前方·南·案山）
                  ↑
                  |
    青龙 ←——— ★ 穴位 ———→ 白虎
   （左·东）     |        （右·西）
                  ↓
              玄武（后方·北·靠山）
```

| 四象 | 方位 | 要求 | 现代对应 |
|------|------|------|---------|
| **玄武** | 后方 | 高·稳·厚实 | 座位/床头有靠·高楼在后 |
| **朱雀** | 前方 | 开阔·低平·有水 | 前方视野好·有广场或水景 |
| **青龙** | 左方 | 略高·环抱 | 左边稍高或有建筑护持 |
| **白虎** | 右方 | 略低·温顺 | 右边稍低·不宜有尖锐物 |

### 住宅外环境

```
✓ 好的外环境：
- 背后有高楼或山丘（靠山）
- 前方开阔（明堂宽广）
- 左右有建筑护持（龙虎抱）
- 附近有水（江河·湖·喷泉）且水流环绕
- 道路弯曲环抱（玉带环腰）

✗ 不好的外环境：
- 直路冲向大门（路冲·枪煞）
- 两楼夹缝对向（天斩煞）
- 反弓路（道路弯曲背面对着房屋）
- 孤楼无依（四面空旷无遮挡）
- 墓地·医院·变电站正对
```

### 本命卦

以出生年确定个人的本命卦，看是否与住宅匹配：

```javascript
function getBenMingGua(year, gender) {
  // 男命公式
  let sum = 0;
  let y = year;
  while (y > 0) { sum += y % 10; y = Math.floor(y / 10); }
  while (sum > 9) { 
    let s = 0; let n = sum;
    while (n > 0) { s += n % 10; n = Math.floor(n / 10); }
    sum = s;
  }
  
  let gua;
  if (gender === 'male') {
    // 2000年前：(10 - sum) || 坎  2000年后：(9 - sum) || 坎
    gua = year < 2000 ? (10 - sum) % 9 : (9 - sum) % 9;
    if (gua === 0) gua = 9;
  } else {
    // 2000年前：(sum + 5 - 1) % 9 + 1  简化
    gua = year < 2000 ? (sum + 5) % 9 : (sum + 6) % 9;
    if (gua === 0) gua = 9;
  }
  
  // 5为中宫，男寄坤(2)，女寄艮(8)
  if (gua === 5) gua = gender === 'male' ? 2 : 8;
  
  const guaNames = {null:null, 1:'坎', 2:'坤', 3:'震', 4:'巽', 6:'乾', 7:'兑', 8:'艮', 9:'离'};
  return guaNames[gua];
}
```

## 五、办公室/书房简易布局

### 办公桌

```
✓ 座位背后有墙（靠山）
✓ 面向门口或窗户（视野开阔·明堂）
✓ 左手边放高物（青龙位·贵人位）
✓ 右手边保持整洁（白虎位·宜静）

✗ 背对门坐（犯背煞·无靠山）
✗ 横梁压顶（压迫感）
✗ 正对厕所门（秽气冲）
✗ 镜子对着座位（反光煞）
```

### 卧室

```
✓ 床头靠实墙
✓ 床头朝生气/天医方
✓ 床的左右两边留空（气流通畅）

✗ 床头对门（冲煞）
✗ 床头对窗（气散）
✗ 镜子对床（精神不宁）
✗ 床头挂大画框（压迫）
✗ 床底堆满杂物（秽气滞留）
```

## 六、本课心法

1. **形势为体，理气为用**：先看大环境，再论方位
2. **趋吉避凶**：风水做不到十全十美，抓主要矛盾
3. **因人制宜**：同一套房，不同命卦的人吉凶不同
4. **动态调整**：运星每20年一变，风水不是一劳永逸
