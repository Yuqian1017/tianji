# 八字十二长生表

> 纯数据。十二长生是判断天干在地支中力量状态的核心工具。

---

## 一、十二长生含义

五行从诞生到消亡的十二个阶段，模拟人的一生：

| 阶段 | 名称 | 比喻 | 力量 | 状态 |
|------|------|------|------|------|
| 1 | **长生** | 出生 | ★★★★ | 有生气·有希望·如婴儿初生 |
| 2 | **沐浴** | 洗礼 | ★★ | 不稳定·桃花·变化·脆弱 |
| 3 | **冠带** | 成年 | ★★★ | 渐强·有志向·初露锋芒 |
| 4 | **临官** | 入仕 | ★★★★ | 强·有地位·事业上升期 |
| 5 | **帝旺** | 极盛 | ★★★★★ | 最旺·鼎盛·但盛极将衰 |
| 6 | **衰** | 老年 | ★★★ | 渐弱·力不从心·但有经验 |
| 7 | **病** | 患病 | ★★ | 弱·有问题·需要调养 |
| 8 | **死** | 死亡 | ★ | 很弱·无力·终结(非真死) |
| 9 | **墓(库)** | 入土 | ★★ | 力量被收藏·有时可用 |
| 10 | **绝** | 断绝 | ★ | 最弱·断根·彻底无力 |
| 11 | **胎** | 受孕 | ★★ | 新生命孕育·潜藏希望 |
| 12 | **养** | 孕育 | ★★ | 被滋养·等待出生 |

## 三、算法实现

```javascript
const CHANGSHENG_LABELS = ['长生','沐浴','冠带','临官','帝旺','衰','病','死','墓','绝','胎','养'];

// 阳干长生起点（地支索引）
const YANG_START = {
  '甲': 11, // 亥
  '丙': 2,  // 寅
  '戊': 2,  // 寅
  '庚': 5,  // 巳
  '壬': 8   // 申
};

// 阴干长生起点（逆行版）
const YIN_START = {
  '乙': 6,  // 午
  '丁': 9,  // 酉
  '己': 9,  // 酉
  '辛': 0,  // 子
  '癸': 3   // 卯
};

/**
 * 查询天干在某地支的十二长生状态
 * @param stem 天干
 * @param branch 地支
 * @returns 长生阶段名称
 */
function getChangsheng(stem, branch) {
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  
  const stemIdx = stems.indexOf(stem);
  const branchIdx = branches.indexOf(branch);
  const isYang = stemIdx % 2 === 0;
  
  let startBranch;
  if (isYang) {
    startBranch = YANG_START[stem];
  } else {
    startBranch = YIN_START[stem];
  }
  
  let offset;
  if (isYang) {
    // 阳干顺行
    offset = (branchIdx - startBranch + 12) % 12;
  } else {
    // 阴干逆行
    offset = (startBranch - branchIdx + 12) % 12;
  }
  
  return CHANGSHENG_LABELS[offset];
}

// 批量查询
function getChangshengRow(stem) {
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  return branches.map(br => ({ branch: br, state: getChangsheng(stem, br) }));
}
```

## 四、十二长生在八字中的应用

### 日主长生状态

日主在各地支的十二长生状态决定了日主的「先天禀赋」：

| 日主状态 | 含义 |
|---------|------|
| 日坐长生 | 精力充沛·有发展潜力 |
| 日坐沐浴 | 桃花·感情丰富·不够稳定 |
| 日坐冠带 | 有志向·能力渐长 |
| 日坐临官 | 有地位·自尊心强 |
| 日坐帝旺 | 精力最旺·但刚强过头 |
| 日坐墓库 | 内秀·藏才·但表现不出来 |
| 日坐病/死 | 身体偏弱·需要扶助 |
| 日坐绝 | 先天不足·但逢生则发 |

### 桃花（沐浴位）

沐浴位又称「桃花位」「咸池」：

```javascript
const TAOHUA = {
  // 年支/日支 → 桃花地支
  '寅': '卯', '午': '卯', '戌': '卯',  // 寅午戌见卯为桃花
  '申': '酉', '子': '酉', '辰': '酉',  // 申子辰见酉为桃花
  '巳': '午', '酉': '午', '丑': '午',  // 巳酉丑见午为桃花
  '亥': '子', '卯': '子', '未': '子'   // 亥卯未见子为桃花
};
```
