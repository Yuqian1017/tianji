## 一、面相基础框架

### 三停（面部纵向三等分）

```
上停：发际线 → 眉毛    代表：少年运（15-30岁）·智慧·祖荫·父母缘
中停：眉毛 → 鼻尖      代表：中年运（31-50岁）·事业·意志·自我
下停：鼻尖 → 下巴      代表：晚年运（51岁后）·财富·子女·晚景

三停均匀：一生运势平稳
上停长：少年得志·学业佳·但中晚年可能平淡
中停长：事业心强·中年发力·自主能力强
下停长：晚年有福·子女孝顺·物质丰足
```

### 五岳（面部横向五区）

```
东岳（左颧）：社交·人际·左边贵人
西岳（右颧）：权力·管理·右边贵人
南岳（额头）：名声·智慧·早年运
北岳（下巴）：根基·晚年·不动产
中岳（鼻子）：自我·财运·中年运

五岳丰隆：一生富贵
某岳凹陷/偏斜：对应方面运势受损
```

### 五行面型

| 面型 | 五行 | 特征 | 性格 | 适合行业 |
|------|------|------|------|---------|
| **金形面** | 金 | 方脸·骨骼突出·面白·线条硬朗 | 果断·重义·刚毅 | 金融·军警·管理 |
| **木形面** | 木 | 长脸·瘦长·额窄下巴尖 | 仁慈·有创意·善思考 | 文教·艺术·研究 |
| **水形面** | 水 | 圆脸·肉多·面色偏黑·下巴圆 | 智慧·灵活·善交际 | 贸易·物流·传媒 |
| **火形面** | 火 | 尖脸·额宽下巴窄·面红·眼神锐 | 热情·急躁·有魄力 | 科技·能源·餐饮 |
| **土形面** | 土 | 方圆·敦厚·面黄·鼻大肉厚 | 稳重·诚信·包容 | 地产·农业·教育 |

## 二、五官详解

### 1. 额头（官禄宫·天庭）

```
宽阔饱满：聪明·少年运好·有领导力
狭窄凹陷：少年辛苦·思虑多
有纹：额纹三条清晰（天纹·人纹·地纹）= 有阅历
发际线整齐：做事有条理
发际线杂乱/美人尖：早年波折·但有个性
```

### 2. 眉毛（兄弟宫·保寿宫）

```
浓密顺畅：精力旺盛·兄弟缘好·重情义
稀疏散乱：孤僻·兄弟缘薄
眉长过目：有寿·聪明
眉短不过目：性急·兄弟少
两眉距离宽（>一指半）：心胸宽·乐观
两眉距离窄（<一指）：心思重·易焦虑
眉尾上扬：有志气·不服输
眉尾下垂：温和·但缺乏冲劲
一字眉：意志坚定·执行力强
柳叶眉：温柔·感性·人缘好
```

### 3. 眼睛（监察官·心之窗）

```
大而有神：聪明·开朗·桃花旺
小而有神：精明·谨慎·城府深
眼神温和：善良·好相处
眼神锐利：决断力强·有威严
眼白多（三白眼/四白眼）：性格极端·需注意
桃花眼（眼尾微翘·水汪汪）：异性缘极佳
丹凤眼：有权威·适合领导
瞳仁大而黑亮：心地善良·有福
眼下卧蚕饱满：子女缘好·异性缘佳
眼下凹陷/泪沟深：子女缘薄·肾气不足
```

### 4. 鼻子（财帛宫·审辨官）

```
鼻梁高挺：自尊心强·有主见·事业心
鼻梁低平：随和·配合性强·但缺乏主见
鼻头圆润有肉：聚财·心地善良
鼻头尖薄：花钱快·心思敏感
鼻翼饱满：存钱能力强
鼻翼薄/鼻孔外露：存不住钱
鼻梁有节/驼峰鼻：性格倔强·中年有波折
鼻歪：心术不正或中年运势偏离
```

### 5. 嘴巴（出纳官·食禄宫）

```
大嘴：慷慨·口才好·社交达人
小嘴：谨慎·保守·精细
嘴唇厚（尤其下唇）：重感情·享受型
嘴唇薄：理性·善言·但薄情倾向
上唇厚下唇薄：付出型
下唇厚上唇薄：接受型·享福
嘴角上扬：乐观·人缘好
嘴角下垂：悲观·不满·晚年孤
唇色红润：健康·有福
唇色暗淡/发紫：脾胃或血液循环问题
```

### 6. 耳朵（采听官）

```
耳大而厚：有福·肾气足·长寿
耳小而薄：肾气弱·精力不足
耳垂大而饱满：有福·晚年好
耳垂薄/无耳垂：物质运一般
耳朵贴脑（正面看不见耳朵）：聪明·善于理财
耳朵招风（正面明显外扩）：个性独立·花钱大方
耳高过眉：智商高·有名望
耳低于眉：性格温和·服从性强
```

## 四、气色判断

```
气色是面相的「动态指标」——面型五官是先天的，气色是当前状态。

红润/黄明：运势好·健康·有喜事
暗沉/灰败：运势低·健康差·需注意
青色：肝气郁结·受惊·有怒气
白色：肺气虚·悲伤·失血
黑色：肾虚·恐惧·重病征兆
黄色：脾虚·消化问题

某个宫位突然变色 → 该宫对应的事务即将有变化
如：印堂发暗 → 近期运势下降
    鼻头发红 → 最近花钱多/有酒色之事
    眼下发青 → 睡眠差/肾虚/子女有事
```

## 五、AI面相分析的关键点

```
AI通过摄像头或照片分析时，重点看：
1. 面型判断（五行归类）
2. 三停比例（上中下哪个突出）
3. 五官逐一分析（大小·形状·位置·对称性）
4. 印堂状态（命宫）
5. 整体气色
6. 痣的位置和含义（可选）
7. 综合给出性格·财运·事业·感情·健康的简评
```

> ⚠️ **重要声明**：面相分析基于传统文化经验，不是科学诊断。AI分析受拍摄角度、光线、表情影响，仅供趣味参考。
