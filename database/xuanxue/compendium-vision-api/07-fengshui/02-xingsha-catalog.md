## 一、外部形煞

### 道路相关

| 煞名 | 形态 | 影响 | 化解 |
|------|------|------|------|
| **路冲（枪煞）** | 直路正对大门或窗户 | 血光·意外·疾病·破财 | 石狮·泰山石敢当·大型盆栽遮挡 |
| **反弓煞** | 弯路外侧对着房屋（路如弓背对我） | 破财·不聚财·人心不稳 | 大型圆叶植物·凸面镜 |
| **镰刀煞** | 天桥或高架路弯曲如镰刀切过 | 血光·损伤 | 铜葫芦·植物遮挡 |
| **割脚煞** | 水或路紧贴楼基而过 | 财来财去·运势反复 | 铜麒麟 |
| **斜路冲** | 道路斜对门窗 | 不安·口舌 | 遮挡·山海镇 |

### 建筑相关

| 煞名 | 形态 | 影响 | 化解 |
|------|------|------|------|
| **天斩煞** | 两栋高楼之间的狭窄缝隙正对 | 血光·手术·意外 | 铜马·龙龟·遮挡 |
| **孤峰煞** | 住宅远高于周围建筑，四面无遮 | 孤独·子女离散·风大气散 | 难化解，选址避免 |
| **探头煞** | 对面建筑后方有更高建筑探头出来 | 盗贼·小人 | 百叶窗·窗帘 |
| **穿心煞** | 大门正对电梯口·楼梯口 | 漏财·健康差 | 屏风·玄关·大叶植物 |
| **壁刀煞** | 对面建筑墙角尖角正对 | 疾病·血光·争吵 | 凸面镜·遮挡 |
| **尖角煞** | 三角形建筑或尖角物正对 | 火灾·口舌·心血管 | 鱼缸（水克火）·植物 |

### 环境相关

| 煞名 | 形态 | 影响 | 化解 |
|------|------|------|------|
| **声煞** | 临近高速路·铁路·机场 | 精神紧张·失眠·心脏 | 隔音·厚窗帘·难根治 |
| **味煞** | 临近垃圾站·工厂·排污口 | 健康·财运差·人气散 | 搬离为上 |
| **光煞** | 对面大厦玻璃幕墙反光 | 血光·眼疾·精神 | 厚窗帘·磨砂玻璃 |
| **天桥煞** | 高架桥压过住宅上方 | 压迫·运势受阻 | 铜葫芦·难化解 |
| **电塔煞** | 住宅近高压电塔·变电站 | 健康·精神 | 距离为上·难化解 |

## 二、内部形煞

### 门的问题

| 煞名 | 形态 | 影响 | 化解 |
|------|------|------|------|
| **门对门** | 两门正对（如卧室门对卫生间门） | 口舌·秽气入室 | 珠帘·长期关门·植物 |
| **门对窗** | 大门正对后窗/阳台（穿堂风） | 漏财·气场不聚 | 屏风·玄关·不透光窗帘 |
| **门对楼梯** | 大门一开就见下行楼梯 | 财运下泄 | 门槛·屏风 |
| **门对柱** | 大门外有柱子正对 | 阻碍·健康 | 大叶植物遮挡 |

### 厨卫问题

| 煞名 | 形态 | 影响 | 化解 |
|------|------|------|------|
| **水火相冲** | 炉灶与水池正对或紧邻 | 家人不和·消化疾病 | 中间放木质隔板（水生木生火） |
| **厕占中宫** | 卫生间位于房屋中心 | 心脏·脾胃·全家健康 | 长期通风·盐灯·难根治 |
| **灶对门** | 炉灶正对厨房门或大门 | 漏财·女主人健康 | 挡板·玄关 |
| **开门见灶** | 入户即见厨房炉灶 | 财露白·漏财 | 屏风遮挡 |

### 卧室问题

| 煞名 | 形态 | 影响 | 化解 |
|------|------|------|------|
| **横梁压床** | 梁正压床头或床中 | 头痛·压迫·失眠 | 假天花·移床·藏梁装修 |
| **镜对床** | 镜子正对床铺 | 惊吓·精神不宁·夫妻不和 | 移走镜子·用布遮挡 |
| **床头空** | 床头不靠墙 | 无靠山·睡眠差·不安 | 床头靠实墙·加厚床头板 |
| **房门对床** | 门开直冲床铺 | 健康·隐私·精神 | 移床·屏风 |

## 三、方位煞（结合飞星年运）

### 2024-2026年方位注意

```
2024年（三碧入中）：
  五黄在正西 → 西方忌动土·红色
  二黑在东南 → 东南方忌求医方向
  三碧在中宫 → 家中忌争吵·绿色过多
  
2025年（二黑入中）：
  五黄在正西偏北 → 西方继续不利
  二黑在中宫 → 全屋病气重·多放铜器
  
2026年（一白入中）：
  五黄在东方 → 东方忌动土
  二黑在西北 → 西北方放铜葫芦
  一白在中宫 → 利桃花·人缘
```

## 四、现代住宅快速风水检查清单

```
外部环境（10分钟）：
□ 大门前是否有直路冲？
□ 周围有无尖角建筑对着？
□ 两楼夹缝是否正对？
□ 背后是否有靠（高楼/山）？
□ 前方是否开阔？
□ 附近有无高压电塔/变电站/垃圾站？

入户第一眼（5分钟）：
□ 开门见什么？（宜见客厅，忌见灶/厕/镜/楼梯）
□ 玄关是否有遮挡？（防穿堂风）
□ 光线是否充足？（阴暗不利）

室内布局（15分钟）：
□ 卫生间是否在中心位？
□ 厨房炉灶与水池关系？
□ 卧室床头是否靠墙？
□ 镜子有无对着门/床？
□ 有无横梁压顶？
□ 各房间通风采光如何？

方位配合（需罗盘）：
□ 坐向属东四宅还是西四宅？
□ 主卧是否在吉方（生气/天医）？
□ 厨房/卫生间是否在凶方？（凶方适合压煞）
□ 当年飞星五黄二黑在哪？
```
