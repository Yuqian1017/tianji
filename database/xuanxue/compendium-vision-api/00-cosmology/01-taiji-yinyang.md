## 一、太极：万物之始

### 核心概念

「太极」是中国哲学描述宇宙原初状态的概念。它不是「无」，而是「未分」——一切可能性浑然一体的状态。

**关键理解**：太极不是静止的。它内含运动的势能，时刻处于将分未分的临界点。这就是为什么太极图不是一条直线分开的黑白两半，而是用S曲线——它在表达「动」。

### 太极的层级性

太极不只存在于宇宙起源。在玄学中，**任何事物都可以作为一个太极来分析**。这是极其重要的方法论：

- 宇宙是一个太极
- 一个人是一个太极
- 一件事是一个太极
- 一个卦是一个太极
- 甚至一个爻也是一个太极（还可以再分）

> 🔮 **开发启示**：在编写占卜程序时，任何分析对象都可以建模为一个包含「阴面」和「阳面」的二元结构。这是所有玄学算法的原子单位。

## 二、阴阳：宇宙的二进制

### 基本定义

阴阳不是两种「东西」，而是**描述事物的两种属性方向**。

| 阳 | 阴 |
|----|----|
| 动 | 静 |
| 热 | 冷 |
| 明 | 暗 |
| 上 | 下 |
| 外 | 内 |
| 刚 | 柔 |
| 奇数 | 偶数 |
| 天 | 地 |
| 男 | 女 |

> *"记住，阴阳不是好坏。太阳很好吧？晒多了中暑。阴凉很差吗？大热天你最想要的就是它。"——师父*

### 阴阳的七大关系

这七种关系是所有玄学推演的逻辑根基：

#### 1. 对立 Opposition
阴阳是相对的。没有绝对的阴或绝对的阳，所有判断都需要参照系。

#### 2. 互根 Interdependence
阴以阳为根，阳以阴为根。没有「动」就无法定义「静」。**在实际推演中，看到极阳要想到阴，看到纯阴要找阳。**

#### 3. 消长 Waxing & Waning
阴阳是此消彼长的。阳长则阴消，阴长则阳消。四季就是最直观的阴阳消长：春夏阳长阴消，秋冬阴长阳消。

#### 4. 转化 Transformation
量变到质变——阳极生阴，阴极生阳。太极图中黑鱼眼白、白鱼眼黑，就是在表达这种转化的种子始终存在。

> 🔮 **排盘应用**：六爻中「老阳变阴、老阴变阳」，八字中「物极必反」的格局判断，都是阴阳转化的直接应用。

#### 5. 包含 Containment
阴中有阳，阳中有阴。任何阴性事物内部都可以再分出阴阳。这就是「太极层级性」的体现。

#### 6. 平衡 Balance
阴阳趋向动态平衡。失衡则生变——这是所有「断吉凶」的底层逻辑：**平衡为吉，失衡为凶。**

#### 7. 交感 Interaction
阴阳相互作用、交感化生万物。「天地交泰」「男女交合」「水火既济」都是阴阳交感的象。

## 三、阴阳在各术数中的体现

| 术数 | 阴阳的体现 |
|------|------------|
| **易经** | 阳爻（—）阴爻（--），构成八卦与六十四卦 |
| **六爻** | 老阳·少阳·老阴·少阴（动静之分） |
| **八字** | 天干分阴阳（甲阳木·乙阴木……），地支分阴阳 |
| **紫微** | 阴阳在星曜属性中（紫微为阴土帝星） |
| **奇门** | 阴遁·阳遁（冬至后阳遁，夏至后阴遁） |
| **风水** | 山为阴水为阳（相对）；阳宅阴宅之分 |

## 四、阴阳的数学表达

> *"你说阴阳是二进制？也行，你们现代人的说法。不过二进制只管开关，阴阳还管大小。"——师父*

### 四象编码

阴阳不只有 0 和 1 两个状态。它的完整编码是**四象**：

| 四象 | 阴阳组合 | 数值 | 含义 | 六爻中 |
|------|----------|------|------|--------|
| 太阳 | 阳 + 阳 | 9 (老阳) | 极阳将变 | ⚊ → 变爻 |
| 少阴 | 阳 + 阴 | 8 (少阴) | 阴方生 | ⚋ → 静爻 |
| 少阳 | 阴 + 阳 | 7 (少阳) | 阳方生 | ⚊ → 静爻 |
| 太阴 | 阴 + 阴 | 6 (老阴) | 极阴将变 | ⚋ → 变爻 |

**关键规则**：老变少不变。9（老阳）和 6（老阴）是动爻，7（少阳）和 8（少阴）是静爻。这是六爻起卦的核心逻辑。

```
🔮 实战：四象判定算法

function fourImage(value) {
  // value: 6, 7, 8, 9
  const isYang = (value % 2 !== 0);  // 奇数为阳，偶数为阴
  const isMoving = (value === 6 || value === 9);  // 老阴老阳为动
  return {
    yinyang: isYang ? 'yang' : 'yin',
    moving: isMoving,
    changeTo: isMoving ? (isYang ? 'yin' : 'yang') : null
  };
}
```
