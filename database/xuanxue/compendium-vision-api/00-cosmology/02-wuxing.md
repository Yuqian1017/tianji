# 练气期 · 第二课 · 五行生克

> **难度标签**：🟢必修：五行生克 | 🟡进阶：旺衰·归类 | 🔴专研：河图洛书

> *师父在地上摆了五样东西：一块木头、一根蜡烛、一捧土、一枚铜钱、一碗水。*
> *"这五样东西之间有什么关系？"*
> *"呃……都是您的？"*
> *"木头能点火，火烧完成灰成土，土里挖出金属，金属表面凝水，水浇灌出木头。这叫生。"*
> *"那克呢？"*
> *"木头扎进土里，土挡住水，水浇灭火，火融化金属，金属砍断木头。简单吧？"*
> *"简单。"*
> *"简单？那我问你——火克金，为什么金首饰要用火来炼？水克火，为什么蒸汽机靠的就是水加火？"*
> *"……"*
> *"五行不是死记硬背就行的。克也分好几种克法。坐下，慢慢说。"*

## 二、五行生克关系

### 相生 (Generation Cycle)

```
木 → 火 → 土 → 金 → 水 → 木
```

- **木生火**：木燃烧生火
- **火生土**：火烧尽成灰成土
- **土生金**：土中蕴藏矿石金属
- **金生水**：金属表面凝结水珠（亦指金属熔化为液态）
- **水生木**：水灌溉滋养木

**生的关系中**：生我者为「母」，我生者为「子」。

在六爻中，这对应**六亲**关系的一部分：
- 生我者 → 父母爻
- 我生者 → 子孙爻

### 相克 (Restriction Cycle)

```
木 → 土 → 水 → 火 → 金 → 木
（木克土，土克水，水克火，火克金，金克木）
```

- **木克土**：树根扎入土地，破土而出
- **土克水**：土筑堤坝阻挡水流
- **水克火**：水浇灭火
- **火克金**：火焰熔化金属
- **金克木**：刀斧砍伐树木

**克的关系中**：克我者为「官鬼」，我克者为「妻财」。

在六爻中：
- 克我者 → 官鬼爻
- 我克者 → 妻财爻

### 快速记忆法

> *"木火土金水，相邻就是生，隔一个就是克。"——师姐*

```
生：顺序紧邻 → 木→火→土→金→水→木...
克：隔一位取 → 木→土→水→火→金→木...
```

## 三、五行生克的深层规则

> *"这才是重点。生克不是简单的加减法，里面有讲究。"——师父*

### 1. 生之有度

生不一定是好事。五行讲究**平衡**：

- **正常之生**：母旺子弱，母生子，吉
- **过度之生**：母太旺而生子太过 → 子反受害（溺爱伤子）
- **无力之生**：母太弱而强行生子 → 母子皆伤

> 🔮 **排盘应用**：八字中「印太重」就是被生太过——过度保护反而压制了日主的能力发挥。

### 2. 克之分类

克有好几种层次：

| 类型 | 条件 | 结果 | 吉凶 |
|------|------|------|------|
| **正克** | 克者与被克者力量适中 | 被克者受制约 | 中性（需看具体情境） |
| **强克** | 克者极旺，被克者极弱 | 被克者被彻底压制 | 多凶 |
| **反克** | 被克者极旺，克者极弱 | 克者反受伤害 | 变局 |
| **有情之克** | 如火克金→金被炼成器 | 约束成器 | 吉 |
| **无情之克** | 如金克木→木被砍碎 | 毁灭性破坏 | 凶 |

> *"你看铁匠打铁，火克金，但铁被火炼了之后变成刀剑器具。这叫有情之克，克出名堂来了。金克木呢？一刀下去树就倒了，这叫无情之克。"——师父*

### 3. 相乘与相侮

- **相乘**：克的过度。正常是木克土，但木太旺疯狂克土 → 相乘（以大欺小）
- **相侮**：反向之克。正常是金克木，但木太旺反过来顶住金 → 相侮（以下犯上）

### 4. 泄与耗

除了生和克，还有两种重要的力量流动：

- **泄**：生的反面。火生土，对火来说是**泄气**——我生出去的东西消耗了我的力量
- **耗**：克的反面。我克别人也需要消耗力量

```
🔮 实战：五行关系判定

function getRelation(me, target) {
  // 五行：wood, fire, earth, metal, water
  const order = ['wood', 'fire', 'earth', 'metal', 'water'];
  const myIdx = order.indexOf(me);
  const tgtIdx = order.indexOf(target);
  const diff = (tgtIdx - myIdx + 5) % 5;
  
  switch(diff) {
    case 0: return { relation: '同类/比和', type: 'sibling' };   // 比肩/劫财
    case 1: return { relation: '我生', type: 'generate' };       // 食神/伤官
    case 2: return { relation: '我克', type: 'restrict' };       // 正财/偏财
    case 3: return { relation: '克我', type: 'restricted_by' };  // 正官/偏官
    case 4: return { relation: '生我', type: 'generated_by' };   // 正印/偏印
  }
}

// 六亲对应（以日主五行为"我"）
// diff=0: 比肩/劫财（阴阳同/异）
// diff=1: 食神/伤官
// diff=2: 正财/偏财
// diff=3: 正官/七杀
// diff=4: 正印/偏印
```

## 四、五行旺衰与四季

五行的力量不是固定的，它随时间（季节/月份）变化：

### 五行旺相休囚死

| 季节 | 旺 | 相 | 休 | 囚 | 死 |
|------|----|----|----|----|------|
| **春（寅卯月）** | 木 | 火 | 水 | 金 | 土 |
| **夏（巳午月）** | 火 | 土 | 木 | 水 | 金 |
| **长夏（辰戌丑未月）** | 土 | 金 | 火 | 木 | 水 |
| **秋（申酉月）** | 金 | 水 | 土 | 火 | 木 |
| **冬（亥子月）** | 水 | 木 | 金 | 土 | 火 |

**规则**：
- **旺**：当令者旺（春天木旺）
- **相**：旺者所生为相（春天木生火，火为相——次旺）
- **休**：生旺者为休（水生木，水在春天休——退休了）
- **囚**：克旺者为囚（金克木，金在春天囚——被关起来了）
- **死**：旺者所克为死（木克土，土在春天死——最弱）

> *"为什么六爻要看月建日辰？因为同样一个金爻，在秋天是虎，在夏天是病猫。时间决定了五行的底气。"——师姐*

```
🔮 实战：五行旺衰判定

const SEASONAL_STRENGTH = {
  // [旺, 相, 休, 囚, 死]
  spring:  ['wood', 'fire', 'water', 'metal', 'earth'],
  summer:  ['fire', 'earth', 'wood', 'water', 'metal'],
  lateSummer: ['earth', 'metal', 'fire', 'wood', 'water'],
  autumn:  ['metal', 'water', 'earth', 'fire', 'wood'],
  winter:  ['water', 'wood', 'metal', 'earth', 'fire']
};

function getStrength(element, season) {
  const order = SEASONAL_STRENGTH[season];
  const idx = order.indexOf(element);
  return ['旺(wang)', '相(xiang)', '休(xiu)', '囚(qiu)', '死(si)'][idx];
}

// 月支 → 季节映射
function monthToSeason(earthlyBranch) {
  const map = {
    '寅': 'spring', '卯': 'spring',
    '巳': 'summer', '午': 'summer',
    '辰': 'lateSummer', '戌': 'lateSummer', 
    '丑': 'lateSummer', '未': 'lateSummer',
    '申': 'autumn', '酉': 'autumn',
    '亥': 'winter', '子': 'winter'
  };
  return map[earthlyBranch];
}
```

## 五、五行的特殊状态

### 五行长生十二宫

每个五行都有从出生到消亡的十二个生命阶段，对应十二地支：

| 阶段 | 含义 | 在八字/六爻中 |
|------|------|---------------|
| **长生** | 出生 | 有生气，渐旺 |
| **沐浴** | 初生，脆弱 | 多主不稳定、桃花 |
| **冠带** | 成长 | 渐强 |
| **临官** | 壮年为官 | 旺盛 |
| **帝旺** | 鼎盛 | 最旺，但物极必反 |
| **衰** | 开始衰落 | 力量减弱 |
| **病** | 生病 | 很弱 |
| **死** | 死亡 | 极弱（但不一定真无用） |
| **墓(库)** | 入墓 | 被收藏/封印，可被冲开 |
| **绝** | 断绝 | 最弱状态 |
| **胎** | 受孕 | 有希望的开始 |
| **养** | 孕育中 | 将要出生 |

## 六、本课心法

> *"五行是玄学的操作系统。你以后学的所有术数，底层跑的都是五行。"——师父*
> *"这么说来，我是在学一个古代操作系统？"*
> *"对。而且这个操作系统到现在还在用——中医、风水、命理，全靠它。版本号都不用更新的那种。"*

1. **万物归五行**：任何事物都可以归到五行之一
2. **生克是动词**：五行之间永远在互动，不存在孤立的五行
3. **旺衰看时令**：离开时间谈五行强弱是耍流氓
4. **平衡定吉凶**：五行全了不一定好，平衡了才好
