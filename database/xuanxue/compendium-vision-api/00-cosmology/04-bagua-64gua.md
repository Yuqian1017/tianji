## 一、八卦基础

### 八卦的诞生

阴阳两仪各自再分阴阳 → 四象 → 四象再分 → 八卦

```
太极 → 两仪(阴/阳) → 四象(太阳/少阴/少阳/太阴) → 八卦
```

每个卦由三个爻（yáo）组成，自下而上排列：

```
上爻 ——   （阳爻：一条长线）
中爻 -- --（阴爻：中间断开的两短线）
下爻 ——  
```

| **乾 ☰** | ≡ | 1 | 6 | 金 | 西北 | 天 | 父 | 头 | 马 | 刚健 |

### 八卦的二进制编码

从下往上读，阳爻=1，阴爻=0：

| 卦 | 下 | 中 | 上 | 二进制 | 十进制 | 先天数 |
|----|----|----|----|----|----|----|
| 坤 | 0 | 0 | 0 | 000 | 0 | 8 |
| 艮 | 1 | 0 | 0 | 001 | 1 | 7 |
| 坎 | 0 | 1 | 0 | 010 | 2 | 6 |
| 巽 | 0 | 0 | 1 | 100→反转→001 | - | 5 |
| ... | | | | | | |

## 二、先天八卦与后天八卦

### 先天八卦（伏羲八卦）

> *"先天八卦讲的是宇宙本来的样子——天地定位，水火相射。"——师父*

```
        ☰ 乾(1)
    ☴巽(5)    ☱兑(2)
  ☵坎(6)        ☲离(3)
    ☶艮(7)    ☳震(4)
        ☷ 坤(8)
```

**先天数口诀**：「乾一兑二离三震，巽五坎六艮七坤」

**对称关系**：先天八卦中对面的两卦数字之和 = 9
- 乾(1) + 坤(8) = 9
- 兑(2) + 艮(7) = 9
- 离(3) + 坎(6) = 9
- 震(4) + 巽(5) = 9

**先天八卦的用途**：梅花易数起卦主要用先天数。

### 后天八卦（文王八卦）

> *"后天八卦讲的是世间万物运行的规律——春夏秋冬，东南西北。"——师父*

```
        ☲ 离(9) 南
    ☴巽(4)        ☷坤(2)
  ☳震(3) 东          ☱兑(7) 西
    ☶艮(8)        ☰乾(6)
        ☵ 坎(1) 北
```

**后天数口诀（洛书数）**：「戴九履一，左三右七，二四为肩，六八为足，五居中央」

**后天八卦的用途**：风水方位、六爻配宫、奇门九宫，用的都是后天八卦。

## 三、六十四卦

### 生成原理

两个八卦（三爻卦）上下叠加 → 六爻卦（六十四卦）

- **下卦（内卦）**：下面三个爻
- **上卦（外卦）**：上面三个爻

8 × 8 = 64 种组合

### 卦序系统

六十四卦有多种排列顺序，不同术数使用不同卦序：

#### 文王序（通行本周易序）
1乾 2坤 3屯 4蒙 5需 6讼 7师 8比 ... 63既济 64未济

用途：解读周易经文。

#### 京房八宫序（六爻必用）

八宫每宫八卦，按世应变化排列：

| 宫 | 本宫卦 | 一世 | 二世 | 三世 | 四世 | 五世 | 游魂 | 归魂 |
|----|--------|------|------|------|------|------|------|------|
| **乾宫(金)** | 乾为天 | 天风姤 | 天山遁 | 天地否 | 风地观 | 山地剥 | 火地晋 | 火天大有 |
| **兑宫(金)** | 兑为泽 | 泽水困 | 泽地萃 | 泽山咸 | 水山蹇 | 地山谦 | 雷山小过 | 雷泽归妹 |
| **离宫(火)** | 离为火 | 火山旅 | 火风鼎 | 火水未济 | 山水蒙 | 风水涣 | 天水讼 | 天火同人 |
| **震宫(木)** | 震为雷 | 雷地豫 | 雷水解 | 雷风恒 | 地风升 | 水风井 | 泽风大过 | 泽雷随 |
| **巽宫(木)** | 巽为风 | 风天小畜 | 风火家人 | 风雷益 | 天雷无妄 | 火雷噬嗑 | 山雷颐 | 山风蛊 |
| **坎宫(水)** | 坎为水 | 水泽节 | 水雷屯 | 水火既济 | 泽火革 | 雷火丰 | 地火明夷 | 地水师 |
| **艮宫(土)** | 艮为山 | 山火贲 | 山天大畜 | 山泽损 | 火泽睽 | 天泽履 | 风泽中孚 | 风山渐 |
| **坤宫(土)** | 坤为地 | 地雷复 | 地泽临 | 地天泰 | 雷天大壮 | 泽天夬 | 水天需 | 水地比 |

> 🔮 **开发要点**：京房八宫序决定了每个卦的**所属宫位**（五行属性）和**世应位置**，是六爻排盘的核心数据。必须用完整的查表实现。

**世应规则**：
- 本宫卦：世在六爻，应在三爻
- 一世卦：世在初爻，应在四爻
- 二世卦：世在二爻，应在五爻
- 三世卦：世在三爻，应在上爻
- 四世卦：世在四爻，应在初爻
- 五世卦：世在五爻，应在二爻
- 游魂卦：世在四爻，应在初爻
- 归魂卦：世在三爻，应在上爻

## 四、卦的变换关系

### 互卦

取二三四爻为下卦，三四五爻为上卦（去掉最上和最下两爻，中间四爻交叉取）。

```
原卦六爻：  6 5 4 3 2 1
互卦上卦：    5 4 3
互卦下卦：      4 3 2
```

### 变卦

动爻（老阳/老阴）阴阳互变后得到的新卦。

### 错卦

所有爻阴阳互换（阳变阴，阴变阳）。

### 综卦

将卦上下翻转（颠倒）。

```
🔮 实战：卦象变换算法

function getHuGua(yaoArray) {
  // yaoArray: [初爻, 二爻, 三爻, 四爻, 五爻, 上爻], 1=阳, 0=阴
  const lower = [yaoArray[1], yaoArray[2], yaoArray[3]]; // 2,3,4爻
  const upper = [yaoArray[2], yaoArray[3], yaoArray[4]]; // 3,4,5爻
  return { upper, lower };
}

function getBianGua(yaoArray, movingLines) {
  // movingLines: 动爻位置数组 [0,2,5] 表示初爻、三爻、上爻动
  const result = [...yaoArray];
  movingLines.forEach(pos => {
    result[pos] = result[pos] === 1 ? 0 : 1;
  });
  return result;
}

function getCuoGua(yaoArray) {
  return yaoArray.map(y => y === 1 ? 0 : 1);
}

function getZongGua(yaoArray) {
  return [...yaoArray].reverse();
}
```

> *"所以我现在相当于在学ABC？"*
> *"差不多。不过你学的是宇宙的ABC。"*
> *"那什么时候能写作文？"*
> *"别急。筑基期讲完易经原理，金丹期你就可以开始算卦了。再坚持两课。"*

1. **先天管数，后天管位**：起卦用先天数，看方位用后天图
2. **上外下内**：上卦为外、为远、为后来；下卦为内、为近、为起初
3. **卦宫定性**：六爻中，一个卦属于哪个宫，决定了它的五行属性
4. **变化是核心**：静卦少见，动爻才是断卦的关键
