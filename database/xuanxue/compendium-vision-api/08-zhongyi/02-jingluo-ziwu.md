# 中医篇 · 第二课 · 经络与子午流注

> **难度标签**：🟢必修：时辰脏腑·十大穴位 | 🟡进阶：十二经脉 | 🔴专研：奇经八脉

> *"经络是人体的'地铁网'。气血沿着经络运行，每条线路有固定的站点和时刻表。"——师父*
> *"时刻表？"*
> *"对。十二条经脉对应十二个时辰，这就是子午流注——气血在不同时辰流经不同脏腑。所以你经常听老人说'几点该干什么'，不全是迷信，是有生理基础的。"*

## 一、十二正经

### 总览

```
手三阴经（从胸走手）：手太阴肺经·手厥阴心包经·手少阴心经
手三阳经（从手走头）：手阳明大肠经·手少阳三焦经·手太阳小肠经
足三阳经（从头走足）：足阳明胃经·足少阳胆经·足太阳膀胱经
足三阴经（从足走腹）：足太阴脾经·足厥阴肝经·足少阴肾经
```

### 经脉流注顺序（大循环）

```
肺(寅)→大肠(卯)→胃(辰)→脾(巳)→心(午)→小肠(未)
 ↑                                              ↓
肝(丑)←胆(子)←三焦(亥)←心包(戌)←肾(酉)←膀胱(申)
```

### 子午流注与术数的对应

```
子午流注的十二时辰 = 十二地支 = 十二经脉
→ 八字时柱可以反映先天体质弱点

例：
- 生于子时（胆经当令）→ 与胆经相关体质特点
- 八字中某五行偏弱 → 对应脏腑可能先天不足
- 大运流年冲克某五行 → 该脏腑易出问题
```

## 三、奇经八脉

除十二正经外，还有八条特殊经脉，不直接配属脏腑。

| 经脉 | 功能 | 关键穴位 | 重要性 |
|------|------|---------|--------|
| **督脉** | 总督诸阳经·行于背正中 | 百会·大椎·命门 | ★★★★★ 阳脉之海 |
| **任脉** | 总任诸阴经·行于腹正中 | 关元·气海·膻中 | ★★★★★ 阴脉之海 |
| **冲脉** | 十二经之海·气血要冲 | 公孙(通) | ★★★★ 血海 |
| **带脉** | 约束纵行诸经·环腰一周 | 足临泣(通) | ★★★ 束腰带 |
| **阳维** | 维系诸阳经 | 外关(通) | ★★ |
| **阴维** | 维系诸阴经 | 内关(通) | ★★★ |
| **阳跷** | 主管下肢运动·司眼合 | 申脉(通) | ★★ |
| **阴跷** | 主管下肢内侧·司眼合 | 照海(通) | ★★ |

### 任督二脉

```
督脉：从尾骨（长强穴）沿脊柱上行至头顶（百会穴）再下至上唇
任脉：从下颌（承浆穴）沿身体前正中线下行至会阴

二脉首尾相连 → 小周天运行的路径
道家内丹功：打通任督二脉 = 小周天通畅
```
