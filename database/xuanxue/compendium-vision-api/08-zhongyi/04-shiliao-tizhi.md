## 一、食物的四气五味

### 四气（寒热温凉）

| 气 | 作用 | 适合体质 | 代表食物 |
|-----|------|---------|---------|
| **寒** | 清热泻火·凉血解毒 | 热证·阴虚 | 苦瓜·西瓜·绿豆·螃蟹·梨 |
| **凉** | 清热较缓·生津 | 偏热体质 | 小麦·绿茶·豆腐·鸭肉·芹菜 |
| **温** | 温中散寒·助阳 | 寒证·阳虚 | 生姜·羊肉·桂圆·韭菜·核桃 |
| **热** | 大补元阳·驱寒 | 极寒证 | 辣椒·花椒·肉桂·白酒·胡椒 |
| **平** | 性质平和·补益 | 任何体质 | 大米·猪肉·山药·红枣·枸杞 |

### 五味归经

| 味 | 五行 | 入脏 | 功能 | 过度之害 |
|-----|------|------|------|---------|
| **酸** | 木 | 肝 | 收敛·固涩·生津 | 伤脾（酸多伤筋） |
| **苦** | 火 | 心 | 泻火·燥湿·坚阴 | 伤肺（苦多伤气） |
| **甘** | 土 | 脾 | 补益·和中·缓急 | 伤肾（甘多伤骨） |
| **辛** | 金 | 肺 | 发散·行气·通窍 | 伤肝（辛多伤皮） |
| **咸** | 水 | 肾 | 软坚·散结·润下 | 伤心（咸多伤血） |

## 二、九大体质调理方案

### 1. 气虚质

**特征**：疲乏无力·气短·易感冒·声音低弱·舌淡胖有齿痕

```
【食疗】
主食：小米粥·山药粥·糯米
肉类：鸡肉·牛肉·黄鳝
蔬果：山药·土豆·南瓜·香菇·红枣·桂圆
药食：黄芪·党参·白术·茯苓

【茶饮】黄芪红枣茶（黄芪15g + 红枣5枚 + 枸杞10g）
【穴位】足三里·气海·关元（艾灸或按揉）
【运动】太极拳·八段锦·散步（忌剧烈运动）
【作息】早睡（22:00前）·午休20分钟
【忌】生冷食物·过度劳累·大汗淋漓的运动
```

### 2. 阳虚质

**特征**：畏寒怕冷·手脚冰凉·精神不振·小便清长·大便溏

```
【食疗】
主食：糯米·高粱
肉类：羊肉·狗肉·鹿肉·虾·海参
蔬果：韭菜·洋葱·南瓜·核桃·板栗·荔枝
药食：肉桂·干姜·附子（需医嘱）

【茶饮】肉桂红糖姜茶（肉桂3g + 生姜3片 + 红糖适量）
【穴位】命门·关元·神阙（艾灸为主，效果最佳）
【运动】慢跑·快走·阳光下运动
【作息】早睡·冬季注意保暖·泡脚
【忌】冰饮·西瓜·螃蟹·空调过低·熬夜
```

### 3. 阴虚质

**特征**：口干·手足心热·盗汗·皮肤干·便秘·舌红少苔

```
【食疗】
主食：小麦·粳米
肉类：鸭肉·猪肉·甲鱼·蛤蜊
蔬果：百合·银耳·梨·藕·黑芝麻·桑葚·枸杞
药食：麦冬·石斛·玉竹·沙参

【茶饮】枸杞菊花茶（枸杞15g + 菊花5朵 + 麦冬10g）
【穴位】三阴交·太溪·涌泉（按揉为主，少灸）
【运动】游泳·瑜伽·太极（忌大汗）
【作息】保证睡眠·减少熬夜·环境保湿
【忌】辛辣·煎炸·烧烤·熬夜·桑拿
```

### 4. 痰湿质

**特征**：体胖腹大·胸闷痰多·口黏·大便黏·舌胖苔腻

```
【食疗】
主食：薏米·赤小豆·玉米
肉类：鲫鱼·鲤鱼（利湿）
蔬果：冬瓜·白萝卜·芹菜·海带·荷叶
药食：陈皮·茯苓·薏仁·白扁豆

【茶饮】薏米陈皮茶（薏米30g + 陈皮6g + 茯苓10g，煮水）
【穴位】丰隆·足三里·阴陵泉（按揉+艾灸）
【运动】快走·跑步·爬山（需要出汗排湿）
【作息】少坐多动·避免潮湿环境
【忌】甜食·肥肉·冰饮·乳制品过量·久坐
```

### 5. 湿热质

**特征**：面垢油光·口苦口臭·大便黏臭·小便黄·易生痤疮·舌红苔黄腻

```
【食疗】
主食：薏米·绿豆
肉类：鸭肉·兔肉（忌牛羊肉）
蔬果：苦瓜·黄瓜·冬瓜·绿豆芽·马齿苋
药食：蒲公英·金银花·栀子·车前草

【茶饮】蒲公英金银花茶（蒲公英10g + 金银花10g + 菊花5g）
【穴位】曲池·合谷·阴陵泉（按揉泻法）
【运动】游泳·中强度有氧（需排汗但不过度）
【作息】清淡饮食·保持皮肤清洁·忌烟酒
【忌】辣椒·煎炸·甜腻·烟酒·火锅·熬夜
```

### 6. 血瘀质

**特征**：面色晦暗·唇色紫暗·舌有瘀斑·皮肤干燥·固定刺痛

```
【食疗】
主食：黑米·紫米·燕麦
肉类：适量红肉补血
蔬果：山楂·黑木耳·洋葱·藕·玫瑰花·桃仁
药食：当归·丹参·红花·川芎·三七

【茶饮】玫瑰山楂茶（玫瑰花6朵 + 山楂5g + 红糖适量）
【穴位】血海·三阴交·太冲（按揉活血）
【运动】有氧运动·拉伸·舞蹈（促进血液循环）
【作息】避免久坐·保持运动习惯·情绪舒畅
【忌】寒凉食物（使血更瘀）·久坐不动·抑郁
```

### 7. 气郁质

**特征**：情绪低落·叹气·胁肋胀痛·咽有异物感·失眠

```
【食疗】
主食：正常饮食
肉类：正常
蔬果：柑橘·佛手·金橘·玫瑰花·薄荷·茼蒿
药食：香附·郁金·合欢花·玫瑰花·薄荷

【茶饮】玫瑰薄荷疏肝茶（玫瑰花6朵 + 薄荷3g + 佛手5g）
【穴位】太冲·期门·膻中（按揉疏肝理气）
【运动】跑步·唱歌·社交运动·瑜伽（需要释放情绪）
【作息】多社交·培养兴趣·避免独处过久·听音乐
【忌】独处·压抑情绪·过度思虑·咖啡因过量
```

### 8. 特禀质（过敏）

**特征**：过敏体质·对花粉/尘螨/食物过敏·荨麻疹·哮喘

```
【食疗】
主食：正常（避开过敏原）
肉类：选择不过敏的种类
蔬果：山药·大枣·黄芪炖汤（固表）
药食：黄芪·防风·白术（玉屏风散组合）

【茶饮】黄芪防风茶（黄芪15g + 防风6g + 白术10g）
【穴位】足三里·肺俞·风池（增强免疫）
【运动】适度运动增强体质
【作息】避免过敏原·保持环境清洁
【忌】已知过敏原·辛辣刺激·环境污染
```

### 9. 平和质（健康维护）

```
【原则】饮食均衡·作息规律·适度运动·情志舒畅
【茶饮】枸杞菊花茶（日常保健）
【穴位】足三里（每日按揉·保健长寿穴）
【运动】每周3-5次中等强度运动
```

## 三、五行食疗对应表

| 五行 | 对应脏 | 偏弱时补什么 | 偏旺时泻什么 |
|------|--------|-------------|-------------|
| **木(肝)** | 肝·胆 | 酸味+绿色食物：菠菜·西兰花·猕猴桃 | 辛味制约：萝卜·薄荷（金克木） |
| **火(心)** | 心·小肠 | 苦味+红色食物：红枣·西红柿·莲子 | 咸味制约：海带·紫菜（水克火） |
| **土(脾)** | 脾·胃 | 甘味+黄色食物：南瓜·小米·山药 | 酸味制约：山楂·乌梅（木克土） |
| **金(肺)** | 肺·大肠 | 辛味+白色食物：百合·银耳·梨·白萝卜 | 苦味制约：苦瓜·莲心（火克金） |
| **水(肾)** | 肾·膀胱 | 咸味+黑色食物：黑芝麻·黑豆·核桃 | 甘味制约：南瓜·大枣（土克水） |

## 四、二十四节气养生要点

| 节气 | 养生重点 | 推荐食物 | 注意事项 |
|------|---------|---------|---------|
| 立春-雨水 | 养肝·升发 | 韭菜·豆芽·香椿 | 忌酸过多（酸收敛不利升发） |
| 惊蛰-春分 | 疏肝·防风 | 菠菜·荠菜·枸杞 | 春困用薄荷·菊花提神 |
| 清明-谷雨 | 祛湿·健脾 | 山药·薏米·白扁豆 | 雨水增多注意祛湿 |
| 立夏-小满 | 养心·清热 | 莲子·苦瓜·绿豆 | 忌过于贪凉伤脾 |
| 芒种-夏至 | 清暑·生津 | 西瓜·冬瓜·酸梅汤 | 出汗多补充盐分 |
| 小暑-大暑 | 防暑·养阴 | 绿豆汤·百合·银耳 | 最热时段忌暴晒 |
| 立秋-处暑 | 润肺·收敛 | 梨·蜂蜜·百合·银耳 | 秋燥伤肺多润燥 |
| 白露-秋分 | 滋阴·润肺 | 山药·莲藕·芝麻 | 减辛增酸 |
| 寒露-霜降 | 补肺·暖胃 | 栗子·核桃·羊肉 | 添衣保暖 |
| 立冬-小雪 | 补肾·藏精 | 黑芝麻·黑豆·羊肉 | 冬季进补开始 |
| 大雪-冬至 | 温阳·进补 | 当归羊肉汤·鹿茸 | 冬至一阳生·最佳进补日 |
| 小寒-大寒 | 固肾·防寒 | 牛肉·狗肉·核桃 | 最冷时段注意心血管 |

> ⚠️ **重要声明**：以上食疗建议为一般性参考，个人体质差异大，有疾病请就医。
