# 紫微斗数 · 十四主星逐星速查

> 每颗主星在命宫时的核心断语。庙旺利陷影响力度，但不改变基本性质。

---

## 一、北斗星系（主体·主攻）

### 紫微星 — 帝王星

```
五行：己土（阴土）| 化气：尊 | 主：官禄
庙：午子 | 旺：丑未寅申 | 利：辰戌 | 陷：巳亥卯酉

命宫断语：
• 领导欲强·自尊心高·不怒自威
• 好面子·不喜被人命令·适合当老板或管理层
• 庙旺：气度恢弘·贵人多·能服众
• 落陷：孤高自傲·难合群·好大喜功
• 最怕：化权后过于霸道·六煞同宫则「君臣不义」

十二宫关键词：
命宫：尊贵·领导   夫妻：配偶条件高·晚婚
财帛：大格局理财   官禄：掌权·高管
田宅：豪宅·家中说了算
```

### 天机星 — 智慧星

```
五行：乙木（阴木）| 化气：善 | 主：兄弟
庙：丑未卯酉 | 旺：子午寅申 | 利：辰戌 | 陷：巳亥

命宫断语：
• 聪明·多思·善策划·好奇心强
• 心思细腻·适合幕僚军师角色·善变
• 庙旺：机智过人·学习能力强·多才多艺
• 落陷：想太多·犹豫不决·小聪明·劳碌
• 最怕：化忌后思虑过重·焦虑失眠

十二宫关键词：
命宫：智慧·善变   夫妻：聪明配偶·但多变
财帛：理财灵活·偏财   官禄：策划·参谋·技术
```

### 太阳星 — 光明星

```
五行：丙火（阳火）| 化气：贵 | 主：官禄·父
庙：寅卯辰巳午 | 旺：未 | 利：申 | 陷：酉戌亥子丑

命宫断语：
• 光明磊落·热心助人·好面子·博爱
• 男命为自己·女命代表父亲和丈夫
• 庙旺（日生人更佳）：贵气十足·仕途顺利·有名望
• 落陷（夜生人更差）：操劳·付出多回报少·眼疾
• 特点：太阳不怕化忌（本就是付出之星）

十二宫关键词：
命宫：光明·贵人   夫妻：男主星·女看丈夫
官禄：政府·公职·名望   父母：父亲缘
```

### 武曲星 — 财星

```
五行：辛金（阴金）| 化气：财 | 主：财帛
庙：辰戌丑未 | 旺：寅申子午 | 利：卯酉 | 陷：巳亥

命宫断语：
• 刚毅果断·重财·实干·不善言辞
• 做事干脆利落·数字敏感·适合金融
• 庙旺：白手起家·理财高手·执行力强
• 落陷：孤克·固执·为财奔波
• 最怕：化忌后破财或为钱伤身

十二宫关键词：
命宫：财气·刚硬   财帛：正财旺·理财能力强
夫妻：配偶务实·但感情淡   官禄：金融·军警·工程
```

### 天同星 — 福星

```
五行：壬水（阳水）| 化气：福 | 主：福德
庙：子午寅申 | 旺：卯酉 | 利：辰戌 | 陷：巳亥丑未

命宫断语：
• 温和·知足·乐观·懒散·享受生活
• 人缘好·随和·但缺乏冲劲
• 庙旺：福气深厚·一生少灾·衣食无忧
• 落陷：好逸恶劳·得过且过·感情困扰
• 先苦后甜型：年轻不顺·中晚年好运

十二宫关键词：
命宫：福气·安逸   福德：享受人生·内心安宁
夫妻：温柔伴侣   官禄：服务业·教育·文艺
```

### 廉贞星 — 次桃花·囚星

```
五行：丙火（阳火）| 化气：囚（又化桃花）| 主：官禄
庙：寅申 | 旺：子午丑未 | 利：卯酉 | 陷：辰戌巳亥

命宫断语：
• 性格复杂·精明能干·好胜心强·桃花重
• 双面性：正则为官·邪则入狱（次桃花又次囚星）
• 庙旺：权谋过人·事业心强·政商能手
• 落陷：感情纠纷·官司·小人
• 化忌最凶：官非·血光·牢狱

十二宫关键词：
命宫：复杂·能干   夫妻：感情波折·多情
官禄：政治·法律·军警   疾厄：泌尿·血液
```

## 二、南斗星系（主用·主守）

### 天府星 — 财库星

```
五行：戊土（阳土）| 化气：贤·令 | 主：田宅·财库
庙：丑未寅申 | 旺：子午卯酉 | 利：辰戌 | 陷：巳亥

命宫断语：
• 稳重保守·善守成·重安全感·有管理才能
• 库存星=会存钱存资源·不喜冒险
• 庙旺：财帛丰厚·管理有方·一生安稳
• 落陷：吝啬·过于保守·错失机会
• 见禄存更旺·天府最喜「府相朝垣」格

十二宫关键词：
命宫：稳健·守成   财帛：善存钱·理财保守
田宅：不动产丰   官禄：管理·行政·银行
```

### 太阴星 — 月亮星

```
五行：癸水（阴水）| 化气：富 | 主：田宅·母
庙：酉戌亥子 | 旺：丑 | 利：未申 | 陷：寅卯辰巳午

命宫断语：
• 温柔·敏感·细腻·内向·富同理心
• 女命为自己·男命代表母亲和妻子
• 庙旺（夜生更佳）：清秀·不动产运好·田宅富
• 落陷（日生更差）：情绪化·多愁善感·皮肤过敏
• 化科最美·化忌则母亲或感情不顺

十二宫关键词：
命宫：柔美·敏感   夫妻：女主星·男看妻子
财帛：不动产·被动收入   田宅：佳·置产运好
```

### 贪狼星 — 欲望桃花星

```
五行：甲木（阳木）| 化气：桃花 | 主：夫妻
庙：丑未辰戌 | 旺：寅申 | 利：子午 | 陷：卯酉巳亥

命宫断语：
• 多才多艺·欲望强·交际广·好奇心旺盛
• 正桃花星：感情丰富·人缘极佳·会玩会享受
• 庙旺：才艺出众·人脉广·能在欲望中求进
• 落陷：贪心·好色·赌博·不专一
• 遇火铃「贪狼火铃」格可暴发

十二宫关键词：
命宫：多欲·多才   夫妻：感情丰富·桃花多
官禄：演艺·公关·销售   福德：追求享受
```

### 巨门星 — 口舌暗星

```
五行：癸水（阴水）| 化气：暗 | 主：疾厄
庙：子午 | 旺：丑未辰戌 | 利：寅申 | 陷：卯酉巳亥

命宫断语：
• 口才好·好辩·疑心重·直言不讳
• 暗星=暗藏·是非·猜疑·但也是专业分析力
• 庙旺：律师·教师·评论家·靠口才吃饭
• 落陷：口舌是非·人际关系差·暗中被害
• 化禄变「石中隐玉」·化忌口舌最多

十二宫关键词：
命宫：口才·是非   夫妻：争吵·但配偶口才也好
官禄：律师·教师·广播·外交   疾厄：肠胃·口腔
```

### 天相星 — 印星

```
五行：壬水（阳水）| 化气：印 | 主：官禄
庙：丑未寅申 | 旺：子午卯酉 | 利：辰戌 | 陷：巳亥

命宫断语：
• 正直·服务精神·善调和·辅佐型人才
• 印星=公文·盖章·管理·信任
• 庙旺：得上级信任·仕途顺·财印相生格更佳
• 落陷：优柔寡断·被人利用·刑忌夹印则灾
• 看天相要同时看两邻宫（受夹宫影响极大）

十二宫关键词：
命宫：服务·辅佐   官禄：公职·秘书·行政
夫妻：体贴·但缺主见   父母：长辈缘好
```

### 天梁星 — 荫星

```
五行：戊土（阳土）| 化气：荫 | 主：父母
庙：子午卯酉 | 旺：丑未寅申 | 利：辰戌 | 陷：巳亥

命宫断语：
• 正义感强·乐于助人·清高·有长辈缘
• 荫星=荫庇·化解灾厄·遇难呈祥
• 庙旺：逢凶化吉·受长辈提携·公职运好
• 落陷：自视清高·好为人师·孤芳自赏
• 「机月同梁」格：适合公教·稳定工作

十二宫关键词：
命宫：正义·清高   父母：长辈荫庇·孝顺
官禄：医疗·宗教·社工·教育   疾厄：逢凶化吉
```

### 七杀星 — 将星

```
五行：庚金（阳金）| 化气：权·肃杀 | 主：迁移
庙：寅申子午 | 旺：丑未辰戌 | 利：卯酉 | 陷：巳亥

命宫断语：
• 刚烈·果断·独立·不服输·开创型
• 将星=冲锋陷阵·孤军奋战·不安于室
• 庙旺：英雄气概·适合创业·军警·运动员
• 落陷：暴躁·冲动·孤克·意外伤害
• 「紫微七杀」同宫=化杀为权，大格局

十二宫关键词：
命宫：刚毅·冒险   官禄：创业·军警·外科
迁移：奔波·适合外地发展   疾厄：外伤·手术
```

### 破军星 — 耗星

```
五行：癸水（阴水）| 化气：耗 | 主：夫妻·子女
庙：子午 | 旺：丑未辰戌 | 利：寅申 | 陷：卯酉巳亥

命宫断语：
• 破旧立新·冒险·变动·不安分
• 耗星=消耗·破坏·但也是革新的力量
• 庙旺：开拓先锋·适合改革·创新行业
• 落陷：破败·浪费·婚姻不稳·人生起伏大
• 化禄后「破军化禄」破而后立·最有爆发力

十二宫关键词：
命宫：变动·革新   夫妻：波折·二婚可能
财帛：大起大落   官禄：开创·技术·破坏性创新
```
