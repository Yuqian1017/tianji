## 一、什么是用神

**用神**是卦中代表所问之事核心对象的那个爻。找对用神是断卦第一步，找错则全盘皆错。

### 用神取法总表

| 问事类型 | 用神取 | 说明 |
|---------|--------|------|
| **问自己** | 世爻 | 世爻永远代表自己 |
| **问对方/外界** | 应爻 | 应爻代表对方或外部环境 |
| **问父母/长辈/文书/房屋/学业** | 父母爻 | |
| **问兄弟/朋友/同事/竞争** | 兄弟爻 | |
| **问子女/晚辈/下属/自由** | 子孙爻 | |
| **问妻子/财运/天气** | 妻财爻 | |
| **问丈夫/官运/疾病/工作** | 官鬼爻 | |
| **问感情（男问女）** | 妻财爻 | |
| **问感情（女问男）** | 官鬼爻 | |
| **问官司** | 官鬼爻(原告) / 世爻(自己角色) | |
| **问失物** | 妻财爻 | |
| **问出行** | 世爻 + 应爻(目的地) | |

### 特殊情况

**卦中无用神**：如果六个爻中没有需要的六亲怎么办？

→ 去**伏神**中找。伏神是本宫卦（八纯卦）对应位置的爻。

**示例**：坎宫的卦中没有妻财爻 → 看坎为水（本宫卦）对应位置是否有妻财。

```
🔮 实战：伏神查找

function findHiddenSpirit(currentGua, targetRelation, palaceName) {
  // 当前卦中找不到用神时，去本宫卦（八纯卦）找
  const pureGua = getPureGuaNajia(palaceName); // 获取本宫卦的纳甲
  
  for (let i = 0; i < 6; i++) {
    const relation = getSixRelation(palaceWuxing, BRANCH_WUXING[pureGua[i].branch]);
    if (relation === targetRelation) {
      return {
        position: i,
        najia: pureGua[i],
        hiddenUnder: i, // 伏在当前卦第几爻之下
        flyingSpirit: currentGua[i] // 飞神（压在伏神上面的爻）
      };
    }
  }
  return null;
}
```

> *"伏神就像藏在地下的宝贝。它在，但被压着。能不能用，要看压在它上面的飞神跟它是什么关系。"——师姐*

**飞伏关系**：
- 飞生伏 → 伏神得助，有希望出来
- 伏生飞 → 伏神泄气，难以出来
- 飞克伏 → 伏神被压制，很难出来
- 伏克飞 → 伏神有力量但受阻
- 飞伏比和 → 关系平和

## 二、月建与日辰

> *"用神找到了，下一步是看它有多少能量。这就要看月建和日辰了。"——师姐*

### 月建（月令）

月建就是起卦时的月份地支，它是**最强的裁判**。

- **月建生用神** → 用神得月生，旺
- **月建克用神** → 用神受月克，弱
- **月建与用神同** → 用神得月令，最旺
- **月建被用神克** → 用神耗力克月（无实际影响，月建不可撼动）

### 日辰（日建）

日辰是起卦当天的地支，它是**第二裁判**，也有特殊功能：

- 日辰可以**冲起**（冲动衰弱的爻，使其暂时有用）
- 日辰可以**冲散**（冲旺相的爻使其暂失作用）
- 日辰可以**合住**（合住的爻暂时不能发挥作用）
- 日辰可以**入墓**（某些爻在日辰入墓，力量被封）

### 旺衰判断流程

```
🔮 实战：旺衰评估

function assessStrength(yao, monthBranch, dayBranch) {
  const yaoWuxing = BRANCH_WUXING[yao.branch];
  const monthWuxing = BRANCH_WUXING[monthBranch];
  const dayWuxing = BRANCH_WUXING[dayBranch];
  
  let strength = 0; // 正数旺，负数衰
  
  // 月建影响（权重最大）
  const monthRelation = getRelation(monthWuxing, yaoWuxing);
  if (monthRelation === '生' || monthWuxing === yaoWuxing) strength += 3;
  else if (monthRelation === '克') strength -= 3;
  else if (monthRelation === '泄') strength -= 1;
  
  // 日辰影响
  const dayRelation = getRelation(dayWuxing, yaoWuxing);
  if (dayRelation === '生' || dayWuxing === yaoWuxing) strength += 2;
  else if (dayRelation === '克') strength -= 2;
  
  // 动爻影响
  // ... 其他爻对它的生克
  
  return {
    level: strength > 2 ? '旺' : strength > 0 ? '相' : strength > -2 ? '休' : '囚',
    score: strength,
    monthEffect: monthRelation,
    dayEffect: dayRelation
  };
}
```

## 三、空亡

> *"空亡是六爻里最诡异的状态。一个爻空亡了，就像一个人魂不守舍——人在心不在。"——师姐*

### 空亡的判定

空亡由**日柱**决定。每个甲子旬（十天一轮）中，有两个地支没有被天干配到，这两个地支就是空亡。

| 旬 | 空亡 |
|----|------|
| 甲子旬（甲子→癸酉） | **戌、亥** |
| 甲戌旬（甲戌→癸未） | **申、酉** |
| 甲申旬（甲申→癸巳） | **午、未** |
| 甲午旬（甲午→癸卯） | **辰、巳** |
| 甲辰旬（甲辰→癸丑） | **寅、卯** |
| 甲寅旬（甲寅→癸亥） | **子、丑** |

```
🔮 实战：空亡判定

function getKongWang(dayStem, dayBranch) {
  const stems = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸'];
  const branches = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  
  const stemIdx = stems.indexOf(dayStem);
  const branchIdx = branches.indexOf(dayBranch);
  
  // 找到当前旬的起点（甲X）
  const xunStart = (branchIdx - stemIdx + 12) % 12;
  
  // 这个旬配了10个地支（从xunStart开始的10个），剩下2个是空亡
  const kong1 = branches[(xunStart + 10) % 12];
  const kong2 = branches[(xunStart + 11) % 12];
  
  return [kong1, kong2];
}

// 示例：壬午日 → stemIdx=8, branchIdx=6
// xunStart = (6-8+12)%12 = 10 → 甲戌旬
// 空亡 = branches[10+10 %12] = branches[8] = 申
//        branches[10+11 %12] = branches[9] = 酉
// → 甲戌旬空亡：申酉 ✓
```

### 空亡的效果

| 状态 | 效果 |
|------|------|
| **旺相而空** | 暂时不起作用，出空后起作用（有其事但延迟） |
| **衰弱而空** | 真空，无力回天（没有这件事） |
| **动爻空** | 动而无功（做了但没结果） |
| **用神空** | 所求之事暂时无着落（但看是真空还是假空） |

**出空**的时机：
- 过了该旬（十天后自然出空）
- 被日辰冲（冲空则实）
- 被月建冲

## 四、入墓

五行入墓（五行力量被封存）：

| 五行 | 墓库 | 说明 |
|------|------|------|
| 木 | 未 | 木入未库 |
| 火 | 戌 | 火入戌库 |
| 土 | 辰 | 土入辰库（有争议，部分流派不用） |
| 金 | 丑 | 金入丑库 |
| 水 | 辰 | 水入辰库 |

**入墓的效果**：力量被封，无法正常发挥。需要被冲开（冲墓库的地支）才能释放。

**注意**：
- 用神入日墓 → 很不利
- 用神入动爻墓 → 被其他爻的动态封住
- 墓库被冲 → 开库，封存的力量释放出来

## 五、旺衰判断总流程

```
🔮 实战：综合旺衰判定流程

1. 找到用神
2. 看月建对用神的关系（月令是第一判断标准）
   - 得月令生扶 → 基本旺
   - 被月令克制 → 基本衰
3. 看日辰对用神的关系
   - 日辰生扶 → 加分
   - 日辰克制 → 减分
4. 看是否空亡
   - 旺相空 → 暂不起作用，出空后有
   - 衰弱空 → 真空，无力
5. 看是否入墓
   - 入墓 → 被封，需冲开
6. 看动爻对用神的影响
   - 有动爻生用神 → 有助力
   - 有动爻克用神 → 有阻碍
7. 综合判断 → 用神旺/衰/中和
```
