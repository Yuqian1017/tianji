## 一、生克：最基本的力量关系

### 六爻中的生克

六爻中的生克遵循五行规则，但有**前提条件**：

**能生克的条件**：
1. 动爻可以生克其他爻（动态力量主动施加）
2. 日辰可以生克任何爻
3. 月建可以生克任何爻
4. 静爻之间不论生克（静的力量不主动发挥）
5. 变爻（动爻变出来的爻）只回头作用于动爻本身

### 贪生忘克

> *"五行生克有优先级——遇到又能生又能克的局面，优先看生。这叫贪生忘克。"——师姐*

当一个动爻同时面对它要克的目标和能生它的来源时：
- 它会**优先接受生**，而**忘记去克**
- 例：金爻动，本要克木，但同时有土动来生金 → 金贪土之生，暂不克木

### 动爻之间的生克

**原则**：两个动爻之间互相作用，看力量对比。

- 两动相生 → 被生的那个爻力量增强
- 两动相克 → 看谁在月令旺，旺者制弱者

## 二、六合

### 六合的种类

| 合的类型 | 内容 | 效果 |
|---------|------|------|
| **地支六合** | 子丑合·寅亥合·卯戌合·辰酉合·巳申合·午未合 | 两个爻合住，暂时失去独立行动能力 |
| **三合局** | 申子辰→水·亥卯未→木·寅午戌→火·巳酉丑→金 | 三个爻合化为一种五行，力量极大 |

### 合的效果

**合住（绊住）**：两个爻合了，好比两个人抱在一起，哪儿都去不了。

- **用神被合** → 事情拖延，被牵制
- **忌神被合** → 忌神被绊住，反而有利
- **日辰合住某爻** → 该爻当日被封（需冲开才能起作用）

**合化**：两个爻合了，且满足一定条件（月令支持），可以合化为新的五行。

> ⚠️ **合化条件严格**：不是合了就能化。需要月令中有化出的五行，或者动爻变出的爻支持化。初学阶段，先掌握「合住」的概念即可。

## 三、六冲

### 六冲的种类

子午冲·丑未冲·寅申冲·卯酉冲·辰戌冲·巳亥冲

### 冲的效果

**冲的本质是对抗和破坏**：

| 情况 | 效果 | 解读 |
|------|------|------|
| 日冲旺相之爻 | **暗动**（激发） | 本来静的爻被日冲激活，相当于动了 |
| 日冲衰弱之爻 | **冲散**（破坏） | 本来就弱，被冲直接溃散 |
| 日冲月合之爻 | **冲开**（解合） | 月合被日冲打破 |
| 动爻冲用神 | 看力量对比 | 旺冲旺：两败俱伤；旺冲衰：被冲方大败 |
| 冲空亡之爻 | **冲实**（填空） | 原本空亡，被冲后变实 |
| 冲入墓之爻 | **冲开墓库** | 封存的力量被释放 |

### 六冲卦与六合卦

**六冲卦**（上下卦爻全冲）：
乾为天·兑为泽·离为火·震为雷·巽为风·坎为水·艮为山·坤为地 + 天雷无妄·雷天大壮

→ 六冲卦主散、主变动、主不持久

**六合卦**：
天地否·地天泰·雷风恒·风雷益·水火既济·火水未济 + 另外几组

→ 六合卦主合、主稳定、主成事

## 四、进神与退神

### 进神

动爻变出的爻，地支按顺序前进，叫进神。

| 动爻 | 变爻 | 进退 |
|------|------|------|
| 寅 → 卯 | 木进 | ✅ 进神 |
| 巳 → 午 | 火进 | ✅ 进神 |
| 申 → 酉 | 金进 | ✅ 进神 |
| 亥 → 子 | 水进 | ✅ 进神 |
| 辰 → 丑 | 土进（争议） | △ |
| 未 → 戌 | 土进（争议） | △ |

**进神的效果**：力量增强，事情向前推进。用神动化进神 → 大吉。

### 退神

动爻变出的爻，地支按逆序后退，叫退神。

| 动爻 | 变爻 | 进退 |
|------|------|------|
| 卯 → 寅 | 木退 | ❌ 退神 |
| 午 → 巳 | 火退 | ❌ 退神 |
| 酉 → 申 | 金退 | ❌ 退神 |
| 子 → 亥 | 水退 | ❌ 退神 |

**退神的效果**：力量减弱，事情后退。用神动化退神 → 不利。

## 五、回头生克

动爻变出的变爻，会回过头来作用于原来的动爻：

| 情况 | 效果 |
|------|------|
| **回头生** | 变爻生动爻 → 动爻增强（吉） |
| **回头克** | 变爻克动爻 → 动爻受制（凶） |
| **化空** | 动爻变出的变爻落入空亡 → 动而无果 |
| **化墓** | 动爻变出的变爻是动爻的墓库 → 力量被封 |
| **化绝** | 动爻变到绝地 → 力量断绝 |

function analyzeChange(dynamicBranch, changedBranch, kongWang) {
  const dynWuxing = BRANCH_WUXING[dynamicBranch];
  const chgWuxing = BRANCH_WUXING[changedBranch];
  
  const results = [];
  
  // 回头生克
  const relation = getRelation(chgWuxing, dynWuxing);
  if (relation === '生') results.push('回头生');
  if (relation === '克') results.push('回头克');
  
  // 化空
  if (kongWang.includes(changedBranch)) results.push('化空');
  
  // 化墓
  const graveMap = { wood: '未', fire: '戌', metal: '丑', water: '辰', earth: '辰' };
  if (changedBranch === graveMap[dynWuxing]) results.push('化墓');

  if (jintuiResult) results.push(jintuiResult);
  
  return results;
}

## 六、反吟与伏吟

### 反吟

**卦的反吟**：变卦与本卦形成六冲关系（每个爻位都冲）。
**爻的反吟**：动爻变出的爻与原爻地支相冲。

**效果**：反复、折腾、来回不定。如果是凶事则反复受灾，吉事则反复波折。

### 伏吟

**卦的伏吟**：变卦与本卦完全相同（如乾为天变乾为天）。
**爻的伏吟**：动爻变出的爻与原爻地支相同。

**效果**：呻吟、痛苦、停滞。事情没有进展，陷在原地。

> *"反吟主来回折腾，伏吟主原地踏步。都不是好事，但反吟至少还在动，伏吟连动都动不了。"——师姐*

## 七、应期（时间判断）

> *"断吉凶是第一步。第二步是断什么时候应验——这叫应期。"——师姐*

### 常见应期规则

| 用神状态 | 应期 |
|---------|------|
| 用神旺相 | 逢合（合住的时候应事）或逢值（用神地支对应的日/月） |
| 用神衰弱 | 逢生（被生扶的时候应事）或逢旺（到旺的月份） |
| 用神被合 | 逢冲（冲开合的时候应事） |
| 用神被冲 | 逢合（合住冲源的时候应事） |
| 用神空亡 | 出空日（过旬）或冲空日 |
| 用神入墓 | 冲墓日/月 |
| 用神动化进神 | 进神到位之日 |

### 应期的时间单位

- 近事（几天内的事）→ 以日计
- 中期（月内的事）→ 以日或旬计
- 远事（年内的事）→ 以月计
- 大事（跨年的事）→ 以年计

> ⚠️ **应期是六爻最难的部分**，需要大量实践积累。以上规则是基本框架，实际断卦中需要综合多种因素判断。

## 九、本课心法

> *师姐合上书。"这些技法你不可能一次全记住。但你需要知道它们存在，断卦遇到了能查到。"*
> *"那什么是最重要的？"*
> *"四个字够了——旺衰生克。用神旺不旺，谁在生它谁在克它。把这个搞清楚，百分之八十的卦你都能断。剩下的技法是用来处理那百分之二十的复杂情况的。"*

1. **生克是核心**：一切技法的底层都是五行生克
2. **动静分明**：动爻主动，静爻被动，变爻回头
3. **合冲对称**：合是聚，冲是散；合住需冲开，冲散需合住
4. **进退看方向**：进神向前（好），退神后退（差）
