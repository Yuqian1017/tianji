## 一、起卦口诀

### 铜钱起卦

```
一个背面是单拆（少阳 ⚊ 值7）
两个背面是拆拆（少阴 ⚋ 值8）
三个背面是重拆（老阴 ⚋→⚊ 值6·动）
三个字面是单连（老阳 ⚊→⚋ 值9·动）
```

### 时间起卦

```
年月日和除以八，得数为上卦之家。
年月日时总除八，余为下卦不用夸。
年月日时总除六，余数便是动爻住。
```

## 二、纳甲口诀

### 八卦纳甲

```
乾纳甲壬外和内，子寅辰配午申戌。
坤纳乙癸分内外，未巳卯配丑亥酉。
震纳庚来起于子，巽纳辛来丑上推。
坎纳戊来寅为始，离纳己来卯为归。
艮纳丙来从辰起，兑纳丁来巳上飞。
```

## 四、用神取用口诀

```
自问己身用世爻，问人问事看应标。
父母文书与长辈，官鬼功名与上朝。
兄弟同辈与竞争，妻财钱财与内交。
子孙福德与后辈，测病子孙是良药。
```

### 分类用神速记

```
问功名：官鬼为用（有无功名看官鬼）
问财运：妻财为用（能否得财看妻财）
问婚姻：男看财女看官（各取所需）
问疾病：世爻为身官为病子为药
问官司：世原告应被告官鬼为法
问出行：世为人应为地看六冲六合
问失物：妻财为用兄弟为贼
问天气：子孙为晴父母为雨
问考试：父母为文凭官鬼为名次
```

## 五、旺衰判断口诀

### 月建

```
月建当权最有力，生扶用神吉更吉。
月建克用神衰败，月建合用神缠绵。
月破之爻如无力，出月方能论可期。
```

### 日辰

```
日辰为爻之根基，冲旺为暗动相宜。
日辰冲衰为冲散，空逢日实空方离。
日建生用力可嘉，日建克用须提防。
```

### 空亡

```
甲子旬中空戌亥，甲戌旬中空申酉。
甲申旬中空午未，甲午旬中空辰巳。
甲辰旬中空寅卯，甲寅旬中空子丑。

旺空有用待时填，衰空无用真空亡。
动空则是有始无终，静空则是虚而不实。
```

## 六、动变口诀

### 动爻

```
动爻为事之机关，静爻无动不相干。
动爻生用吉神助，动爻克用凶煞缠。
多爻齐动看世用，独发一爻断分明。
```

### 变爻（回头）

```
动化回头生力强，动化回头克力殃。
动化进神事渐好，动化退神事渐伤。
动化空亡终不就，动化墓库力封藏。
```

### 进退神

```
寅化卯来巳化午，申化酉来亥化子——进神
卯化寅来午化巳，酉化申来子化亥——退神

进神有力前程远，退神无力后路难。
```

## 七、六冲六合口诀

### 六合

```
子丑合土寅亥木，卯戌合火辰酉金。
巳申合水午未日，六合之爻喜相亲。
合则缓也事迟缓，用神逢合暂不行。
忌神逢合则被绊，解合须待冲来临。
```

### 六冲

```
子午冲来丑未冲，寅申冲来卯酉冲。
辰戌冲来巳亥冲，六冲之卦事不宁。
冲旺则为暗中动，冲衰则是散无踪。
冲空出空填实论，冲墓开墓力相逢。
```

### 三合

```
申子辰来合水局，亥卯未来合木局。
寅午戌来合火局，巳酉丑来合金局。
三合成局力量大，化出五行论吉凶。
```

## 八、应期口诀

```
用神旺相——逢值逢合日应（近期事）
用神休囚——逢生逢旺月应（远期事）
用神受克——去克之日方应
用神逢合——冲合之日方应
用神空亡——出空填实方应
用神入墓——冲墓之日方应
用神化进——到位之日方应
用神化退——退位之日不应
```

### 应期单位判断

```
近事看日——三五日内
中事看旬——旬内应期
远事看月——月内应期
大事看年——年内应期
```

## 十、断卦十六字诀

> *"我把断卦的核心流程浓缩成了十六个字。每次断卦前默念一遍。"——师姐*

```
定用神、辨旺衰、察动变、断吉凶。
```

**展开**：
1. **定用神**：根据所问确定哪个爻是用神
2. **辨旺衰**：月建日辰对用神的生克关系
3. **察动变**：动爻的方向、变爻的回头作用
4. **断吉凶**：综合以上，给出结论和应期

## 十一、特殊卦断口诀

### 六冲卦

```
六冲卦来事难成，纵然成了也不宁。
冲中逢合有转机，看是何爻来合停。
```

### 六合卦

```
六合卦来万事和，谋事求财皆可贺。
合中逢冲须注意，好事多磨终可过。
```

### 游魂卦

```
游魂之卦心不定，东奔西走难安宁。
测行必去未必归，测事反复费精神。
```

### 归魂卦

```
归魂之卦主归来，在外思乡心不开。
测行不去留原地，测事终须归本源。
```
