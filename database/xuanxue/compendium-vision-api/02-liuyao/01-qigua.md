## 一、起卦的基本原则

### 核心规则

1. **一事一卦**：一次只问一件事
2. **心诚意专**：起卦时专注于所问之事
3. **不重复起卦**：同一件事不要反复起卦（「初筮告，再三渎，渎则不告」）
4. **记录时间**：起卦时间的年月日时干支非常重要，影响旺衰判断

### 起卦时需要记录的信息

## 二、铜钱法（最经典）

### 操作方法

取三枚铜钱（或硬币），双手合拢摇晃后掷出，记录正反面：

- 正面（有字） = 3
- 反面（无字/花） = 2

三枚铜钱的和：

| 组合 | 值 | 阴阳 | 类型 |
|------|------|------|------|
| 三个反面 (2+2+2) | **6** | 老阴 ⚋ | **动爻**（阴变阳）|
| 两反一正 (2+2+3) | **7** | 少阳 ⚊ | 静爻 |
| 两正一反 (3+3+2) | **8** | 少阴 ⚋ | 静爻 |
| 三个正面 (3+3+3) | **9** | 老阳 ⚊ | **动爻**（阳变阴）|

**共摇六次**，第一次为初爻，第六次为上爻（从下往上排）。

```
🔮 实战：铜钱法起卦

function castWithCoins() {
  const hexagram = [];
  
  for (let i = 0; i < 6; i++) {
    // 模拟三枚铜钱：2(反) 或 3(正)
    const coin1 = Math.random() < 0.5 ? 2 : 3;
    const coin2 = Math.random() < 0.5 ? 2 : 3;
    const coin3 = Math.random() < 0.5 ? 2 : 3;
    const sum = coin1 + coin2 + coin3; // 6, 7, 8, 9
    
    hexagram.push({
      value: sum,
      yinyang: (sum % 2 !== 0) ? 'yang' : 'yin',  // 奇阳偶阴
      moving: (sum === 6 || sum === 9),              // 老变少不变
      line: (sum % 2 !== 0) ? 1 : 0                  // 1=阳爻, 0=阴爻
    });
  }
  
  return hexagram; // [初爻, 二爻, ..., 上爻]
}
```

> ⚠️ **概率说明**：铜钱法中四种结果的概率不均等——
> - 6 (老阴)：1/8 = 12.5%
> - 7 (少阳)：3/8 = 37.5%
> - 8 (少阴)：3/8 = 37.5%  
> - 9 (老阳)：1/8 = 12.5%
> 
> 这意味着动爻（6和9）出现的概率较低（各12.5%），符合「变为少数」的哲学。

## 三、时间起卦法

用起卦时间的干支数字来起卦：

### 方法一：年月日时起卦

```
上卦 = (年数 + 月数 + 日数) ÷ 8 取余数 → 对应先天八卦
下卦 = (年数 + 月数 + 日数 + 时辰数) ÷ 8 取余数 → 对应先天八卦
动爻 = (年数 + 月数 + 日数 + 时辰数) ÷ 6 取余数 → 第几爻动

余数为0时，上下卦取坤(8)，动爻取6(上爻)
```

**数字对应**：
- 年数：地支序数（子1丑2寅3……亥12）
- 月数：农历月份（正月1……十二月12）
- 日数：农历日期
- 时辰数：地支序数（子1丑2……亥12）

```
🔮 实战：时间起卦算法

function castByTime(yearBranch, lunarMonth, lunarDay, hourBranch) {
  const branchOrder = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥'];
  const yearNum = branchOrder.indexOf(yearBranch) + 1;
  const hourNum = branchOrder.indexOf(hourBranch) + 1;
  
  const sumWithoutHour = yearNum + lunarMonth + lunarDay;
  const sumWithHour = sumWithoutHour + hourNum;
  
  // 先天八卦：乾1 兑2 离3 震4 巽5 坎6 艮7 坤8
  const xiantianMap = ['', '乾', '兑', '离', '震', '巽', '坎', '艮', '坤'];
  
  let upperNum = sumWithoutHour % 8;
  let lowerNum = sumWithHour % 8;
  let movingLine = sumWithHour % 6;
  
  if (upperNum === 0) upperNum = 8;
  if (lowerNum === 0) lowerNum = 8;
  if (movingLine === 0) movingLine = 6;
  
  return {
    upper: xiantianMap[upperNum],
    lower: xiantianMap[lowerNum],
    movingLine: movingLine  // 1=初爻, 6=上爻
  };
}
```

## 四、数字起卦法

用任意数字起卦，适合日常应用：

### 两数起卦

取两个数：
- 第一个数 ÷ 8 取余 → 上卦（先天数）
- 第二个数 ÷ 8 取余 → 下卦（先天数）
- 两数之和 ÷ 6 取余 → 动爻

### 三数起卦

取三个数：
- 第一个数 ÷ 8 取余 → 上卦
- 第二个数 ÷ 8 取余 → 下卦
- 第三个数 ÷ 6 取余 → 动爻

### 手机号/车牌号起卦

- 取前半段数字之和 → 上卦
- 取后半段数字之和 → 下卦
- 全部数字之和 → 动爻

```
🔮 实战：数字起卦

function castByNumbers(num1, num2, num3 = null) {
  const xiantianMap = ['', '乾', '兑', '离', '震', '巽', '坎', '艮', '坤'];
  
  let upperNum = num1 % 8 || 8;
  let lowerNum = num2 % 8 || 8;
  let movingSum = num3 !== null ? num3 : (num1 + num2);
  let movingLine = movingSum % 6 || 6;
  
  return {
    upper: xiantianMap[upperNum],
    lower: xiantianMap[lowerNum],
    movingLine
  };
}
```

## 五、其他起卦方法

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **蓍草法** | 古法，用50根蓍草反复分拨 | 庄重占卜，概率与铜钱法不同 |
| **报数法** | 让人随口说数字 | 为他人起卦 |
| **字占法** | 以文字笔画数起卦 | 梅花易数常用 |
| **声音法** | 以声响次数起卦 | 偶然听到敲击声等 |
| **方位法** | 以来人方位起卦 | 来人占 |
| **外应法** | 以突发事件起卦 | 梅花易数特色 |

## 六、本课心法

1. **起卦即定局**：起卦那一刻，卦就定了，不要犹豫
2. **一事一卦**：贪多嚼不烂
3. **记录完整**：年月日时干支必须记录，否则无法判断旺衰
4. **方法自选**：铜钱法最经典、最随机；时间法最方便；数字法最灵活

> *"好，现在你会起卦了。但起出来的只是六个数字。下一步是把这六个数字'装'成一个完整的卦——纳甲、装世应、配六亲、安六神。这一步是六爻的核心工程。"——师姐*
