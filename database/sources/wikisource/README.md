# Wikisource Zhouyi validation snapshot

- Fetch date: 2026-07-10
- API: `https://zh.wikisource.org/w/api.php`
- Scope: the 64 subpages under `周易`
- Stored file: `zhouyi-64-mediawiki-2026-07-10.json`
- Purpose: a pinned transcription witness for offline comparison, not the sole textual authority

The snapshot preserves page titles, page IDs, fetch time, and MediaWiki source for 64 pages. The underlying received `Zhouyi` text is public domain. Wikisource transcription and markup reuse remains subject to Wikisource/Wikimedia terms; attribution is preserved through page title and page ID.

The root Wikisource text was marked 50% complete when sampled. Every substantive difference therefore requires adjudication against another source. This project uses the Chinese Text Project only for the differing passages and does not bulk-copy its corpus.
